'''南里 侑香'''（なんり ゆうか、[[1984年]][[3月13日]] - ）は、[[日本]]の[[俳優|女優]]、[[声優]]、[[歌手]]。[[長崎県]][[佐世保市]]出身<ref>{{Cite web |url=http://ameblo.jp/nanri-yuuka/entry-11200547626.html |title=南里侑香オフィシャルブログ「ゆうかcome home★」 |accessdate=2017-2-03 }}</ref>。所属事務所は[[スペースクラフト|スペースクラフト・エンタテインメント]]。所属レーベルは[[フライングドッグ|FlyingDog]]。身長157cm。血液型はA型。[[FictionJunction YUUKA]]のボーカル・'''YUUKA'''としても活動している。

== 略歴 ==
1984年、長崎県佐世保市に生まれる。5歳の時、家族の転居に伴い[[東京]]に移住<ref name="bank">{{Cite web |url=http://www.talent-databank.co.jp/interview/2006/0401/interview_01.html |title=タレントデータバンク |accessdate=2017-2-03 }}</ref>。通っていた学校の合唱発表の際に口パクだったことを心配した母親の勧めにより、小学2年で地域の合唱団に入団、周囲に感化され歌うことの楽しさに目覚める<ref>「人きらり インタビュー」、『Mainichi INTERACTIVE』 2000年7月28日</ref><ref name="mainichi">「小学生だったよ」、『[[毎日小学生新聞]]』 2000年8月4日</ref>。8歳、合唱団の知人らと家族ぐるみで芝居を観劇するようになり、その時観たファミリーミュージカルに衝撃を受け自らも舞台に立ちたいと願うようになる<ref name="pia">{{Cite web |url=http://t.pia.jp/interview/music/nanriyuuka.jsp |title=チケットぴあインタビュー |accessdate=2017-2-03 }}</ref><ref name="official">{{Cite web |url=http://www.nanriyuuka.com/profile.html |title=南里侑香公式サイト |accessdate=2017-2-03 }}</ref>。家族を説得し10歳でオーディションを受け[[スペースクラフト|東京児童劇団]]に所属<ref name="bank"/>、[[子役]]としてCMや[[NHK教育テレビ]]『[[うたってオドロンパ]]』等に出演していた<ref name="mainichi"/><ref>NHKビデオ『うたってオドロンパ 2』、[[日本コロムビア]]、COVC-4922</ref>。

[[1996年]]、ミュージカル『[[グッバイガール|THE GOODBYE GIRL]]』ルーシー・マクフェダン役で12歳にして念願の初舞台を踏む<ref>『月刊ミュージカル』 1997年3月号、ミュージカル出版社、18頁</ref>。[[1997年]]、[[南青山少女歌劇団]]に7期生として入団。2000年以降頭角を現し、2001年の卒団までに主演3作、劇団の中心メンバーとして活躍した<ref>『シアターガイド』 2001年8月号、モーニングデスク、150頁</ref>。[[2001年]]、スペースクラフトに[[緒方恵美]]と[[ホーリーピーク|堀尾裕樹]]が移籍してきたことにより声優部門が開拓される<ref>{{Cite web |url=http://ameblo.jp/horio-dazo/entry-11207049707.html |title=堀尾裕樹（ホーリーピーク）ブログ |accessdate=2017-2-03 }}</ref>。この時南里は舞台経験者でキャストが固められた『[[パワーパフガールズ]]』日本語吹き替え版の主役の一人の座を射止め<ref>『パワーパフ・ガールズ Vol.1』、[[ワーナー・ホーム・ビデオ]]、2005年9月2日、DLT-7389</ref>、声優としての第一歩を踏み出す<ref name="pia"/>。[[2003年]]、以前から南里の声に注目していたという[[梶浦由記]]によって、アニメ『[[機動戦士ガンダムSEED]]』挿入歌「[[暁の車]]」のボーカリストに抜擢される<ref name="song">『アニソンマガジン Vol.1』、[[洋泉社]]MOOK、2007年6月8日、46頁、ISBN 978-4-86248-160-3</ref>。[[FictionJunction YUUKA]]名義<ref group="注">「FictionJunction featuring vocal YUUKA（初オンエア時）」「FictionJunction featuring YUUKA（コンピレーションCDリリース時）」「FictionJunction YUUKA（シングルCDデビュー時）」とユニット名が変遷している。</ref>でのデビューとなったこの楽曲は公称20万枚<ref>{{Cite web |url=http://www.jvcmusic.co.jp/flyingdog/-/Profile/A018153.html |title=FlyingDog公式サイト内FictionJunction YUUKAプロフィール |accessdate=2017-8-30 }}</ref>のヒット作となり、南里のプロ歌手活動のきっかけとなっている<ref name="pia"/>。梶浦との最初の出会いは[[1998年]]とされ、当時は南青山少女歌劇団の音楽制作・歌唱指導者と劇団員の一人という関係であった<ref>飯田一著「物語は鳴りはじめる　梶浦由記ヒストリア」、『[[ユリイカ (雑誌)|ユリイカ]]』 2015年11月号、[[青土社]]、ISBN 978-4-7917-0297-8</ref>。梶浦は南里の初主演作品<ref group="注">2000年 ミュージカル『ハイスクールRevolution〜愛と勇気の旅立ち〜』</ref>で組んだ際「なんて悲しい声なんだろう！」という印象を持ったといい、更に後年には「真ん中にある芯に、ギューっと音が集まっていく、日本語を乗せるための細かく刻むメロディーが歌える声」と評した<ref name="song"/>。

また、時を同じくして2003年に[[千葉紗子]]との歌手ユニット[[tiaraway]]の活動が始まる。シングル3枚アルバム1枚をリリースするも、2005年3月の1stコンサートにおいて電撃的に解散を発表<ref name="tiaraway">『[[声優グランプリ]]』 2005年5月号、[[主婦の友社]]、127頁</ref>、以降一切の活動を休止した。解散の理由は明言されていないが、関連する事実として[[サイトロン・デジタルコンテンツ]]のtiaraway担当プロデューサー[[志倉千代丸]]と担当ディレクター[[斎藤滋]]<ref>『TWO:LEAF』、サイトロン・デジタルコンテンツ、2005年1月26日、KDCA-0021</ref>が直後に同社を退社<ref>{{Cite web |url=https://www.barks.jp/artist/?id=2000036837 |title=志倉千代丸 BARKSアーティスト |accessdate=2017-2-03 }}</ref><ref>{{Cite web |url=http://www.esp.ac.jp/epv/exp/trial_detail/120728.html |title=特別講師 斎藤滋さんプロフィール |accessdate=2017-2-03 }}</ref><ref>{{Cite web |url=http://www.hanasakuiroha.jp/interview/interview_39.html |title=インタビュー｜アニメ「花咲くいろは」公式サイト |accessdate=2017-2-03 }}</ref>していることが挙げられる。志倉はtiarawayに関し、「正直、心の底から解散したくなかった」と後年になり語っている<ref>{{Cite web |url=http://news.mynavi.jp/articles/2009/12/12/chiyomaru/003.html |title=志倉千代丸トークショー in 法政大学 |accessdate=2017-2-03 }}</ref>。一方、声優としては第59回[[毎日映画コンクールアニメーション映画賞]]を受賞した{{Refnest|group="注"|同年ノミネートの[[宮崎駿]]監督作品『[[ハウルの動く城]]』を抑えての受賞<ref>[http://mainichi.jp/enta/cinema/mfa/etc/history/59.html 毎日新聞「毎日映画コンクール：コンクールの歴史」] 2017年2月26日閲覧。</ref>。}}[[2004年]]公開作品『[[雲のむこう、約束の場所]]』でヒロインの沢渡佐由理役を演じている。作品を手掛けた監督の[[新海誠]]は、デモテープを多数聞いた中から第一希望で南里に出演依頼を出しており、その理由について「手の届かない、初恋の象徴のようなイメージにぴったりだった」と語っている<ref>『雲のむこう、約束の場所』 映像特典「新海誠インタビュー」、[[コミックス・ウェーブ・フィルム]]、2005年2月17日、MZDV-0005</ref>。また、「サユリは南里さんに演じてもらうことを前提に、本編のキャラクターをつくった<ref>『雲のむこう、約束の場所 コンプリートブック』、[[角川書店]]、2005年02月24日、12頁、ISBN 4-04-853836-5</ref>{{Refnest|group="注"|[[2002年]]発表のパイロット版の段階で主役3名の配役が決定していた為、本編の脚本は[[吉岡秀隆]]・[[萩原聖人]]・南里の声をイメージして執筆されている<ref>『雲のむこう、約束の場所』 付属ブックレット、コミックス・ウェーブ・フィルム、2005年2月17日、MZDV-0005</ref>。}}」とも述べている。[[2009年]]、ソロシンガーとしての継続的活動を開始<ref name="official"/>。[[2010年]]、当時スペースクラフトに所属していた[[結城アイラ]]と歌手ユニット[[Fairy Story]]を結成し<ref>{{Cite web |url=http://www.animatetimes.com/news/details.php?id=1307886896 |title=アニメイトタイムズ |accessdate=2017-2-03 }}</ref>アルバム2枚をリリース。[[2014年]]、自身のブログで入籍を発表<ref>{{Cite web |url=http://ameblo.jp/nanri-yuuka/entry-11957910277.html |title=南里侑香オフィシャルブログ「ゆうかcome home★」 |accessdate=2014-11-28}}</ref>、その後も芸能活動を継続している。
