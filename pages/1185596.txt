'''中津 真莉'''（なかつ まり、[[1983年]][[7月8日]] - ）は、[[日本]]の[[声優]]、[[俳優|女優]]。本名および旧芸名は中津 真莉子。

[[大阪府]][[東大阪市]]出身。[[アクロス エンタテインメント]]預かり所属。

== 来歴 ==
[[樟蔭中学校・高等学校]]卒業。[[大阪芸術大学]]を経て[[帝塚山大学]]に編入学、卒業。

[[2001年]]から[[2003年]]まで、『フェアリーセクターテントー』に'''ブルーテントー'''役で出演し、主題歌「SILENT BLUE」を歌う。

[[2004年]]より[[NHK教育テレビジョン|NHK教育テレビ]]『[[みてハッスルきいてハッスル]]』に女忍者'''花影あやめ'''役で出演、女王さま風の声と演技で人気を得る。『みてハッスルきいてハッスル』では、「どうする?ゆうきくん」の登場人物'''あおいちゃん'''の声を担当しているほか、恋の妖精'''ラブリル'''など、本編に登場するゲスト人物の声をあてることがある。

また、[[赤星憲広]]のウェブサイトにて、赤星のオフィシャルグッズ「RSナンバージャージ」のモデルとして写真が掲載されている<ref>{{Cite web|date=|url=http://www.redstar53.com/goods/mesh.html|title=redstar53|work=|author=|publisher=redstar53|accessdate=2007-10-20|archiveurl=https://web.archive.org/web/20070914095539/http://www.redstar53.com/goods/mesh.html|archivedate=2007年9月14日|deadlinkdate=2017年9月}}</ref>。

このほか、「[[野球少年の詩]]」（[[ガガガSP]]）や「空フル〜虹を架けろ〜」（[[SHALE APPLE]]）の[[ミュージック・ビデオ]]にも出演している<ref>{{Cite web|date=2008-01-29|url=http://yaplog.jp/hangao-neko/archive/9|title=プロモーションビデオ|work=中津真莉子☆半顔のネコにご用心!|author=中津真莉子|publisher=[[yaplog!]]|accessdate=2008-02-25}}</ref>。

[[2005年]]の2月頃までは[[グレース (芸能プロダクション)|グレース]]に所属していた。[[2009年]]9月より[[サンミュージックプロダクション]]所属<ref>{{Cite web |url=http://yaplog.jp/hangao-neko/archive/617 |title=中津真莉子♪♪ :: 中津真莉子☆半顔のネコにご用心！ |accessdate=2012-09-15}}</ref>。

[[2013年]]6月から[[2014年]]7月まで、[[秋奈]]、[[杜野まこ]]と共に「[[プリンセスケッツ|プリケッツ]]」という声優ユニットを組んでいた。ピュアホワイト・右ケッツ担当。

2014年8月8日に、芸名を『中津 真莉子』から『中津 真莉』へ変更した<ref>{{Cite web |url=http://ameblo.jp/hangao-mari/day-20140808.html|title=☆中津真莉です☆::半顔のネコ参上！ |accessdate=2014-08-08}}</ref>。

[[2015年]]9月、サンミュージックプロダクションとの契約を終了<ref>{{Cite web |url=http://ameblo.jp/hangao-mari/entry-12067281982.html|title=ご報告です。::半顔のネコ参上！ |accessdate=2015-09-05}}</ref>。同年11月、[[アクロス エンタテインメント]]に所属することを発表<ref>{{Cite web |url=http://ameblo.jp/hangao-mari/entry-12090513853.html|title=★☆お知らせです☆★::半顔のネコ参上！ |accessdate=2015-11-01}}</ref>。

[[2017年]][[7月7日]]、自身のブログで元芸能人の男性と結婚したことを発表した。<ref>{{Cite web|url=http://ameblo.jp/hangao-mari/entry-12284105014.html|title=☆☆201777ご報告です☆☆|publisher=半顔のネコ参上!|date=2017-07-07|accessdate=2017-07-07}}</ref>。

2018年1月18日、自身のブログにで無事に出産するしたことを報告した<ref>{{Cite web|url=https://ameblo.jp/hangao-mari/entry-12345242191.html|title=出産しました！！|publisher=半顔のネコ参上!|accessdate=2018-01-28}}</ref>。

== 人物 ==
声優になろうと思ったきっかけは6歳のころに大病を患い、病室でアニメをよく観ていたから<ref>[http://seigura.com/senior/20131119_164.html 声優未来予想図17(nakatsu) - 声優グランプリweb]</ref>。

ニックネームは「まりにゃん」。趣味は[[ハウス (ダンス)|ハウスダンス]]と[[イラストレーション|イラスト]]。幼少期に[[バレエ#クラシック・バレエ|クラシックバレエ]]。[[ネコ]]好きでアスランというネコを飼っている<ref>{{Cite web |url=http://yaplog.jp/hangao-neko/archive/2 |title=ネコ王子 :: 中津真莉子☆半顔のネコにご用心！ |accessdate=2012-10-18}}</ref>。2級[[小型船舶操縦士]]免許を取得している<ref>{{Cite web|url=http://ameblo.jp/hangao-mari/entry-11382937347.html|title=小型船舶操縦免許証GET☆船長です！|publisher=半顔のネコ参上!|date=2012-12-18|accessdate=2013-04-09}}</ref>。

[[特別支援教育]]や[[発達障害]]について、日本LD学会に参加するなどして勉強したり、指導教室でボランティアをしていた<ref>{{Cite web|url=http://ameblo.jp/hangao-mari/entry-11020802444.html#main|title=日本ＬＤ学会　第２０回大会|publisher=半顔のネコ参上!|date=2011-09-17|accessdate=2013-04-09}}</ref><ref>{{Cite web|url=http://yaplog.jp/hangao-neko/archive/346|title=日本発達障害ネットワーク|publisher=中津真莉子☆半顔のネコにご用心！|date=2008-12-15|accessdate=2013-04-09}}</ref>。

実家は書道用品店。

好きなモノは猫と海で、ファッションではスカルデザインのものや[[クロムハーツ]]のアクセサリーを愛用している。
好きな色は白と紫。

オフィシャルブログは毎日休まずに更新している（日に2回更新している時もある）
