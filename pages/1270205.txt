'''折笠 奈緒美'''（おりかさ なおみ、[[1981年]][[3月3日]]<ref name="Blog profile">{{Cite web |url=http://profile.ameba.jp/naomimi-mama/ |title=なおママのプロフィール |accessdate=2012-12-03}}</ref> - ）は、[[日本]]の女性[[声優]]。[[東京都]]出身。血液型は[[ABO式血液型|O型]]<ref name="Blog profile" />。夫はゲームクリエイターの[[時田貴司]]。

2006年までは[[ドラマチック・デパートメント]]に所属していたが、現在はフリーで活動している。

== 人物 ==
2007年9月、自身のブログで結婚したことを発表<ref>{{Cite web |url=http://yaplog.jp/tomatopasta/archive/98 |title=◆ご報告があります◆ :: 折笠奈緒美屋　＊＊せかんどエディション＊＊ |accessdate=2012-12-03}}</ref>。2008年には男の子<ref>{{Cite web |url=http://yaplog.jp/tomatopasta/archive/189 |title=☆☆大報告☆☆ :: 折笠奈緒美屋　＊＊せかんどエディション＊＊ |accessdate=2012-12-03}}</ref>、2011年には女の子を出産した<ref>{{Cite web |url=http://ameblo.jp/naomimi-mama/entry-11019642496.html |title=◆◆ご報告◆◆｜おさんぽ日和 |accessdate=2012-12-03}}</ref>。
