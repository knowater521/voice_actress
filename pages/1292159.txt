'''榛田 安芸'''（はるた あき、[[1975年]][[1月3日]]<ref>{{Cite web |url=http://katari-za.cside.com/kataribt/akiha/index.htm|title=「語人」秋葉 好美|publisher=語座|accessdate=2013-02-15 |archiveurl=http://web.archive.org/web/20040831040357/http://katari-za.cside.com/kataribt/akiha/index.htm|archivedate=2004-08-31 }}</ref> - ）は、[[日本]]の[[女性]][[声優]]。所属は[[フリーランス|フリー]]。

== 人物 ==
[[広島県]]出身<ref name="kikakukaigi"/><ref name="profile">{{Cite web |url=http://www02.geocities.co.jp/AnimeComic-Palette/4858/page002.html|title=はじめに・・・。|publisher=アキメキア（秋葉好美公式サイト）|accessdate=2013-02-15 |archiveurl=http://web.archive.org/web/20040411045943/http://www.geocities.co.jp/AnimeComic-Palette/4858/page002.html|archivedate=2004-04-11 }}</ref>。中学3年時に[[神奈川県]]に移住<ref name="profile"/>。

高校中退後、劇団所属<ref>{{Cite web |url=http://www02.geocities.co.jp/AnimeComic-Palette/4858/page065.html|title=（元）声優志望者に50の質問、その2|publisher=アキメキア（秋葉好美公式サイト）|accessdate=2013-02-15 |archiveurl=http://web.archive.org/web/20030425101338/http://www02.geocities.co.jp/AnimeComic-Palette/4858/page065.html|archivedate=2003-04-25 }}</ref>、[[映像テクノアカデミア]]研修クラス2期生、[[九プロダクション]]、[[ぷろだくしょん★A組]]を経て、[[2005年]][[7月1日]]からフリー<ref name="kikakukaigi">{{Cite web |url=http://tvproject.fc2web.com/radio/staff.html|title=Staff Profile|publisher=TV-Project企画会議|accessdate=2013-02-15 |archiveurl=http://web.archive.org/web/20070620201502/http://tvproject.fc2web.com/radio/staff.html|archivedate=2007-06-20 }}</ref>。

[[槇大輔]]が代表を務める「語座」の座員でもあった。

旧名は'''秋葉 秋'''（あきは あき）、'''秋葉 好美'''（あきは このみ）。フリーになると同時に現在の芸名に改名<ref name="kikakukaigi"/>。安芸は本名。最初の芸名の名付け親は九プロダクションの社長。特技は歌、ダンス、エレキベース、CGイラスト。

主に外画の吹き替えで活動し、演じる役は少年役が多い<ref name="kikakukaigi" />。
