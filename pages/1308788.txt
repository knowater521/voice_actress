'''目黒 未奈'''（めぐろ みな、[[1973年]][[3月27日]]<ref name="W98-0644">{{Cite web |url=http://www.vip-times.co.jp/?talent_id=W98-0644 |title=目黒未奈 |work=日本タレント名鑑 |publisher=VIPタイムズ社 |accessdate=2017-04-29}}</ref><ref name="t2000042934">{{Cite web |url=http://www.talent-databank.co.jp/search/t2000042934 |title=目黒未奈 |publisher=タレントデータバンク |accessdate=2017-04-29}}</ref> - ）は、日本の[[俳優|女優]]、[[声優]]<ref name="t2000042934" />。[[東京都]]出身<ref name="W98-0644" /><ref name="bungakuzaprof">{{Cite web |url=http://www.bungakuza.com/member/prof/meguro-mina.htm |title=目黒未奈 |publisher=文学座 |accessdate=2017-04-29}}</ref>。身長164cm、血液型はA型<ref name="bungakuzaprof" />。[[文学座]]所属<ref name="bungakuzaprof" />。

== 略歴 ==
[[大妻女子大学]]中野女子高等学校（現・[[大妻中野中学校・高等学校]]）を卒業後<ref name="unionpedia">{{Cite web |url=http://ja.unionpedia.org/大妻中野中学校・高等学校 |title=目黒未奈 |publisher=unionpedia |accessdate=2018-01-09}}</ref>、1991年、文学座付属演劇研究所へ入所<ref name="bungakuzaprof" />。本科生<ref name="neoapo">{{Cite web |url=http://neoapo.com/voiceartists/1857 |title=目黒未奈 |publisher=NeoApo |accessdate=2018-01-09}}</ref>、研修科生<ref name="neoapo" />、準座員を経て<ref name="neoapo" />、1996年、文学座の座員となる<ref name="bungakuzaprof" />。1994年、文学座アトリエ公演「シンガー」にて初舞台<ref name="bungakuzaprof" />。2001年度、『心破れて』のフィリーマ役、『マイシスター インディスハウス』のクリスティーヌ役の演技に対し、第8回[[読売演劇大賞]]の優秀女優賞、杉村春子賞を授与された<ref name="bungakuzaprof" />。大妻女子大学中野女子高等学校時代の演劇部の友人に声優の[[園崎未恵]]がいる<ref name="sonozaki">{{Cite web |url=http://yaplog.jp/mie_kingyo/daily/200910/14/1 |title=園崎未恵のゆらゆら日記。 |publisher=園崎未恵のゆらゆら日記。 |accessdate=2018-01-09}}</ref>。
