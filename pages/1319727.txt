== 概要 ==
=== 来歴・人物 ===
[[1974年]][[8月4日]][[三重県]][[青山町]]（現・[[伊賀市]]）に生まれる。{{要出典範囲|中学生時代から声優になることが夢で|date=2017年3月}}、{{要出典範囲|[[三重県立名張桔梗丘高等学校|名張桔梗丘高校]]卒業|date=2017年3月}}後に{{要出典範囲|単身で上京|date=2017年3月}}、[[劇団すごろく]]に入団し<ref name="aoniprof" />、{{要出典範囲|[[アルバイト]]をしながら声優を目指す|date=2017年3月}}。[[1996年]]（平成8年）、21歳で劇場アニメ『[[トイレの花子さん]]』の上岡山大介役で{{要出典範囲|声優デビューを果たす|date=2017年3月}}。

「わさび」という芸名は{{要出典範囲|実家の庭に自生していた[[ワサビ|山葵（わさび）]]が由来で、本名が言い難い（読み辛い）ことから名付けられた|date=2017年3月}}。命名したのは所属していた[[劇団]]（劇団すごろく）の当時の座長であった[[緒方賢一]]<ref>{{Cite web |url=http://www.tv-asahi.co.jp/reading/wasadora93/29522/ |title=緒方さん帽子 |work=【アニメ・ドラえもん　水田わさび公式】わさドラブログ |publisher=テレビ朝日 |date=2012-07-06 |accessdate=2015-05-03}}</ref>。{{要出典範囲|劇団の公演で馬の役をやっていたところを、たまたま見に来ていた当時ジャイアン役の[[たてかべ和也]]に発掘されて、声優の仕事をするようになった|date=2017年3月}}。

[[広島東洋カープ]]のファン<ref name="wasadoraprof" />。{{要出典範囲|『[[ジョブジョブ]]』（広島のローカル番組）にゲスト出演した時にそのことをアピールしたことがある|date=2017年3月}}。また、「[[週刊ベースボール]]」[[2007年]]（平成19年）4月2日号の「こだわり野球熱伝」のコーナーにも登場し、その熱烈ぶりを語っている。

舞台役者としては劇団すごろく→{{要出典範囲|演劇部隊チャッターギャングで活動していたが、育児・声優業などに忙殺され休止していた|date=2017年3月}}{{要出典範囲|。『[[武装錬金]]』で競演した[[小山剛志]]との縁で|date=2017年3月}}KOYA-MAPの舞台『喝采』で久しぶりに舞台活動を行う。

既婚者で、家族は{{要出典範囲|夫と2人の娘の4人家族|date=2017年3月}}。{{要出典範囲|ブログでは、家族のことをわさ夫さん（夫）・子わさ（上の娘）・わさ子（下の娘）と呼んでいる|date=2017年3月}}。

[[2010年]]（平成22年）に第4回[[声優アワード]]キッズファミリー賞を受賞<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_04 |title=第四回声優アワード受賞者 |publisher=声優アワード |accessdate=2017-03-28}}</ref>。

また[[東京声優アカデミー]]で講師を務め、後輩育成にも力を注いでいる<ref>{{Cite web |url=http://www.tmaa.ac.jp/tma/pc/sp_teacher-mizuta.html |title=水田わさび |work=声優講演会 |publisher=東京声優アカデミー |accessdate=2017-03-28}}</ref>。

=== ドラえもん関連 ===
[[2005年]]（平成17年）[[4月15日]]より、[[テレビ朝日]]系列のアニメ『[[ドラえもん (2005年のテレビアニメ)|ドラえもん]]』で、[[大山のぶ代]]に代わり主人公の[[ドラえもん (キャラクター)|ドラえもん]]の声を担当するという大役を務める。内定の瞬間号泣したという<ref name="walkerplus20150214">{{Cite web |url=http://news.walkerplus.com/article/55342/ |title=ドラえもん旧声優陣からのエールに水田わさびが号泣！ |work=ニュースウォーカー |publisher=KADOKAWA |date=2015-02-14 |accessdate=2017-03-28}}</ref>。{{要出典範囲|同年[[7月]]の時点で大山とは面識が無いと言っていた|date=2017年3月}}。ただし、大山らとは一緒に録ってはいないが、リニューアル前の『[[ドラえもん (1979年のテレビアニメ)|ドラえもん]]』においても「赤ちゃんほんやく機」での赤ちゃん役等、ゲストキャラクターの声を何度か担当しており、作品上での共演はあった。また、水田は大山がドラえもんを演じた26年間で一度も声をあてることのなかった「耳のある黄色いドラえもん」にも声をあてている。

リニューアル当初（水田がドラえもん役を引き継いだ2005年頃）は視聴者に深く根付いた大山が演じるドラえもんからの急な転換に視聴者が戸惑い、自身への[[バッシング]]を耳にしたことがあったという<ref name="walkerplus20150214" />。また、当時は「（ドラえもん役を）クビになるかも」と思っていたが、水田が演じる映画ドラえもんシリーズ第1作『[[ドラえもん のび太の恐竜2006|のび太の恐竜2006]]』（[[2006年]]（平成18年））の制作が決まり「ドラえもんを続けさせてもらえるんだ」と安堵したという<ref name="walkerplus20150214" />。このことから自身の思い出の映画シリーズに『のび太の恐竜2006』をあげている<ref name="walkerplus20150214" />。

リニューアルから年月が過ぎるにつれ、少しずつ水田が演じるドラえもんが定着し、現代の子供たちを中心に「ドラえもんの声＝水田」となっている<ref>{{Cite web |url=http://www.oricon.co.jp/special/47709/ |title=声優交代から10年…新たに確立した新世代の『ドラえもん』 |work=ORICON NEWS |publisher=オリコン |date=2015-02-22 |accessdate=2017-03-28}}</ref>。[[2015年]]（平成27年）にはリニューアル10周年というひとつの区切りをむかえ、ドラえもんを演じ続けている。
