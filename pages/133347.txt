'''大本 眞基子'''（おおもと まきこ、<!-- 生年の編集を行う方はノートページをご覧ください。 -->[[1973年]]<ref>{{Cite web |url=https://web.archive.org/web/20171222052216/https://talent.thetv.jp/person/0000083850/ |title=大本眞基子のプロフィール、出演情報、スケジュール｜【スタスケ】芸能人・タレントスケジュール|accessdate=2018-02-14}}</ref>[[2月1日]] - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]。[[岡山県]][[倉敷市]]出身。[[アトミックモンキー]]所属<ref>{{Cite web |url= http://www.atomicmonkey.jp/jp/archives/talent/omotomakiko |title=アトミックモンキー|大本眞基子 |accessdate=2018-2-14}}</ref>。[[青二塾]]東京校14期卒業。

代表作に、[[星のカービィシリーズ]]の[[カービィ]]、『[[コレクター・ユイ]]』の春日結、『[[ゾイド -ZOIDS-]]』のフィーネ・エレシーヌ・リネ、『[[クレヨンしんちゃん (アニメ)|クレヨンしんちゃん]]』のミッチー(2代目)、[[戦国無双シリーズ]]の[[稲姫]]、『[[ダンガンロンパ 希望の学園と絶望の高校生]]』の舞園さやか<ref>{{Cite web |url=http://www.seigura.com/enjoy/directory/19700101_-146.html |title=『月刊声優グランプリ』3月号・4月号付録「声優名鑑」　大本眞基子 |accessdate=2018-1-27}}</ref><ref>{{Cite web |url= http://kikubon.jp/sp/product.php?aKey=108|title= 朗読者情報|accessdate=2018-1-27}}</ref>などがある。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
岡山県倉敷市で生まれる。[[就実高等学校]]<ref name="dairy060607">{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2006.01-.html |title=『HORSE & HATOOK!!』「Diary」2006年6月7日 |accessdate=2015-5-12 |archiveurl=https://web.archive.org/web/20110712084116/http://www5f.biglobe.ne.jp/~tinowa/diary2006.01-.html |archivedate=2011年7月12日 }}</ref>卒業後、[[嵯峨美術短期大学]]<ref name="housoubunka200007">『放送文化』 NHK出版　2000年7月号　p3</ref>に入学。大学卒業後、声優を目指して上京し、[[青二塾]]に入塾。青二塾卒業後、[[青二プロダクション]]に所属。

* 1994年8月、『[[ストリートファイターII MOVIE]]』の「電話の声」役で、声優デビュー<ref name="twitter150718" />。
* 1997年2月、『[[キューティーハニーF]]』の「秋夏子」役で、テレビアニメ初レギュラー<ref>{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2007.10-.html |title=『HORSE & HATOOK!!』「Diary」2007年11月9日 |accessdate=2015-5-12 |archiveurl=https://web.archive.org/web/20090417022015/http://www5f.biglobe.ne.jp/~tinowa/diary2007.10-.html |archivedate=2009年4月17日 }}</ref>。
* 1999年1月、『[[ニンテンドウオールスター! 大乱闘スマッシュブラザーズ]]』より、[[カービィ]]役を担当。
* 1999年4月、『[[コレクター・ユイ]]』の春日結役でテレビアニメ初主演。
* 2004年6月、『[[クレヨンしんちゃん (アニメ)|クレヨンしんちゃん]]』第503話から、前任の[[草地章江]]に代わって「鳩ヶ谷ミッチー」役を担当。
* 2008年7月24日をもって[[青二プロダクション]]を退所し、[[フリーランス|フリー]]で活動。
* 2009年6月、著作絵本『ぼくの旅』出版。
* 2012年10月、13世紀の詩人、[[ジャラール・ウッディーン・ルーミー]]作詩の朗読CD『ただひとつの息吹 -Only Breath-』発売。
* 2017年12月、[[埼玉県]][[八潮市]]応援大使に就任<ref name="twitter171210">{{Cite web |url= https://twitter.com/Makiko_Ohmoto/status/939829232712028160 |title= 大本眞基子公式twitter 2017年12月10日 |accessdate=2017-12-23 }}</ref>。
* 2018年2月1日より[[アトミックモンキー]]に所属<ref>{{Cite web|url=http://www.atomicmonkey.jp/jp/archives/2191|title=新加入のお知らせ|publisher=アトミックモンキー|accessdate=2018-02-01}}</ref>。

== 人物 ==
=== 声優活動 ===
得意な役柄として、天然キャラ（男女問わず）や頭の良い男の子、天才少年を挙げている。『仰天人間バトシーラー』ではメインキャラを含み、10役以上を演じ分け、自キャラ同士の会話を行う場面もこなした<ref>{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2007.01-.html |title=『HORSE & HATOOK!!』「Diary」2007年1月12日 |accessdate=2015-5-24 |archiveurl=https://web.archive.org/web/20110818024935/http://www5f.biglobe.ne.jp/~tinowa/diary2007.01-.html |archivedate=2011年8月18日 }}</ref>。

演じるにあたって苦労した役に『[[サイボーグクロちゃん]]』のマタタビを挙げており、同作品の音響監督の[[松浦典良]]に「君はもっと低い声のポテンシャルがあるはず」と指導を受けながら演じていたという<ref>{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2007.10-.html |title=『HORSE & HATOOK!!』「Diary」2007年10月27日 |accessdate=2015-5-12 |archiveurl=https://web.archive.org/web/20090417022015/http://www5f.biglobe.ne.jp/~tinowa/diary2007.10-.html |archivedate=2009年4月17日 }}</ref>。また、松浦とは後に主演した『[[星のカービィ (アニメ)|星のカービィ]]』でも現場を共にしている。

今まで演じた自身の最も高い声の役に[[星のカービィシリーズ]]の[[カービィ]]、最も低い声の役に『サイボーグクロちゃん』のマタタビを挙げている<ref>{{Cite web |url=https://twitter.com/Makiko_Ohmoto/status/688016829172826112
 |title=大本眞基子 twitter 2016年1月15日 |accessdate=2016-1-26}}</ref>。

今まで演じたキャラクターで最も印象に残ってるキャラクターとして、『[[高橋留美子劇場 人魚の森]]』の真人と『陰陽大戦記』のウツホを挙げている。前者は外見と性格のギャップ、後者は外見と年齢のギャップを理由として挙げており、他にラストボスを務めた役が印象に残っているという。

=== 声優になるまで ===
* 幼少時代から言葉で表せないものを表現するのが好きだったため、暇さえあれば絵を描いていたが、中学2年生の時に声優という職業を知り、「これだ！」と思ったという。声の仕事は、年齢や性別に縛られることなく幅広い表現を可能にし、あるときは「顔出しの演技」以上に様々な疑似体験を可能にし、精神的な表現の限界に挑戦することができるということに、とても大きな可能性を感じたことにより、声優を志した<ref>{{Cite web |url=http://www.koyoerc.or.jp/antei136.html |title=『職業研究』2010年春季号 「しごとインタビュー」 声優・ナレーター 大本眞基子さん|accessdate=2015-5-13}}</ref>。
* 高校は、放送文化部目当てで就実高等学校（全国総合文化祭放送部門で1・2・3位を独占するほどの強豪校である上、[[日本放送協会|NHK]]のアナウンサーの直接指導が受けられ、OB・OGにアナウンサーを輩出する実績のある部であった）に進学した。また、高校では美大受験のために掛け持ちで美術部にも所属していた<ref>{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2007.01-.html |title=『HORSE & HATOOK!!』「Diary」2007年6月27日 |accessdate=2015-5-13 |archiveurl=https://web.archive.org/web/20110818024935/http://www5f.biglobe.ne.jp/~tinowa/diary2007.01-.html |archivedate=2011年8月18日 }}</ref>。大学では、演劇部と漫研に所属していた<ref>{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2006.01-.html |title=『HORSE & HATOOK!!』「Diary」2006年5月6日 |accessdate=2015-5-13 |archiveurl=https://web.archive.org/web/20120823135112/http://www5f.biglobe.ne.jp/~tinowa/diary2006.01-.html |archivedate=2012年8月23日 }}</ref>。
* 役者になる事は決めていたが、「東京に出たら勘当する」と言われていたため、1クッション置くために、元々絵を描くのが好きだったことから、美大の[[嵯峨美術短期大学]]へ進学した。本人曰くこれが「まんまと家を出る口実」となり、その卒業後に上京した<ref name="dairy060607" />。

=== 演じた役について ===
* コンピュータゲームシリーズおよびそれを原作としたアニメ『[[星のカービィシリーズ|星のカービィ]]』の主人公・[[カービィ]]の役は海外版もすべて大本が演じている。また『星のカービィ』の原作者（現在は開発元を退社）であるゲームデザイナーの[[桜井政博]]と交流があり、桜井の主催するイベントに呼ばれたりもしている<ref>{{Cite web |url=http://www.nintendo.co.jp/n01/n64/software/nus_p_nalj/smash/flash/orchestraconcert/ |title=大乱闘スマッシュブラザーズDX オーケストラコンサート|accessdate=2015-7-19}}</ref><ref>{{Cite web |url=http://www.famitsu.com/news/201409/17061551.html |title=ゲーム音楽のオーケストラコンサート“PRESS START 2014”は『スマブラ』の楽曲尽くし！　ファン歓喜の模様をリポート|accessdate=2015-7-19}}</ref>。
* カービィの声は『[[ニンテンドウオールスター! 大乱闘スマッシュブラザーズ]]』で初めてつき、その声は[[オーディション]]で『星のカービィ』の原作者（現在は開発元を退社）であるゲームデザイナーの桜井政博が決めた。その決め手はナチュラルボイスであった<ref>{{Cite web |url=https://twitter.com/sora_sakurai/status/28486042206081024 |title=桜井政博 twitter 2011年1月21日 |accessdate=2015-6-30}}</ref>。同作品に出演する[[ネス (MOTHER2)|ネス]]も、同オーディションで決まった<ref>{{Cite web |url=https://twitter.com/sora_sakurai/status/28501748175863810 |title=桜井政博 twitter 2011年1月21日 |accessdate=2015-6-30}}</ref>。
* 『[[ゾイド -ZOIDS-]]』のフィーネ・エレシーヌ・リネは、他の役とは異なり、人と会話する時になるべく相手と掛け合わないように独り言になるように意識して演じていた<ref>{{Cite web |url=http://tinowa.cocolog-nifty.com/mugendai/2013/01/post-b4c6.html |title=『ホース＆ハトック　∞　-無限大-』 2013年1月25日 |accessdate=2015-7-09}}</ref>。
* [[戦国無双シリーズ]]の稲姫は、透明感を意識し、凛としているが、粗野にはならないように、女っぽくも男っぽくもなりすぎないように、少し中性的なイメージで演じている<ref>{{Cite web |url=http://tinowa.cocolog-nifty.com/mugendai/2014/05/post-3e27.html |title=『ホース＆ハトック　∞　-無限大-』 2014年5月20日 |accessdate=2015-7-09}}</ref>。また、稲姫を演じる上で、稲姫を知るために[[森雅裕]]の『化粧槍とんぼ切り』を読み、役作りのヒントにした<ref>{{Cite web |url=http://tinowa.cocolog-nifty.com/mugendai/2011/02/post-ff50.html|title=『ホース＆ハトック　∞　-無限大-』 2011年2月10日 |accessdate=2015-7-09}}</ref>。
* 『[[陰陽大戦記]]』のウツホは、すでに登場していた消雪のタンカムイの回の収録終わりに、現場のスタッフに「ちょっと残って」と言われ、怒られるのかな？と思ったが、そこでウツホ役の話をもらい、担当することになった<ref>{{Cite web |url=https://twitter.com/makiko_ohmoto/status/517143304861806593 |title=大本眞基子 twitter 2014年9月30日 |accessdate=2015-7-11}}</ref>。
* 『[[ドラえもん (2005年のテレビアニメ)]]』の[[キャスト]]の一新に伴うオーディションで[[野比のび太]]役を受け、最終オーディションまで残っていた<ref>{{Cite web |url=http://www5f.biglobe.ne.jp/~tinowa/diary2005.html |title=『HORSE & HATOOK!!』「Diary」2005年3月29日 |accessdate=2015-6-30 |archiveurl=https://archive.is/20150630201504/http://www5f.biglobe.ne.jp/~tinowa/diary2005.html |archivedate=2015-06-30}}</ref>。
* 『[[ダンガンロンパ 希望の学園と絶望の高校生]]』の舞園さやかは、ボイスサンプルを新しくするために収録に行ったスタジオで、偶然本作品のオーディションをしており、「良かったら受けていかれませんか？」と誘われてオーディションを受けたところ、起用が決まった<ref>{{Cite web |url= https://twitter.com/Makiko_Ohmoto/status/700920865764978689 |title=大本眞基子 twitter 2016年2月19日 |accessdate=2016-06-04}}</ref><ref>{{Cite web |url= https://twitter.com/Makiko_Ohmoto/status/700922375978639360 |title=大本眞基子 twitter 2016年2月19日 |accessdate=2016-6-4}}</ref>。
* 『[[新・光神話 パルテナの鏡]]』の自然王ナチュレは、本作のディレクターである桜井政博がカービィ関連の縁でゲーム開発段階における仮音声全キャラクター分を大本に頼み、キャストを正式に決めることとなった際にオーディションは行ったものの大本の音声以上にしっくりくるものがなく、そのまま本採用になったという経緯がある<ref>『週刊ファミ通』2012年4月19日号{{要ページ番号|date=2017年7月}}</ref>。
* 『[[ダンガンロンパ 希望の学園と絶望の高校生|ダンガンロンパ 希望の学園と絶望の高校生 THE ANIMATION]]』の舞園さやかの[[キャラクターソング]]である『ネガイゴトアンサンブル』は、「舞園ちゃんのすべての想いが詰まった「アイドル」という仕事」であるため、「彼女（舞園さやか）が一番表現したかったんじゃないかな？と思う「希望」とか「元気」が伝わればいいな」と思いながら歌った<ref>{{Cite web |url=http://tinowa.cocolog-nifty.com/mugendai/2013/08/cd-7749.html |title=『ホース＆ハトック　∞　-無限大-』 2013年8月27日 |accessdate=2015-7-09}}</ref>。また、『モノクローム・アンサー』は、「苗木君への想いや内面の表現」を意識して歌った<ref>{{Cite web |url= https://twitter.com/Makiko_Ohmoto/status/382705632970686464 |title=大本眞基子 twitter 2013年9月24日 |accessdate=2015-10-03}}</ref>。
* 『[[星のカービィ トリプルデラックス]]』のクィン・セクトニアは、「高貴で、気高くて、でも少し狂気を孕んだ物悲しさ」を意識して演じた<ref>{{Cite web |url=http://tinowa.cocolog-nifty.com/mugendai/2014/03/post-afbb.html |title=『ホース＆ハトック　∞　-無限大-』 2014年3月1日 |accessdate=2015-7-09}}</ref>。
