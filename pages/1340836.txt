'''小見川 千明'''（おみがわ ちあき、[[1989年]][[11月11日]]<ref name="prof"/> - ）は、[[日本]]の[[声優]]、[[舞台]][[俳優|女優]]<!--削除はノートの議論終了後にお願いします-->。[[クロコダイル (声優事務所)|クロコダイル]]所属。

[[神奈川県]]出身<ref name="prof"/>。[[日本芸術高等学園]]卒業<ref>{{Cite web |url=http://j-highschool.nichigei.ac.jp/01.html |title=日本芸術高等学園の特色 |publisher=日本芸術高等学園 |accessdate=2015-02-17}}</ref>。身長158cm<ref name="prof"/>。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]<ref name="prof"/>。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
4歳より[[劇団]]に入り舞台活動を開始、10歳で退団した後は、自らオーディションを受け舞台活動を続け<ref name="biography">{{Cite web|date=|url=http://omigawa.net/biography/01.html |title=Biography|work=小見川千明オフィシャルウェブサイト|author=|publisher=|accessdate=2017-08-04 |archiveurl=https://web.archive.org/web/20150418005835/http://omigawa.net/biography/01.html |archivedate=2015-04-18}}</ref>、[[2006年]]に[[ヒラタオフィス]]に所属<ref name="biography"/>、[[2008年]]にテレビアニメ『[[ソウルイーター]]』の主人公「マカ＝アルバーン」役で声優デビューする<ref>{{Cite web |url=http://news.mynavi.jp/articles/2008/08/22/souleater/ |title=『ソウルイーター』DVDが発売開始 - ヒロイン・マカを演じる小見川千明にいろいろ聞いてみた! (1) 4歳のころにデビュー |work=マイナビニュース |accessdate=2015-02-17}}</ref>。
[[2016年]][[12月31日]]、所属していたヒラタオフィスを離れて[[フリーランス|フリー]]で活動することを発表。[[2017年]]8月16日より[[クロコダイル (声優事務所)|クロコダイル]]に所属<ref>{{Twitter status2|omigawa_chiaki|897659879430782976|accessdate=2017-08-16}}</ref>。

== 人物 ==
特技は[[少林寺拳法]]（アクション）、人の背中で腹筋をすること<ref name="prof">{{Cite web|date=|url=http://omigawa.net/profile/profile.html|title=Profile|work=|author=小見川千明|publisher=小見川千明オフィシャルウェブ|accessdate=2013-07-09|archiveurl=https://web.archive.org/web/20130929081358/http://omigawa.net/profile/profile.html|archivedate=2013年9月29日|deadlinkdate=2017年10月}}</ref>。趣味は[[読書]]、[[詩]]を書く事、[[ダンス]]（[[ジャズダンス]]、[[バレエ|クラシックバレエ]]）、[[歌唱]]（音域：下のソ〜1オクターブ上のシ）<ref name="prof"/>。習い事として、バレエ、ジャズダンス、[[ヒップホップ (ダンス)|ヒップホップダンス]]などをやっていた<ref name="hanasakuiroha">{{Cite web |url=http://www.hanasakuiroha.jp/interview/interview_21.html |title=インタビュー |work=「花咲くいろは」公式サイト |accessdate=2015-02-17}}</ref>。好きな作家に[[宮沢賢治]]を挙げている<ref>{{Cite web |url=http://repotama.com/2014/12/78235/ |title=『ゆるふわ』は永久に不滅です！？ 『小見川千明のゆるっとふわっと』最終回レポート |work=れポたま！ |accessdate=2015-02-17}}</ref>。

地声は高い（俗に言うアニメ声）が、声優デビュー当初から低い声（一般的な地声に近い）での演技の方が多いため、声のギャップに驚かれ、初めて会った人に自己紹介をする時など、地声とのギャップのためにそれらのキャラクターを演じている人だと信じてもらえない事も多い<ref>{{Cite web |url=http://zendesu.chunsoft.jp/weekly/interview/ |title=小見川千明さんインタビュー |work=週刊善人シボウデス |accessdate=2015-02-17}}</ref>。ラジオに出演した際、低い声のマカを演じる小見川の地声があまりにも可愛らしい事に驚いたリスナーから、次々と驚きの声がメールで寄せられた<ref>{{Cite web |url=http://news.mynavi.jp/articles/2008/08/22/souleater/ |title=『ソウルイーター』DVDが発売開始 - ヒロイン・マカを演じる小見川千明にいろいろ聞いてみた! (1) 4歳のころにデビュー |work=マイナビニュース |accessdate=2015-02-17}}</ref>。

引退した[[大前茜]]の役を引き継いでいる。

『[[花咲くいろは]]』公式サイトのインタビューにて、かなりの[[アルバイト|アルバイター]]であったことを告白。その内容も[[ファーストフード|ファーストフード店]]、[[コンビニエンスストア|コンビニ]]、[[居酒屋]]、[[デパ地下]]の販売員、劇場アナウンスなど多岐に渡る<ref>{{Cite web |url=http://www.hanasakuiroha.jp/interview/interview_21.html |title=インタビュー |work=「花咲くいろは」公式サイト |accessdate=2015-02-17}}</ref>。

=== 天然キャラクター ===
「[[STARCHILD FESTIVAL|スターチャイルドフェスティバル2009・Spring]]」における『[[夏のあらし!]]』と『[[アスラクライン]]』の合同イベントのクイズコーナーで、「最終兵器」と称されるほどの珍回答を連発した<ref>{{Cite web |url=http://news.mynavi.jp/articles/2009/05/30/starchild/001.html |title=約2年ぶりの開催でトークにライブに会場は大熱狂! 「STARCHILD FESTIVAL 2009・spring」 (2) 作品コーナーは春の新番組、そして夏へ |work=マイナビニュース |accessdate=2015-02-17}}</ref><ref>{{Cite web |url=http://www.animate.tv/news/details.php?id=1241741447 |title=2年ぶり開催『STARCHILD FESTIVAL 2009.. |work=アニメイトTV |accessdate=2015-02-17}}</ref>。[[内山昂輝]]は小見川と初めて会った時、天然なキャラを「作っている」ように見えて「苦手なタイプかな」と思ったが、実はそれが自然体だったと語っている<ref>{{Cite journal|和書|journal=[[アニカン|別冊アニカンR]]|volume=006|issue=|pages=|author=若林和弘|title=|date=2008-6|publisher=エムジーツー|id=}}</ref>。

=== 交友関係 ===
憧れの声優は、[[桑島法子]]と[[坂本真綾]]<ref>{{Cite journal|和書|journal=[[オトナアニメ]]|volume=Vol.11|issue=|pages=|author=若林和弘|title=|date=2009-01-10|publisher=[[洋泉社]]|id=ISBN 978-4-86-248364-5}}</ref>。桑島に関しては、[[朗読夜]]で自身も好きな宮沢賢治の朗読をライフワークにしていることに由来し、桑島が朗読しているプラネタリウムの『[[銀河鉄道の夜]]』を観た小見川は感動のあまり号泣し、DVD-BOXを買っている<ref>{{Cite journal|和書|journal=[[声優アニメディア]]|issue=2009年2月号|title=|ソウルイーター“チィちゃんウッチーのアナタの魂伺いま〜す第9回|publisher=[[学研パブリッシング]]|date=2009-01-10}}</ref>。

『[[夏のあらし!]]』や『[[荒川アンダー ザ ブリッジ]]』などで共演している[[杉田智和]]には、現場で元ネタのある台詞を教えてもらったりする<ref>{{Cite journal|和書|journal=[[オトナアニメ]]|volume=Vol.19|issue=|pages=|author=小見川千明|title=|date=2011-01-08|publisher=洋泉社|id=ISBN 978-4-86-248679-0}}</ref>など、親しく会話できる関係である。
