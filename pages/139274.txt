'''清水 マリ'''（しみず まり、[[1936年]][[6月7日]] - ）は、[[日本]]の[[女性]][[声優]]、[[俳優|女優]]。[[劇団俳優座]]付属養成所7期、[[劇団朋友|劇団新人会]]、[[高松事務所]]、[[青二プロダクション]]、[[ぷろだくしょんバオバブ]]を経て、[[2016年]][[7月1日]]から[[81プロデュース]]に所属<ref name="onkyo"/>。2003年から個人事務所である[[清水マリ事務所]]でも活動している<ref name ="bio"/>。声優ミュージアム名誉館長。

[[埼玉県]][[浦和市]]（現：[[さいたま市]][[浦和区]]）出身<ref name="voice3"/>。現在も浦和区在住<ref>[http://www.sankei.com/region/news/150708/rgn1507080062-n1.html 浦和から飛んだ「鉄腕アトム」　声優・清水マリさんが自伝出版]</ref>。[[浦和市立仲本小学校]]、[[浦和市立原山中学校]]、[[埼玉県立浦和西高等学校]]卒業<ref name ="bio"/>。

[[水垣洋子]]、[[古谷敏]]、[[西塔紅美]]、[[木澤雅博]]らとの、朗読劇の公演を開始している。父親は俳優の[[清水元]]。通称“声優の母”。

== 経歴・人物 ==
中学1年時に父親の劇団で『[[ピノキオ]]』の[[舞台]]を上演することになったが、劇団に[[子役]]がいなかった為、清水がピノキオ役で出演することになり、それがきっかけで役者の道に進む<ref name="voice1">{{Cite web|url=http://seigura.com/senior/road/20131119_110/1.html|title=清水マリの声優道 みんなでひとつの舞台を作っていくという感覚が好きだった|publisher=声優グランプリweb|accessdate=2014-06-28}}</ref>。父親は役者になることについては反対していた<ref name="voice1"/>。

高校では演劇部に所属し、卒業後に俳優座の養成所に入所<ref name="voice1"/>。卒業後は劇団新人会に入団<ref name="onkyo"/>。女優業の他、[[渡辺美佐子]]の付き人や、演出助手などを経験<ref name="voice1"/>。日本初の30分アニメの主役、『[[鉄腕アトム]]』のアトム役を演じる。これは手塚プロに清水元の劇団の出身者がおり、[[手塚治虫]]からアトムのモチーフがピノキオであると聞かされた事から、中学生時代にピノキオを演じた清水にオファーを出したとされる<ref name="voice1"/>。その後もほぼ一貫してアトム役を演じていたが、[[2003年]][[4月6日]]をもってアトム役を降りることを決める<ref name="voice3">{{Cite web|url=http://seigura.com/senior/road/20131119_110/3.html|title=清水マリの声優道 感情を表現するにはたくさんの経験を積まないといけない|publisher=声優グランプリweb|accessdate=2014-06-28}}</ref>。

[[虫プロダクション]]のアトムの仕事が終わった後は劇団の活動を中心にするつもりだったが、番組放映期間中に子供が生まれて舞台活動を続けていくことが困難になり<ref name="voice2"/>、[[1966年]]に新人会を退団<ref name="bio"/>。[[大山のぶ代]]の誘いで<ref name="voice2"/>、高松事務所に所属<ref name="bio"/>。声優に転身した<ref name="voice2">{{Cite web|url=http://seigura.com/senior/road/20131119_110/2.html|title=清水マリの声優道 不思議なご縁でさまざまな素晴らしい作品に巡り会えた|publisher=声優グランプリweb|accessdate=2014-06-28}}</ref>。

バオバブの附属養成所 B・A・Oや<ref name ="bio">{{Cite web |url=http://www2.odn.ne.jp/mari-collection/newpage2.htm|title=プロフィール|publisher=清水マリ公式ホームページ|accessdate=2014-06-28|archiveurl=https://web.archive.org/web/20131017021630/http://www2.odn.ne.jp/mari-collection/newpage2.htm|archivedate=2013-10-17 }}</ref>、[[音響芸術専門学校]]演習・演劇研究講師の講師でもある<ref name="onkyo">{{Cite web|url=http://www.onkyo.ac.jp/seiyu/teacher.html|title=主な講師|publisher=音響芸術専門学校|accessdate=2014-06-28}}</ref>。

2006年、第5回[[東京アニメアワード]]功労賞を受賞<ref name="onkyo"/>。2017年、第11回[[声優アワード]]功労賞を受賞<ref>{{Cite news|publisher=アニメイト|work=アニメイトタイムズ|url=http://www.animatetimes.com/news/details.php?id=1487652061|title=「第11回 声優アワード」受賞者の一部を先行発表! 小林清志さん・清水マリさん・堀絢子さん・中尾隆聖さん・島本須美さんが受賞決定|date=2017-02-21|accessdate=2017-02-21}}</ref>。

== エピソード ==
* アトムを演じていた頃に、[[アフレコ]]収録を行う[[スタジオ]]へ[[タクシー]]で移動した際に、その自宅までの往復分の運賃で、1話分のギャラが全部なくなってしまったという。当時の声優の待遇の悪さを物語るエピソードである。
* 父親の清水元が出演していた<ref>{{Twitter status|hashimotoichi23|831219968427503616}}</ref>『マグマ大使』の主題歌の歌詞の中で、'''「SOS SOS カシン カシン カシン」'''の声を担当していた<ref>{{Twitter status|hashimotoichi23|831204764385898496}}</ref>。
