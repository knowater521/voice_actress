'''朴 璐美'''（ぱく ろみ、[[1972年]][[1月22日]]<ref name="t2000043895">{{Cite web |url=http://www.talent-databank.co.jp/search/t2000043895 |title=朴{{JIS2004フォント|璐}}美 |publisher=タレントデータバンク |accessdate=2017-05-05}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]、[[俳優|女優]]、[[歌手]]<ref name="t2000043895" />。国籍は日本<ref name="kanpou">官報（2015年12月25日付）</ref>。[[東京都]]出身<ref name="enprof">{{Cite web |url=http://www.enkikaku.jp/women'sprofile/park-prof.html |title=朴璐美 |publisher=円企画 |accessdate=2017-05-05}}</ref><ref name="comprof">[https://web.archive.org/web/20120712001248/http://park-romi.com/prof.html プロフィール - 朴{{JIS2004フォント|璐}}美 旧公式サイト]（2012年7月12日時点の[[インターネットアーカイブ|アーカイブ]]）</ref>。身長161cm<ref name="t2000043895" /><ref name="enprof" />。血液型AB型<ref>[http://ameblo.jp/romi-nee/day-20110201.html AB型] 朴{{JIS2004フォント|璐}}美オフィシャルブログ 2011年2月1日付</ref>。[[演劇集団 円]]を経て、現在は個人事務所LALを立ち上げ[[フリーランス]]<ref>[http://www.romi-park.com/posts/3242161 ご報告です],朴 璐美 オフィシャルウェブサイト,2017年11月15日</ref>。

代表作は『[[∀ガンダム]]』（[[ロラン・セアック]]）、『[[鋼の錬金術師]]』シリーズ（[[エドワード・エルリック]]）、『[[NANA#テレビアニメ|NANA（テレビアニメ版）]]』（大崎ナナ）など<ref name="t2000043895" />。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
1990年に[[和洋国府台女子高等学校]]を卒業後、[[桐朋学園芸術短期大学]]演劇科に入学し、1992年に卒業<ref name="comprof" />。その後の半年間は父祖の母国である[[大韓民国|韓国]]・[[延世大学校 韓国語学堂|延世大学の韓国語学堂]]に留学した<ref name="comprof" />。1993年に円演劇研究所へ入り、2年後の1995年に[[演劇集団 円]]の会員に昇格した<ref name="enprof" /><ref name="comprof" />。

「[[東京国際アニメフェア]]2004」にて声優賞を受賞<ref name="enprof" /><ref name="comprof" />。「第1回[[声優アワード]]」で、『[[NANA]]』の大崎ナナ役で主演女優賞を受賞した<ref name="enprof" /><ref name="comprof" /><ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_01 |title=第一回声優アワード受賞者 |publisher=声優アワード |accessdate=2017-05-05}}</ref>。

2006年からは歌の仕事も行い、CDをリリースしている<ref name="avexnet">[https://web.archive.org/web/20110918075813/http://avexnet.jp/id/bakur/discography/ ディスコグラフィー - エイベックス]（2011年9月18日時点の[[インターネットアーカイブ|アーカイブ]]）</ref>。

2013年には[[第25回東京国際映画祭]]、日本映画・ある視点部門ノミネート作品『[[あかぼし]]』で実写映画に初主演した<ref name="">{{Cite web |url=http://eiga.com/movie/77563/ |title=あかぼし : 作品情報 |publisher=映画.com |accessdate=2017-05-05}}</ref>。

2015年12月25日、日本に帰化した<ref name="kanpou" />。

2016年、出演予定だった舞台『箱の中身2016』を体調不良のため降板<ref>{{cite news|url=http://www.sponichi.co.jp/entertainment/news/2016/03/24/kiji/K20160324012272530.html|title=「NANA」声優・パク・ロミ、体調不良で舞台降板「とても舞台に立てる状態では…」|newspaper=スポニチアネックス|date=2016-03-24|accessdate=2016-03-24}}</ref>。

2017年11月15日付で22年間在籍した演劇集団 円（円企画）を退所、個人事務所LAL（ラル）を設立して、今後はフリーとして活動することを発表した<ref>[http://www.romi-park.com/posts/3242161 ご報告です],朴 璐美 オフィシャルウェブサイト,2017年11月15日</ref>。

== 人物・特色 ==
趣味や特技は[[スキューバダイビング]]、[[少林寺拳法]]、[[ピアノ]]、[[水泳]]、[[朝鮮語|韓国語]]<ref name="enprof" />。

もともと、演劇に興味を持っていたわけではなく、先輩に誘われたのが理由だったとのこと<ref name="paku1">{{Cite web |url=http://seigura.com/senior/road/paku_seiyudo/1.html |title=朴{{JIS2004フォント|璐}}美の声優道 ①一生を賭して演劇をやっていくなんて考えもしなかった |work=声優グランプリweb |publisher=主婦の友インフォス情報社 |accessdate=2017-05-05}}</ref>。舞台でも声優でも、老若男女幅広い役を演じている<ref name="paku1" />。特にアニメでは、ティーンエイジの少年役に起用される事が多く、様々な役柄を演じている<ref name="paku2">{{Cite web |url=http://seigura.com/senior/road/paku_seiyudo/2.html |title=朴{{JIS2004フォント|璐}}美の声優道 ②作品を通してさまざまな人と知り合った |work=声優グランプリweb |publisher=主婦の友インフォス情報社 |accessdate=2017-05-05}}</ref>。

尊敬する声優は『[[シャーマンキング]]』で共演した[[高山みなみ]]と[[林原めぐみ]]<ref name="paku2" />。高山を父、林原を母であると公言している<ref name="paku2" />。声優の[[川上とも子]]は桐朋学園短大時代の同期生であり、川上が2011年6月に亡くなった際には自らのブログで彼女への思いを語り、また通夜にかけつけた<ref>[http://ameblo.jp/romi-nee/day-20110614.html ゴメンね] 朴{{JIS2004フォント|璐}}美オフィシャルブログ 2011年6月14日付</ref>。

本人の名前には[[JIS X 0208]]外の文字（「・」の字：[[王部|王偏]]に「路」、[[JIS X 0213]]にはある）が使われている。自身の公式サイトでは「ぱくろみ」と書くことがあり、表示できないウェブサイトなどでは「朴ロ美」、「朴王路美」などと表記されることがある<ref name="avexnet" /><ref>{{Cite web |url=http://www.kinenote.com/main/public/cinema/person.aspx?person_id=163260 |title=朴ロ美 - 略歴・フィルモグラフィー |work=KINENOTE |publisher=キネマ旬報社 |accessdate=2017-05-05}}</ref>。

== 事業家 ==
2010年から2012年までダイニングカフェ・シーラカンスを[[ゲッターズ飯田]]と共同経営していた<ref>[http://ameblo.jp/romi-nee/day-20121022.html ありがとうございました] 朴{{JIS2004フォント|璐}}美オフィシャルブログ 2012年10月22日付</ref>。

2013年から声優育成私塾studio Cambria（スタジオ カンブリア）を開校<ref>{{Cite web |url=http://www.voice-romi.jp/ |title=･ 2013 studio Cambria (最下部に表記） |publisher=スタジオ カンブリア |accessdate=2017-05-05}}</ref>。
