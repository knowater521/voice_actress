'''橘田 いずみ'''（きった いずみ、[[11月27日]]<ref name="Hibiki">{{Cite web |url=http://hibiki-cast.jp/kitta/ |title=響 HiBiKiの所属声優一覧 |accessdate=8 Novebber, 2015|date= |work=Bushiroad Music }}</ref> - ）は、[[日本]]の[[女性]][[声優]]である。[[響_(芸能プロダクション)|響]]所属<ref name="Hibiki"/>。また[[餃子]][[評論家]]としても活動している<ref name="TS"/>。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 来歴 ==
[[2007年]]から声優養成学校に通い、[[2008年]]より声優としての活動を始める<ref name="TS"/>。[[ドワンゴクリエイティブスクール]]1期生。以前は[[ドワンゴアーティストプロダクション]]に所属していた。このほか、[[声優ユニット]]「[[ミルキィホームズ (声優ユニット)|ミルキィホームズ]]」のメンバーとしても活動している<ref name="Hibiki"/>。また書籍の刊行、「[[ズームイン!!サタデー]]」等でのテレビ出演<ref name="Hibiki"/>、「[[お願い!ランキング|お願い!ランキング GOLD]]」等でのランキング監修<ref name="Hibiki"/>、[[宇都宮市]]が開催する餃子関連のイベントの出演など<ref>[https://web.archive.org/web/20140213230309/http://www.gyozakai.com/information/3812.html/ 宇都宮餃子祭り２０１３　開催のお知らせ！！],宇都宮餃子会公式サイト,2013年10月4日</ref>、餃子評論家としても活動する<ref name="TS"/>。著書に『橘田いずみのザ・餃子』『本日も餃子日和』がある。

== 人物 ==
『[[私がモテないのはどう考えてもお前らが悪い!]]』で演じた黒木智子（もこっち）役が地声に近く、小さいころから自分の低い声が好きではなかったと語っている<ref name="aud"/><ref name="da">{{Cite web|url=http://ddnavi.com/serial/159185/|title=『ワタモテ』もこっちを熱演している橘田いずみさんに直撃！【後編】|work=[[ダ・ヴィンチ]]電子ナビ|publisher=[[KADOKAWA]]|date=2013-08-24|accessdate=2013-10-04}}</ref>。そのため人前では[[裏声]]で話している<ref name="Hibiki"/>。また大人になってから声をほめられたことが声優という職業を意識したきっかけであると語っている<ref name="aud">{{Cite journal |last = [[Audition]]編集部 |title = Audition（オーディション）2012年1月号 |publisher = [[白夜書房]] |year = 2011}}</ref>。

趣味として餃子研究、餃子作り、[[百合]]を挙げている<ref name="Hibiki"/>。こと餃子に関しては筋金入りで、物心ついたころから好きな食べ物はずっとギョーザとしか答えなかったと語る<ref>{{Cite book|last = いずみ|first = 橘田|title = 橘田いずみのザ・餃子|publisher = [[ブシロードメディア]]|year = 2014|isbn= 978-4-04-899404-0-C0076
}}</ref><ref name="TS">{{Cite journal|last = [[東京スポーツ]]|title= ギョーザ大好き　オススメ店|publisher = 東京スポーツ|year = 2015|date = 23 February}}</ref>。2011年2月26日には池袋餃子スタジアム（現[[ナムコ・ナンジャタウン]]）の1日町長に任命されている。それを皮切りに2013年には[[TBSテレビ|TBS]]『[[ランク王国]]』<ref name="Hibiki"/>、『[[ズームイン!!サタデー]]』<ref name="Hibiki"/>、宇都宮餃子祭り2013[[トークショー]]出演など<ref name="Hibiki"/>、その餃子愛と知識から数々の仕事をこなしている<ref>{{Cite web |url=http://prtimes.jp/main/html/rd/p/000000010.000014827.html |title=声優・橘田いずみが[[パッケージ]]に！『宇都宮餃子会監修 水餃子』発売 |accessdate=8 November, 2015 |date=1 September |work=PR TIMES Inc.}}</ref>。[[月刊ブシロード]]誌上で自身の餃子コーナーを持つに止まらず、『[[ロボットガールズZ]]』では宇都宮市の[[マスコットキャラクター|マスコットキャラ]]の「ミヤリー」を演じる<ref name="Hibiki"/>。なお彼女の弁によると宇都宮移住を本気で考えているそうだが、「事務所に止められている<ref>[http://ameblo.jp/miyaryblog/entry-11935184274.html ミヤリー日記 宇都宮市民じゃないのに愉快市民に任命♪餃子人間橘田いずみちゃん]</ref>」という話である。2013年には、宇都宮餃子PR特命[[アンバサダー]]に任命<ref name="Hibiki"/><ref name="TS"/>、2014年には餃子グルメ本『ザ・餃子』を出版、そして宇都宮愉快市民に任命された<ref>[http://eiga.com/news/20141004/1/ 「ミルキィホームズ」声優・橘田いずみ、宇都宮愉快市民に就任！]</ref>。

前述のように百合を趣味に挙げており<ref name="Hibiki"/>、自他共に認める「百合[[オタク|オタ]]」<ref name="tpia">{{Cite journal| last = [[声優パラダイスR|声優パラダイス]]編集部| title = 声優パラダイス（PARADISE）vol.12| publisher =  	 [[グライドメディア]]| year = 2012| isbn= 9784813081852| p=53}}</ref>。それが高じて、2012年には[[PigooHD]]にて[[三上枝織]]とともに[[パーソナリティ]]を務める『百合魂-ゆりイズム-』という番組を持っていた<ref name="tpia"/>。

[[カードゲーム]]を特技として挙げている<ref name="Hibiki"/>。養成所時代に[[木谷高明]]から『[[サンデーVSマガジン カードゲームラジオ|サンデーVSマガジン]]』の[[ラジオ]]をやらないかと誘われたことがカードゲームをはじめるきっかけとなった<ref name="CG7">{{Cite journal|last = |title = [[カードゲーマー]] vol.7|publisher = [[ホビージャパン]]|year = 2012|isbn= 9784798605067}}</ref>。
TBSで放送された『[[カード学園]]』では自ら[[講師]]役で出演していた<ref>{{Cite web| url= http://www.tbs.co.jp/card-gakuen/cast.html| title = TBS「カード学園」 / [[キャスト]]| work = Tokyo Broadcasting System Television, Inc| accessdate= 8 November, 2015
}}</ref>。また、[[サンデーVSマガジンTCG]]で全国大会準優勝の実績がある<ref name="CG7"/><ref>{{Cite web|date=2010-12-07|url=http://www.d-cs.jp/special/setumei/yarikata.html |title=[[ドワンゴクリエイティブスクール]]サイト|work=ドワンゴクリエイティブスクール|author=|publisher=[[ドワンゴ]]|accessdate=2013-04-03}}{{リンク切れ|date=2013年4月3日 (水) 04:53 (UTC)}}</ref>。
