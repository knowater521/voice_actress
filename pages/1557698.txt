'''小林 ゆう'''（こばやし ゆう、<!-- [[1982年]]<ref name="結婚発表">{{Cite web|date=2017-12-29|url=https://www.oricon.co.jp/news/2103365/|title=声優・小林ゆう、一般男性と結婚発表「これからも精一杯精進」|publisher=[[オリコン]]スタイル|accessdate=2017-12-29}}</ref><ref group="注">2017年12月29日の結婚発表報道の時点で35歳</ref>-->[[2月5日]] - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]、元[[モデル]]。[[ホーリーピーク]]所属。

[[東京都]]出身。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]。愛称は'''ゆうちゃん'''、'''画伯'''など。公式ファンクラブは「'''White Wing'''」。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
高校生の頃スカウトされ[[モデル (職業)|モデル]]として活動を始める<ref>{{Cite web |date=2007-05-29 |url=http://www.famitsu.com/anime/news/2007/05/29/681,1180407364,72428,0,0.html |title=小林ゆうインタビュー：役作りにかける並々ならぬ情熱 |work=ファミ通.com |publisher=KADOKAWA |accessdate=2014-12-18}}</ref>。[[2003年]]に[[声優]]として活動を始め、[[2004年]]放送の「[[DAN DOH!!]]」で初主演。[[2008年]]には、第2回[[声優アワード]]にて新人女優賞を受賞。

[[2006年]][[6月20日]]、ファンクラブ『White Wing』を発足し、[[2007年]]、「[[空のコトバ]]」で歌手デビュー。[[2010年]]2月からロックユニット「'''[[Crush Tears]]'''」として音楽活動を開始し、[[2013年]]7月26日に[[小野坂昌也]]・元[[AAA (音楽グループ)|AAA]]の[[後藤友香里]]らと[[アニソン]]EDM（ADM）ユニット「'''[[EMERGENCY]]'''」を結成し「CRAZY YU」の名で活動を開始。

[[2012年]]8月1日に[[落語]]CD「モエオチ!」を発売。

2017年12月29日、自身のTwitter<ref>https://twitter.com/holy_kobayashi/status/946690074980237313</ref>で一般人男性と入籍したことを発表した<ref name="結婚発表"/>。

== 人物 ==
* 好きな言葉は「Never too late（決して遅すぎることはない）」。
* [[栄養士]]の資格を持っている。「料理に興味はある」と言う<ref>{{Cite web|date=2013-02-11|url=http://www.famitsu.com/blog/yu/1219453_1827.html|title=赤松健先生×小林ゆうのクロストーク（その4）|work=ゆうのお部屋|author=|publisher=[[エンターブレイン]]|accessdate=2013-07-18}}</ref>。
* 個性的な絵を書くことで知られる（[[#画業|後述]]）
* 自身が初めて歌った『[[DAN DOH!!]]』のキャラクターソング「My Dream」では作詞も担当している。

=== 趣好・友好関係 ===
* 好きな音楽はロック。好きなアーティストとして[[クイーン (バンド)|Queen]]や[[Red Hot Chili Peppers]]をあげている。ライブイベントではそのロック魂を存分に発揮しており、[[大槻ケンヂ]]は「声優界とロック界をつなぐ逸材」と評した。現在、ロックバンド[[Crush Tears]]のヴォーカルと作詞を務める。
* 大の豚好き（動物としても、食べ物としても）。ラジオ番組『[[小林ゆうの（仮）]]』では豚のぬいぐるみに囲まれて放送している。
* [[大槻ケンヂ]]や[[田村直美]]との親交が深い。大槻とはお互いのライブやイベント、楽曲に参加したりしている。また、田村は小林のボイストレーナーもしている。
* 『[[ONE PIECE (アニメ)|ONE PIECE]]』、『[[ドラゴンボール]]』の大ファン。
* [[落語]]が好きで[[立川談志]]のファン。2012年には落語CDを発売。『昭和元禄落語心中』では小夏役のオーディションに受かってあまりの嬉しさで泣き出したという。また、自ら『萌えオチ落語』という古典をアレンジした咄を行い、『小林ゆうの落語でありんす』という番組を持つほど造詣が深い。
* 『[[KERA]]』の愛読者でモデルとしても不定期に登場している。

=== 特色 ===
* 独特のハスキーボイスで男女どちらとも演じることができ、少年役や姉御肌的な女性、中性的な役柄を担当することが多い。
* 役作りに対する姿勢はデビュー当時から現在に至るまで非常に真面目かつ積極的で、『[[DAN DOH!!]]』で共演した[[中原茂]]は自分の予想を超えた小林の熱心な姿を自身のHPのエッセイに綴っている。
* ララ・ゴンザレス（『[[スクールランブル (アニメ)|スクールランブル]]』）、猿飛あやめ（『[[銀魂 (アニメ)|銀魂]]』）や桜咲刹那（『[[ネギま!?]]』）、木村カエレ（『[[さよなら絶望先生 (アニメ)|さよなら絶望先生]]』シリーズ）など、[[高松信司]]、[[新房昭之]]両監督の作品においてエキセントリックなキャラクターを演じる事が多い。
* 「芝居にすごくパワーがある」と脚本家の[[小黒祐一郎]]に評され、『さよなら絶望先生』の木村カエレは「役者のおかげでキャラクターが膨らんだ」と副監督の[[龍輪直征]]に言われるほど、演技に関しては評価が高い<ref>『【俗・】さよなら絶望先生 絶望案内正本』懺悔部屋1 スペシャルインタビューより。</ref>。
* 動物の鳴きまねが得意で、ラジオなどで度々披露している。猿や豚の鳴きまねをすることが多い。『[[メタルギアソリッド ピースウォーカー]]』に出演した際はこれを受けてシナリオが追加され、ゲーム内で[[ケツァール]]などの鳴きまねを聞くことができる<ref>「METAL GEAR SOLID PEACE WALKER」 公式サイト メイキング映像 Vol.6 声優インタビューより</ref>。また[[佐藤聡美]]は幅広く演じられる小林の動物の鳴きまねをとても魅力的であるとし「小林ZOO」と称した<ref>「生徒会役員共」BR第2巻購入特典映像 佐藤聡美インタビューより。</ref>。

== 画業 ==
きわめて独創的な絵<ref>番組や雑誌などで披露されているものは黒のマジックやサインペンで描かれたキャラクターに血のような赤で彩りが加えられていることが多い。モデルを原型どおりに捉えたものがほとんど存在しないのも共通する特徴である。</ref>を描くことで知られる。各種出演アニメイベントなどで絵を披露している。魔法先生ネギま!のラジオ企画で絵を描いてから話題になり、「'''画伯'''」と呼ばれるほどになる。以下がこれまでの作品例である。

; 「First Cast 声優編 Vol.02」
: 発売元「D-project」での[http://www.d-project.info/index.html 通販]特典である特製ジャケット裏に共演の[[小林沙苗]]、[[河原木志穂]]のものと共に、それぞれの自画像が印刷されている。
; [[インターネットラジオ]]『麻帆良学園中等部2-A・ヒミツの放課後』
: 「恐怖!お絵かき対決!」のコーナーでネギま!のキャラクターを[[佐藤利奈]]やゲストと描いていた。一般に、小林の画伯伝説はこのコーナーから始まったと言われている。ここで描かれた絵が下記の単行本に収録された。
; 『[[魔法先生ネギま!]]』第16巻限定版
: クラスメイト31人全員のイラスト+作者である[[赤松健]]の似顔絵を担当。
; テレビアニメ『[[ネギま!?]]』
: このアニメでは毎回最後の提供バックにその回のメインキャラクターを演じる声優が描いた絵が登場するのだが、刹那がメインの第6話では小林が担当。予想通りの出来であり、絵が出る直前には「画伯!?」というテロップも表示された。また本編で刹那がリアクションをとる際、小林が描いた姿になることもあり『ネギま!?』のOVAでもその姿を見ることができる。さらに第11話では[[教会]]らしき場所の[[ステンドグラス]]に、彼女の絵らしきものが描かれていた。それは "Yuu Kobayashi" のサイン付きであり、クレジットにも表記された。
; 『魔法先生ネギま!』 オフィシャルファンブック「ネギパ!」
: その独創的な絵のセンスを買われ、毎号[[次回予告]]のイラストを担当している。また自身が演じている桜咲刹那が特集された第5号では、「画伯タッチの絵描き歌」まで登場。さらに、同書では「'''魅惑の絵師'''」「'''驚愕の絵師'''」「'''驀進の絵師'''」などと称されている。
; 『ネギま!?』 iPod「ネギPon!?」
: 彼女の絵が数十枚入っているほか、「小林ゆう画伯塾」なるビデオまで収録されている。
; [[Wii]]用ソフト『ネギま!? ネオ・パクティオーファイト!! 』
: ある条件を満たすと隠しキャラクターとして彼女の書いた刹那が使用可能になる。
; [[ニンテンドーDS|DS]]用ソフト『ネギま!?超麻帆良大戦チュウ』
: ソフト購入者の中から抽選で100名に、小林が描いた「アーティファクト・ジャケット」がプレゼントされた。モチーフは前作「ネギま!?超麻帆良大戦」のパッケージイラストである<ref>{{Cite web|date=|url=http://www.mmv.co.jp/special/game/ds/negi_ds2/treasure_old.html|title=秘宝館|work=「ネギま!?超麻帆良大戦チュウ チェックイ〜ン全員集合!やっぱり温泉来ちゃいましたぁ」公式サイト|author=|publisher=[[マーベラスAQL]]|accessdate=2013-07-18}}</ref>。
; 『魔法先生ネギま!』 23巻限定版カバー下
: 23巻限定版のカバーを外すとクウネル・サンダースの絵が描かれている。赤松健はビームを発射する巨神兵のようと。
; インターネットラジオ『[[小島幸子と小林ゆうの日本橋天女放送局]]』
: 『[[落語天女おゆい]]』のWebラジオで『ヒミツの放課後』同様に絵を描くコーナー「日本橋学園高校お絵かき部」があり、[[小島幸子]]とともに作品に登場しそうな妖魔を描いた。画像は公式サイトのラジオ通信のページに掲載されている。
; [[スクールランブル|スクラン祭りだよ、全員集合!!]]
: [[2006年]][[7月2日]]に行われた同名のイベントにおいては『ルチャドール』（『ルチャ・ドール』）という（本人も認める）破壊力抜群の歌を披露。。
; [[RADIOアニメロミックス|RADIOアニメロミックス 〜ひぐらしのなく頃に・こぼれ話編〜]]
: 2006年11月 - 12月に放送の[[アニメイト|animate.tv]]の同名番組内で、彼女の描いた絵のお題を当てるコーナーがあった。
; テレビアニメ『[[Saint October]]』
: 共演している[[榎あづさ|片岡あづさ]]、[[福井裕佳梨]]の[[ブログ]]に[[マクドナルド]]のキャラクター[[マクドナルド#イメージキャラクター|ドナルド]]を書いた絵がアップされた。あまりの絵に2人とも「天才」というコメントを残した。
