'''三上 枝織'''（みかみ しおり、本名同じ、[[1989年]][[1月6日]]<ref name="profile_aoni" /><ref name="mattari20100107">{{Cite web |url=http://blog.livedoor.jp/kirax2_day/archives/1222986.html |title=０から１へ |quote=昨日で無事に21歳になりました。 |author=kirax2_day |work=まったりぶろぐ |publisher=三上枝織 |date=2012-01-07 |accessdate=2015-06-20}}</ref> - ）は、[[日本]]の[[女性]][[声優]]である。[[青二プロダクション]]所属。[[青森県]]出身。身長160cm<ref name="profile_aoni" />。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]。[[専門学校東京アナウンス学院|東京アナウンス学院放送声優科]]卒業。

== 人物 ==
=== 経歴 ===
1989年1月6日、[[青森県]]生まれ。この誕生日は平成改元2日前の昭和64年である（ただし、学年上は昭和63年生まれと同学年であり、昭和63年度の次は平成元年度となる）。中学ではソフトテニス部<ref name="seiyunowa28">{{Cite web |url=http://ddnavi.com/anime-bu/210791/ |author=はるのおと |title=三上枝織さんに50の質問「声優の輪」【第28回】 |publisher=ダ・ヴィンチニュース |date=2014-10-19 |accessdate=2015-06-25}}</ref>、高校では茶華道部に所属。高校では放送委員の委員長を経験<ref name="seiguraweb20131119">{{Cite web |url=http://seigura.com/senior/20131119_137.html |title=第３回 三上枝織さん｜先輩から学ぼう！ |work=声優グランプリweb |publisher=主婦の友インフォス情報社 |date=2013-11-19 |accessdate=2015-06-20}}</ref>。

2007年に上京<ref>{{Cite web |url=http://blog.livedoor.jp/kirax2_day/archives/1002531.html |author=三上枝織 |title=4月。 |work=まったりぶろぐ |date=2009-04-02 |accessdate=2015-06-20}}</ref>し、[[専門学校東京アナウンス学院|東京アナウンス学院]]に入学。同年11月には、[[文化放送]]・[[超!A&G+]]『[[智一・美樹のラジオビッグバン]]』第7期BGPとしてラジオ初レギュラー<ref>{{Cite web |url=http://www.tohogakuen.ac.jp/news/news_view.php?msg_id=192 |title=「智一・美樹のラジオビッグバン」アシスタントに放送声優科１年三上枝織さんが決定！ |work=東京アナウンス学院 ニュース |publisher=学校法人 東放学園 |date=2007-11-14 |accessdate=2015-06-20}}</ref>。

2008年、第2回[[声優アワード|声優アワード新人発掘オーディション]]で7社からのオファーを獲得し<ref>{{Cite web |url=http://www.tohogakuen.ac.jp/news/news_view.php?msg_id=247 |title=「声優アワード・新人発掘オーディション」で学生が健闘！ |work=東京アナウンス学院 ニュース |publisher=学校法人 東放学園 |date=2008-03-17 |accessdate=2015-06-20}}</ref>、在学中のまま[[青二プロダクション]]に所属。

2009年、東京アナウンス学院放送声優科卒業<ref>{{Cite web |url=http://blog.livedoor.jp/kirax2_day/archives/995833.html |title=卒業式(*^ω ^*) |author=三上枝織 |work=まったりぶろぐ |date=2009-03-20 |accessdate=2015-06-20}}</ref>。

2011年、『[[ゆるゆり]]』シリーズの赤座あかり役に抜擢、同アニメに出演するメイン声優4人で[[声優ユニット]]「[[七森中☆ごらく部]]」を結成。[[オリコン]]週間6位（3rdシングル「[[いぇす!ゆゆゆ☆ゆるゆり♪♪]]」）をはじめ好調なセールスを記録する。

2012年、第6回[[声優アワード]]で新人女優賞を受賞<ref>{{Cite web |url=http://www.tohogakuen.ac.jp/news/news_view.php?msg_id=1223 |title=第6回声優アワードで卒業生声優・三上枝織さんが受賞！ |work=東京アナウンス学院 ニュース |publisher=学校法人 東放学園 |date=2012-03-06 |accessdate=2015-06-20}}</ref>。

2014年、Webラジオ番組『[[A&G NEXT GENERATION Lady Go!!]]』の出演者5人で声優ユニット「[[A&G NEXT GENERATION Lady Go!!#.lady.|.lady.]]（ドットレディ）」を結成、リーダーを担当<ref>{{Cite web|url=http://columbia.jp/dotlady/ |title=.lady.(ドットレディ) |work=日本コロムビア |accessdate=2015-06-20}}</ref>。

=== 愛称 ===
愛称「みかしー」は[[セガ]]のプロデューサーが名付け親であるといい<ref name="ladygo20101007" /><ref name="seharadi20150406">文化放送・超!A&G+「セハガールのハードなSEGA情報RADIO『セハラジ』」 第26回、2015年04月06日配信分。</ref><ref>{{Cite web |url=https://twitter.com/marioversion2/status/154960712412831744 |quote=「みかしー」可愛いよね。似合ってるー。名付け親として誇らしい |author=中山雅弘 |title=中山雅弘さんはTwitterを使っています |publisher=Twitter |date=2012-01-06 |accessdate=2015-06-20}}</ref><ref>{{Cite web |url=https://twitter.com/marioversion2/status/552419201777172480 |quote=みかしーのしーは枝織のしーと、merciのしーのイメージ |author=中山雅弘 |title=中山雅弘さんはTwitterを使っています |publisher=Twitter |date=2015-01-06 |accessdate=2015-06-20}}</ref>、『ゆるゆり』のサウンドトラックで「みかしー」の名を冠する楽曲が存在するなど浸透している<ref>『ゆるゆりのおんがく♪YURUYURI ORIGINAL SOUNDTRACK』34曲目「みかしーオワタ」</ref>。「みかしー」が浸透する前は主に「しおりん」と呼ばれており、事務所の先輩でもある[[神田朱未]]<ref name="fma201212" />や、[[超!A&Gミュージック+TV#OToGi8|OToGi8]]で共演したメンバーなどが用いている<ref>{{Cite web |url=http://blog.livedoor.jp/afiliasaga/archives/1401292.html |title=(類´_ゝ`)｛ときめけ！！ |quote=しおりん(三上枝織ちゃん)☆ |author=ルイズ・スフォルツア |work=アフィリア・サーガ公式ブログ |date=2010-06-24 |accessdate=2015-06-20}}</ref>。

=== 特技 ===
主な特技に幼少から習っていた[[ピアノ]]があり、[[ゆるゆり]]のライブイベントにおいて約5,000人の観客の前でソロの弾き語りを披露したことがある<ref>『[[七森中♪ふぇすてぃばる]]』、[[2013年]][[2月24日]]。</ref>。ほかの特技には中学の部活で行っていた[[ソフトテニス]]など<ref name="profile_aoni" /><ref name="seiyunowa28" />。

保有資格は[[運転免許|普通運転免許]]（マニュアル）、[[漢字検定|漢字検定2級]]、[[全国珠算教育連盟主催珠算検定|そろばん1級]]、[[全国珠算教育連盟主催暗算検定|暗算1級]]<ref name="profile_aoni" />

出身地にちなみ[[津軽弁]]を得意とする。普段は[[標準語]]だが、ラジオやイベントでたびたび津軽弁を披露することがある<ref>{{Cite web |url=http://ameblo.jp/mikami-shiori/entry-11170823260.html |title=青森。 |work=三上枝織 オフィシャルブログ 「わたしめぐり。」 |date=2012-02-20 |accessdate=2015-06-20}}</ref>。また、テレビアニメ『[[てさぐれ!部活もの|てさぐれ!部活もの すぴんおふ プルプルんシャルムと遊ぼう]]』では津軽弁のキャラクターを演じ、主題歌にも津軽弁の合いの手を入れる<ref>{{Cite web |url=http://www.ntv.co.jp/tesapuru/character/rin.html |title=キャラクター｜てさぐれ！部活もの すぴんおふ プルプルんシャルムと遊ぼう｜日本テレビ |work=日本テレビ |accessdate=2015-06-20}}</ref>など、作品内で披露する機会もある。

=== 趣味・嗜好 ===
好きな食べ物には焼肉を挙げ、2014年だけで少なくとも45回以上行ったという<ref>{{Cite web |url=http://ameblo.jp/mikami-shiori/entry-11971313292.html |title=年末感。 |work=三上枝織 オフィシャルブログ 「わたしめぐり。」 |date=2014-12-31 |accessdate=2015-06-20}}</ref>。

甘いもの（特に[[チョコレート]]、[[パフェ]]）も好きである。飲酒も好きで、[[梅酒]]、[[白ワイン]]、[[日本酒]]を好んで嗜む<ref name="seiyunowa28" />。

好きな生き物は猫。過去に飼ったことはないため猫雑誌や猫カフェに行くなどして接しているが、将来的には猫と一緒に暮らすのが夢と語っている<ref>{{Cite web |url=http://ameblo.jp/mikami-shiori/entry-11021446226.html |title=朝だよー！！  |work=三上枝織 オフィシャルブログ 「わたしめぐり。」 |date=2011-09-18 |accessdate=2015-06-20}}</ref>。

嫌いな生き物には[[テントウムシ]]を挙げ、「生きている生物の中で一番苦手」と告白している<ref>{{Cite web |url=http://ameblo.jp/mikami-shiori/entry-11270061296.html |title=＼ほら／＼ほら／ |work=三上枝織 オフィシャルブログ 「わたしめぐり。」 |date=2012-06-06 |accessdate=2015-06-20}}</ref>。

=== エピソード ===
長所は「マイペース」、短所は「マイペース過ぎる」<ref name="profile_ameba">{{Cite web |url=http://profile.ameba.jp/mikami-shiori/ |title=三上枝織 プロフィール |work=三上枝織のプロフィール｜Ameba (アメーバ) |accessdate=2015-06-20}}</ref>。

座右の銘は「私らしく行こう」<ref name="seiguraweb20131119" />、「なるようになるさ」<ref name="seiyunowa28" />。

声優を志すきっかけとなったアニメに、祖母の家で鑑賞した『[[セイバーマリオネットJ]]』を挙げている<ref name="seiguraweb20131119" />。また、『[[ポポロクロイス物語 (アニメ)|ポポロクロイス]]』を観ていたという<ref name="seiyunowa28" />。
ほとんど[[テレビゲーム]]のプレイ経験は無いというが、プレイしたことのあるゲームに『[[ポポロクロイス物語]]』<ref name="seiyunowa28" />『ぷよぷよ』『[[夢見館の物語|真説・夢見館 扉の奥に誰かが…]]』を挙げている<ref name="seharadi20150406" />。一方、自身が声優として参加している[[ソーシャルゲーム]]などをプレーすることがある<ref>{{Cite web |url=https://twitter.com/mikami_shiori/status/568653897167818752 |author=三上枝織 |title=三上枝織さんはTwitterを使っています |publisher=Twitter |date=2015-02-20 |accessdate=2015-06-20}}</ref>。

実家は自動車整備工場である。同工場を経営している父は[[ダートトライアル]]選手でもあり、兄も競技に出場することがある。本人も自動車趣味があり、DVD『声優ゆめ日記』では[[BMW・Z4]]に乗車し自身の運転で[[東京湾アクアライン]][[海ほたるパーキングエリア]]から[[首都高速湾岸線]][[大井パーキングエリア]]まで走行している。

『[[東奥日報]]』の夕刊一面で本人が特集されたことがある<ref>{{Cite news|title=あおもり輝き人（８）「三上枝織さん」|newspaper=[[東奥日報]]夕刊|date=2014-11-14|publisher=東奥日報社|page=1|language=日本語}}</ref>。これがきっかけで青森県内のイベントや[[NHK青森放送局]]のラジオ番組など地元でのイベント出演が増えているという<ref name="ameblo_20150121">{{Cite web |url=http://ameblo.jp/mikami-shiori/entry-11978628214.html |title=新年。 |work=三上枝織 オフィシャルブログ 「わたしめぐり。」 |date=2015-01-21 |accessdate=2015-06-20}}</ref><ref>{{Cite web |url=https://twitter.com/mikami_shiori/status/613909781485498369 |quote=イベントに声をかけて頂くまでの流れを聞いて、本当に全部が繋がってるんだなと嬉しくなりました。昨年の青森県立美術館さんでのイベントから、東奥日報さん、そして今年のNHK青森放送局さん、全ての繋がりに感謝しながら、イベント頑張ります(｀_´)ゞ |author=三上枝織 |title=三上枝織さんはTwitterを使っています |publisher=Twitter |date=2015-06-25 |accessdate=2015-06-25}}</ref>。

[[2017年]][[10月8日]]朝、イベントのために訪れていた[[青森市]]内で偶然サルを目撃し、その様子をスマートフォンで撮影。撮影した映像は[[青森朝日放送]]に提供され、同日夕方の[[テレビ朝日]]系報道番組『[[スーパーJチャンネル]]』で本人のインタビューと共に放送された<ref>{{Cite web |url=http://news.tv-asahi.co.jp/sphone/news_society/articles/000111772.html |title=サル出没　場所は人が行き交う青森駅前 |work=テレビ朝日 |date=2017-10-09 |accessdate=2017-10-15}}</ref><ref>{{Cite web |url=https://twitter.com/aoike/status/916974393317658625 |quote=三上枝織が今朝イベント前に散歩に出かけたら、サルと遭遇し、写真と動画を撮ったら、青森朝日放送さんが、その動画と写真をぜひ提供して欲しいとお願いされ、それだけにとどまらず、取材スタッフさんが来て、夕方のニュースに出演するという珍事件！みかしー持ってるわ！ |author=Ikeda |title=IkedaさんはTwitterを使っています |publisher=Twitter |date=2017-10-08 |accessdate=2017-10-15}}</ref><ref>{{Cite web |url=https://twitter.com/mikami_shiori/status/916993454969823232 |quote=とうとうニュースのお世話になってしまいました…笑 |author=三上枝織 |title=三上枝織さんはTwitterを使っています |publisher=Twitter |date=2017-10-08 |accessdate=2017-10-15}}</ref>。

=== 交友関係 ===
『[[ゆるゆり]]』で共演する[[七森中☆ごらく部]]のメンバー（[[大坪由佳]]、[[津田美波]]、[[大久保瑠美]]）や、『[[A&G NEXT GENERATION Lady Go!!]]』のパーソナリティ（[[上坂すみれ]]、[[小松未可子]]、大久保瑠美、[[高森奈津美]]）とはイベントやネットメディアを通じて露出する機会が多い<ref name="seiyunowa28" />。

同じ事務所の同期である[[佐々木愛 (声優)|佐々木愛]]、[[藤堂真衣]]、[[斉藤佑圭]]、[[丸山未沙希]]、[[西村麻弥]]、また先輩の[[佐藤聡美]]などと親しい。特に佐藤は三上と同じく東京アナウンス学院卒業であり、前述の『ラジオビッグバン』のBGP出身者でもある。また、DVD発売イベントや動画など共演も多い。他に、[[小野恵令奈]]<ref>{{Cite web |url=http://ameblo.jp/mikami-shiori/entry-11410907354.html |title=あまあま。 |work=三上枝織 オフィシャルブログ 「わたしめぐり。」 |date=2012-11-23 |accessdate=2015-06-20}}</ref><ref>{{Cite web|url=http://yaplog.jp/lp-erena/archive/177|title=優女になるにはどーすれば！！！|work=小野恵令奈オフィシャルブログ|date=2012-11-22|accessdate=2015-06-20|archiveurl=https://web.archive.org/web/20130617224704/http://yaplog.jp/lp-erena/archive/177|archivedate=2013年6月17日}}</ref>や、[[アフィリア・サーガ|ユカフィン・ドール]]（[[アフィリア・サーガ]]）、[[櫻井里佳|美月リカ]]<ref>{{Cite web |url=https://twitter.com/miduki_rika/status/377770597482500096 |quote=きゅーきゅーのグループLINEが楽しすぎるし、超癒し。あ、きゅーきゅーメンバーはえれな、ユカフィン、みかしー、わたしね。 |author=美月リカ |title=美月リカさんはTwitterを使っています |publisher=Twitter |date=2013-09-11 |accessdate=2015-06-25}}</ref>との交流もみられる。

また、2017年に入ってからはテレビアニメ『[[てさぐれ!部活もの|てさぐれ!部活もの すぴんおふ プルプルんシャルムと遊ぼう]]』で共演し「明坂三上のよたんぼぱやぱや」で親しくなった[[明坂聡美]]との交流もみられる。
