'''冨岡 美沙子'''（とみおか みさこ、[[1988年]][[8月13日]]<ref name="acrossprof">{{Cite web |url=http://across-ent.com/talent/women/misako_tomioka.html |title=冨岡美沙子 |work=タレントプロフィール |publisher=アクロス エンタテインメント |accessdate=2013-08-21}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[高知県]]出身<ref name="acrossprof"/>。身長は156cm<ref name="aoni">{{Cite web |url=http://www.aoni.co.jp/junior/ta/pdf/tomioka-misako.pdf |title=冨岡美沙子 |format=PDF |publisher=青二プロダクション |accessdate=2013-08-21 |archiveurl=https://web.archive.org/web/20110705192905/http://www.aoni.co.jp/junior/ta/pdf/tomioka-misako.pdf |archivedate=2011年7月5日 |deadlinkdate=2017年9月 }}</ref>。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]<ref name="seigura"/>。[[アクロス エンタテインメント]]所属<ref name="im-free">{{Cite web |author=冨岡美沙子 |date=2012-07-02 |url=http://im-free-now.jugem.jp/?eid=608 |title=事務所が決まりました! |work=GO GIRL |accessdate=2013-08-21}}</ref>。

== 人物 ==
=== 経歴 ===
高知県内の高校を卒業後、[[青二塾]]東京校に入学し<ref name="pg106">{{Cite web |url=http://enmusic.jp/pg106.html |title=生徒さんの声 |publisher=en music |accessdate=2013-08-21 |archiveurl=https://web.archive.org/web/20130905000429/http://enmusic.jp/pg106.html |archivedate=2013-09-05}}</ref>第28期卒業<ref name="aoni"/>。[[2008年]]4月から<ref>{{Cite web |author=冨岡美沙子 |date=2009-04-19 |url=http://im-free-now.jugem.jp/?eid=163 |title=１年 |work=GO GIRL |accessdate=2013-08-21}}</ref>[[2011年]][[12月31日]]まで<ref>{{Cite web |author=冨岡美沙子 |date=2012-02-17 |url=http://im-free-now.jugem.jp/?eid=592 |title=お知らせ |work=GO GIRL |accessdate=2013-08-21}}</ref>、[[青二プロダクション]]（ジュニア）に所属。6か月のフリー期間を経て、[[2012年]][[7月1日]]から[[アクロス エンタテインメント]]（預かり）所属<ref name=im-free/>。

=== 趣味・嗜好===
趣味は、ライブに行くこと<ref name=pg106/>、[[ネイルアート]]、[[音楽]]鑑賞、[[映画]]鑑賞、[[読書]]<ref name="aoni"/>。
