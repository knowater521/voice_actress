'''島 ゆいか'''（しま ゆいか、[[1996年]][[9月20日]] - ）は、[[日本]]の[[俳優|女優]]、[[歌手]]。[[熊本県]]出身。[[アミューズ]]所属。可憐Girⅼsの元メンバー。

== 略歴 ==
2008年6月、テレビ東京系アニメ[[絶対可憐チルドレン]]で、1年限定で結成されたユニット[[可憐Girl's]]にAYAMI（[[武藤彩未]]）SUZUKA（[[中元すず香]]）と共に'''YUIKA'''名義で参加。2009年3月31日に番組終了とともに活動も終了した<ref>{{Cite news |title=2008年のJ-POPを牽引するのは、可憐な小学生？|url=http://www.barks.jp/news/?id=1000040810 |date=2008-06-11 |work=Barks|publisher= |accessdate=2014-04-08}}</ref><ref>{{Cite news |title=任務完了！さようなら可憐Girl's |url=http://www.barks.jp/news/?id=1000047171|date=2009-02-18|work=Barks |publisher= |accessdate=2014-04-08}}</ref>。また、このアニメ番組の中でYUIKA役として[[声優]]に初挑戦している。

2012年、遠山昇司監督の映画「NOT LONG,AT NIGHT ‐夜はながくない‐」に出演、女優としての活動は本作が初演となる。遠山監督が"盲目の少女"という神秘的な役柄を演じる女の子として、未知数の可能性を秘めた女の子を探していた際、撮影監督の森賢一の推薦により出演が決まった<ref>{{Cite web |title= 『NOT LONG AT NIGHT 夜は長くない』映画サイト|url=http://not-long-at-night.com/cast/cast_page.html |date=2012-06-11 |work=|publisher= |accessdate=2014-04-08}}</ref>。同映画は、[[第25回東京国際映画祭]] 日本映画・ある視点にノミネートされた<ref>{{Cite web |title= 『第25回東京国際映画祭』サイト|url=http://2012.tiff-jp.net/ja/lineup/works.php?id=135 |date=2012-06-11 |work=|publisher= |accessdate=2014-04-08}}</ref>。

2014年4月2日開始のアニメ『[[くつしたがだるだるになっちゃうわけ 〜イマドキ妖怪図鑑〜|くつだる。]]』（[[NHK教育テレビジョン|NHK Eテレ]]）のメインキャストである「立花くらら」の声優を担当<ref name="natalie1">{{Cite web |url=http://natalie.mu/music/news/112020|title=さくら学院・堀内＆佐藤「くつだる。」で声優に挑戦|accessdate=2015-8-20|work=[[ナタリー (ニュースサイト)|ナタリー]]|publisher=株式会社ナターシャ|date=2014年3月14日}}</ref><ref name="dogatch">{{Cite web|url=http://dogatch.jp/news/nhk/23998|title=堀内まり菜、金井美樹、佐藤日向、島ゆいかの“あるある話”とは!?新アニメ『くつだる。』スピンオフ企画決定|accessdate=2015-8-20|work=[[テレビドガッチ]]|publisher=株式会社プレゼントキャスト|date=2014-3-31|archiveurl=https://web.archive.org/web/20150708094454/http://dogatch.jp/news/nhk/23998|archivedate=2015年7月8日|deadlinkdate=2017年10月}}</ref><ref name="cdjournal2">{{Cite web |url=http://www.cdjournal.com/main/news/-/57692|title=新作TVアニメ「くつだる。」キャストに金井美樹、島ゆいか、堀内まり菜、佐藤日向|accessdate=2015-8-20|work=CDJournal|publisher=音楽出版社|date=2014.3.10}}</ref>。同アニメの声優を担当する4名（[[堀内まり菜]]、[[金井美樹 (1996年生)|金井美樹]]、[[佐藤日向]]、島ゆいか）でユニット「マボロシ☆ラ部」を結成し、同年11月19日に主題歌と挿入歌を収録したデビューシングル「Merry Go World」を発売<ref name="entamenext1">{{Cite web |url=http://www.entamenext.com/topics_detail2/id=1109|title=元さくら学院・堀内まり菜、佐藤日向が参加するマボロシ☆ラ部が、「NO MUSIC, NO IDOL?」ポスターに初登場！|accessdate=2015-8-20|work=エンタメNEXT|publisher=株式会社 [[徳間書店]]|date=2014年11月11日}}</ref>。また、同年8月より、[[飯田來麗]]と共に[[Ustream]]にて番組「スピカの夜」を開始し、MCを務める。2015年2月1日、同番組の公開生放送で飯田來麗と共に音楽プロジェクトを始動させる事を発表<ref name="oricon">{{Cite web |url=http://www.oricon.co.jp/news/2048004/full/|title=島ゆいか&飯田來麗が音楽活動「震えた」 さくら学院・可憐Girl’s卒業生の新星|accessdate=2015-8-20|work=[[ORICON STYLE]]|publisher=[[オリコン]]株式会社|date=2015-2-1}}</ref><ref name="natalie2">{{Cite web |url=http://natalie.mu/music/news/137522|title=島ゆいか＆飯田來麗、ヒゲドライバー迎え音楽プロジェクトを始動|accessdate=2015-8-20|work=[[ナタリー (ニュースサイト)|ナタリー]]|publisher=株式会社ナターシャ|date=2015年2月1日}}</ref><ref name="amob">{{Cite web |url=https://d.amob.jp/mob/pageShw.php?site=D&cd=tokusyu_spica|title=スピカの夜 特集|accessdate=2015-8-20|work=A! MUSIC|株式会社[[アミューズ]]}}</ref>。ユニット名は番組名と同名の「スピカの夜」<ref name="realsound">{{Cite web |url=http://realsound.jp/2015/03/post-2712.html|title=島ゆいか＆飯田來麗ユニット、デビュー曲MV公開　アミューズの先輩多数出演の豪華映像に|accessdate=2015-8-20|work=Real Sound|publisher=株式会社blueprint|date=2015.3.12}}</ref>。3月18日に1stデジタルシングル「SPICA」を[[iTunes]]とアミューズ公式音楽配信サイト「A! MUSIC」にて発売<ref name="realsound"/>。

== 人物 ==
{{出典の明記|section=1|date=2015年8月14日 (金) 21:41 (UTC)}}
