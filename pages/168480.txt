'''平田 裕香'''（ひらた ゆか、[[1983年]][[9月15日]]<ref name="niiba">{{Cite web |url=http://www.nigun-niiba.co.jp/hirata/ |title=プロフィール |publisher=オフィスニグンニイバ |date=2013-1-18 |accessdate=2013-1-18 |archiveurl=https://web.archive.org/web/20121122062801/http://www.nigun-niiba.co.jp/hirata/ |archivedate=2012年11月22日 |deadlinkdate=2017年10月 }}</ref><ref name="across">{{Cite web |url=http://across-ent.com/talent/women/yuka_hirata.html |title=プロフィール |publisher=アクロス エンタテインメント |accessdate=2016-11-01}}</ref> - ）は、[[日本]]の[[俳優|女優]]、[[声優]]、[[タレント]]、[[グラビアアイドル]]である。

[[北海道]][[夕張郡]][[栗山町]]<ref>{{Cite web |author=平田裕香(@Hirata_Yukaco) |date=2016-10-29|url=https://twitter.com/Hirata_Yukaco/status/792364127326867456 |title=栗山町出身者として、栗山監督にありがとうを伝えたい。|publisher=Twitter |accessdate=2016-11-01}}</ref>出身、過去の所属は[[オフィスニグンニイバ]]<ref name="niiba"/>。[[アクロス エンタテインメント]]（業務提携<ref name="Twitter20140403" />）所属。

== 来歴 ==
1983年9月15日、[[札幌市]]の病院で誕生<ref name="bomb"/>。1993 - 1995年、小学4年生の時に[[夕張郡]][[栗山町]]に移ったため、同町を出身地としている<ref name="bomb"/>。

初めてオーディションを受けたのは中学2年生の時。その後1998年、ミュージカルの[[ピーター・パン (ミュージカル)|ピーターパン]]のオーディションで最終選考まで残ったが、この時は落選<ref name="bomb"/>。この直ぐ後、同年の[[ジャパン・アーチスト・オフィス]]「第32回新人プレゼンテーション」を契機に芸能界入りした<ref name="mynavi">{{Cite web |url=http://news.mynavi.jp/news/2013/01/21/002/ |title=平田裕香、30代突入でもグラビア--壇蜜については「彼女は彼女」ときっぱり |publisher=マイナビニュース|date=2013-01-21 |accessdate=2016-11-01}}</ref>。

1999年、「第10回 YJ全国女子高生制服コレクション」にノミネートされた<ref name="mynavi"/>。2003年3月、[[東京都立代々木高等学校]]卒業<ref>[[宝島社]]「NIPPONアイドル探偵団2000」p.190</ref>。

2013年1月28日、自身のブログに「月日を重ね、年を重ね、考えることはたくさんありました。やりたいことも増えました。やれないジレンマもありました。事務所と何度も話し合いを重ね、たくさん心配をかけたりもしましたが、前に進むための背中を押してもらい、こうして、皆さんへのご報告の形となりました」と書込み、[[オフィスニグンニイバ]]を退社したことを発表した<ref>{{Cite web |url=http://www.narinari.com/Nd/20130120349.html |title=グラドル平田裕香がフリーに、10年以上在籍の所属事務所を退所。 |publisher=Narinari.com |date=2013-01-29 |accessdate=2013-01-29}}</ref>。

2014年4月3日から、[[アクロス エンタテインメント]]と業務提携（声の仕事のみ）をしている<ref name="Twitter20140403">{{Cite web |author=平田裕香(@Hirata_Yukaco) |date=2014-04-03 |url=https://twitter.com/Hirata_Yukaco/statuses/451549472892276737 |title=業務提携として、声の仕事を【アクロスエンターテイメント】にマネージメントしていただくことになりました。|publisher=Twitter |accessdate=2014-04-03}}</ref>。2017年、「海道いちご」名義で小学館のマンガサイト「[[サンデーうぇぶり]]」にて連載のアイドルをテーマとしたマンガ『[[栃ノ木のどかのアイドル日常]]』に原案協力として参加していた。しかも、日本一のアイドルを目指して上京した女子高生・栃ノ木のどかのモデルにされているから、[[あだち充]]が彼女の由来などを考えてそのペンネームを名付けた<ref>{{Cite web |author=[[MANTANWEB]] |date=2017-11-10 |url=https://mantan-web.jp/article/20171109dog00m200008000c.html |title=平田裕香：「ゲキレンジャー」戦隊女優はアイドルマンガの原案協力者だった　あだち充がペンネーム命名 |publisher=MANTANWEB |accessdate=2017-11-10}}</ref>。
