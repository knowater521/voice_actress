'''藤東 知夏'''（ふじとう ちか、[[7月16日]] - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]。[[スーパーシャークエンターテイメント]]所属<ref name="SSE">{{Cite web |url=http://yaplog.jp/natsuiro_usagi/archive/1049 |title=事務所所属のご報告です。 - 夏色ウサギ|accessdate=2015-07-12}}</ref>。[[2015年]][[3月31日]]までは[[エイベックス・プランニング&デベロップメント]]に所属していた<ref name="free">{{Cite web |url=http://yaplog.jp/natsuiro_usagi/archive/1040 |title=ご報告です。 - 夏色ウサギ|accessdate=2015-04-11}}</ref>。

[[兵庫県]][[明石市]]出身。身長152cm。[[ABO式血液型|血液型]]は[[ABO式血液型|AB型]]。[[兵庫県立明石高等学校]]卒業。エイベックス・アーティスト・アカデミー、声優タレントコース修了生<ref name="academy">{{Cite web |url=http://ameblo.jp/aaa-bh/entry-11074590937.html |title=声優修了生・藤東知夏さん活動情報! - エイベックス・アーティストアカデミーオフィシャルブログ|accessdate=2013-01-01}}</ref>。

== 人物、来歴 ==
中学生の時に観た『[[デジモンアドベンチャー02]]』の影響を受けて声優になる事を志す。その後、兵庫県立明石高等学校を卒業後、大学に進学し、大学2年時より大阪の声優養成所に通うようになるが、それから二年後、その大阪の養成所を退所し上京。エイベックス・アーティスト・アカデミーに入学し、アカデミー在籍中に初めて受けたアニメオーディション『[[けいおん!]]』で、真鍋和役に抜擢され声優デビューとなった<ref name="koegura">{{Cite web |url=http://seigura.com/senior/20131128_757.html |title=声グラ『ただいま修行中』Vol.9 藤東知夏|accessdate=2013-01-01}}</ref>。その後はアカデミーの課程を修了し<ref name="academy"></ref>、[[2015年]][[3月31日]]までエイベックス・プランニング&デベロップメントに所属。一時的に[[フリーランス]]として活動したのち<ref name="free"></ref>、現在はスーパーシャークエンターテイメントに所属している<ref name="SSE"></ref>。

憧れの声優は[[坂本千夏]]と[[折笠富美子]]（共に『デジモンシリーズ』出演者）<ref name="koegura"></ref>。
