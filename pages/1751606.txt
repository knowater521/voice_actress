'''巽 悠衣子'''（たつみ ゆいこ、[[1987年]]<ref>{{cite news|url=https://www.sponichi.co.jp/entertainment/news/2017/12/30/kiji/20171230s00041000133000c.html|title=声優の巽悠衣子　一般男性との結婚を報告「私が結婚出来る日がくるなんて…」|newspaper=スポニチアネックス|date=2017-12-30|accessdate=2018-03-13}}</ref>[[9月17日]] - ）は、[[日本]]の[[女性]][[声優]]。[[アイムエンタープライズ]]所属。

[[大阪府]]出身。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]。[[日本ナレーション演技研究所]]卒業。

== 経歴 ==
姉が一人おり、その姉が声優志望だったためその影響で自身も声優を目指したと語っている<ref>{{Cite web|publisher=ダ・ヴィンチ電子ナビ|date=2013-09-11|url=http://ddnavi.com/serial/162241/|title=声優・巽悠衣子さんに50の質問！〜声優の輪〜|accessdate=2013-09-18}}</ref>。

高校3年の頃に養成所の資料を請求し、大学入学と同時に[[声優養成所]]に通い始めた<ref name = "syugyochu">{{Cite web|url = http://seigura.com/senior/20131128_752.html
|title = 先輩から学ぼう！ Vol.8 巽悠衣子|work = [[声優グランプリ|声グラ]] web|author = |publisher = 主婦の友インフォス情報社|date = 2009-05-10|accessdate = 2013-10-13}}</ref>。最初に入った養成所は方向性が合わず1年で辞め、大学2年ヘ進級する際に友人が通っていた[[日本ナレーション演技研究所]]の入所面接を受け本科に合格<ref name = "syugyochu" />。同養成所の大阪校に入所し、研修科への進級時に[[アイムエンタープライズ]]所属となった<ref name = "syugyochu" />。

事務所所属をきっかけに大学を辞めて上京することを考えていたが、養成所スタッフや事務所のマネージャーから「学業優先」を理由に引き止められたため、卒業に必要な全単位を大学3年次までに取得し、[[2009年]]春に上京した<ref name = "syugyochu" />。上京後は同養成所東京校へ移籍し<ref name = "syugyochu" />、[[2010年]]3月に大学、養成所を共に卒業した<ref group="PR">{{Cite web|url = http://ameblo.jp/yuihinablog/entry-10485418045.html|title = 旅立ち。|work = ゆいこのひなたミチ|author = 巽悠衣子|publisher = [[アメーバブログ|Ameba]]|date = 2010-03-19 01:51:32|accessdate = 2010-03-26}}</ref><ref group="PR">{{Cite web|url = http://ameblo.jp/yuihinablog/entry-10491109035.html|title = ただいま(*>д<*)|work = ゆいこのひなたミチ|author = 巽悠衣子|publisher = [[アメーバブログ|Ameba]]|date = 2010-03-25 19:23:12|accessdate = 2010-03-26}}</ref>。

2017年12月30日、自身のブログにて一般男性と結婚したことを発表した<ref>{{Cite news|url-https://www.sponichi.co.jp/entertainment/news/2017/12/30/kiji/20171230s00041000133000c.html|title=声優の巽悠衣子 一般男性との結婚を報告「私が結婚出来る日がくるなんて…」|date=2017-12-30|accessdate=2017-12-31|newspaper=Sponichi Annex|publisher=スポーツニッポン新聞社}}</ref>。

== 人物 ==
自分の性格について一言でいうと「ネガティブ」、好きな言葉は「備えあれば憂いなし」と語っている<ref name="アニメマシテ">{{Cite web|publisher=アニメマシテ|date=2016-07-11|url=http://www.tv-tokyo.co.jp/anime/animemashite/backnumber/|title=ここだけプロフィール 巽悠衣子|accessdate=2016-07-19}}</ref>。

仲のいい声優として[[金元寿子]]、[[山本希望]]を挙げ<ref>{{Cite web|publisher=ダ・ヴィンチ電子ナビ|date=2013-09-11|url=http://ddnavi.com/serial/162241/|title=声優・巽悠衣子さんに50の質問！〜声優の輪〜|accessdate=2013-09-18}}</ref>、衝撃を受けた声優として[[高山みなみ]]を挙げている<ref name="アニメマシテ" />。

日本ナレーション演技研究所事務局チーフの神部功一は、入所面接で基礎科ではなく本科に合格するケースは「あまりない」とし、巽を「努力家」と評している<ref name = "syugyochu" />。また、巽の性格について、やる気に溢れているのはいいが先走り過ぎてしまう、気さくで明るく現場で好かれるタイプとも述べている<ref name = "syugyochu" />。
