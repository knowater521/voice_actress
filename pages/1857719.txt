'''麻志那 恂子'''（ましな じゅんこ、[[1946年]][[11月6日]]<ref>{{Cite book |和書 |title=日本音声製作者名鑑2004 vol.1 |publisher=[[小学館]] |page=241 |date=2004-04-10 |isbn=4-09-526301-6}}</ref> - [[2013年]][[5月3日]]<ref name="facebook" />）は、日本の[[俳優|女優]]、[[声優]]。本名は'''池田 佳子'''（いけだ よしこ）<ref>{{Cite book |和書 |title=声優名鑑 |publisher=[[成美堂出版]] |page=260 |date=1999-08-10 |isbn=4-415-00878-X}}</ref>。旧名は'''麻志奈 純子'''（ましな じゅんこ）。[[神奈川県]]出身<ref name="bungakuzaprof">{{Cite web |url=http://www.bungakuza.com/member/prof/mashina-junko.htm |title=麻志那恂子 |publisher=文学座 |accessdate=2017-03-25}}</ref>。身長166cm<ref name="bungakuzaprof" />。体重53kg<ref name="pp14915">{{Cite web |url=http://talent.yahoo.co.jp/pf/profile/pp14915 |title=麻志那恂子（出典：日本タレント名鑑） |work=Yahoo!検索（人物）|publisher=Yahoo Japan |accessdate=2017-03-25 |archiveurl=https://archive.is/20130501150747/http://talent.yahoo.co.jp/pf/profile/pp14915 |archivedate=2013-05-01}}</ref>。

== 人物 ==
1967年に文学座研究所入所、1972年に[[文学座]]の座員となる<ref name="bungakuzaprof" />。

2013年5月3日死去<ref name="facebook">{{Cite web |author=森島朋美 |date=2013-05-07 |url=http://www.facebook.com/permalink.php?id=100002154788162&story_fbid=456763297738826 |title=麻志那恂子さんのご逝去の報、謹んでお悔やみを申し上げます。 |publisher=facebook |accessdate=2017-03-25}}</ref>。
