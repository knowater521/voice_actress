'''味里'''（みさと、[[6月14日]] - ）は、[[日本]]の女性[[声優]]。

[[神奈川県]]出身。フリー。かつて、[[プロダクション・エース]]、[[オフィスCHK]]に所属していた。[[アミューズメントメディア総合学院]]声優学科卒業。

== 人物 ==
[[アミューズメントメディア総合学院]]出身<ref>{{Cite web |url=http://www.amgakuin.co.jp/contents/?p=8938&author=8&cat=3 |title=【産学共同プロジェクト】映画「ねこタクシー」に在校生・卒業生が多数参加!! |accessdate=2010-06-19}}</ref>。

[[2010年]]5月、[[咲乃藍里]]との声優ユニット「''''etoile'''」結成を発表<ref>{{Cite web |url=http://ameblo.jp/eto-ile/entry-10543944346.html |title=★ユニット結成いたしました★ |accessdate=2010-06-17}}</ref>。同年9月に、[[オフィスCHK]]から[[プロダクション・エース]]へ移籍。また、[[2012年]]には『[[うぽって!!]]』で共演した[[野水伊織]]、[[富樫美鈴]]、[[佐土原かおり]]と共に声優ユニット「[[sweet ARMS]]」を結成<ref>{{Cite web|publisher=アニメ!アニメ!|url=http://animeanime.jp/article/2012/09/04/11329.html|title=「うぽって!!」から生まれたユニットsweet ARMS　ファーストライブ開催決定|accessdate=2014-02-14}}</ref>。[[2015年]][[11月30日]]付けでプロダクション・エースを退所し、[[12月1日]]から[[フリーランス|フリー]]で活動<ref>{{Cite web |url= http://ameblo.jp/ajility/archive1-201512.html |title= 【皆様に大切なお知らせ】｜ajility-store |accessdate=2015-12-03}}</ref>。

金魚すくいが得意でお祭りで124匹すくったことがある<ref>{{Cite web |url=http://ameblo.jp/ajility/entry-10171892006.html |title=描こうか迷ったけれど。。。｜ajility-store |accessdate=2012-08-05}}</ref>。
