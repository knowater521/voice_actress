'''楠見 藍子'''（くすみ あいこ、[[1983年]]<ref>『タレントデータバンク2010』、793項、株式会社タレントデータバンク、2010年、ISBN 978-4990457112</ref>[[5月11日]] - ）は、[[日本]]の[[声優]]。[[マウスプロモーション]]所属。[[千葉県]]出身。身長159cm。

== 来歴・人物 ==
[[桐朋学園芸術短期大学]]専攻科演劇専攻卒業。2008年にマウスプロモーションに所属。

父は同じ事務所に所属する[[楠見尚己]]。[[演劇集団キャラメルボックス]]に所属する[[多田直人]]と結婚したが<ref>{{Cite web |url=http://ameblo.jp/megane-diary/entry-10818229502.html |title=引っ越しをした。それと…｜眼鏡日記 |accessdate=2011-03-02}}</ref><ref>{{Cite web |url=http://ameblo.jp/ohimesama-gokko/entry-10818815485.html |title=お引越とご報告。 |accessdate=2012-04-13}}</ref>、
2011年12月9日に離婚<ref>{{Cite web |url=http://ameblo.jp/ohimesama-gokko/entry-11121386566.html |title=個人的なご報告。  |accessdate=2011-12-30}}</ref>。その後、2014年5月2日に一般男性と再婚<ref>{{Cite web |url=https://twitter.com/minnie_aiko/status/539432563110989824 |title=本人のTwitterより。 |accessdate=2014-12-01}}</ref>したことを発表し、翌年3月29日に第一子となる長男を出産<ref>{{Cite web |url=https://twitter.com/minnie_aiko/status/582026796796772352 |title=本人のTwitterより。 |accessdate=2015-03-29}}</ref>した。
