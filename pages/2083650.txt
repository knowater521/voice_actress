'''津田 美波'''（つだ みなみ、[[1989年]][[6月8日]]<ref>{{Cite web|url=http://profile.ameba.jp/00dpd/|title=プロフィール|work=津田さんは明日もがんばるよ。|accessdate=2011-09-12}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[青二塾]]東京校29期卒業<ref name="aoniprofile">{{Cite web |url=https://www.aoni.co.jp/search/tsuda-minami.html |title=津田美波プロフィール |publisher=青二プロダクション |accessdate=2016-11-25}}</ref>。[[青二プロダクション]]所属<ref>{{Cite web|publisher=青二プロダクション|url=http://www.aoni.co.jp/actress/ta/|title=女性タレント た|accessdate=2013-04-01}}</ref>。
[[神奈川県]]出身。身長156cm 靴23.5cm<ref>{{Cite web|url=http://www.aoni.co.jp/actress/ta/pdf/tsuda-minami.pdf|title=津田美波|publisher=青二プロダクション|accessdate=2013-04-14}}{{リンク切れ|date=2017年1月}}</ref>。 [[ABO式血液型|血液型]]は[[ABO式血液型|A型]]。

== 来歴 ==
子供の頃から将来は声優になろうと思っていたが、[[アニメ]]に特別な思い入れがあった訳ではなく、姉が[[アニメーター]]になるという夢があったため、姉が作ったアニメに声をつけたいという想いから声優になる決意をする。その後歌が好きということもあり、小学4年生から高校3年生まで児童合唱団で活動したほか、中学、高校とも部活は合唱部に所属した。[[俳優|女優]]・[[歌手]]である恩師の[[伊東恵里]]の紹介で知り合った[[富沢美智恵]]に声優になる方法を直接問い合わせると「[[青二塾]]に入りなさい」とアドバイスを受けたことを機に、声優の道に進むことを決め、[[青二塾]]に入塾した後、オーディションにより青二プロダクションに所属<ref name="HMV">{{Cite web|publisher=HMV ONLINE|url=http://www.hmv.co.jp/news/article/1109140044/|title=『ゆるゆり』 津田美波インタビュー！|accessdate=2011-09-17}}</ref>。

『[[フラクタル (テレビアニメ)|フラクタル]]』のフリュネに代表される透明感のある少女役、『[[ゆるゆり]]』の船見結衣のように低いトーンの少女役ほか、少年役や人間以外のキャラクターの声もこなす。声優以外でも、『[[忍たま乱太郎]]』のミュージカルに出演したり、『[[七森中☆ごらく部]]』の一員として[[バラエティ番組]]に出演するなど、活躍の場を広げている。

[[2012年]]、[[10月24日]]に理沙+ミドリ Starring 津田美波+[[三上枝織]]名義で「ボーダーライン／モノローグ」をリリースした<ref>{{Cite web|date=2012-08-16|url=http://listen.jp/store/musicnews_40522_all.htm|title=「携帯彼氏」シリーズ発の限定ユニットがシングルをリリース!|work=|author=|publisher=リッスンジャパン|accessdate=2012-08-18 09:37}}</ref>。

2013年4月、青二プロダクション「ジュニア所属」から「準所属」になった<ref>{{Cite web|url=http://ameblo.jp/00dpd/entry-11502617843.html|title=お知らせいっぱい　嘘ついてません！ここでは・・・ね |work=本人のブログより|accessdate=2014-10-17}}</ref>。

== 人物 ==
資格は普通免許（ＡＴ限定）を持っている。趣味は食レポ、観劇、男装。特技は[[相対音感]]<ref name="aoniprofile"/>。

== エピソード == 
趣味の1つに舞台鑑賞を挙げており、『[[忍たま乱太郎]]』のミュージカルで知り合った舞台仲間の舞台を観に行ったりしている。『[[ゆるゆり]]』の共演者達が自分の舞台を観に来てくれたりもするという<ref name="HMV"/>。

自身が『[[ポケットモンスター ベストウイッシュ]]』に出演していることもあってゲーム『[[ポケットモンスター ブラック・ホワイト]]』をプレイしており、特に自身が演じる[[キバゴ]]は「進化させるキバゴ」、「キバゴのまま強くするキバゴ」、「観賞用のキバゴ」と3匹捕まえている<ref>{{Cite web |author=津田美波 |url=http://ameblo.jp/00dpd/entry-10662957146.html |title=津田は混乱してしまった！ |work=津田さんは明日もがんばるよ。 |publisher=アメーバブログ |date=2010-09-30 |accessdate=2013-10-10 |archiveurl=http://archive.is/Kfk9 |archivedate=2012-07-26}}</ref>。

犬よりも猫の方が好きであるが、本人は猫アレルギーである。過去に犬を飼っていたこともある<ref name="HMV"/>。

『ゆるゆり』の船見結衣を演じるに際し、髪型を結衣に合わせて短くした<ref>{{Cite web|url=http://ameblo.jp/00dpd/entry-10887068031.html|title=お知らせ|work=津田さんは明日もがんばるよ。|date=2011-05-10|accessdate=2013-04-14}}</ref>。
