'''黒沢 ともよ'''（くろさわ ともよ、[[1996年]]<ref>{{Cite web|url=http://ameblo.jp/kurosawa-tomoyo/entry-11820006375.html|title=はじまっちゃいますっ|work=黒沢ともよオフィシャルブログ「もよちーのよちよち歩き」|date=2014-04-11|accessdate=2014-05-02}}</ref>[[4月10日]] - ）は、[[日本]]の[[俳優|女優]]、[[声優]]、元[[子役]]、[[歌手]]。[[マウスプロモーション]]所属。[[埼玉県]][[秩父市]]出身<ref>{{Cite web |url=https://mausu.net/talent/kurosawa-tomoyo.html |title=マウスプロモーション公式サイト｜所属タレント |publisher=[[マウスプロモーション]] |accessdate=2013-01-19}}</ref><ref name="seiburail" />。

== 人物・経歴 ==
{{年譜のみの経歴|date=2017年3月}}
3歳頃から演技の勉強を始め<ref>『[[宇宙ショーへようこそ]]』ブルーレイディスク音声特典[[オーディオコメンタリー]]より</ref>、[[2000年]]頃から子役として[[テレビドラマ]]、[[テレビコマーシャル|CM]]、[[舞台]]へ出演。2000年、NHK[[大河ドラマ]]『[[葵 徳川三代]]』の[[市姫]]で[[テレビドラマ]]初出演。[[2003年]]、舞台『[[奇跡の人]]』出演時に、役者人生を歩むことを決意<ref>『[[A&G NEXT BREAKS FIVE STARS|A&G NEXT BREAKS 黒沢ともよのFIVE STARS]]』第15回放送より</ref>。同年、舞台『新・美空ひばり物語』にて幼少期の[[美空ひばり]]を演じる<ref name=misora>[http://www.kisoji.com/kisobunka/event-record/H15event/H15_e_misor.html 新・美空ひばり物語]</ref>。

[[2005年]]、『[[モーツァルト!]]』のアマデ役で[[ミュージカル]]初出演<ref>[http://www.umegei.com/mozart/intro.html 「モーツァルト！」イントロダクション]</ref>。また[[中川晃教]]ら演者の歌唱に刺激を受け、本作品の歌唱指導に師事した事が歌を始めるきっかけとなった<ref>ラジオCD『ブラック・ブレット 〜延珠＆ティナの天誅ラジオ〜』より</ref>。自身の人生において大きな部分を占め、この作品なしでは今の自分は存在しないと語っている<ref>[http://ameblo.jp/kurosawa-tomoyo/entry-11828210935.html もよちーのよちよち歩き:でもこの『モーツァルト！』なしでは今の私はないと断言できるほど、私の人生においてとても大きな部分を占めている作品です。]</ref>。[[2006年]]、『[[ポンキッキ]]』内で放送された[[CGアニメーション]]作品『[[ポンポン ポロロ]]』の主題歌を担当<ref>『[[マリー・アントワネット (ミュージカル)|マリー・アントワネット]]』パンフレットより</ref>。同年[[前進座]]の公演『謎帯一寸徳兵衛』の団七娘お市役で[[歌舞伎]]初出演。

[[2008年]]より[[Sound Horizon]]のPVやコンサートにモーションアクターとして出演。[[2010年]]の7th Story CD『[[Märchen]]』ではボーカル・ボイスアクターなども務める。この際の縁で[[TM NETWORK]]の[[宇都宮隆]]を敬愛していると語る<ref>自身がパーソナリティーを務める『[[A&G NEXT BREAKS FIVE STARS]]』にて、宇都宮隆が番組初ゲストとして出演した際の発言。{{Cite web |url=https://twitter.com/ag_five_stars/status/602767688922898433 |title=Twitter / ag_five_stars: 黒沢ともよのFIVE STARS #8」 初ゲストに宇都宮隆さんをお迎えします。|accessdate=2015-05-25}} {{Cite web |url=https://twitter.com/TomoyoKurosawa/status/602750957915492352 |title=Twitter / TomoyoKurosawa: 大好きなウツさんを迎えてのとっても楽しいけど慌ただしかった30分！！！|accessdate=2015-05-25}}</ref>。[[2009年]]、ミュージカル『[[グレイ・ガーデンズ (ミュージカル)|グレイ・ガーデンズ]]』に[[ジャクリーン・ケネディ・オナシス|ジャクリーン・ブーヴィエ]]役で出演。

[[2010年]]、劇場アニメ『[[宇宙ショーへようこそ]]』の主人公小山夏紀役にて声優デビュー<ref>監督の[[舛成孝二]]は、「演技がある程度のレベルに達していなければ、主人公達に大人の配役も考えていたが（中略）ともよちゃん達は僕の想像を超えた魅力を引き出してくれた」と語る。『[[宇宙ショーへようこそ]]』付属パンフレットより。</ref>。[[スペースクラフト|東京児童劇団]]を退団後、[[2012年]]より[[マウスプロモーション]]に所属。声優業を活動の中心とする。[[2015年]]、大学に進学。同年4月より放送の『[[響け! ユーフォニアム]]』の黄前久美子役でテレビアニメ初主演。[[2016年]]、舞台『アイワズライト』に出演、[[ストレートプレイ]]は7年ぶりの出演となる。

[[2018年]][[3月3日]]、第12回[[声優アワード]]主演女優賞受賞。

[[涼風真世]]や[[井上芳雄]]、[[山口祐一郎]]など共演歴のある俳優とは舞台観劇を通じて今でも交流がある<ref>{{Cite web |url=https://twitter.com/TomoyoKurosawa/status/637179206674153474 |title=Twitter / TomoyoKurosawa: 会いたい人にたくさん会えて嬉しかった〜。祐様も禅さんも今さんも涼風ママンも…！|accessdate=2015-08-28}} {{Cite web |url=https://twitter.com/TomoyoKurosawa/status/634350818431385600 |title=Twitter / TomoyoKurosawa: 芳雄にいさまの初トートは、とてもエモーショナルでした。|accessdate=2015-08-20}} {{Cite web |url=https://twitter.com/TomoyoKurosawa/status/637205145009328129 |title=Twitter / TomoyoKurosawa: エリザや貴婦人の楽屋でデュエットCDのお話しをしたら『ちびルドとアマデが…大きくなって…泣』とたくさん驚愕されました。|accessdate=2015-08-28}} [http://ameblo.jp/kurosawa-tomoyo/entry-11828210935.html もよちーのよちよち歩き:久しぶりに、あっきーさん、由美子さんに会えてほんっとーーーに嬉しかったです！]</ref>。[[大竹しのぶ]]を尊敬し、7歳のころに大竹から「いい役者になりたかったら、いい人間になりなさい」と言葉をかけられた<ref name="voice2017nen2gatsu">『B.L.T VOICE GIRLS』VOL.29、109頁。</ref>。趣味は料理で、週に1回は煮込み料理を作る<ref name="voice2017nen2gatsu"/>。高校時代は軽音部に所属し、[[ギター]]を担当していた。
