'''林 沙織'''（はやし さおり、[[1985年]][[11月10日]]<ref>{{Cite web |date=2012-11-12 |url=http://www.joqr.co.jp/comike/2012/11/1112.html |title=11月12日放送分　収録後記 - めちゃこみ☆ |publisher=[[文化放送]] |accessdate=2014-02-08}}</ref> - ）は、[[日本]]の元[[女性]][[声優]]、元[[舞台]][[俳優|女優]]。[[神奈川県]]出身。[[アトミックモンキー]]声優・演技研究所第3期生<ref name="prof">{{Cite web |url=http://www.atomicmonkey.jp/jp/amprofile/hayashi.html |title=林沙織 公式プロフィール |accessdate=2014-06-24 |date=2013-11-10 |work= |publisher=アトミックモンキー |archiveurl=http://web.archive.org/web/20131110113055/http://www.atomicmonkey.jp/jp/amprofile/hayashi.html |archivedate=2013-11-10 }}</ref>を経て、アトミックモンキー所属で活動した後、[[アクセルワン]]に所属していた。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:声優#冒頭部分]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 来歴 ==
高校卒業時に[[看護師国家試験|准看護師免許]]を取得し、進学先の[[文教大学]][[臨床心理学科]]<ref name="blogprof">{{Cite web |author=林沙織 |date=2014-06-07 |url=http://profile.ameba.jp/nikushokumiffy/ |title=林沙織 プロフィール |accessdate=2015-05-25 |work=♪ひよっこ日和♪ |archiveurl=http://archive.is/bQEzg |archivedate=2014-06-23}}</ref>にて[[心理学]]を学んだ<ref>{{Cite web |author=林沙織 |date=2011-05-22 |url=http://ameblo.jp/nikushokumiffy/entry-10900083578.html |title=大好きな後輩達へ☆★ |work=ひよっこ日和 |accessdate=2016-09-15 |archiveurl=http://archive.is/mzXe |archivedate=2012-07-12}}</ref>後、演劇を志して、アトミックモンキー声優・演技研究所へ入学し、同養成所の第3期生となった<ref name="prof"></ref>。養成所在籍中の[[2009年]]より、『[[智一・美樹のラジオビッグバン]]』に第9期BGPメンバーとして出演し、[[2010年]]より声優として本格的に活動を始めた。

ラジオ番組ではリスナーから「癒される」「声や動きが癒し系」などの評価を得、『[[超!A&Gミュージック+TV]]』[[2011年]][[2月23日]]にゲスト出演した[[関智一]]により『うんぴー』と命名され、それ以降ラジオでは『うんぴー（うんP）』と呼ばれていた<ref>{{Cite web |author=林沙織 |date=2011-02-23 |url=http://ameblo.jp/nikushokumiffy/entry-10810983619.html |title=ラジオのお弁当♪♪ |work=ひよっこ日和 |accessdate=2016-09-15 |archiveurl=http://archive.is/0AFe |archivedate=2012-07-07}}</ref><ref>{{Cite web |author=林沙織 |date=2011-08-15 |url=http://ameblo.jp/nikushokumiffy/entry-10986964256.html |title=まなみんぐー♪♪♪ |work=ひよっこ日和 |accessdate=2016-09-15 |archiveurl=http://archive.is/6DeG |archivedate=2012-07-07}}</ref>。

2013年[[12月31日]]付けで、所属していたアトミックモンキーを離れ、フリー転向し<ref>{{Cite web |author=林沙織 |date= 2014-01-05 |url=http://ameblo.jp/nikushokumiffy/entry-11743361788.html |title=☆★大切なご報告★☆ |work=♪ひよっこ日和♪ |accessdate=2014-01-06 |archiveurl=http://web.archive.org/web/20140106181536/http://ameblo.jp/nikushokumiffy/entry-11743361788.html |archivedate=2014-01-06}}</ref>、[[2014年]][[4月14日]]からアクセルワンに移籍<ref>{{Cite web |author=林沙織 |date= 2014-04-13 |url=http://ameblo.jp/nikushokumiffy/entry-11821865886.html |title=★☆アクセルワン☆★ |work=♪ひよっこ日和♪ |accessdate=2016-09-15 |archiveurl=http://web.archive.org/web/20140416093354/http://ameblo.jp/nikushokumiffy/entry-11821865886.html |archivedate=2014-04-16}}</ref>。

=== 人物 ===
特技は幼少期より続けているという[[ピアノ]]<ref name="prof" />。

趣味は大学在学時に在籍したサークルで始めた[[自転車]]旅行<ref name="prof" />で、[[ロードバイク]]に乗って走りまわっている。

飼っている[[ペット]]を溺愛しており、ブログやかつての[[Twitter]]アカウントに写真をよく載せていた。[[2013年]][[11月]]には保健所から子犬を引き取り里親になるなど、動物愛護に関心が強い。

=== 自殺未遂騒動とそれに伴う引退 ===
アクセルワンに移籍して間もない[[2014年]][[6月3日]]、自身のブログ記事において[[自殺]]を示唆する発言をしたが<ref>{{Cite web |author=林沙織 |date=2014-06-03 |url=http://ameblo.jp/nikushokumiffy/entry-11869186935.html |title=たくさんのありがとう |accessdate=2014-06-24 |work=♪ひよっこ日和♪ |publisher= |archiveurl=http://megalodon.jp/2014-0603-1110-11/ameblo.jp/nikushokumiffy/entry-11869186935.html |archivedate=2014-06-03 }}<!-- 記述が消されたりしているため、時間も表記しておきます。 --></ref><ref>{{Cite news |title=声優・林沙織が自殺ほのめかすブログ「天国から、皆の幸せを願って、祈っています |newspaper=[[東スポWeb]] |date=2014-06-03 |url=http://www.tokyo-sports.co.jp/entame/anime/273393/ |accessdate=2014-06-24 |publisher=[[東京スポーツ新聞社]] }}</ref>、当日中に無事が確認され<ref>{{Cite news |title=自殺ほのめかした声優・林沙織の所属事務所「無事が確認できた」 |newspaper=[[東スポWeb]] |date=2014-06-03 |url=http://www.tokyo-sports.co.jp/entame/entertainment/273444/ |accessdate=2014-06-24 |publisher=[[東京スポーツ新聞社]] }}</ref>、該当の記事は削除された。騒動を受けて、アクセルワンより謝罪文が発表されたが、その後の動向に関しては、「林本人がまだ説明できる状態にない」として明確な言及を避けた<ref>{{Cite web |url=http://www.axl-one.com/news/956.html |title=ファンの皆様ならびに関係者の皆様へ |accessdate=2015-05-25 |date=2015-06-03 |publisher=[[アクセルワン]] }}</ref>。

その後、林当人から、[[6月23日]]付けで先述の騒動に関する謝罪のコメントが公式ブログに投稿され、自身が[[うつ病|鬱病]]を患っていることや実際に自殺を図ったことなどを明かした上で、自殺未遂によるケガや後遺症などを理由に声優を引退し、アクセルワンから退社したことも公表した<ref>{{Cite web |author=林沙織 |date=2014-06-23 |url=http://ameblo.jp/nikushokumiffy/entry-11882839153.html |title=最後に |work=♪ひよっこ日和♪ |accessdate=2014-06-23 |archiveurl=http://megalodon.jp/2014-0623-1404-01/ameblo.jp/nikushokumiffy/entry-11882839153.html |archivedate=2014-06-23 }}<!-- 記述が消されたりしているため、時間も表記しておきます。 --></ref>。しかし、この時点ではまだアクセルワンの公式サイトにプロフィールが記載されており、おたくま経済新聞による騒動に関する取材があった時点では、電話の相手の回答から少なくとも彼女の発言内容の真偽について確証が得られず<ref>{{Cite news |title=声優・林沙織、ブログで自殺未遂告白と引退発表 事務所は沈黙のまま |newspaper=おたくま経済新聞 |date=2014-06-24 |url=http://otakei.otakuma.net/archives/2014062401.html |accessdate=2014-06-28 |publisher=シー・エス・ティー・エンターテインメント }}</ref>、正式な扱いも明言されなかった<ref name="menscyzo2014624">{{Cite news |title=「首を吊ったせいでこのような顔に」自殺未遂騒動の声優・林沙織、ブログで引退発表も即削除にファンの動揺広がる |newspaper=[[メンズサイゾー]] |date=2014-06-24 |url=http://www.menscyzo.com/2014/06/post_7935.html |accessdate=2014-06-27 |publisher=[[サイゾー]] }}</ref>。

その後も、自身のブログに自殺願望を仄めかす文章を何度も掲載しては削除していた<ref name="menscyzo2014624" />が、2014年6月26日付で公式ブログが閉鎖された。ブログの閉鎖によって林の現状や今後が不明となり、過去に担当していた一部の持ち役のキャスト交代や、出演予定だったメディア作品の公式ページよりキャスト変更に関するアナウンスが相次いで発表され、ほぼ全ての出演作を事実上降板<ref name="monica" /><ref name="natsumi" /><ref name="megaCD2" />。その後も事務所・スタッフ側からの説明がないまま、同年12月26日、公式サイトより林のプロフィールが削除された<ref group="注">{{Wayback |url=http://www.axl-one.com/talent/hayashi.html |title=林 沙織｜タレント｜アクセルワン |date=20150204065034}}</ref>。
