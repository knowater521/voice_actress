'''安野 希世乃'''（やすの きよの、[[1989年]]<ref name="GP1107"/><ref name="cd_jounal">{{Cite web|url=http://artist.cdjournal.com/a/yasuno-kiyono/909124|title=安野希世乃|work=CDJournal|publisher=音楽出版社|accessdate=2017-04-17}}</ref>[[7月9日]]<ref name=="srimix"/> - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]。[[エイベックス・ピクチャーズ]]（事務所）、[[フライングドッグ]]（音楽レーベル）に所属。[[宮城県]]出身。旧芸名は、'''目澤 希世乃'''（めざわ きよの）<ref name="cd_jounal"/>。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。プロジェクト:声優#冒頭部分にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
[[京都府]]で生まれて間もなく[[北海道]][[函館市]]、[[宮城県]]へと転居を重ね、小学6年生から[[東京都]]で暮らす<ref name="GP1107"/>。

声優を目指したきっかけは、9歳の頃に『[[怪盗セイントテール]]』の特別番組を見たこと<ref name="GP1107"/>。アフレコ密着映像を見てアニメの裏側で活躍している声優の存在を知り、この職業に就きたいと感じた<ref>{{Cite web|url=http://ddnavi.com/news/198552/a/|title=「安野希世乃」声優インタビュー&撮り下ろしグラビア【声優図鑑】|accessdate=2015-10-25|publisher=ダ・ヴィンチニュース}}</ref>。

高校生（15歳）の頃、[[エイベックス・プランニング&デベロップメント|エイベックス・アーティストアカデミー]]声優タレントコースに特待生として入学<ref name="GP1107"/><ref>{{Cite web |date= |url=http://aaa.avex.jp/course/voiceactor/ |title=声優スクール |publisher=エイベックス・アーティストアカデミー |accessdate=2017-04-20}}</ref>。21歳の時、アニメ映画『[[キズナ一撃]]』の轟キズナ役で主役デビューを果たす<ref name="GP1107"/>。以後、『[[冴えない彼女の育てかた]]』の加藤恵、『[[うしおととら]]』の井上真由子、『[[あんハピ♪]]』の久米川牡丹などの主要キャラクターを演じる。

歌手としては[[アイドルマスター シンデレラガールズ]]（木村夏樹役）、[[ワルキューレ (マクロスΔ)|ワルキューレ]]（カナメ・バッカニア役）など複数の[[声優ユニット]]に所属し、2017年7月にミニアルバム『[[涙。]]』で[[フライングドッグ]]からソロデビューした<ref>{{Cite web|date=2017-04-02|publisher=安野希世乃オフィシャルブログ「きよのがたり」|url=http://ameblo.jp/kiyono-yasuno/entry-12261770614.html|title=春一番の。|accessdate=2017-04-20}}</ref>。

== 人物 ==
趣味は自転車とバスに乗ること<ref>{{Cite web|url=http://avex.jp/kiyono-yasuno/profile/|title=声優 安野希世乃オフィシャルサイト|accessdate=2015-10-25|publisher=エイベックス・ピクチャーズ}}</ref>、漫画<ref name="blog_profile">{{Cite web |url=http://profile.ameba.jp/kiyono-yasuno/ |title=プロフィール |work= |publisher=安野希世乃オフィシャルブログ「きよのがたり」 |accessdate=2017-07-28}}</ref>、[[日本酒]]<ref>{{Cite web |url=http://www.jvcmusic.co.jp/flyingdog/-/Profile/A025792.html |title=安野希世乃 プロフィール |publisher=フライングドッグ |accessdate=2017-07-28}}</ref>。愛用の[[電動アシスト自転車]]で仕事場に通ったり、オフには[[ポタリング]]を楽しんだりする<ref>「おもいでポタリング 安野希世乃」『声優グランプリ 2017年8月号』、主婦の友インフォス情報社、2017年。</ref>。日本酒は[[新政酒造]]の「[http://www.aramasa.jp/collection/no.6.html No.6]」という銘柄が気に入っている<ref name="nizisuta"/>。ほかに好きな飲み物は[[カフェラテ]]<ref>{{Cite web |date=2015-09-04 |url=http://www.animatetimes.com/news/details.php?id=1441329904 |title=安野希世乃さんが実践しているボイトレ方法とは!?　『中川翔子☆アニメが好ぎだーーーっ!!』第38回放送レポート |publisher=アニメイトタイムズ |accessdate=2017-07-28}}</ref>。

特技は似顔絵と歌<ref>「声優名鑑2015女性編」『声優グランプリ 2015年3月号』付録、主婦の友インフォス情報社、57ページ。</ref>、書道<ref name="GP1107"/>。サインを書くときには、その時に演じたキャラクターのイラストをサインの左下に添えることが多い。

自分の性格は「のんびり、ゴーイングマイウェイ<ref name="GP1107"/>」、長所は「変なところでポジティブ<ref name="blog_profile"/>」、短所は「うっかり屋<ref name="blog_profile"/>」。座右の銘は「愛と笑いが世界を救う」<ref name="GP1107"/><ref name="blog_profile"/>。
