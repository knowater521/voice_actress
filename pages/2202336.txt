'''新田 恵海'''（にった えみ、[[1985年]]<ref name=shinmai20151210>{{cite news |title=長野市出身 新田恵海、初の紅白へ |newspaper=[[信濃毎日新聞]] |page={{要ページ番号|date=2016年8月}} |date=2015-12-26}}</ref>[[12月10日]] - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]。[[長野県]][[長野市]]出身<ref name ="shimmai"/><ref name="post-40fa.html">{{cite news |title=声に夢乗せて 信州発 若手声優3 |newspaper=[[信濃毎日新聞]] |page={{要ページ番号|date=2016年8月}}  |date=2011-11-19 |accessdate=2013-05-24}}</ref>。[[ABO式血液型|血液型]]は[[ABO式血液型|B型]]。ディファレンス（事務所）、[[ブシロードミュージック]]（レコード会社）に所属。

愛称は'''えみつん'''（レーベル名の「'''emitsun'''」も、自身の愛称<ref name ="emitsun"/>から採られている）。[[μ's]]のメンバーでもある。公式ファンクラブは、「'''EmiRing◎'''」<ref>{{Cite web |url=http://www.emiring.jp/ |title=EmiRing◎｜新田恵海 OFFICIAL FANCLUB |accessdate=2017-09-03}}</ref>。ファンの呼称は「'''恵海人（えみんちゅ）'''」

== 略歴 ==

== 活動 ==
幼少時からずっと歌が大好きだった一方で、歌が周りにあることが「当たり前になりすぎて」おり、歌手を目指すことは当初は「まったく考えていなかった」<ref name="LisAni_143">『リスアニ!』vol.18.1、143頁（新田恵海のインタビューページ）。</ref>。小学校高学年のときにクラブ活動で演劇部に入ったことがきっかけで、「お芝居って楽しい！」と思うようになり、そのころからずっと役者に憧れを持っていた<ref name="LisAni_143"/>。高校生で進路を決める際、「やっぱり自分は音楽が大好きだ」という想いをずっと抱いていたことに気づき、実際にその時点でも既に曲を書いたり自分で歌ったりしている<ref name="LisAni_144">『リスアニ!』vol.18.1、144頁（新田恵海のインタビューページ）。</ref><ref group="注">ただし本人曰く、「趣味の延長ですけど」と述べている。</ref>。音楽大学への進学を目指した理由としては、母が「やっぱりあなたは音楽で表現してるときがいちばん活き活きしているし、もっと勉強したいならすればいいと思う。音大っていう選択肢もあるよ」と後押ししてくれたことが大きかったこと<ref name="LisAni_144"/>、及び、「自分という楽器の鳴らし方を覚えたらそれが楽しくて、もっと声楽を勉強したいなと思って音大への進学を決めましたね」といったエピソードを挙げている<ref name="LisAni_144"/>。

大学生活は音楽漬けの毎日で、学生ミュージカルに熱中していた。大学卒業が迫って進路を考えるにあたり、「やっぱり自分は歌が好きだと」ということの他に、「ハスキーで高めの声」で歌える歌を模索していたときに、[[石田燿子]]の「[[乙女のポリシー]]」が大好きであることを思い出し、[[アニメソング]]の世界を志した<ref name="LisAni_147">『リスアニ!』vol.18.1、147頁（新田恵海のインタビューページ）。</ref>。後に「アニソン オーディション」と検索をかけると、偶然にもSのオーディションが開催されており、締め切りギリギリで駆け込んでオーディションに間に合った<ref name="LisAni_147"/>。

『[[ラブライブ!]]』では高坂穂乃果役を演じているが、作品にも役にも思い入れが強く{{refnest|group="注"|実際、2015年8月3日、高坂穂乃果の誕生日とされる日には、「誕生日おめでとう」という趣旨のツイートを2回している<ref>{{Cite web |author=新田恵海(@Nitta_Emi) |date=2015-08-03 |url=https://twitter.com/Nitta_Emi/status/627857270685085701 |title=穂乃果ー！ お誕生日おめでとー！！ |publisher=Twitter |accessdate=2015-09-30}}</ref><ref>{{Cite web |author=新田恵海(@Nitta_Emi) |date=2015-08-03 |url=https://twitter.com/Nitta_Emi/status/628213838597353472 |title=Happy Birthday HONOKA♡ |publisher=Twitter |accessdate=2015-09-30}}</ref>。}}、インタビュー記事でも、「高坂穂乃果 CV.新田恵海」とファンからは認識されていると思う<ref name="LisAni_148">『リスアニ!』vol.18.1、148頁（新田恵海のインタビューページ）。</ref>、あるいは、「えみつんは穂乃果そのまんまだよね」とファンから結構言われる、と発言<ref name="LisAni_148"/>。また、「[[μ's]]」としてのデビュー作品である「[[僕らのLIVE 君とのLIFE]]」は、「素直になんの癖もなくまっすぐ新田恵海として歌っている歌」と述べている<ref name="LisAni_148"/>。また、高坂穂乃果については、「前を向いているような、それでいて並んでくれているような、ほんとに太陽のような存在だなあってすごく思います」<ref>『cut』2015年8月号、15頁（新田恵海のμ'sとしてのインタビューページ）。</ref>、「あなたに出会って、私の人生は大きく変わりました。あなたの輝きの一部分でいられることが、私の誇りで、私のいのちです」<ref>『cut』2015年8月号、33頁（新田恵海のμ'sとしての手紙形式での特集ページ。新田恵海が、高坂穂乃果に充てて手紙を綴る形式になっている）。</ref>と、思いを綴っている。

歌手活動においては、[[Elements Garden]]及び[[畑亜貴]]との結び付きが強い<ref>『リスアニ!』vol.18.1、148頁、150頁（新田恵海のインタビューページ）。</ref>。Elements Gardenは、事務所に入って以来、声優活動と並行して提供楽曲の仮歌やコーラスを担当してきており<ref>『リスアニ!』vol.18.1、150頁（新田恵海のインタビューページ）。</ref>、畑亜貴とは、μ'sとして（実質的に）すべての歌詞を提供を受けてきた間柄である<ref group="注">詳細は[[ラブライブ!のディスコグラフィ]]をも併せて参照のこと。</ref>。また、ソロ歌手としてのメジャーデビュー以後も、シングル曲である「笑顔と笑顔で始まるよ!」<ref group="注">『リスアニ!』vol.18.1、148頁でのインタビューにて、「もちろんいまの私の歌でもあるんですけど、例えば10年前の私が、『もっと音楽やりたいな、もっと楽しい音楽に出会いたいな、たくさん歌を歌いたいな』って思った気持ちを、今現在の私が歌ったという感じじゃないかなとも思っていて」と述べている。</ref>「探求Dreaming」でElements Gardenから楽曲提供を受けており<ref group="注">「笑顔と笑顔で始まるよ!」「探求Dreaming」は、いずれも[[上松範康]]が作曲している。</ref>、畑亜貴からは、歌詞提供を受けている。

== 人物 ==
