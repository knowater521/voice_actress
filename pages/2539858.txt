'''高井 舞香'''（たかい まいか、[[1993年]]<ref>{{Cite web |url=https://twitter.com/takaichan18/status/765905243691298817 |title=2016年8月17日付、高井のツイートより|accessdate=2016-09-12}}</ref>[[8月18日]] - ）は、[[日本]]の[[女性]][[声優]]。[[徳島県]]出身<ref>{{Cite web |url=http://gigazine.net/news/20111009_newtype_anime_award_machiasobi7/ |title=驚異的な強さで「まどかマギカ」が12部門制覇、ニュータイプ・アニメ・アワード 2011発表・授賞式 |accessdate=2012-06-11}}</ref>。[[マウスプロモーション]]付属養成所27期生<ref>{{Cite web |url=http://ameblo.jp/mausu-y/entry-11260170971.html |title=「心の引き出し」27B高井舞香｜マウスプロモーション付属養成所ブログ |accessdate=2012-06-11}}</ref>経て、マウスプロモーションに所属。

== 経歴 ==
[[2010年]]、[[徳島市]]で開催された「[[マチアソビ|眉山山頂秋フェスタ×マチ★アソビvol.4]]」にてアニメーション制作会社[[ユーフォーテーブル]]制作[[みのりスクランブル!]]のアニメ声優の公開オーディションが開催され、[[2011年]]1月の「[[マチアソビ|マチ★アソビ vol.5]]」にて同アニメの登場人物である「晃」役に抜擢されたことが発表された<ref>{{Cite web |url=http://www.animate.tv/news/details.php?id=1296712130 |title=【マチ★アソビ】雪中で感激の結果発表！　徳島の女子高生・高井舞香さんと、阿久津加菜さん、五十嵐裕美さんが出演決定した『みのりスクランブル』オーディション発表をレポート！ |accessdate=2012-3-19}}</ref><ref>{{Cite web |url=http://gigazine.net/news/20110131_minori_audition_machiasobi5/ |title=「みのりスクランブル！」の声優オーディション結果発表は驚きの現役女子高生起用 |accessdate=2012-3-19}}</ref>。

[[2012年]][[3月18日]]、徳島市に開設された映画館「[[ufotable CINEMA]]」オープニングイベントの司会を務めた<ref>{{Cite web |url=http://gigazine.net/news/20120318-ufotable-cinema-opening/ |title=数百人がオープン前に詰めかけた徳島県知事も期待の「ufotable CINEMA」オープンレポート |accessdate=2012-3-19}}</ref>。

== 人物 ==
* 趣味・特技は、[[アコースティックギター]]、[[サッカー]]、食べること、[[Twitter]]、[[ガンダム]]や[[ロボットアニメ]]<ref>{{Cite web |url=https://mausu.net/talent/takai-maika.html|title=高井舞香 公式プロフィール |accessdate=2016-11-03}}</ref>。
* 大学では、フランス文学を専攻していた<ref>{{Cite web|url=https://nizista.com/views/article2?id=bce6e360780a11e793ff276fb22737ff|title=高井舞香、作曲・作詞を手掛けたオリジナルアルバム『Magistrale』インタビュー in Paris|publisher=NIZISTA|accessdate=2017-12-02}}</ref>。
* 事務所の同期には[[桑原由気]]、[[高野麻里佳]]、[[水間友美]]がいる<ref>{{Cite web |url=http://pigoo.jp/pigoohd/matsuiga/|title=まついがプロデュース #3|accessdate=2016-11-03}}</ref>。
