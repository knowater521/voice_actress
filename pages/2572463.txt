'''加隈 亜衣'''（かくま あい、[[1988年]]<ref>{{Cite web |url=https://twitter.com/aimausu/status/376735810038407168 |title=2013年9月9日付、加隈のツイートより|accessdate=2013-09-9}}</ref>[[9月9日]]<ref name="profile">{{Cite web |url=https://mausu.net/talent/kakuma-ai.html|title=マウスプロモーション公式サイト：所属タレント|accessdate=2013-04-16}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[マウスプロモーション]]所属<ref name="profile" />。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]<ref name="profile" />。[[福岡県]]出身<ref name="mausu blog">{{Cite web |url=http://ameblo.jp/mausu-y/entry-10924568648.html |title=初☆26期Aクラス加隈亜衣｜マウスプロモーション付属養成所ブログ |accessdate=2012-05-13}}</ref><ref name="famitsu">{{Cite web |url=http://www.famitsu.com/news/201201/30009039.html |title=7次元さきの世界に住む少女とコミュニケーションせよ！ ガスト新作『シェルノサージュ 〜失われた星へ捧ぐ詩〜』発表 - ファミ通.com|accessdate=2012-05-15}}</ref>。身長164cm<ref name="famitsu" />。

== 経歴 ==
家族に迷惑をかけたくない理由で、高校卒業後には4年間[[OL]]として勤務していたが、声優になる想いを捨てきれず上京し、マウスプロモーション付属養成所の第26期生<ref name="mausu blog" />を経て、2013年4月よりマウスプロモーションの所属となる<ref name="profile" /><ref>{{Cite web |url=https://twitter.com/aimausu/status/319047227161063424|title=2013年4月2日付、加隈のツイートより|accessdate=2013-04-02}}</ref>。

2011年秋からラジオ番組やゲームソフトに出演している。『[[パパのいうことを聞きなさい!]]』のおねえさん役でアニメデビュー<ref>{{Cite web |url=http://ddnavi.com/news/204077/|title=『加隈亜衣さんに50の質問「声優の輪」【第25回】|accessdate=2014-12-04}}</ref>。

2013年10月から12月まで、[[水瀬いのり]]と期間限定ユニット「いのりっくま」を結成し、徳島、福岡、東京でイベントを開催した。

== 人物 ==
福岡県の出身であり、[[博多弁]]を話す。かつては[[大阪弁]]も話せていた<ref name="famitsu" />。

声優になろうと思ったきっかけは小学校6年生の頃に観たアニメ『[[犬夜叉]]』を挙げていて、憧れの先輩もヒロイン・日暮かごめを演じた[[ゆきのさつき|雪野五月]]である。

特技は[[水泳]]や[[卓球]]。他には[[パーソナルコンピュータ|パソコン]]のタイピングも得意とする<ref name="famitsu" />。

[[運転免許|自動車免許]]、[[日本漢字能力検定|漢字検定]]、[[全商珠算・電卓実務検定|電卓検定]]、[[全国商業高等学校協会|簿記検定]]、[[全商ワープロ実務検定|ワープロ検定]]の資格を持ち、タレントプロフィール『趣味・特技』欄にある「カッパの捕獲」は2012年11月に岩手県遠野市で『カッパ捕獲許可証』を取得（購入）したことによる<ref name="profile" /><ref>{{Cite journal|和書|journal=[[電撃PlayStation]]|issue=Vol.517|page=107|publisher=[[アスキー・メディアワークス]]|date=2012-11-8}}</ref><ref>{{Cite journal|和書|journal=電撃PlayStation|issue=Vol.531|page=91|publisher=アスキー・メディアワークス|date=2012-11-29}}</ref>。

2015年5月4日には、[[マチ★アソビ]]にて行われた事務所の先輩である[[市来光弘]]と、友人である[[井ノ上奈々]]の両名の公開結婚式の司会を担当した<ref>{{Cite web |url=http://gigazine.net/news/20150504-wedding-machiasobi14/|title=声優・市来光弘さん＆井ノ上奈々さんの公開結婚式レポート|accessdate=2015-05-15}}</ref><ref>{{Cite news
