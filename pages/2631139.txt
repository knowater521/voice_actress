'''小岩井 ことり'''（こいわい ことり、[[2月15日]]<ref name="pgprofile">{{Cite web |url=http://pg-wcf.co.jp/profile/04/ |title=小岩井 ことり |publisher=ピアレスガーベラ |accessdate=2016-12-15}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]。[[京都府]][[京都市]]出身<ref name="seigura" />。[[ピアレスガーベラ]]所属<ref name="pgprofile" />。[[ABO式血液型|血液型]]は[[ABO式血液型|A型]]<ref>[[血液型くん!|血液型くん!2]] 第8話「血液型ちゃんと大好きな先生」 クレジットタイトルより。</ref>。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。プロジェクト:声優#冒頭部分にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 来歴 ==
高校生の時にナレーションの仕事をしたことがきっかけで声優の仕事を志す。知人の紹介でメイクモデルのバイトをした事があり、その繋がり・紹介から声優になるきっかけを掴むために関西でテレビやCMのナレーションなど様々な仕事を経て、声優になった<ref name="seigura">{{Cite journal|和書|url=http://seigura.com/senior/20131119_170.html|title=声優未来予想図|journal=[[声優グランプリ]]|issue=2012年8月号|publisher=[[主婦の友社]]|date=2012-07-10|pages=96-97}}</ref>。

[[2011年]]末、テレビアニメ『[[青の祓魔師 (アニメ)|青の祓魔師]]』吉国役で[[テレビアニメ]]デビュー<ref name="seigura" />。翌[[2012年]]、テレビアニメ『[[じょしらく]]』にて自身初のメインキャラクターを務める<ref name="kigurumi">{{Cite journal |和書 |journal=[[別冊少年マガジン]] |issue=2012年6月号 |publisher=[[講談社]] |page={{要ページ番号|date=2016年12月}} |date=2012-05}}</ref>。[[2013年]]、テレビアニメ『[[断裁分離のクライムエッジ]]』にて初のヒロインを務める<ref name="iwai">{{Cite web |url=http://crimeedge.com/#/staff |title=スタッフ ／ キャスト |work=「断裁分離のクライムエッジ」アニメ公式サイト|accessdate=2013-01-22}}</ref>。

また作詞家・作曲家としても活動しており、声優の[[田中理恵 (声優)|田中理恵]]に楽曲提供をしていることを公表している<ref name="seigura" />。[[金田朋子]]がMCを務める『金朋声優ラボ』に出演した際は番組をイメージした楽曲「天才!金朋ラボ」を作詞・作曲して披露した。自身がパーソナリティを務めるラジオ番組『[[断裁分離のクライムエッジ#Webラジオ|切と祝の断裁分離のクライムレディオ]]』でも楽曲を披露している。

[[MIDI検定]]2級を取得しており<ref>{{Cite web |author=小岩井ことり |date=2016-04-21 |url=http://ameblo.jp/koiwai-kotori/entry-12152616549.html |title=MIDI検定2級合格！ |work=小岩井ことりオフィシャルブログ「ゆめはキミの太陽。」 |accessdate=2016-04-21}}</ref><ref>{{Cite web |author=[[藤本健]] |date=2016-08-30 |url=http://www.dtmstation.com/archives/51982658.html |title=難関MIDI検定2級、99点のトップ合格者は声優の小岩井ことりさんだった！ |work=藤本健の“DTMステーション” |accessdate=2016-11-19}}</ref>、2016年11月にはMIDI検定1級も取得したことを公表している<ref>{{Cite web |author=小岩井ことり |date=2016-11-29 |url=http://ameblo.jp/koiwai-kotori/entry-12224114876.html |title=MIDI検定1級！ |work=小岩井ことりオフィシャルブログ「ゆめはキミの太陽。」 |accessdate=2016-11-29}}</ref>。2018年1月にはダイビングのライセンスを取得したことをブログで報告している<ref>{{Cite web |author=小岩井ことり |date=2018-01-08 |url=https://ameblo.jp/koiwai-kotori/entry-12342877969.html|title=あけましておめでとうございます！|work=小岩井ことりオフィシャルブログ「ゆめはキミの太陽。」 |accessdate=2018-01-08}}</ref>。

== 人物 ==
愛称は、こっこちゃん<ref>{{Cite web |url=http://ameblo.jp/koiwai-kotori/entry-12121185527.html |title=LTD04リリースLIVE！  |accessdate=2017-1-7 |date=2016-1-24 |work=小岩井ことりオフィシャルブログ「ゆめはキミの太陽。」 }}</ref>など。膝まである長い黒髪をしている<ref>{{Cite web |author=小岩井ことり |date=2013-01-13 |url=http://ameblo.jp/koiwai-kotori/entry-11448228794.html |title=声優アニメイト |work=小岩井ことりオフィシャルブログ「ゆめはキミの太陽。」 |accessdate=2013-04-30}}</ref>。

アニメ『[[ポケットモンスター (アニメ)|ポケットモンスター]]』の[[ピカチュウ (サトシのポケモン)|ピカチュウ]]が鳴き声だけで気持ちを伝えていることに感動し声優に憧れる<ref name="seigura" /><ref>{{Cite web |url=http://www.animatetimes.com/news/details.php?id=1362040118 |title=『2.5次元てれび』に『じょしらく』メンバー最後のゲスト・小岩井ことりさんが登場 |accessdate=2017-1-7 |date=2013-2-28 |work=animate Times |publisher=[[アニメイト]]}}</ref>。「声優という仕事が大好きで100歳になっても続けることが夢」と語っている<ref name="seigura" />。

生まれ育った家庭環境ゆえに京都弁、大阪弁、広島弁の3つの方言を話すことができる<ref name="seigura" />。
