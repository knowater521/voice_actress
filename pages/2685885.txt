'''洲崎 綾'''（すざき あや、[[1986年]][[12月25日]]<ref name="tower"/> - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]。
[[アイムエンタープライズ]]所属。愛称は主に'''あやっぺ'''。[[石川県]][[金沢市]]出身<ref name="mro" />。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]<ref name="profile" />。[[東京学芸大学]]卒業。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 略歴 ==
[[東京学芸大学]]<ref>[http://recochoku.jp/artist/2000221410/album/ 洲崎綾 アルバム一覧] [[レコチョク]] 2014年12月24日</ref>在学中に[[日本ナレーション演技研究所]]に入所<ref>{{Cite web |url=http://nichinare.com/about/native.html |title=主な出身者 |publisher=日本ナレーション演技研究所 |accessdate=2015-02-15}}</ref>。大学卒業後は[[OL]]を経て<ref name="tower"/>、2010年に『[[バクマン。 (アニメ)|バクマン。]]』で声優デビュー。作品内でヒロインの亜豆美保とユニットを組む声優・天音ミサヨ役を担当。亜豆美保役の[[早見沙織]]、松田桃子役の[[春宮茉由]]とともに、聖ビジュアル女学院合唱部として歌唱も担当した<ref name="davinci">[http://ddnavi.com/news/187105/ 洲崎綾さんに50の質問「声優の輪」【第16回】] - ダ・ヴィンチニュース</ref>。

2011年に[[アイムエンタープライズ]]への所属が決まる<ref>{{Cite web |url=http://www.imenterprise.jp/info.php?cate_no=9 |title=information |publisher=アイムエンタープライズ |accessdate=2015-02-15}}</ref>。

2013年、『[[たまこまーけっと]]』の北白川たまこ役でテレビアニメ初主演。また、映画の主題歌「[[たまこまーけっとのディスコグラフィ#プリンシプル|プリンシプル]]」の歌唱を担当し、個人名義でのシングルを発売<ref>[http://tamakolovestory.com/staff-cast/ たまこラブストーリー Staff&Cast]</ref><ref>{{Cite web|url=http://artist.cdjournal.com/a/suzaki-aya/915787|title=洲崎綾|work=CDJournal|accessdate=2017-12-04}}</ref>。

2015年には第9回[[声優アワード]]新人女優賞を受賞する<ref>{{Cite web|publisher=[[声優アワード]]|url=http://seiyuawards.jp/201503/06170044.php|title=第九回声優アワード受賞者一覧|accessdate=2015-03-06}}</ref>。同年11月公開のアニメ映画、『[[RWBY|RWBY Volume1]]』（ノーラ・ヴァルキリー役）にて、吹き替えに初挑戦<ref>『[[RWBY|RWBY Volume2]]』劇場版パンフレットより。打越純一（演出）のインタビューにて言及されている。</ref><ref group="注">アメリカにて制作されたアニメを、日本語吹き替え版として限定イベント上映した作品。なお、演出担当の打越曰く、「非常にスピーディーにしゃべれる方で、原語版のセリフ量やセリフ回しを忠実に対応できていてすごいと思った」という趣旨の発言をしている。</ref>。

2017年1月1日、自身の公式Twitterを開設した<ref>[https://twitter.com/suzaki_aya/status/815211082432135170]2017年1月28日閲覧。</ref><ref name="ayappenewyear2017">{{Cite web|publisher=Clown's Pocket！|url=http://ameblo.jp/clown-happy/entry-12233792352.html|title=あけましておめでとうございます！|accessdate=2017-01-28}}</ref><ref group="注">自身の公式Twitter開設後の最初のツイートは、2017年1月1日の0時ちょうどにアップされているが、プロフィール欄には「2016年12月に登録」と記されている。</ref>。

== 人物 ==
=== 人物的特徴 ===
声優を目指したきっかけは、中学時代に『[[名探偵コナン (アニメ)|名探偵コナン]]』が好きで、[[灰原哀]]役の[[林原めぐみ]]に憧れて[[声優]]という仕事に興味を持ったことによる<ref>{{Cite web |url=http://recochoku.jp/artist/2000221410/album/ |title=洲崎綾 アルバム一覧 |work=レコチョク |accessdate=2015-02-15}}</ref><ref name="davinci" /><ref group="注">しかし、洲崎は小学生時代には既に声優になりたいと思っており、小学生のころに[[こども電話相談室|子供電話相談室]]にて林原と直接会話しており、声優になるためのアドバイスなどを受けている。</ref>。

大学卒業後もなかなか所属事務所が決まらず、一般企業に就職し働きながら養成所に通い続けた。教員免許を取得したものの、「教師やりながら声優の勉強は無理だなって、教育実習のとき思った」ため、「（レッスンに）土日をフルに使えるOLになった」と話している<ref name="sidonia7">[http://www.onsen.ag/program/sidonia/ ラジオ　シドニアの騎士〜綾と綾音の秘密の光合成〜] - 第7回配信</ref>。
本人曰く、声優デビュー後も数年間はアルバイトをしながらの生活だったと述べている<ref>[http://www.kill-la-kill.jp/blu-ray_dvd/ キルラキル Blu-ray&DVD 第5巻]</ref>。

出演した作品の中でも、特に『[[たまこまーけっと]]』（北白川たまこ役）及び『[[キルラキル]]』（満艦飾マコ役）は、特に思い入れの強い作品とのことであり、2016年12月の自身のバースデーイベントは、この2作品のスタッフの協力で「幸せすぎる一日」になったとのことである<ref name="ayappenewyear2017" />。

資格は[[実用英語技能検定|英検3級]]、[[珠算|珠算初段]]、[[普通自動車免許]]、[[華道]]（古流柏葉会師範）、[[PADI|PADIオープンウォーターダイバー]]、[[幼稚園]]・[[小学校]]・[[特別支援学校]][[教育職員免許状|教諭免許]]を取得している<ref name="imenterprise">{{Cite web |url=http://www.imenterprise.jp/data.php?id=94 |title=洲崎綾 Talent Profile |publisher=アイムエンタープライズ |accessdate=2015-03-08}}</ref>。

特技は[[トランペット]]と[[華道]]<ref name="imenterprise" />。好きなスポーツは、[[ソフトテニス]]、[[スクーバダイビング|スキューバダイビング]]を挙げている<ref name="clown-happy">{{Cite web |url=http://profile.ameba.jp/clown-happy/ |title=洲崎綾のプロフィール |work=Clown's Pocket！ |accessdate=2015-02-15}}</ref>。小・中学校時代には隣家に住む一つ年上の友人の影響でマーチングバンド、ソフトテニス部に所属していた<ref>{{Cite web |url=http://www.koepota.jp/news/2014/12/08/0702.html |title=洲崎綾のインタビュー到着！ドラマCD「りりくる - LIly LYric cyCLE -」Vol.4が2月25日に発売！ |accessdate=2015-03-06}}</ref>。

好きな食べ物は[[すき焼き]]<ref>「声優グランプリ2015年2月号」別冊の「声優読本2015」より。</ref>、[[うどん]]、[[沖縄そば]]、[[タコライス]]。嫌いな食べ物は[[納豆]]、[[ヤマノイモ|山芋]]（食べるとのどが痒くなる）、[[イクラ]]、[[キウイフルーツ|キウイ]]<ref name="clown-happy" />、[[蜂蜜]]<ref name="sidonia">[http://www.onsen.ag/program/sidonia/ ラジオ　シドニアの騎士〜綾と綾音の秘密の光合成〜] - 第28回　2015年1月9日 配信</ref>。嫌いな飲み物は[[コーヒー]]（飲むと具合が悪くなる）<ref name="sidonia"/>。

好きな芸能人は[[堺雅人]]。好きなアーティストは[[夏川りみ]]と[[T.M.Revolution]]の[[西川貴教]]<ref name="davinci" /><ref name="clown-happy" />である。

尊敬する声優に林原めぐみを挙げている<ref name="davinci" />。また、「私のテーマ曲」という質問に林原の「ふわり」を挙げている<ref>「林原さんの歌のなかで一番最初に好きになった曲です。『がんばれ！』と背中を押す感じの曲が多いなか、いい意味で力が抜けていて気持ちが軽くなる優しさと暖かさがあって、中高生時代に悩んだときによく聴いていました。今でも大好きな一曲です。」声優パラダイス2015年2月号より。</ref>。

「ごえもん」という名前の猫(ブルーの[[ブリティッシュショートヘア]])を飼っていて、時々成長の様子がTwitterやブログ等で報告されている<ref name="ayappenewyear2017" /><ref group="注">2017年1月現在、自身の公式Twitterのプロフィール欄には、位置情報として「ごえもんのとなり」と記されている。</ref>。また、弟がいるとのことである<ref>{{Cite web |date=2012-01-10 |url=http://ameblo.jp/clown-happy/entry-11452614435.html |title=Where there is a will, there is a way. |work=Clown's Pocket！ |publisher=サイバーエージェント |accessdate=2015-02-22}}</ref>。

=== 交友関係 ===
声優の[[佳村はるか]]とは養成所で知り合い<ref>[http://ddnavi.com/news/187850/ 佳村はるかさんに50の質問「声優の輪」【第17回】] </ref>、所属事務所も同じでプライベートでも長く親交がある。

その他に仲の良い声優として、[[金子有希]]や[[長妻樹里]]ら『[[たまこまーけっと]]』で共演したメンバー（3人共『たまこまーけっと』がアニメ初レギュラー作である）、[[津田美波]]や[[種田梨沙]]ら『[[ロウきゅーぶ!|ロウきゅーぶ!SS]]』で共演したメンバーを挙げている<ref name="davinci" />。

またWEBラジオ番組『[[洲崎西]]』で共にパーソナリティをしている[[西明日香]]とも仲が良く、TVアニメ『[[てさぐれ!部活もの]] あんこーる』のBD/DVD第1巻ではコラボ企画として共演を果たした<ref>{{Cite web |url=http://www.vap.co.jp/tesabu/ |title=DVD・Blu-ray BOX |publisher=vap |accessdate=2015-02-15}}</ref>。

『[[シドニアの騎士]]』や『[[トリニティセブン 7人の魔書使い|トリニティセブン]]』で共演した同事務所の[[佐倉綾音]]とも親交を深めている<ref>[http://ddnavi.com/news/223500/ 「洲崎綾」新人声優のインタビュー＆ミニグラビア【新人声優図鑑】]</ref>。

===こども夢質問箱===
声優になる以前、NHKラジオ第1放送『[[きょうも元気でわくわくラジオ]]』内の『こども夢質問箱』コーナーを林原めぐみが担当した回（2003年9月27日放送）にかけた電話が繋がり、憧れていた林原めぐみと直接話をする機会を得る<ref name="suzakinishi">[http://seaside-c.jp/program/suzakinishi/ 洲崎西] - 第2回配信 2013/07/09</ref>。

声優デビューから3年後、[[西明日香]]と共にパーソナリティを務めるWEBラジオ『[[洲崎西]]』内で林原のラジオ番組やこども夢質問箱に関するエピソードを話した<ref name="suzakinishi" />ところ、番組を聞いていた『[[林原めぐみのHeartful Station]]』のリスナーが林原宛に投稿し、第1138回放送内で取り上げられた。洲崎の相談に回答した事は覚えていなかったものの、「今でも考えは変わっていない」と述べている。

洲崎自身も第1138回の放送を聞いており、「大興奮しちゃって。嬉しくて嬉しくて」「普段は12時ぐらいに寝るのに、3時まで寝付けなかった」と喜びを顕にしていた<ref>[http://seaside-c.jp/program/suzakinishi/ 洲崎西] - 第5回配信</ref>。その後、洲崎自身があらためて声優になった報告とお礼を『[[林原めぐみのHeartful Station|Heartful Station]]』宛に投稿、第1140回内で読み上げられた。林原からは「送っちゃおうか、おもちゃ箱。たまにはリスナーに戻りたまえ」と番組グッズが贈られた。

===林原めぐみのラジオ番組にゲスト出演===
2014年4月、洲崎綾がヒロイン・星白閑役を演じるアニメ『[[シドニアの騎士]]』の番組宣伝も兼ねて、『[[林原めぐみのHeartful Station|Heartful Station]]第1176回』及び『[[林原めぐみのTokyo Boogie Night|Tokyo Boogie Night]]第1146回』にゲスト出演する運びとなった。

林原のラジオ番組の製作・提供をしている[[キングレコード]]が、『[[シドニアの騎士]]』の製作陣にも加わっている縁もあり、林原との対面が実現した。

ゲスト出演後に『[[洲崎西]]』内で「この番組をやってなかったら、林原さんに知ってもらうことも無かった。勇気を持ってお礼の手紙を書くことも出来なかった」「本当に幸せだったなあ。人生で一番会いたかった人にやっと会えた」と感想を述べている。共にパーソナリティを務める[[西明日香]]は、ゲスト出演時の終始緊張している洲崎の様子を「完全に借りてきた猫状態だった」と評している<ref>[http://seaside-c.jp/program/suzakinishi/ 洲崎西] - 第44回配信 2014/04/29</ref>。
