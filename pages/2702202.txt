'''玉木 佑和'''（たまき ゆな、[[1990年]]<ref>{{Cite web |url=http://ameblo.jp/hatsukoikikaku/entry-11526937731.html |title=3rdキャストインタビューVol.11｜初恋企画 制作日誌 |accessdate=2013-10-20}}</ref>[[4月28日]] - ）は、[[日本]]の[[声優]]、[[俳優|舞台女優]]、[[脚本家]]。旧芸名は'''田中 里和'''（たなか さとわ）。[[81プロデュース]]所属。[[茨城県]]出身。[[ABO式血液型|血液型]]は[[ABO式血液型|B型]]<ref name="Blog profile">{{Cite web |url=http://profile.ameba.jp/b-rank-diamond3100/ |title=田中里和のプロフィール |accessdate=2013-01-09}}</ref>。

声域は[[ソプラノ]]<ref name="Blog profile" />。[[花やしきアクターズスタジオ]]出身<ref>{{Cite web |url=http://www.hanayashiki.net/as/voice/workshop/ |title=花やしきアクターズスタジオ |accessdate=2012-12-08}}</ref>。趣味・特技は[[カラオケ]]、[[剣道]]、[[相対音感|相対性音感]]、[[タロット占い]]<ref>[http://www.81produce.co.jp/list.cgi?lady+3141112135910 81プロデュースによるプロフィール]より。</ref>。

== 来歴 ==
[[2010年]]、花やしきアクターズスタジオからオーディションを受け、[[81プロデュース]]が運営する81アクターズスタジオに入所。

[[2011年]]、81プロデュース所属。

[[2012年]]に小林悠、[[中島ヨシキ]]、平山友紀子と共に演劇集団「初恋企画」を結成<ref>{{Cite web |url=http://ameblo.jp/hatsukoikikaku/entry-11181168055.html |title=始動です！｜初恋企画 制作日誌 |accessdate=2013-01-09}}</ref>。演者として出演するだけではなくリーダー（主宰<ref>{{Cite web |url=http://ameblo.jp/hatsukoikikaku/entry-11188300218.html |title=はつこい〜｜初恋企画 制作日誌 |accessdate=2013-01-09}}</ref>）兼脚本も担当している<ref>{{Cite web |url=http://ameblo.jp/b-rank-diamond3100/entry-11210853364.html |title=こんなんやります｜Ｂらんく☆ダイアモンド |accessdate=2013-01-09}}</ref>。

[[2016年]][[2月4日]]、田中里和から改名<ref>{{Cite web |url=http://www.81produce.co.jp/ |title=81produce |accessdate=2016-02-05 |publisher=[[81プロデュース]]}}</ref><ref>{{Twitter status|satowa_t3100|695170367908806657|本人による2016年2月4日のツイート}}</ref>。
