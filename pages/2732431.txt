'''諏訪 彩花'''（すわ あやか、[[1988年]]<ref name="nana-monky">{{Cite web |author=諏訪彩花 |url=http://ameblo.jp/nana-monky/ |title=プロフィール |work=諏訪彩花の日常未来☆彡 |accessdate=2016-11-19}}</ref>[[5月27日]]<ref name="artsvision">{{Cite web |url=http://www.artsvision.co.jp/talent/832/ |title=諏訪 彩花 |publisher=アーツビジョン |accessdate=2016-11-19}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[アーツビジョン]]所属<ref name="artsvision" />。[[愛知県]][[名古屋市]]生まれ、[[日進市]]出身<ref name="prof">{{Cite web |url=http://www.tv-tokyo.co.jp/anime/animemashite/backnumber/index_38.html |title=ここだけのプロフィール 諏訪彩花 |work=アニメマシテ |publisher=テレビ東京 |accessdate=2016-11-19 |archiveurl=https://web.archive.org/web/20161119045039/http://www.tv-tokyo.co.jp/anime/animemashite/backnumber/index_38.html |archivedate=2016-11-19}}</ref>。[[ABO式血液型|血液型]]は[[ABO式血液型|A型]]<ref name="nana-monky" />。

== 来歴 ==
幼少期からアニメを見るのが好きで、小学生の頃にやっていた[[ミュージカル]]で演技の楽しさを実感。声優業を初めて知ったのは、中学生時代に『[[テニスの王子様 (アニメ)|テニスの王子様]]』の[[越前リョーマ]]に恋をしたのがきっかけ<ref name="shinjin">{{Cite journal |和書 |author=田中克佳 |title=新人声優 NOTEBOOK |journal=[[声優グランプリ]] |issue=2014年4月号付属小冊子 |publisher=[[主婦の友社]] |date=2014-03-10 |page=5}}</ref>。
声優という仕事の幅広さに魅力を感じ、高校、大学で演劇に携わって、周りの友人の後押しもあり、声優を目指すことを決めた<ref name="shinjin" />。

出身地である[[名古屋]]には声優の養成所や専門学校があまりなかったため、名古屋校があった[[日本ナレーション演技研究所]]に入所。大学に通いながら週1回のレッスンを受けた<ref name="shinjin" />。日本ナレーション演技研究所を経て現事務所に所属。

2011年『[[グラール騎士団|グラール騎士団 Evoked THE Beginning white]]』の下級生、女子生徒役で声優デビュー<ref name="question">{{Cite web |url=http://ddnavi.com/anime-bu/205916/a/ |title=諏訪彩花さんに50の質問「声優の輪」【第26回】 |work=ダ・ヴィンチニュース |publisher=KADOKAWA |date=2014-08-31 |accessdate=2016-11-19}}</ref>。[[2014年]]4月より放送されたテレビアニメ『[[悪魔のリドル]]』で自身初となる主役の東兎角を演じた<ref name="question"/>。

== 人物 ==
中学時代は放送部部長。高校時代は演劇部に所属<ref>{{Cite web |author=諏訪彩花 |date=2013-02-24 |url=http://ameblo.jp/nana-monky/entry-11477520730.html |title=学ラン派です☆ |work=諏訪彩花の日常未来☆彡 |publisher= |accessdate=2016-11-19}}</ref>。

2014年4月期に放送された[[テレビアニメ]]『[[僕らはみんな河合荘]]』の常田美晴役では[[美濃弁|岐阜弁]]で演じた。演じるにあたって[[岐阜県|岐阜]]に住む友人から岐阜弁のレクチャーを受けた<ref>{{Cite web |url=http://www.tbs.co.jp/anime/kawaisou/story/story11.html |title=第11話あらすじ |work=僕らはみんな河合荘 公式ホームページ |publisher=TBSテレビ |accessdate=2016-11-19}}。</ref>。文化庁Webサイト内にて公開されている「方言アフレコ体験教室」では、{{要出典範囲|[[青森県]]八戸市、[[岩手県]]釜石市、[[宮城県]]仙台市の東北3市の[[東北弁]]を披露|date=2016年11月}}。講師の音源を念入りに聴いて指導も受けたとのこと<ref>{{Cite web |author=諏訪彩花(@Suwa_Ayaka) |date=2014-06-25 |url=https://twitter.com/Suwa_Ayaka/status/481695881054281729 |title=愛知県出身です！講師の方の音源をめちゃめちゃ聞きまくって指導も頂きました！！ |publisher=Twitter |accessdate=2016-11-19}}</ref>。

目標にしている声優は先輩全員で、同期や後輩も含め尊敬できる人はたくさんいるが、なかでも憧れている声優は[[井上麻里奈]]、[[坂本真綾]]、[[沢城みゆき]]だという<ref name="question"/>。

==== 悪魔のリドルに関して ====
[[2014年]]4月より放送された、テレビアニメ『[[悪魔のリドル]]』で自身初となる主役の東兎角を演じた<ref name="question"/>。

[[オーディション]]は受けたいキャラを2人選ぶ形式で行われ、最初は番場真昼と桐ヶ谷柩を受けたが結果は不合格。その結果に落ち込んでいた矢先、東兎角の再オーディションの話が舞い込んだ。そこでは東兎角が一ノ瀬晴を守るシーンがあり、息を切らせる芝居が必要だったところを自分で台詞を作らないといけないと勘違いし、「こいつを傷つけるな！」など[[アドリブ]]で叫んだところ関係者をざわつかせたが、結果は合格であった<ref>{{Cite book |和書 |chapter=諏訪彩花×金元寿子対談 |title=悪魔のリドル 〜黒組公式ガイドブック〜 |publisher=[[角川書店]] |date=2014-07-23 |pages=62-63 |isbn=978-4-04-102097-5}}</ref>。

[[音泉]]（および[[ニコニコ生放送|ニコ生配信]]）で配信されたWEBラジオ『「悪魔のリドル」ラジオ〜黒組通信〜』ではメイン[[ラジオパーソナリティ|パーソナリティー]]を担当。同年齢の事務所の先輩で共演した[[沼倉愛美]]が歌った『concentration』を流す際に何気なく「イントロを自ら口ずさみながら曲の紹介」をした所<ref name="party">{{Cite web |url=http://www.koepota.jp/news/2014/09/19/0204.html |title=諏訪彩花、金元寿子ら10年黒組＆内田真礼が熱唱！「悪魔のリドル」トーク＆ライブイベント『黒組PARTY！』 |work=こえぽた |publisher=MFS |date=2014-09-19 |accessdate=2014-10-07}}</ref>、以降ラジオで曲を流す前に諏訪がその自己流イントロを披露する流れになり<ref name="suwaru">{{Cite news |url=http://news.mynavi.jp/articles/2014/06/15/riddle/ |title=TVアニメ『悪魔のリドル』、諏訪彩花と沼倉愛美が登場! EDテーマミニアルバム発売記念イベント「10年黒組臨時授業」1・2限目 |newspaper=マイナビニュース |publisher=マイナビ |date=2014-06-15 |accessdate=2014-10-07}}</ref>、更にライブイベント『黒組Party!』では諏訪がメインキャスト全員の曲のイントロを事前にレコーディングの上で流すクイズ『黒組諏訪トロ PARTY!!』なるコーナーまで行われるに至った<ref name="party"/>。これらを収録した『[[#諏訪トロ Party!|諏訪トロ Party!]]』が2014年10月15日より[[レコチョク]]および[[iTunes Store]]にて配信が開始された<ref>{{Cite web |url=http://www.lisani.jp/news/id104376 |title=『悪魔のリドル』、TV未放送第13話の上映＆トークイベント開催決定！そして「諏訪トロ」音源、本日より悲願の配信開始！ |work=リスアニ！WEB |publisher=エムオン・エンタテインメント |date=2014-10-15 |accessdate=2016-11-19 |archiveurl=https://web.archive.org/web/20141020161401/http://www.lisani.jp/news/id104376 |archivedate=2014-10-20}}</ref>。また、番組内では「[[すべり芸|すべる]]」を「諏訪る」と言い換えられている<ref name="question"/><ref name="suwaru"/>。
