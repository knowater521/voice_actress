'''若井 友希'''（わかい ゆうき、[[1995年]][[10月30日]]<ref name="irisprofile">{{Cite web |url=http://iris.dive2ent.com/profile/profile_member.php?id=1000222 |title=若井友希 |work=iRis OFFICIAL WEB SITE |publisher=エイベックス・ミュージック・クリエイティヴ |accessdate=2016-11-25}}</ref><ref name="Facebookprofile">{{Cite web |url=https://www.facebook.com/media/set/?set=a.434486423261088.98689.429637317079332&type=3 |title=若井友希 |publisher=Facebook |accessdate=2013-06-12}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]、[[アイドル]]。アイドルグループ・[[i☆Ris]]のメンバー。[[81プロデュース]]所属<ref>{{Cite web |url=http://www.81produce.co.jp/list.cgi?lady+9111027303120 |title=所属俳優プロフィール |publisher=81プロデュース |accessdate=2013-06-12}}</ref>。

[[岐阜県]]出身<ref name="irisprofile"/><ref name="Facebookprofile"/>。身長145cm<ref name="irisprofile"/><ref name="Facebookprofile"/>、血液型はA型<ref name="irisprofile"/><ref name="Facebookprofile"/>。

== 人物 ==
大手レコード会社[[エイベックス・グループ|エイベックス]]と、声優事務所[[81プロデュース]]がタッグを組んで開催した「アニソン・ヴォーカルオーディション」の合格者6名にて結成されたユニット、[[i☆Ris]]のメンバーとして活動している。

特技は[[ピアノ]][[弾き語り]]<ref name="irisprofile"/>で、好きなアニメソングは[[きただにひろし]]の「[[ウィーアー!]]」<ref name="irisprofile"/>。

ソロ名義の[[シングル]]「Destiny Sky」でi☆Risと同時にソロデビューを果たしている。

2016年、第10回[[声優アワード]]にて、i☆Risの一員として歌唱賞を受賞<ref>{{Cite web |url=http://www.animatetimes.com/news/details.php?id=1457773181 |title=「第十回 声優アワード」受賞者発表!助演男優賞&パーソナリティ賞に鈴村健一さん、最多得票賞に5年連続で神谷浩史さん |work=アニメイトタイムズ |publisher=animateLAB |date=2016-03-12 |accessdate=2016-11-25}}</ref>。

[[SKE48]]の[[高柳明音]]は[[アニメージュ]]での対談をきっかけに仲が良い<ref>{{Cite web |author=若井友希 |date=2015-08-21 |url=http://ameblo.jp/iris-official-blog/entry-12064207869.html |title=ゆうき*ちゅりちゃんとおでーと！ |work=i☆Ris OFFICIAL BLOG |publisher=Cyber Agent |accessdate=2016-06-04}}</ref>。
