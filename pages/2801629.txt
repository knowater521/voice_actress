== 経歴 ==
『[[美少女戦士セーラームーン (テレビアニメ)|美少女戦士セーラームーン]]』と『[[新世紀エヴァンゲリオン]]』を観て、全く異なるキャラクターを演じた[[三石琴乃]]（[[月野うさぎ]]と[[葛城ミサト]]）に憧れ、声優になることを目指すようになった<ref name="日ナレ紹介_後"> {{Cite web |date= 2009-06-03 |url= http://www.animate.tv/school/interview_details.php?s_id=A01&id=1251108986 |title= 日本ナレーション演技研究所 日笠陽子さん 後編 |work= [[アニメイトTV]] |publisher= [[日本ナレーション演技研究所]] |accessdate= 2009-08-14 }}</ref><ref name="声優プリンセス">{{Cite journal|和書 |author= |year= 2009 |title= PICK-UP FRESH VOICE ACTRESS |journal= 声優プリンセス |pages=79-81 |publisher= [[近代映画社]] | id=ISBN 978-4-7648-7153-3}}</ref>。高校卒業後、[[被服学|被服]]（[[洋裁]]）の[[専修学校|専門学校]]と両立し、[[日本ナレーション演技研究所]]に通った<ref group="注">入所を決めた理由は、学校と両立できる[[カリキュラム]]があり、また憧れである[[三石琴乃]]も卒業したことによるという。</ref><ref name="日ナレ紹介_後" />。本科生のときに「無料新人育成オーディション」に合格し、[[特待生]]に選ばれる。

[[2007年]]、『[[スケッチブック (漫画)|スケッチブック 〜full color's〜]]』の根岸みなも役でアニメデビュー。[[2009年]]に『[[けいおん!]]』の秋山澪役で注目を浴び、次第に主要な役柄を演じることが増えた。そして、同年7月放送の『[[世紀末オカルト学院]]』の神代マヤ役でアニメ初主演。アニメのほか、吹き替えやナレーションの仕事もこなしている。

[[2010年]][[3月6日]]に発表された第4回[[声優アワード]]にて、[[テレビアニメ]]『けいおん!』の劇中にて結成したユニット「放課後ティータイム」として、[[豊崎愛生]]、[[佐藤聡美]]、[[寿美菜子]]、[[竹達彩奈]]とともに歌唱賞を受賞。

[[2013年]]、テレビアニメ『[[進撃の巨人 (アニメ)|進撃の巨人]]』でエンディングテーマを歌うと同時に、歌手として活動することを発表<ref>{{Cite web|date=2012-12-08 22:42|url=http://www.cinematoday.jp/page/N0048456|title=けいおん!」“秋山澪”役の声優・日笠陽子がソロプロジェクトを始動!|work=|author=|publisher=シネマトゥデイ|accessdate=2012-12-12 06:41}}</ref>。同年[[2月8日]]に『進撃の巨人』のエンディングテーマである「[[美しき残酷な世界]]」を皮切りとして3か月連続での音楽作品リリースが発表された<ref>{{Cite web|date=2013-02-08 00:00|url=http://www.oricon.co.jp/news/music/2021488/full/|title=“てへぺろ”日笠陽子、5・8ソロ始動！ 怒とうの3ヶ月連続リリース|work=ORICON STYLE|author=|publisher=[[オリコン]]|accessdate=2012-02-08 11:58}}</ref>。同年10月27日には自身初となるワンマンライブを[[STUDIO COAST]]で開催した。

[[2015年]][[12月30日]]、自身のブログで結婚を報告した<ref>{{Cite web|date=2015年12月30日|url=http://blog.livedoor.jp/hiyonikki/archives/2070038.html|title=ごあいさつ|work=日笠陽子のひよっ子記 |author=日笠陽子 |publisher=[[livedoor Blog]] |accessdate=2015-12-30}}</ref>。

=== てへぺろ ===
「'''てへぺろ(・ω<)'''」という持ちギャグをラジオやブログを中心に各所で多用しており<ref group="注">『おどろき戦隊 モモノキファイブ』第7話（2009年5月21日放送）が初披露とされる。</ref>、ラジオやアニメで共演する声優などにも徐々に広まる。[[声優グランプリ|声優グランプリweb]]による企画「声グラwebが勝手に選ぶ流行語2009」（第1回）に選ばれ、情報発信者として表彰された<ref>{{Cite web |date=2010-03-10 |url=http://www.seigura.com/topics/show/1373 |title=日笠陽子さんが受賞！ 「声グラwebが勝手に選ぶ流行語2009」:声優グランプリweb |work=[[声優グランプリ|声優グランプリweb]] |publisher=[[声優グランプリ]] |accessdate=2010-03-11 |archiveurl=https://web.archive.org/web/20100322213536/http://www.seigura.com/topics/show/1373 |archivedate=2010年3月22日 |deadlinkdate=2017年10月 }}</ref>。これを基に[[azusa (シンガーソングライター)|azusa]]がミニオリジナルソングを作った。

さらには、[[渡辺麻友]]などの[[AKB48]]のメンバーや[[はんにゃ]]の[[金田哲]]も使うことがあり、女子中高生のブログでも頻繁にみられるようになった。2011年には「[[女子中高生ケータイ流行語大賞]]2011」で銀賞<ref>{{Cite web |date=2011-12-01 |url=http://www.p-nest.com/research/rst.php?seq=2300 |title=2011年の女子中高生ケータイ流行語大賞が決定しました。 |work=P-NESTリサーチ |publisher=ピーネスト|accessdate=2011-12-01 }}</ref><ref>{{Cite web |date=2011-12-01 |url=http://sankei.jp.msn.com/life/news/111201/trd11120106060001-n2.htm |title=金賞は「ポポポポ〜ン」　ネット流行語大賞2011|work=MSN産経ニュース |publisher=[[産経新聞社]] |accessdate=2011-12-01 }}</ref>、「GRP AWARD 2011」の「ギャルの流行語トップテン」で第2位<ref>{{Cite web |date=2011-12-02 |url=http://mdpr.jp/021160385 |title=2011年度「ギャルの流行語大賞」発表 |work=モデルプレス |publisher=ネットネイティブ |accessdate=2011-12-04 }}</ref>に選ばれた。前者については『映画けいおん!』公開と重なったため、12月3日放送の『[[情報7days ニュースキャスター]]』にて、映画の舞台挨拶で披露した模様と合わせて取り上げられたほか、5日放送の『[[めざましテレビ]]』、9日放送の『[[はなまるマーケット]]』の「[[本上まなみ]]の気になる金曜日」でも「気になるキーワード」のひとつとしてこの言葉が取り上げられ、本人もVTR出演した<ref>{{Cite web|url=http://www.tbs.co.jp/hanamaru/weekly/fri_20111209.html|title=TBS「はなまるマーケット」|archiveurl=https://web.archive.org/web/20120213224738/http://www.tbs.co.jp/hanamaru/weekly/fri_20111209.html|archivedate=2012年2月13日|accessdate=2013-05-13|deadlinkdate=2017年10月}}</ref>。

2012年2月から放送の[[ソフトバンクモバイル|ソフトバンク]]のCM「[[白戸家]]」シリーズの「ハワイからの留学生編」で、鳥取タダ（演・[[トリンドル玲奈]]）が使用していた<ref>{{Cite web |date=2012-02-01 |url=http://mantan-web.jp/2012/02/01/20120131dog00m200037000c.html |title=トリンドル玲奈 ： “留学生のタダ”役で白戸家にホームステイ 「てへぺろ」などギャル語駆使 |work=まんたんウェブ |publisher=毎日新聞デジタル |accessdate=2013-10-09 }}</ref>。2012年4月15日の『[[笑点]]』の[[大喜利 (笑点)|大喜利]]にも使用された。2012年6月に放送された『[[THEクイズ神]]』のダイジェストでも出題されている。2012年11月には『[[現代用語の基礎知識]]』2013年版に掲載され、2012年12月には「女子中高生ケータイ流行語大賞2012」で金賞を受賞している<ref>{{Cite web|date=|url=http://www.p-nest.com/research/rst.php?seq=2400|title=2012年の女子中高生ケータイ流行語大賞が決定しました。|work=|author=|publisher=ピーネスト|accessdate=2012-12-12 16:07}}</ref>。

=== 歌手活動 ===
新人時代当初より歌唱力には定評があり、歌手活動に対する意欲も強く、ソロデビューの3年前、『[[けいおん!|けいおん!!]]』に出演していた頃から歌手デビューのオファーが来ており、その後もプロデューサーが間を置いて幾度か話を持ちかけていたものの、当初は「音楽だけで頑張っていたり、音楽をやりたい人はたくさんいて。自分は“役者”として命をかけていて。。。それってどうなのかな？と考えてしまった時間がありました。」との思いもあり、そのつど固辞していた。<ref name="mcplus201305">{{Cite web|date=2013-5-08|url=http://www.allnightnippon.com/mcplus/info.php?yyyy=2013&mm=5&dd=8|title=ミューコミ+プラス|accessdate=2013-05-12}}</ref><ref name="lisani13">{{Cite journal|和書|journal=[[リスアニ!]]|issue=Vol.13|publisher=[[エムオン・エンタテインメント]]|date=2013-04-25|pages=158-165}}</ref>

アニメ『[[ロウきゅーぶ!]]』でのユニット「[[RO-KYU-BU!]]」活動をきっかけに、その心境に変化が生じ転機となる。歌やライブなど、活動する中で非常に楽しい経験をしたことから、「またライブがしたい」という気持ちを持つようになり、その結果、真摯に「やりませんか？」と声をかけ続けてくれたプロデューサーの想いを受け止め、「やらないで後悔するなら、やって後悔した方がいい」と考え、ソロデビューを決心した。<ref name="mcplus201305" /><ref name="lisani13" />

デビュー以前にも数多くのキャラクターソングを歌うも、それまではレコーディングのためスタジオに行って歌うだけであり、CD制作の裏側は知らなかった。自身で関わってみて初めて、様々な人達が一つひとつ作り上げていく大変な工程やこだわりを肌身で感じる。また、個人名義のCDは自身の納得するものができないと出さないと言い切っている。

日笠陽子ソロプロジェクトは、始動日である2013年5月8日の約1年半前から計画されていた。また、ソロデビュー後の初アルバム作品となる『[[Glamorous Songs]]』には以前から報じられていた[[小室哲哉]]<ref group="注">この作品で自身初の声優への楽曲提供となる。</ref>のほか、活動休止中である男女音楽デュオ・[[TWO-MIX]]の音楽担当・[[永野椎菜]]等が楽曲提供を行った。
