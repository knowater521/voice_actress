'''中村 繪里子'''（なかむら えりこ、[[1981年]]<ref>アイマスタジオ第33回（ラジオCD「iM@STUDIO」Vol.5に収録）などより。</ref>[[11月19日]]<ref name="artsvisionprof">{{Cite web |url=http://www.artsvision.co.jp/talent/697/ |title=中村繪里子 |publisher=アーツビジョン |accessdate=2017-01-05}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[舞台女優]]、[[歌手]]。[[神奈川県]]出身<ref name="artsvisionprof" />。[[アーツビジョン]]所属<ref name="artsvisionprof" />。代表作に『[[THE IDOLM@STER]]』（[[天海春香]]）<ref name="oriconprofile">{{Cite web |url=http://www.oricon.co.jp/prof/376758/profile/ |title=中村繪里子（出典：VIPタイムズ社） |work=ORICON STYLE |publisher=オリコン |accessdate=2017-01-05}}</ref>、『[[宇宙戦艦ヤマト2199]]』（桐生美影）<ref name="oriconprofile" />、『[[恋と選挙とチョコレート]]』（住吉千里）<ref name="oriconprofile" />、『[[星空へ架かる橋]]』（中津川初）<ref name="oriconprofile" />などがある。<!--代表作の追加・削減・変更はリンク先に変更があった場合か、信頼できる情報源に基づく出典の明記を伴った上でお願いします。-->{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 来歴 ==
[[日本ナレーション演技研究所]]<ref name="artsvisionprof" />を経て、アーツビジョンに所属。19歳くらいの頃、初めて受けたオーディションに合格する<ref name="日経84">{{Cite book|和書|year=2015|title=[[日経エンタテインメント!]] アニメSpecial 声優バイブル2016|page=84|publisher=[[日経BP]]|isbn=978-4-8222-7251-7}}</ref>。これがのちに「[[アイドルマスターシリーズ]]」となり天海春香役で10年以上演じ続けることになるゲーム作品だが、当時は「[[プロトタイプ]]だから有名じゃない子がいい」と社内[[コンペティション|コンペ]]に出すためのオーディションで、タイトルも決まっていなかった<ref name="日経84"/>。商品化の際、中村は下ろされる可能性もあったが、春香役続投を望まれる<ref name="日経84"/>。中村は「とにかく歌が怖くて」といった状態で、歌だけは別人に変更する案も出たが、音楽チームから「歌声に力があれば大丈夫」と言われ、再度オーディションを受けた<ref name="日経84"/>。

2012年10月に『[[月刊ニュータイプ|Newtype]]×[[マチ★アソビ]] アニメアワード2012』にて女性声優賞1位となる<ref>{{Cite web |date=2012-10-08 |url=http://gigazine.net/news/20121008-moving-picture-award-machiasobi9/ |title=「ニュータイプアニメアワード2012」はアイドルマスターとFate/Zeroの一騎打ちに |work=GIGAZINE |accessdate=2017-01-05}}</ref>。

2014年10月に『Newtype×マチ★アソビ アニメアワード2014』にて女性声優賞1位となる<ref>{{Cite web |date=2014-10-12 |url=http://gigazine.net/news/20141012-anime-award-machiasobi-13/ |title=「ニュータイプアニメアワード2014」結果発表、今年も熾烈な戦いに |work=GIGAZINE |accessdate=2017-01-05}}</ref>。

2014年10月、とくしまアニメ大使に任命される<ref>{{Cite web |date=2014-10-11 |url=http://www.famitsu.com/news/201410/11063428.html |title=“マチ★アソビ vol.13”は3日間のイベントを2日に凝縮−−開会式では声優の中村繪里子さんが“とくしまアニメ大使”に任命【マチ★アソビ vol.13】 |work=ファミ通.com |accessdate=2017-01-05}}</ref>。
