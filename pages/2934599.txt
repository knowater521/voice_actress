'''都丸 ちよ'''（とまる ちよ、[[1991年]]<ref name="swg">{{Cite web |url=https://web.archive.org/web/20140708005536/http://www.s-w-g.co.jp/artists02-01.html |title=Artists 都丸ちよ |publisher=Still Wood Garden|accessdate=2017-06-03}}（2014年7月8日時点の[[インターネットアーカイブ|アーカイブ]]）</ref>[[9月28日]]<ref name="profile">{{Cite web|publisher=EARLY WING|url=http://www.earlywing.co.jp/talent/tomaruchiyo/|title=都丸ちよ | 声優事務所EARLY WING|accessdate=2017-06-03}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[埼玉県]]出身<ref name="swg" /><ref name="profile" />。

[[東京アニメーター学院]]卒業<ref>{{Cite web |url=http://www.tag.co.jp/tokyo/event/event_20160718.html |title=東京アニメーター学院卒業生声優イベント「東アニの目覚め」 |accessdate=2017-06-03 |publisher=東京アニメーター学院}}</ref>。[[EARLY WING]]所属<ref>{{Cite web|publisher=EARLY WING|url=http://earlywing.co.jp/talent_w.html|title=所属タレント 女性|accessdate=2018-03-03}}</ref>。かつては[[スチール・ウッド・ガーデン]]に所属していた。

== 経歴・人物 ==
芝居がしたくて当初は女優に憧れていたが、恥ずかしがり屋だったので表に出ない声優になろうと思い専門学校に入った<ref name ="fresh">{{Cite web|url=http://www.at-x.com/at-x_contents/summerfes2015/fresh04.html|title=フレッシュ夏フェス隊！ | ウチの夏フェス2015!|accessdate=2017-06-03|publisher=[[アニメシアターX]]}}</ref><ref name ="fresh2">{{Cite web|url=https://www.at-x.com/at-x_contents/at-x_summerfes2017/fes/076.html|title=ウチの夏フェス2017特設サイト｜フレッシュ夏フェス隊 ー 都丸ちよ|accessdate=2017-09-07|publisher=[[アニメシアターX]]}}</ref>。

趣味は旅行<ref name="swg" />、ご当地[[カルピス]]収集<ref name="swg" /><ref name="profile" />、麻雀<ref name="profile" /><ref name ="fresh" />、競馬<ref name ="fresh2" />。

特技・資格は書元会毛筆（特待位）<ref name="swg" />。

4匹の猫を飼っている<ref name ="fresh" /><ref>{{Cite web|url=http://www.earlywing.co.jp/blog/%E9%83%BD%E4%B8%B8%E3%81%A1%E3%82%88%E3%80%90%E3%82%B2%E3%83%BC%E3%83%A0%E3%80%91%E3%80%8C%E3%82%A2%E3%82%A4%E3%83%89%E3%83%AB%E3%83%9E%E3%82%B9%E3%82%BF%E3%83%BC%E3%82%B7%E3%83%B3%E3%83%87%E3%83%AC/|title=都丸ちよ：アイドルマスターシンデレラガールズ 新Sレアが登場しました！|date=2016-10-20|accessdate=2017-06-20|publisher=EARLY WING}}</ref>。
