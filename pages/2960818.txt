'''小澤 亜李'''（おざわ あり、[[1992年]]<ref>{{Cite web|date=2014-08-10|url=http://ameblo.jp/ari-step/entry-11907867888.html|title=お祝いありがとうござます！|work=『青空ステップ!』|author=小澤亜李|publisher=[[サイバーエージェント]]|accessdate=2014-08-19}}</ref>[[8月10日]]<ref name="ari-step1" /> - ）は、[[日本]]の[[女性]][[声優]]。[[東京都]]出身。[[アイムエンタープライズ]]所属。

== 来歴 ==
中学1年生の時『[[魔法先生ネギま! (アニメ)|魔法先生ネギま!]]』を観て声優に憧れ<ref name="ddnavi">{{Cite web|publisher=ダ・ヴィンチニュース|url=http://ddnavi.com/news/208242/|title=「小澤亜李」新人声優のインタビュー＆ミニグラビア【新人声優図鑑】 | ダ・ヴィンチニュース:|accessdate=2015-05-13}}</ref>、高校1年次に親が[[保育士]]として働く[[保育園]]で保育助手の[[アルバイト]]を1年間務めて資金を獲得すると<ref name="ddnavi" />、高校2年次に[[日本ナレーション演技研究所]]へ入所し、2013年4月[[アイムエンタープライズ]]所属となる。

2014年に放送された『[[月刊少女野崎くん]]』の佐倉千代役で初のメインキャラクターを演じる<ref name="ddnavi" />。

2017年には、第11回[[声優アワード]]で新人女優賞を受賞する<ref>{{Cite web|publisher=アニメイト|work=アニメイトタイムズ|url=http://www.animatetimes.com/news/details.php?id=1489817579|title=第11回 声優アワード：受賞者 発表!主演男優・女優賞は『君の名は。』主人公＆ヒロインがW受賞、歌唱賞にAqours(アクア)|date=2017-03-18|accessdate=2017-03-18}}</ref>。

== 人物・交友 ==
事務所に所属する以前に正社員の経験を積もうと、[[調理師免許]]の[[資格]]を取得している<ref name="ddnavi" />。特技は[[イラストレーション|イラスト]]<ref name="kurakura"/>。'''そら'''という名前の犬を飼っている<ref>{{Cite web |url=http://ameblo.jp/ari-step/entry-11742146306.html|title=新しい家族 |work=青空ステップ！ |accessdate=2015-06-13}}</ref>。甘い物が苦手であり、煎餅などを好んでいる<ref>{{Cite web |url=http://ameblo.jp/ari-step/entry-11632849353.html|title=お砂糖少なめで |work=青空ステップ！ |accessdate=2016-02-22}}</ref>。三姉妹の次女で姉と妹がいる<ref>{{Cite web |url=http://now.ameba.jp/ari-step/|title=昨日は両親が家に来て掃除手伝ってくれたから、きれいなキッチンで料理したくなったんです！(笑)今日は姉と妹に会うよー( ・∇・)|work=小澤亜李 なう|date=2016-05-01|accessdate=2016-05-01}}</ref><ref name=":0" />。親から、本人の名前もアキにしたら良かったと言われた<ref name=":0" />。なお、名前の漢字の類似（「李」と「季」）から、実際に名前が「あき」だと間違われることも多い。モットーとしていることは、「'''ありのまま、いつもまっすぐわたしらしく'''」である<ref name="ari-step">{{Cite web |url=http://profile.ameba.jp/ari-step |title=小澤亜李のプロフィール |work=青空ステップ！ |accessdate=2015-05-08}}</ref>。

仲が良い声優は、同期である[[長縄まりあ]]<ref>{{Cite web|date=2014-06-03|url=http://ameblo.jp/ari-step/entry-11869710547.html|title=ぬ、ますみん|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2015-05-05}}</ref>、[[田澤茉純]]<ref>{{Cite web|date=2014-06-03|url=http://ameblo.jp/ari-step/entry-11869710547.html|title=ぬ、ますみん|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2015-05-05}}</ref>、[[本渡楓]]、『[[ローリング☆ガールズ]]』などで共演した[[花守ゆみり]]<ref>{{Cite web|date=2015-03-12|url=http://ameblo.jp/ari-step/entry-12000799430.html|title=ゆみりちゃんと映画|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2015-05-05}}</ref>が挙げられる。同じ事務所の[[雨宮夕夏]]<ref>{{Cite web|date=2013-10-22|url=http://ameblo.jp/ari-step/entry-11647499799.html|title=ピッッッッザ☆|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2015-05-26}}</ref>や[[大西沙織]]<ref>{{Cite web|date=2014-01-22|url=http://ameblo.jp/ari-step/entry-11755369168.html|title=さおりんとご飯|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2015-05-26}}</ref>、[[アーツビジョン]]所属の[[芽衣]]<ref>{{Cite web|date=2013-10-31|url=http://ameblo.jp/ari-step/entry-11660867777.html|title=はぴはぴ芽衣ちゃん|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2016-02-02}}</ref>、クロコデイル所属の[[甘束まお]]<ref>{{Cite web|date=2016-03-24|url=http://ameblo.jp/ari-step/entry-12142490972.html|title=まおちゃーん|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2016-03-28}}</ref>、マウスプロモーション所属の[[秦佐和子]]<ref>{{Cite web|date=2014-08-09|url=http://ameblo.jp/ari-step/entry-11907220339.html|title=からあげ食べ放題|work=|author=小澤亜李|publisher=[[アメーバブログ]]|accessdate=2016-02-02}}</ref>とも交友があり、[[井澤美香子]]とは『わかば*ガール』で共演する以前から[[プライベート]]での親交がある<ref>{{Cite web|date=2016-02-02|url=http://ameblo.jp/izawamike/entry-12123913727.html|title=“はじめて”パーティ！！|work=|author=井澤美香子|publisher=[[アメーバブログ]]|accessdate=2016-02-02}}</ref>。
