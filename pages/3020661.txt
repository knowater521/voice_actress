'''郁原 ゆう'''（かはら ゆう、[[2月19日]]<ref name="spellboundprof">[https://archive.is/20140713004855/http://www.spell-bound.jp/%E3%83%97%E3%83%AD%E3%83%80%E3%82%AF%E3%82%B7%E3%83%A7%E3%83%B3division/%E3%82%B9%E3%83%9A%E3%83%AB%E3%83%90%E3%82%A6%E3%83%B3%E3%83%89%E6%89%80%E5%B1%9E%E5%A3%B0%E5%84%AA/ スペルハウンド｜所属声優]（2014年7月13日時点の[[archive.is|アーカイブ]]）</ref> - ）は、[[日本]]の[[女性]][[声優]]<ref name="seigura20140423" />。[[千葉県]]出身<ref>{{Cite web |url=http://www.koepota.jp/news/2014/02/26/0201.html |title=【4週連続企画・声優を目指すあなたへ】第4回 郁原ゆう「私の演技やセリフで元気になってもらいたいんです」  |work=こえぽた |publisher=MFS |date=2014-02-26 |accessdate=2017-04-30}}</ref><ref name="gamer20150225">{{Cite web |url=http://www.gamer.ne.jp/news/201502250060/ |title=「ミリオンライブ！」「しんぐん」に出演！「あやがみっくす」主題歌CDもリリースした新人声優・郁原ゆうさんにスペシャルインタビュー！ |work=Gamer |publisher=イクセル |date=2015-02-25 |accessdate=2017-04-30}}</ref>。身長156cm、血液型はA型<ref name="spellboundprof" />。2016年4月より[[フリーランス]]で活動<ref name="ameblo20160401" />。愛称は「ゆうちゃ」<ref name="gamer20150225" /><ref>{{Cite web |url=http://ch.nicovideo.jp/voicegarage/blomaga/ar1062168 |title=ゆうちゃのゆう遊自的 |work=ニコニコチャンネル |publisher=DWANGO |accessdate=2017-04-30}}</ref>。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。プロジェクト:声優#冒頭部分にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 来歴 ==
{{独自研究|date=2017年4月|section=1}}
高校を卒業後、{{要出典範囲|東京メディアアカデミー（現：[[東京声優アカデミー]]）の声優養成科|date=2017年4月}}で2年間通う。{{要出典範囲|卒業後[[ラムズ]]の準所属となり、付属養成所「RAMS Professional Education」で学ぶ|date=2017年4月}}。

{{要出典範囲|2013年、ラムズ倒産後、[[スペルバウンド]]に所属する|date=2017年4月}}。同年、ソーシャルゲーム「[[アイドルマスター ミリオンライブ!]]」のエミリー スチュアート役で声優デビューする<ref name="seigura20140423">{{Cite web |url=http://seigura.com/senior/message/20140423_6575/1.html |title=声優グランプリWEB |publisher=[[主婦の友社]] |accessdate=2014-07-08}}</ref>。

2014年6月8日、中野サンプラザで開催された「THE IDOLM@STER MILLION LIVE! 1stLIVE HAPPY☆PERFORMANCE!!」に出演。

2016年3月31日、[[スペルバウンド]]を退所してフリーになる<ref name="ameblo20160401">{{Cite web |author=郁原ゆう |date=2016-4-01 |url=http://ameblo.jp/yukahara219/entry-12145474901.html |title=お知らせ |work=ことだまだより |publisher=Ameba |accessdate=2016-4-01}}</ref>。4月28日よりインターネットラジオ「ゆうちゃのゆう遊自的」がスタート<ref name="">{{Cite web |url=http://www.koepota.jp/news/2016/04/26/0501.html |title=郁原ゆうの新番組「ゆうちゃのゆう遊自的」の生放送第1回が4月28日にニコ生にて放送！ |work=こえぽた |publisher=MFS |date=2016-04-26 |accessdate=2017-04-30}}</ref>。
