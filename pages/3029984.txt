'''今村 彩夏'''（いまむら あやか、[[1993年]][[8月5日]] - ）は、[[日本]]の[[女性]][[声優]]。[[大阪府]]出身。[[EARLY WING#WITH LINE|WITH LINE]]所属。

== 経歴 ==
*声優を目指したきっかけは、アニメ・[[漫画]]が好きだった学生時代に「声がかわいいね」と褒められたためである。その後、[[演劇]]部での活動を経て、高校卒業後に声優の[[専門学校]]・[[放送芸術学院専門学校|大阪アニメーションスクール専門学校]]に入った<ref name="ddnavi">[http://ddnavi.com/news/222859/ 「今村彩夏」新人声優のインタビュー＆ミニグラビア【新人声優図鑑】(1/3)] ダ・ヴィンチニュース 2015年1月19日、同2月18日閲覧。</ref>。
*声優を意識したのは、中学時代に友人に勧められ、ゲーム『[[戦国BASARA]]』をプレイした時であり、人の心に届く[[演技]]ができる声優を目指したいと語っている<ref name="animemashite">{{Cite web|url=https://web.archive.org/web/20171015160142/http://www.tv-tokyo.co.jp/anime/animemashite/backnumber/index_26.html|title=アニメマシテ BACK NUMBER (September 2015)|accessdate=2017-10-16|publisher=[[テレビ東京]]}}（2017年10月15日時点の[[インターネットアーカイブ|アーカイブ]]）</ref><ref name="ニジスタ前編">{{Cite web |url = https://nizista.com/views/article2?id=e8b52c40b9f011e787728b477125cd74|title = 上京する前にお母さんと見た流れ星は素敵な思い出ですー今村彩夏インタビュー（前編）（1/2）|publisher = [[ニジスタ]] |accessdate = 2017-10-31 }}</ref>。
*アニメデビューに先がけて、[[2014年]][[7月]]から自身の名前を冠した[[ラジオ番組]]『[[今村彩夏 STAY GOLD]]』が始まった<ref>{{Cite web |url = http://www.agqr.jp/pginfo/2014/07/-stay-gold710.php |title = 【新番組】７月９日スタート！期待の新人声優！『今村彩夏 STAY GOLD』 |publisher = [[文化放送]] |accessdate = 2014-07-17 }}</ref>。
*初めて受けたオーディションは『[[アブソリュート・デュオ]]』<ref>{{Cite web|url=https://nizista.com/views/article2?id=e8b52c40b9f011e787728b477125cd74&page=2|title=上京する前にお母さんと見た流れ星は素敵な思い出ですー今村彩夏インタビュー（前編）（2/2）|publisher = ニジスタ|accessdate=2017-10-31}}</ref>。

== 人物 ==
*[[おたく|オタク]]を自認している。オタクになった切っ掛けは、中学1年生の時に絵が上手くなりたいと、模写のためにたまたま買った[[安西信行]]の『[[MAR (漫画)|MÄR]]』<ref name="ddnavi"/>。
*好きな食べ物は一蘭のラーメン、フルーツ<ref name="ニジスタ後編">{{Cite web |url = https://nizista.com/views/article2?id=1a54c870b9f811e7bce35d8760fa1ca1|title = 常に成長した姿を見てもらい、目指すは格好いい人間ー今村彩夏インタビュー（後編）|publisher = [[ニジスタ]] |accessdate = 2017-10-31 }}</ref>。
*物心ついた頃から[[水泳]]をやっており、高校生の時は卓球部、演劇部に所属していた<ref name="ニジスタ前編" />。他にも[[テニス]]などのラケット種目を得意としている<ref name="ニジスタ前編" />。
*苦手なスポーツは[[サッカー]]、[[バスケットボール]]といった敵味方、入り乱れたスポーツだと語っている<ref name="ニジスタ前編" />。
*好きな言葉・[[座右の銘]]は「夢は逃げない、逃げるのは自分」<ref name="animemashite" />。
*かつて同じ事務所に所属していた[[大下菜摘]]とは専門学校時代からの友人である。また、[[渡辺はるか]]とも仲が良い<ref>{{Cite web|url=https://twitter.com/yakararadio/status/641060330475356160|title=Twitter / @yakararadio: …そして初ゲスト！今村彩夏、大下菜摘の大親友「渡辺はるか」さん登場です！…|date=2015-09-07 18:27|accessdate=2015-10-15}}</ref>。
*衝撃を受けた声優には[[諏訪彩花]]を挙げており、歌っている時と普段との[[ギャップ]]に驚いたという<ref name="animemashite" />。また、目指すべき役者の理想像の声優に[[沢城みゆき]]を挙げている<ref name="ニジスタ後編" />。
*[[読売ジャイアンツ|巨人]]ファンである。巨人ファンは親の影響によるもの。
*2015年5月8日、個人ブログ「GAME CENTER イマムラ」を開設した。

== 趣味 ==
*[[趣味]]は[[メダルゲーム]]、[[読書]]、[[競馬]]、お酒、ネットカフェ巡りなど<ref name="ニジスタ後編" />。メダルゲームに関しては都内各所の[[ゲームセンター]]を巡る日々と言う<ref name="ddnavi_3">[http://ddnavi.com/news/222859/3/ 「今村彩夏」新人声優のインタビュー＆ミニグラビア【新人声優図鑑】(3/3)] ダ・ヴィンチニュース 2015年1月19日、同2月18日閲覧。</ref>。
*[[競馬]]は[[アーケードゲーム]]「[[STARHORSE|スターホース]]」（セガ）をきっかけにして覚え、ラジオ番組[[今村彩夏 STAY GOLD|『STAY GOLD]]』も好きな[[ステイゴールド]]からとられている<ref name=":0">カドカワ「週刊ファミ通」2017年1月19日発行号（No.1468）「音どけ」（232頁）より</ref>。
*実際の競馬を見るのは[[ディープインパクト (競走馬)|ディープインパクト]]が入り口。[[ブログ]]では競馬予想もしているが穴馬券好きなので、ガチガチなレースほど当たらないという<ref name=":0" />。
*2017年、「スターホース」のスマホ版である「[[スターホースポケット]]」において秘書・日吉遥役を担当。「競馬好きを公言していてよかったな、と心から思いました（笑）」とコメントした<ref name=":0" />。
*乗馬ライセンス5級も所持している<ref name="ニジスタ後編" />。
