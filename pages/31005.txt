'''斎賀 みつき'''（さいが みつき、[[1973年]][[6月12日]]<ref name="W01-0245">{{Cite web |url=http://www.vip-times.co.jp/?talent_id=W01-0245 |title=斎賀みつき |work=日本タレント名鑑 |publisher=VIPタイムズ社 |accessdate=2017-01-06}}</ref><ref>{{Cite book|和書|year=2004|title=日本音声製作者名鑑2004|volume=Vol.1|pages=185|publisher=[[小学館]]|id=ISBN 978-4095263014}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]、[[歌手]]である。[[埼玉県]]出身<ref name="W01-0245" />。身長は168cm、[[ABO式血液型|血液型]]は[[ABO式血液型|AB型]]<ref name="W01-0245" />。[[賢プロダクション]]所属<ref>{{Cite web |url=http://www.kenproduction.co.jp/talent/member.php?mem=w22 |title=斎賀みつき |publisher=賢プロダクション |accessdate=2017-01-06}}</ref>。

代表作に『[[天元突破グレンラガン]]』（ロシウ・アダイ）<ref name="oriconprofile">{{Cite web |url=http://www.oricon.co.jp/prof/284096/profile/ |title=斎賀みつき（出典：VIPタイムズ社） |work=ORICON STYLE |publisher=オリコン |accessdate=2017-01-06}}</ref>、『[[07-GHOST]]』（テイト＝クライン）<ref name="oriconprofile" />、『[[海月姫]]』（鯉淵蔵之介）<ref name="oriconprofile" />、『[[イナズマイレブンGO]]』（神童拓人）などがある<ref name="oriconprofile" />。

以前[[伊藤健太郎 (声優)|伊藤健太郎]]主宰の[[劇団K-Show]]のメンバーであった（[[2008年]][[9月23日]]に自身のブログにて退団を公表<ref>[https://web.archive.org/web/20160315162947/http://blog.animate.tv/saiga/index.php?blogid=47&archive=2008-09 一歩、踏み出す！]（2016年3月25日時点の[[インターネットアーカイブ|アーカイブ]]） - 「地球の子だもの」アニメイトTVブログ、2008年9月23日。</ref>）。

[[2008年]]に、第2回[[声優アワード]]サブキャラクター女優賞を受賞<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_02 |title=第二回声優アワード受賞者 |publisher=声優アワード |accessdate=2017-01-06}}</ref>。[[2010年]]に、第4回声優アワード海外ファン賞を受賞<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_04 |title=第四回声優アワード受賞者 |publisher=声優アワード |accessdate=2017-01-06}}</ref>。

== 人物 ==
=== 声優として ===
小学生の頃、親戚の家で[[アニメ雑誌]]の付録の台本を見て「大きくなったら私はこの仕事をするんだ」と漠然と思い、声優を志すようになる<ref name="2005p59">{{Cite book|和書|year=2005|title=別冊ぱふ 声優インタビュー2004|publisher=[[雑草社]]|page=65}}</ref>。高校生になっても迷いはなく、3年時の三者面談でも「声優の専門学校に行く」と教師に伝え、面談は1分で終了したという<ref name="2005p59"/>。

[[代々木アニメーション学院]]在籍中、賢プロダクションに所属する<ref>{{Cite journal|和書|journal=[[声優アニメディア]]|issue=2011年5月号|page=110|publisher=[[学研]]}}</ref>。所属経緯は、事務所の養成所に入るためのオーディションを受けていたがすべて落選し、そんなときに先生から合格したらすぐに所属できると誘われて賢プロダクションのオーディションに参加した<ref name="声グラ">{{Cite web|url=http://seigura.com/senior/saiga_mitsuki.html|title=斎賀みつきの声優道|work=声優グランプリWEB|publisher=主婦の友インフォス情報社|date=2015-09-29|accessdate=2017-04-29}}</ref>。

=== 趣味、嗜好 ===
以前から天体観測活動をしており<ref>[http://blog.animate.tv/saiga/index.php?itemid=5454 「曇ってもうたよ…」]ブログ『地球の子だもの。』2008年9月17日。</ref>、2009年8月22日『[[宙のまにまに]]』出演者・スタッフらによる天体観測会に参加<ref>[http://blog.animate.tv/saiga/index.php?itemid=6808 「8月22日」]ブログ『地球の子だもの。』2009年8月23日。</ref>して以降は「天文部」「星見会」などと称し、同じく声優の[[間島淳司]]、[[安元洋貴]]、[[伊藤かな恵]]、[[寺島拓篤]]、[[川原慶久]]、[[ささきのぞみ]]、[[山口清裕]]らと天体観測に度々繰り出している<ref>『宙のまにまに』原作者・[[柏原麻実]]の『カシマミはらっぱブログ』2010年3月6日付の記事[http://mmk.typepad.jp/1999/2010/03/%E7%AC%AC%EF%BC%94%E5%9B%9E%E5%A3%B0%E5%84%AA%E3%82%A2%E3%83%AF%E3%83%BC%E3%83%89.html 「第4回声優アワード」]の中に「その後我々のリアル天文部でもみんなをまとめてくださってる斎賀みつきさん。」という表現がある。</ref>。
