'''佐藤 日向'''（さとう ひなた、[[1998年]][[12月23日]]<ref name="amuse"/><ref name="oricon1"/><ref name="sakuragakuin">{{Cite web|url=http://www.sakuragakuin.jp/members/hinata.html|title=さくら学院 生徒紹介 - 佐藤 日向|accessdate=2015-08-13|work=[[さくら学院]]|publisher=株式会社[[アミューズ]]|archiveurl=https://web.archive.org/web/20140316133153/http://www.sakuragakuin.jp/members/hinata.html|archivedate=2014年3月16日|deadlinkdate=2017年9月}}</ref> - ）は、[[日本]]の[[アイドル]]、[[モデル (職業)|モデル]]、[[女優]]<ref name="oricon1"/>。[[芸能事務所|所属事務所]]は[[アミューズ]]<ref name="amuse"/>。[[さくら学院]]の元メンバー<ref name="amuse"/>。[[新潟県]]出身<ref name="amuse"/>{{Refnest|group="注"|VIPタイムズ社（ORICON STYLE）及びさくら学院での紹介によると[[神奈川県]]<ref name="oricon1"/><ref name="sakuragakuin"/>}}。

== 略歴 ==
小学生低学年の頃からテレビCM等に出演。
2010年4月より、アイドルユニット[[さくら学院]]のメンバーとしての活動を開始<ref name="amuse"/>。同年12月、さくら学院のメンバーとしてシングル「[[夢に向かって / Hello ! IVY]]」でメジャーデビュー。その後さくら学院を中心に派生ユニット「バトン部 [[Twinklestars]]」、2012年から「科学部 [[科学究明機構ロヂカ?]]」でも活動。2013年5月5日に開催された「さくら学院2013年度・転入式」にてさくら学院の「ムード委員長」に任命される<ref name="nikkansports">{{Cite web |url=http://www.nikkansports.com/entertainment/news/f-et-tp0-20130505-1122789.html|title=さくら学院に2人の転入生！新制服も披露|accessdate=2015-08-13|publisher=[[日刊スポーツ]]新聞社|date=2013-05-05}}</ref><ref name="barks1">{{Cite web |url=http://www.barks.jp/news/?id=1000090243|title=【Kawaii girl Japan/ライブレポート】さくら学院 2013年度 新生徒会長は堀内まり菜に決定。「今年度のさくら学院も大きな桜を咲かせます」|accessdate=2015-08-13|work=[[BARKS]]|publisher=ジャパンミュージックネットワーク株式会社|date=2013-05-13}}</ref>。2014年3月、さくら学院を卒業{{Refnest|group="注"|さくら学院は中学校を卒業すると同時にグループからも卒業となる<ref name="natalie2"/>。}}<ref name="natalie2">{{Cite web |url=http://natalie.mu/music/news/113419|title=さくら学院、12人の“絆”が見えた卒業式|accessdate=2015-08-13|work=[[ナタリー (ニュースサイト)|ナタリー]]|publisher=株式会社ナターシャ|date=2014-03-31}}</ref><ref name="sanspo">{{Cite web |url=http://www.sanspo.com/geino/news/20140331/ido14033104020001-n1.html|title=さくら学院・堀内まり菜「いつか恩返しできる日が来ますように」|accessdate=2015-08-13|work=[[サンケイスポーツ]]|publisher=株式会社産経デジタル|date=2014-03-31}}</ref>。

2014年4月2日開始のアニメ『[[くつしたがだるだるになっちゃうわけ 〜イマドキ妖怪図鑑〜|くつだる。]]』（[[NHK教育テレビジョン|NHK Eテレ]]）のメインキャストである「柄沢りん」の声優を担当<ref name="natalie1">{{Cite web |url=http://natalie.mu/music/news/112020|title=さくら学院・堀内＆佐藤「くつだる。」で声優に挑戦|accessdate=2015-08-13|work=[[ナタリー (ニュースサイト)|ナタリー]]|publisher=株式会社ナターシャ|date=2014-03-14}}</ref><ref name="dogatch">{{Cite web |url=http://dogatch.jp/news/nhk/23998|title=堀内まり菜、金井美樹、佐藤日向、島ゆいかの“あるある話”とは!?新アニメ『くつだる。』スピンオフ企画決定|accessdate=2015-08-13|work=[[テレビドガッチ]]|publisher=株式会社プレゼントキャスト|date=2014-03-31}}</ref><ref name="cdjournal2">{{Cite web |url=http://www.cdjournal.com/main/news/-/57692|title=新作TVアニメ「くつだる。」キャストに金井美樹、島ゆいか、堀内まり菜、佐藤日向|accessdate=2015-08-13|work=CDJournal|publisher=音楽出版社|date=2014-03-10}}</ref>。同アニメの声優を担当する4名（[[堀内まり菜]]、[[金井美樹 (1996年生)|金井美樹]]、佐藤日向、[[島ゆいか]]）でユニット「マボロシ☆ラ部」を結成し、同年11月19日に主題歌と挿入歌を収録したデビューシングル「Merry Go World」を発売<ref name="entamenext1">{{Cite web |url=http://www.entamenext.com/topics_detail2/id=1109|title=元さくら学院・堀内まり菜、佐藤日向が参加するマボロシ☆ラ部が、「NO MUSIC, NO IDOL?」ポスターに初登場！|accessdate=2015-08-13|work=エンタメNEXT|publisher=株式会社 [[徳間書店]]|date=2014-11-11}}</ref>。

2015年8月、[[東京国際フォーラム]] ホールCにて公演の[[ミュージカル]]『[[グッバイガール|グッバイ・ガール]]』にてシンシア役として出演<ref name="goodbye">{{Cite web |url=http://www.good-bye-girl.com/cast.html|title=キャスト&スタッフ|accessdate=2015-08-13|work=ミュージカル グッバイ・ガール ＜オフィシャルサイト＞|publisher=[[フジテレビジョン]]}}</ref>。
