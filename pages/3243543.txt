'''高田 憂希'''（たかだ ゆうき、[[1993年]]<ref>{{Twitter status|sunflower930316|577358017542287360}}</ref>[[3月16日]]<ref name="profile">{{Cite web |url=https://mausu.net/talent/takada-yuuki.html|title=マウスプロモーション公式サイト：所属タレント|accessdate=2013-07-05}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[マウスプロモーション]]所属<ref name="profile" />。[[ABO式血液型|血液型]]は[[ABO式血液型|AB型]]<ref>{{Twitter status|sunflower930316|582478310552174592}}</ref><ref>{{Twitter status|sunflower930316|582480091185184768}}</ref>。[[福岡県]][[北九州市]]出身<ref name="profile"/>{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。プロジェクト:声優#冒頭部分にてガイドラインが制定されていますので、そちらも参照して下さい。}}。

== 経歴 ==
{{雑多な内容の箇条書き|section=1|date=2016年7月}}
幼少期からテレビアニメを好んで視聴しており、『[[シャーマンキング]]』『[[BLEACH]]』のテレビアニメ版がきっかけで声優業に関心を持ち、高校生になってから[[演劇部]]に所属して演劇を学ぶ傍ら、[[代々木アニメーション学院]]福岡校の体験入学におけるアフレコ体験を機に、声優を本格的に志すようになり、同校へ入学<ref name="nizista">{{Cite web |url=https://nizista.com/views/article2?id=8b78b0505eec11e6b392a5d167e4b118|title=高田憂希インタビュー（前編）「一人前の声優を目指して、挫折を味わったこともありました」|accessdate=2016-08-12}}</ref>。

[[2013年]]、代々木アニメーション学院福岡校からの卒業を機に東京へ移住し、マウスプロモーション付属俳優養成所（第28期生）に入学<ref name="profile" /><ref>{{Cite web |url=http://ameblo.jp/mausu-y/entry-11536343531.html|title=「芝居心」28A高田憂希|accessdate=2016-11-10}}</ref>。同養成所在籍中に、劇場版アニメ『[[BAYONETTA Bloody Fate]]』に出演して声優デビューを果たし<ref name="debut">{{Twitter status|sunflower930316|380719153385988096}}</ref>、[[2015年]]より現事務所所属となる。

2015年[[7月5日]]、株式会社オブジェクト主催・企画のイベント『こえずか 〜声優図鑑に出てくれた最高なメンバーをがむしゃらに揃えましたSP〜【夜の部】』にて優勝。

2017年[[11月4日]]、[[アニうた KITAKYUSHU|北九州ポップカルチャーフェスティバル]]のオープンセレモニーにて、[[北九州市]]観光大使に任命される<ref>https://twitter.com/sunflower930316/status/926760890543644673</ref>。

北九州市観光大使かつ声優なのは、高田憂希が'''初めて'''である。

== 人物 ==
* 自身のコラムにて、[[天然パーマ]]だと語っている<ref>{{Cite journal|和書|journal=[[娘TYPE]]|issue=2016年6月号|publisher=Newtype|date=2016-4-30|page=85}}</ref>。
* 趣味・特技は、[[明太子]]の食べ比べ、歌うこと、[[靴下]]集め<ref name="profile"/>、ひとりカラオケ、あいさつ作り<ref>{{Cite journal|和書|journal=VOICE Channel|issue=Vol.1|publisher=コスミック出版|date=2017-10-28|isbn=978-4-7747-8413-7}}</ref>。
* 好きな食べ物は[[博多の女]]<ref>{{Cite web |url=http://ameblo.jp/mausu-y/entry-11602591269.html|title=「ご飯はしっかり食べよう」28A高田憂希|accessdate=2016-11-10}}</ref>、[[カレーライス]]、苦手な食べ物は[[いちご]]（いちご味の食品は大丈夫だが、いちごそのものがダメ）である<ref>{{Twitter status|sunflower930316|716250204622561285}}</ref>。
* 『[[美少女戦士セーラームーン]]』が好きで、特に[[セーラーヴィーナス]]が好きだと語っている<ref>{{Cite journal|和書|journal=[[娘TYPE]]|issue=2016年3月号|publisher=Newtype|date=2016-1-30|page=69}}</ref>。
* 動物の[[キリン]]が好きである<ref>{{Cite journal|和書|journal=[[娘TYPE]]|issue=2016年7月号|publisher=Newtype|date=2016-5-30|page=85}}</ref>ほか、[[レモン]]、[[ひまわり]]が好きである<ref name="di">{{Cite web |url=http://www.joqr.co.jp/kilehi/ |title=ラジオどっとあい　きりんとレモンとひまわりと高田憂希 公式サイト|accessdate=2015-07-09}}</ref>。
* オフの日は、1人[[カラオケ]]に行くことが多く、その際、[[LiSA]]や[[UNISON SQUARE GARDEN]]の楽曲を歌っていると語っている<ref name="nizista後編">{{Cite web |url=https://nizista.com/views/article2?id=4fac5440603711e6b681c95beba2a649|title=高田憂希インタビュー（後編）「先輩たちのような素敵な人になりたい」|accessdate=2016-08-22}}</ref>。
* 自身の曲がアニメの主題歌に起用されることに憧れを持っており、[[水樹奈々]]や[[戸松遥]]、[[堀江由衣]]、[[スフィア (声優ユニット)|スフィア]]に憧れを感じている<ref name="nizista後編" />。
* 交友のある声優には[[下地紫野]]の名前をあげている<ref name="nizista後編" />。
