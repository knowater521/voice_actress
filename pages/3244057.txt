'''篠田 みなみ'''（しのだ みなみ、[[1994年]]<ref>{{Cite web |url= https://twitter.com/minami_shinoda/status/605754755160236032|title=2015年6月3日付、篠田みなみのツイートより|accessdate=2015-07-07}}</ref>[[6月3日]]<ref name="profile">{{Cite web |url= http://vims.co.jp/talent_profile_detail.php?id=34|title=ヴィムス公式サイト：所属タレント|accessdate=2013-07-07}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[ヴィムス]]所属。[[千葉県]]出身<ref name="profile"/>。

== 来歴 ==
『[[シャーマンキング]]』のアニメを見たころから声優という職業を意識し始めたが、元々は[[ディズニーランド]]のダンサーやミュージカル俳優も目指して幼少期にダンスを習って育った。だが、身長が147cmであることから職業柄制限される場合が多く、容姿に関係なくさまざまな表現ができる声優という職業を選んだ<ref>[[声優グランプリ]]2015年4月号 新人声優NOTEBOOKより{{要ページ番号|date=2018-01-21}}</ref>。

小学校の頃は[[シンガポール]]にて暮らしており、小学校3年生の頃は[[ソーラン節]]を、2年後にはジャズダンスを始めており、10年以上も習っている<ref name="ddnavi">{{Cite web |date=2016-05-10|url=http://ddnavi.com/news/300702/a/|title=「篠田みなみ」声優インタビュー＆ミニグラビア【声優図鑑】|publisher=[[ダ・ヴィンチ]]ニュース|accessdate=2016-05-10}}</ref>。このため、小学校の文集には「ダンサーになりたい」と書くほどだった<ref name="ddnavi" />。また、学生時代には軽音楽部でギターも経験している。

高校1年生の頃にお芝居を勉強したいと日本ナレーション演技研究所に通い、高校3年生になる頃には声優業を仕事にしたいと意識したことを語っている<ref name="ddnavi" />。在学中には、篠田みなみと[[諏訪ななか]]と栗山りさによるラジオ番組『アニラブ！ガチャガチャ<ref>{{Cite web |url=http://ondemand.joqr.co.jp/AG-ON/contents/gachagacha20121009.php|title=2012年10月9日、アニラブ!ガチャガチャAG-ON|accessdate=2015-07-07}}</ref>』で初めての仕事を獲得し、『アニラブ!ガチャガチャ』のオーディションは『[[本気!アニラブ]]』第1期のオーディション落選後に合格した。『アニラブ!ガチャガチャ』の放送終了後、2013年の春にヴィムス所属となる。

[[2014年]][[2月19日]]、スマートフォン向けアプリケーションゲーム『[[Tokyo 7th シスターズ]]』がリリースされ、春日部ハル役として初のメインキャラクターを演じる。この役は篠田が声優のオーディションで初めて合格した役にもなった<ref name="ddnavi" />。また、2014年4月には『[[一週間フレンズ。]]』にて初のテレビアニメ出演を果たす。さらに、同月から開始した『本気!アニラブ』では、第4期生の火曜日担当として初の個人ラジオ番組を経験した<ref>{{Cite web |url=http://www.joqr.co.jp/alb/2014/04/|title=2014年04月、本気!アニラブ|accessdate=2015-11-23}}</ref>。

2015年春には、株式会社MICのあすかと篠田みなみがパーソナリティを務めるラジオ番組『篠田みなみ・MICあすかのクリエイターズサロン!』が放送された。また、同時期に『[[声優グランプリ]]』2015年4月号の別冊付録『新人声優NOTEBOOK 2015』にて初めての雑誌掲載インタビューと撮影が掲載された。同年6月には、『t7s 1st Anniversary Live』で初のライブイベント出演を果たす。

2016年4月、『[[ふらいんぐうぃっち]]』の木幡真琴役で初のテレビアニメ主演を果たす。

2018年1月、『[[きららファンタジア]]』の呉織あぎり役を[[高部あい]]から引き継ぐ。

== 人物==
* 資格は英語検定準2級、漢字検定3級、韓国語能力検定初級を所持<ref name="profile"/>。特技は、手を合掌させてうねうねさせること<ref>{{Cite web|url=http://www.at-x.com/at-x_contents/summerfes2015/fresh03.html#fresh_s02|title=フレッシュ夏フェス隊 - 篠田 みなみ|accessdate=2016-01-28}}</ref>。
* 好きな食べ物は[[チャーハン]]、ツブ貝などの貝類。
* 声優にとって必要だと思うことについて、1番目は「好奇心」、2番目は「ワクワクすること」、3番目は「真剣」であること、また仕事の現場での対応力も声優として必要なスキルと語っている。また、これから声優を目指す人には、学校の授業等のこともすべていつか活かせるものだと思って頑張ってほしいとも語っていた<ref>2015年[[声優グランプリ]]4月号 新人声優NOTEBOOKより</ref>。
* 憧れの声優には[[矢島晶子]]を挙げている<ref name="ddnavi" />。また、仲の良い声優には、[[井澤美香子]]、[[中島唯 (声優)|中島唯]]、[[高田憂希]]、[[道井悠]]、[[井ノ上奈々]]、[[大地葉]]、[[橋本ちなみ]]、[[山本希望]]、[[高井舞香]] などを挙げている。井澤とは事務所に所属する前に共演して以来の仲であり<ref name="ddnavi" />、[[山本希望]]とは趣味仲間でもある。
* 事務所の同期には、[[野上翔]]、[[八代拓]]、橋本ちなみ、泊明日菜がいる<ref>{{Cite web|url=https://twitter.com/minami_shinoda/status/747443054668189696|title=今日は、野上翔さん、泊明日菜さん、橋本ちなみさんとごはんしてきました…|accessdate=2016-06-28}}</ref>。
