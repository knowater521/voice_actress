== 来歴 ==
中学生時代まで、シュガープリンセスルームのモデル、雑誌[[JSガール]]の専属モデルなどジュニアモデルとして活躍した。<!--モデルを始めたきっかけなどについて出典があれば追記してください。ニコ☆プチでのカリスマプチ読、雑誌mariaでの表紙、キッズファッションRONIでのカタログモデルについて出典があれば追記しても構いません。-->

[[阪急百貨店うめだ本店]]のシュガープリンセスルームのモデルを1年間務め<ref name="blog10960001208">{{Cite web |author=中島由貴 |date=2011-07-21 |url=http://ameblo.jp/gorogoro1022/entry-10960001208.html |title=シュガープリンセスモデル募集！ |work=中島由貴オフィシャルブログ「いつも笑顔で ～happy smile～」 |publisher=サイバーエージェント |accessdate=2017-03-07}}</ref>、2011年9月11月のイベントをもってシュガープリンセスルームのモデルを卒業した<ref name="blog11047041357">{{Cite web |author=中島由貴 |date=2011-10-15 |url=http://ameblo.jp/gorogoro1022/entry-11047041357.html |title=撮影☆ディアブルコート♪（追記あり） |work=中島由貴オフィシャルブログ「いつも笑顔で ～happy smile～」 |publisher=サイバーエージェント |accessdate=2017-03-07}}</ref>。

JSガールの創刊号である『JSガール Vol.1』を見て読者モデルに応募する<ref name="blog11121831112">{{Cite web |author=中島由貴 |date=2011-12-31 |url=http://ameblo.jp/gorogoro1022/entry-11121831112.html |title=１月～３月♪ＪＳガール＆由貴の１年間 |work=中島由貴オフィシャルブログ「いつも笑顔で ～happy smile～」 |publisher=サイバーエージェント |accessdate=2017-03-07}}</ref>。2011年5月に発売された『JSガール Vol.2』で表紙を飾り<ref name="blog11121918514">{{Cite web |author=中島由貴 |date=2011-12-31 |url=http://ameblo.jp/gorogoro1022/entry-11121918514.html |title=４月～６月♪ＪＳガール＆由貴の１年間 |work=中島由貴オフィシャルブログ「いつも笑顔で ～happy smile～」 |publisher=サイバーエージェント |accessdate=2017-03-07}}</ref>、以降読者モデルとしてJSガールに登場した<ref name="JSgirl3">{{Cite web |url=http://www.sun-a.com/magazine/detail.php?pid=4993 |title=JSガール2011 Vol.3 |work=JSガール |publisher=三栄書房 |accessdate=2017-03-07}}</ref><ref name="JSgirl4">{{Cite web |url=http://www.sun-a.com/magazine/detail.php?pid=5113 |title=JSガール2011 Vol.4 |work=JSガール |publisher=三栄書房 |accessdate=2017-03-07}}</ref><ref name="JSgirl5">{{Cite web |url=http://www.sun-a.com/magazine/detail.php?pid=5200 |title=JSガール2011 Vol.5 |work=JSガール |publisher=三栄書房 |accessdate=2017-03-07}}</ref>。2011年12月、JSガールの専属モデルに選ばれたことが発表された<ref name="blog11112626928">{{Cite web |author=中島由貴 |date=2011-12-21 |url=http://ameblo.jp/gorogoro1022/entry-11112626928.html |title=ＪＳガール専属モデル発表！！ |work=中島由貴オフィシャルブログ「いつも笑顔で ～happy smile～」 |publisher=サイバーエージェント |accessdate=2017-03-07}}</ref>。2011年10月22日発売の『JSガール Vol.11』をもってJSガール専属モデルを卒業した<ref name="blog11385662773">{{Cite web |author=中島由貴 |date=2012-10-25 |url=http://ameblo.jp/gorogoro1022/entry-11385662773.html |title=JSガール専属モデルを卒業しました |work=中島由貴オフィシャルブログ「いつも笑顔で ～happy smile～」 |publisher=サイバーエージェント |accessdate=2017-03-07}}</ref>。

プロダクションアミューズに所属し、[[子役]]としてCM、ドラマなどに出演した<ref>{{Cite web |url=http://artist.amuse.co.jp/artist/nakashima_yuki/ |title=中島 由貴 |accessdate= |work=所属アーティスト |publisher=[[アミューズ]] |archiveurl=https://web.archive.org/web/20110315220248/http://artist.amuse.co.jp/artist/nakashima_yuki/ |archivedate=2011-03-15 |deadlinkdate=2017-02-28}}</ref>。

[[2014年]]にアース・スター エンターテイメントが開催した国民的声優グランプリにて3000名応募の中、グランプリを受賞する<ref>{{Cite web |author= |date=2015-03-14 |url=http://blog.nicovideo.jp/niconews/ni051987.html |title=国民的声優グランプリ受賞者から結成！『アース・スター ドリーム』チャンネル |work=ニコニコインフォ |publisher=[[ドワンゴ]] |accessdate=2017-02-28}}</ref>。その後[[アース・スター ドリーム]]のメンバーとしてデビューするとともに、アース・スター エンターテイメントに所属した。

『[[てーきゅう]]』アニメ4期の大喜利の司会者役で[[声優]]デビューし<ref>{{Twitter_status2|earthstar_dream|553719897306312704|accessdate=2017-03-03}}</ref>、同作品5期の田中きなこ役で初めて名前のある役を演じる。また、アニメに先行し、舞台版においてもきなこを演じ、舞台デビューも果たしている<ref>{{Twitter_status2|earthstar_dream|613264903361527808|accessdate=2017-03-03}}</ref>。

2015年に日本きものシステム協同組合のオーディションで第4期ドリームエンジェルとして選ばれ、同組合のCM、パンフレットなどで活躍した<ref name="JKS">{{Cite web |url=http://www.furisodeshop.com/trend/dream.html |title=Dream Angel |accessdate=2017-02-28 |year=2015 |month= |work=トレンド2015 |publisher=日本きものシステム |archiveurl=https://web.archive.org/web/20151218201133/http://www.furisodeshop.com/trend/dream.html |archivedate=2015-12-18 }}</ref>。

高校は2年まで地元の高校に通学し、3学年より[[日出中学校・高等学校|日出高等学校]]に通学、2016年3月に卒業した<ref>{{cite web|url=http://ameblo.jp/gorogoro1022/entry-12233485935.html|title=2016年振り返り！①1月～3月✨|work=アメーバブログ|date=2016-12-31|accessdate=2017-02-28<!--|archiveurl=https://web.archive.org/web/20170228112734/http://ameblo.jp/gorogoro1022/entry-12233485935.html|archivedate=2017-02-28-->}}</ref>。

2016年9月よりORICON GAMEの公式プレイヤーとして活動している<ref name="oricon" />。

2017年10月に、12月末をもってアース・スター ドリームを卒業することを発表し<ref>{{Cite web |date= 2017-10-09|url= http://comic-earthstar.jp/esd/news.php#201710091848-1|title= 中島由貴に関する重要なお知らせ|publisher= アース・スター ドリーム|accessdate=2017-11-02}}</ref>、アース・スター ドリーム自体も12月末に活動休止となった。
