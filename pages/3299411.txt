'''和氣 あず未'''（わき あずみ、[[1994年]]<ref>{{Cite episode

}}</ref>[[9月8日]] - ）は、[[日本]]の[[女性]][[声優]]。[[東京俳優生活協同組合]]所属。[[東京都]]出身。

== 経歴 ==
* 13歳の時に好きであったアニメに出演していた声優のアフレコ映像を見て、声優業に興味を持った<ref name="SPインタビュー">{{cite web|url=http://www.anime.ac.jp/employment_debut/special-interview/special-interview04.php|title=「和氣あず未」 スペシャルインタビュー|publisher=東京アニメ・声優専門学校|accessdate=2016-06-09}}</ref>。
* [[東京アニメ・声優専門学校]]第3期卒業生である<ref name="SPインタビュー" />。中学時代は[[キャビンアテンダント]]を目指していたが、高校に入ると、[[英語 (教科)|英語]]が苦手なことから声優志望に転向した<ref name="ddnavi">{{Cite web |date=2015-10-14|url=http://ddnavi.com/news/262783/a/|title=「和氣あず未」声優インタビュー＆撮り下ろしグラビア【声優図鑑】|publisher=[[ダ・ヴィンチ]]ニュース|accessdate=2015-10-17}}</ref>。
* [[2014年]]に俳協に所属<ref name="haikyo">[http://www.haikyo.co.jp/profile/profile.php?ActorID=12722 東京俳優生活協同組合によるプロフィール]</ref><ref>{{Cite press release |title=新しく所属になりました。|publisher=東京俳優生活協同組合|date=2015-01-13|url=http://www.haikyo.co.jp/information/info15/info15.html|accessdate= 2015-10-17}}</ref>。
* 元々は人前で何かすることが苦手であったと語っている<ref name="SPインタビュー" />。
* 所属後、初めてオーディションを受けた作品が『[[アイドルマスター シンデレラガールズ]]』であった<ref name="ddnavi" />。
* 初ライブとなった「[[アイドルマスター シンデレラガールズ|THE IDOLM@STER CINDERELLA GIRLS 3rdLIVE シンデレラの舞踏会]]」には、養成所の名誉会長にあたる[[津川雅彦]]からフラワースタンドが贈られている。

== 人物 ==
* 名前の「'''あず未'''」は本名だが、[[長野県]]の[[安曇野]]から取られた名前だと自身の[[Twitter]]にて語っている<ref>{{Cite web |date=2016-04-26|url=https://twitter.com/azumi_waki/status/724778694766551040|title=長野県楽しみだなぁ〜！ ちなみに長野県の安曇野って場所から、あずみって名前をとったらしい(^-^) 綺麗な場所なんだって〜〜…|publisher=和氣あず未|accessdate=2016-04-26}}</ref>。
* 年の離れた兄が3人いる<ref>{{Cite web |date=2016-05-11|url=https://twitter.com/azumi_waki/status/730365866320109568|title=私は歳の離れた兄がいるから…|publisher=和氣あず未|accessdate=2016-05-11}}</ref><ref>{{Cite web |date=2016-05-11 |url=https://twitter.com/azumi_waki/status/754263551263412224 |title=4人兄妹の中の長男の結婚式でした♪… |publisher=和氣あず未 |accessdate=2016-08-26}}</ref>。
* 中学、高校と[[テニス]]部に所属していたため、運動神経は良いと自負している<ref name="ddnavi" />。
* 趣味はお菓子作りとイラスト。
* 寝る事が好きである<ref name="ddnavi" />。
* [[ハンバーガー]]、[[ゾンビ]]も好きであり、LIVERTINE AGEとコラボした際にはそれらをモティーフにTシャツやパーカーのデザインをしている<ref>{{Cite web |date=2017-09-22 |url=https://twitter.com/livertineage/status/911173070433378306|title=本日スタート！！　大人気声優和氣あず未さんとのコラボ商品の受注予約を開始しました… |publisher=LIVERTINE AGE |accessdate=2017-09-23}}</ref>。
* 好きなアニメは『[[鋼の錬金術師 (アニメ)|鋼の錬金術師]]』である<ref name="ddnavi" />。この作品に登場する[[鋼の錬金術師の主要な登場人物#ロイ・マスタング|ロイ・マスタング]]のようにちょっとSなキャラクターが好きと語っている<ref name="ddnavi" />。
* ファンから友達感覚でいてもらえるような親しみやすい声優を目指している<ref name="ddnavi" />。
