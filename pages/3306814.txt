'''鈴木 みのり'''（すずき みのり、[[1997年]][[10月1日]]<ref>{{Cite web |url=http://e-stone.victormusicarts.jp/minoringo/biography |title=BIOGRAPHY |publisher=鈴木みのり Official Website |accessdate=2016-03-28}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]。[[愛知県]]出身。[[e-stone music]]所属<ref>{{Cite web|url=http://e-stone.victormusicarts.jp/minoringo/contact|title=CONTACT|work=鈴木みのり Official Website|publisher=
[[ビクターミュージックアーツ]]|accessdate=2016-05-25}}</ref>。血液型はA型<ref>{{Cite web |url=http://e-stone.victormusicarts.jp/minoringo/archives/754 |title=じいじとばあば|publisher=鈴木みのり Official Website |accessdate=2016-06-09}}</ref>。

== 略歴 ==
小さい頃から歌うことが大好きで、「声」に関する仕事（歌手・ナレーション・芝居）に興味を持っていた<ref name="times180125">{{Cite news |title=声優・鈴木みのりのルーツ――アニメ好き、声優を目指したキッカケとは／インタビュー前編 |url=https://www.animatetimes.com/news/details.php?id=1516775318 |newspaper=アニメイトタイムズ |date=2018-01-25 |accessdate=2018-03-07}}</ref>。小学校高学年の頃に[[深夜アニメ]]を観るようになり、「声優になれば自分のやりたいことが全部できる」と憧れを抱くようになる<ref name="VST05">{{Cite web |date=2016-03 |url=http://v-storage.bandaivisual.co.jp/vstorage_archives/vol5/index.html |title=フレイア・ヴィオン役 鈴木みのりインタビュー |work=V-STORAGE Vol.5 |publisher=バンダイビジュアル |pages=6-7 |accessdate=2016-03-30}}</ref>。高校1年生の時にアルバイトでお金を貯め、高校2年生の1年間は週に1度東京の声優養成学校に通い、レッスンを受けていた<ref name="souken">{{Cite news |title=鈴木みのりがソロデビュー。「FEELING AROUND」には、みのりんごの笑顔が詰まっている！ |url=https://akiba-souken.com/article/32832/ |newspaper=アキバ総研 |date=2018-01-23 |accessdate=2018-03-07}}</ref><ref name="NT1802">『月刊ニュータイプ 2018年2月号』、KADOKAWA、2018年、51頁。</ref>。

2014年末に[[マクロスシリーズ]]の新作テレビアニメの新歌姫・声優オーディション<ref>{{Cite news |title=『マクロスΔ（仮）』歌姫オーディションの応募受付スタート。『マクロス』シリーズ最新作の歌姫を目指せ！ |newspaper=電撃オンライン |date=2014-12-01 |author= |url=http://dengekionline.com/elem/000/000/967/967472/ |accessdate=2016-03-30}}</ref>に応募し、応募者約8,000人の中から合格<ref>{{Cite news |title=新作『マクロスΔ』は『F』の8年後が舞台、新歌姫は少女5人組グループに |newspaper=マイナビニュース |date=2015-10-29 |author= |url=http://news.mynavi.jp/news/2015/10/29/400/ |accessdate=2016-03-30}}</ref>。2016年4月にテレビアニメ『[[マクロスΔ]]』の[[ワルキューレ (マクロスΔ)#フレイア・ヴィオン|フレイア・ヴィオン]]役で芸能活動を開始し、作中のアイドルユニット「[[ワルキューレ (マクロスΔ)|ワルキューレ]]」名義での音楽活動も行う。2018年にはシングル「FEELING AROUND」をリリースしてソロデビューした<ref>{{Cite news |title=明日で20歳の鈴木みのり「ラーメン大好き小泉さん」OP曲でソロデビュー決定 |url=http://natalie.mu/music/news/250929 |newspaper=音楽ナタリー |date=2017-09-30 |accessdate=2017-09-30}}</ref>。

声優デビュー後、『ときめきアイドル』の月島美奈都役で初めてオーディションに合格する<ref>{{Cite news |title=日岡なつみ、鈴木みのり、和久井優らが登壇！　コナミ、『ときめきアイドル』ステージレポート |url=https://www.voice-style.jp/archives/51787 |newspaper=Voice-Style |date=2017-09-25 |accessdate=2018-03-09}}</ref>。

== 人物 ==
=== 愛称 ===
*小学生の頃に友達から付けられた'''みのりんご'''という愛称を使っている<ref>{{Cite news |title=マクロスTVシリーズ最新作『マクロスΔ』放送直前ルンピカ上映会イベントレポート |newspaper=V-STORAGE |date=2016-03-25 |author= |url=http://v-storage.bandaivisual.co.jp/report/event/36472/ |accessdate=2016-03-30}}</ref><ref name="souken"/>。また、その頃母親が考えてくれたサインを今でも使っている<ref name="souken"/>。
*『マクロスΔ』オーディション最終審査のため上京し、[[原宿駅]]からビクタースタジオに向かう途中「みのりんご<ref>[http://www.minoringo.jp/index.html みのりんごホームページ]。男性店長も子供の頃「みのりんご」という愛称で呼ばれていた。</ref>」というカレーショップがあることを知り、また、オーディション本番ではヒロイン（フレイア）がリンゴ好きという設定を知り、運命的なものを感じたという<ref>「おもいでさんぽ 鈴木みのり」『[[声優グランプリ]] 2018年3月号』、主婦の友社、2018年、113頁。</ref>。

=== 趣味・嗜好 ===
* 最初に好きになったアニメは、6歳の頃に観た『[[マーメイドメロディー ぴちぴちピッチ]]』<ref name="NT1802"/>。
* 声優を目指す前は『[[ちゃお]]』の漫画家になりたいと思っていたり、[[ドールハウス]]職人になりたいと思っていた<ref >{{Cite web |title=むかしの夢|url=http://e-stone.victormusicarts.jp/minoringo/archives/760|publisher=鈴木みのり Official Website|accessdate=2016-06-02}}</ref>。
* 好きな食べ物として挙げているものは、[[カツ丼]]と[[りんご]]<ref name="prof">{{Cite web |date=2016-03-28 |title=愛知県出身鈴木みのり18歳！人生初のブログ第一回目！！|url=http://e-stone.victormusicarts.jp/minoringo/archives/366|publisher=鈴木みのり Official Website|accessdate=2016-03-29}}</ref>。
* 好きな人物には、[[坂本真綾]]、[[中島愛 (声優)|中島愛]]、[[やなぎなぎ]]<ref name="prof" />、[[大泉洋]]<ref name="saima">{{Cite web |title=さいまさんと|url=http://e-stone.victormusicarts.jp/minoringo/archives/709|publisher=鈴木みのり Official Website|accessdate=2016-05-22}}</ref>をあげている。坂本については中学校の文化祭で「[[Driving in the silence|誓い]]」をカヴァーして弾き語りしてから大ファンになり<ref name="NT1802"/>、ファンクラブに入会し、CDを全部揃えている<ref>{{Cite web |date=2016-03-31 |url=http://e-stone.victormusicarts.jp/minoringo/archives/394 |title=3月31日 |work= |publisher=鈴木みのり Official Website|accessdate=2016-08-16}}</ref>。デビュー後は[[フライングドッグ]]のレーベルメイトとなり、坂本が書いた歌詞を歌いたいという希望が実現した<ref>『マクロスΔ』挿入歌「風は予告なく吹く」、『[[あまんちゅ!〜あどばんす〜]]』OPテーマ「Crosswalk」の歌詞提供を受けている。</ref>。
* 趣味は音楽鑑賞で、上記の三人のほか、[[星野源]]や[[サカナクション]]、[[Galileo Galilei]]、[[青葉市子]]<ref name="prof" />、[[北川勝利]]<ref name="hashiru">{{Cite web |date=2016-05-03 |title=走る|url=http://e-stone.victormusicarts.jp/minoringo/archives/597|publisher=鈴木みのり Official Website|accessdate=2016-05-03}}</ref>の楽曲も好んでいる。アニソン関連では坂本や中島のほかに[[水樹奈々]]、[[宮野真守]]、[[GRANRODEO]]のライブに参加していた<ref name="times180125"/>。また、日課として声優の[[丹下桜]]のラジオを聴く事を挙げている<ref name="manga">{{Cite web |date=2016-04-09 |title=漫画のお話ペラペーラ！|url=http://e-stone.victormusicarts.jp/minoringo/archives/457|publisher=鈴木みのり Official Website|accessdate=2016-04-10}}</ref>。
*一番好きな漫画は『[[彼氏彼女の事情]]』<ref name="times180125"/>。好きな漫画家に[[今日マチ子]]、[[CLAMP]]、[[小川麻衣子]]を挙げている<ref name="manga" />。
* 学生時代は陸上部に所属しており、走ることには自信がある。上京後も[[ランニング]]を習慣としている<ref name="hashiru" />。
* 自分が感じたこと、誰かとのメールで大切だと思ったことはメモやプリントアウトしてファイルし、忘れないようにしている<ref>{{Cite web |date=2016-04-26 |title=振り返る|url=http://e-stone.victormusicarts.jp/minoringo/archives/551|publisher=鈴木みのり Official Website|accessdate=2016-05-03}}</ref>。

=== 家族・交友関係 ===
* 家族は父親・母親と四歳年下の妹がいる<ref name="NT1802"/>。もともと父がアニメ好き、母が音楽好きだったが、母親も声優について調べるうちにアニメファンになり、親子一緒でライブに参加したりしている<ref name="times180125"/>。
* デビュー作の『マクロスΔ』で共演している[[西田望見]]や[[瀬戸麻沙美]]と仲が良い<ref>{{Cite web |title=緊急！|url=http://e-stone.victormusicarts.jp/minoringo/archives/454|publisher=鈴木みのり Official Website|accessdate=2016-05-03}}</ref>。また、[[中野さいま]]<ref name="saima" />や[[井上ほの花]]<ref>{{cite tweet |user=minoringo_staff |author=鈴木みのり official |number=817585079350243333 |date=2017-01-06 |title=そういえば! }}</ref>、[[田辺留依]]<ref>{{cite tweet |user=minoringo_staff |author=鈴木みのり |number=945122313585934337 |date=2017-12-24 |title=私のサンタさんは田辺氏でした }}</ref>とも交友がある。

=== マクロス関連 ===
小学生当時、「[[星間飛行]]」や「[[ダイアモンド クレバス/射手座☆午後九時 Don't be late|ダイアモンドクレバス]]」などの[[アニソン]]から興味を持ち、『[[マクロスF]]』（2008年）を夢中で観ていた<ref name="VST05"/>。作中で[[ランカ・リー]]役を演じた[[中島愛 (声優)|中島愛]]に憧れ、ファンとして握手会に参加したこともある<ref>{{Cite web |date=2016-03-30 |title=盛り沢山！Anime Japan 2016！|url=http://e-stone.victormusicarts.jp/minoringo/archives/385 |publisher=鈴木みのり Official Website |accessdate=2016-03-30}}</ref>。母親も『マクロスF』が好きで、オーディションへの挑戦を応援してくれたという<ref name="CASTINT">{{Cite web |date= |url=http://macross.jp/delta/cast/index02.php |title=ヒロイン"フレイア・ヴィオン役 " 鈴木みのりインタビュー |publisher=MACROSS PORTAL SITE |accessdate=2016-03-30}}</ref>。歌唱審査ではランカのあとを継ぎたいという願いを込めて「ホシキラ」を歌い<ref>{{Cite web |date=2016-04-03 |url=http://e-stonemusic.com/minoringo/archives/410 |title=マクロスのお話  |work=鈴木みのり オフィシャルブログ |publisher=e-stone music |accessdate=2017-08-31}}</ref>、結果的に中島と同じく、マクロスシリーズの歌姫として声優・歌手デビューすることになった。2017年と2018年のワルキューレの[[横浜アリーナ]]ライブでは、シークレットゲストの中島と『星間飛行』をデュエットした<ref>{{cite tweet |user=mamegu_staff |author=中島愛 official |number=825689712262213633 |date=2017-01-28 |title=ワルキューレのみんな、最高のライブお疲れさまでした！ }}</ref>。

『マクロスΔ』の[[河森正治]]総監督は、選考理由について「前向きでアクティブで、フレイアの性格に非常にあっていた<ref>{{Cite web |date=2015-10-30 |url=http://hobby.dengeki.com/news/121347/ |title=新バルキリーと新歌姫が公開! 『マクロスΔ（デルタ）』超時空発表会! |publisher=電撃ホビーウェブ |accessdate=2016-07-07}}</ref>」と述べている。オーディションの応募動画の段階から有力候補のひとりだったが、最終面接で受け答えする姿は、シナリオから抜け出してきたのではないかという程フレイアだったという<ref>「MACROSSΔ STAFF INTERVIEW」『SANKYO Presents 戦術音楽ユニット「ワルキューレ」Walkure Attack! 1st Live in Zepp パンフレット』、ムービック、2016年、24-25頁。</ref>。鈴木自身も見た目以外はフレイアと似ていないところがないくらいだと思っている<ref name="VST05"/>。なお、『マクロスΔ』の好きなキャラクターは空中騎士団の[[マクロスΔ#ボーグ・コンファールト|ボーグ・コンファールト]]である<ref>『V-STORAGE Vol.6』、バンダイビジュアル、2016年、12頁。</ref>。
