'''三咲 麻里'''（みさき まり、[[3月18日]]<ref>{{Cite web |url=https://twitter.com/msk_v3v/status/578167280791474176|title=三咲 麻里twitter|accessdate=2015-12-21}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。

== 来歴 ==
かつては[[シグマ・セブンe]]に所属していたが、[[2015年]][[1月31日]]に退社。2ヶ月間の[[フリーランス|フリー]]期間<ref>{{Cite web |url=https://twitter.com/msk_v3v/status/561728819808043008|title=三咲 麻里twitter|accessdate=2015-12-21}}</ref>を経て同年[[4月1日]]より[[プロダクション・エース]]に所属<ref>{{Cite web |url=https://twitter.com/msk_v3v/status/588218449924648960|title=三咲 麻里twitter|accessdate=2015-12-21}}</ref>。

2014年10月より[[大亀あすか]]とともに[[日本プロ麻雀連盟]]に所属<ref>[https://twitter.com/Msk_v3v/status/521278944054484993 三咲麻里のツイート]</ref><ref>{{Cite web |date=2014-10-13 |url=http://getnews.jp/archives/681547 |title=声優・大亀あすか、『咲-Saki-』きっかけでプロ雀士に！ |publisher=ガジェット通信 |accessdate=2015-12-22}}</ref><ref>{{Cite web |url=http://www.ma-jan.or.jp/guide/member/tokyo.html |title=日本プロ麻雀連盟東京本部 連盟員名簿 |publisher=[[日本プロ麻雀連盟]] |accessdate=2015-12-22}}</ref>。

2017年3月末にて日本プロ麻雀連盟を退会。2017年5月末にてプロダクション・エース退所と同時に結婚を発表<ref>{{Cite web |url=https://twitter.com/Msk_v3v/status/870177657664618497
|title=三咲 麻里twitter|accessdate=2017-06-01}}</ref>。
