'''竹下 礼奈'''（たけした れいな、[[1989年]]<ref>{{Cite web |url=https://twitter.com/reiny440/status/755433407820115968|title=26歳最後の日は、がっきゅばたこみわ〜んと遊んですごしました…|accessdate=2016-07-20}}</ref>[[7月20日]]<ref name="プロフィール">{{Cite web |url= http://www.leopardsteel.com/talent/Takeshita_Reina/ |title= 竹下礼奈 |accessdate=2016-05-27}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[鹿児島県]]出身。[[ABO式血液型|血液型]]は[[ABO式血液型|A型]]。

== 略歴 ==
[[松濤アクターズギムナジウム]]（18期生）<ref>{{Cite web |url= lhttps://www.actorschool.jp/sag/issue/180401/ |title= 18期宣材撮影レポート |publisher= https://www.actorschool.jp |accessdate=2016-03-04}}</ref>を経て、[[レオパード・スティール]]に所属。

阿佐ヶ谷アニメストリート内にある[[カフェ]]SHIROBACOにてシロバコタマ5として勤務していた（3/31付で卒業）<ref>{{Cite web |url= http://shirobaco.com/cast/ |title= CAST |publisher=  http://shirobaco.com/ |accessdate=2016-03-04}}</ref>。

== 人物 ==
趣味・特技は[[ダンス]]、[[映画鑑賞]]、台本の丸暗記、歌うことである<ref name="プロフィール" />。また、[[スキューバダイビング]]の資格を持っている<ref name="プロフィール" />。
