'''藤原 夏海'''（ふじわら なつみ、[[1994年]]<ref>{{Cite web|url=https://www.sponichi.co.jp/entertainment/news/2018/03/13/kiji/20180313s00041000194000c.html|title=「メジャーセカンド」試写会　主人公役声優が感涙「泣くなって言われていたのに…」|work=|author=|publisher=スポニチアネックス|date=2018-03-13|accessdate=2018-03-13}}</ref>[[6月2日]]<ref name="prof">{{Cite web |url=http://www.artsvision.co.jp/talent/3817/|title=藤原夏海 公式プロフィール|work=アーツビジョン|publisher=|accessdate=2016-04-16}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[アーツビジョン]]所属。[[静岡県]]出身。[[日本ナレーション演技研究所]]出身。

== 経歴 ==
デビュー2年目<ref>{{Cite web |url=http://www.voice-style.jp/archives/13884|title=【アニメフォーカス第19回】藤原夏海さんと三宅健太さんにフォーカス！|work=Voice-Style|accessdate=2016-06-13}}</ref>となる2016年春放送のTVアニメ『[[少年メイド]]』の小宮千尋役で初めて主役を演じた<ref name="少年メイド">{{Cite web |url=http://www.animatetimes.com/news/details.php?id=1457990898|title=藤原夏海さん、島﨑信長さん、前野智昭さんがみどころを語る！　『少年メイド』キャストインタビュー|work=アニメイトタイムズ|accessdate=2016-06-13}}</ref>。この作品で共演した[[前野智昭]]からは「（小宮千尋という）年齢も性別も違うキャラクターを演じるというプレッシャーを一切感じさせないで堂々と演じている」と評価されている<ref name="少年メイド" />。

== 人物 ==
趣味・特技は[[歌唱]]、[[キャッチボール]]、[[鉄棒]]である<ref name="prof" />。同期には、[[本渡楓]]がいる<ref>{{Cite web |url=http://ameblo.jp/fruits-box-blog/entry-12147934558.html|title=本渡｢竹井宏わん！(出演について)｣|work=とれたて！フルーツボックス ～あかお・いわい・ほんど・みやけ直送～|accessdate=2016-06-21}}</ref>。
