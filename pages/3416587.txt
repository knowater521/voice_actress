'''藍原 ことみ'''（あいはら ことみ、[[1990年]]<ref>{{cite web|url=http://tower.jp/artist/2440233/%E8%97%8D%E5%8E%9F%E3%81%93%E3%81%A8%E3%81%BF|title=藍原ことみ - TOWER RECORD ON LINE|publisher=TOWER RECORD ON LINE|date=2017-06-26|accessdate=2017-12-20|author=TOWER RECORD}}</ref>[[5月2日]]<ref name="profile" /> - ）は、[[日本]]の[[女性]][[声優]]。

== 経歴 ==
アトミックモンキー声優・演技研究所第7期生。ソーシャルゲームやアニメーションで活躍する。

小学5年生の時に『[[カードキャプターさくら]]』や『[[HUNTER×HUNTER (1999年のアニメ)|HUNTER×HUNTER]]』を見て「2次元の男の子」になりたいと思い、少しずつ声優を目指すようになった<ref name ="fresh">{{Cite web|url=http://www.at-x.com/at-x_contents/summerfes2015/fresh.html|title=フレッシュ夏フェス隊！ | ウチの夏フェス2015!|accessdate=2017-06-03|publisher=[[アニメシアターX]]}}</ref><ref name ="zenpen">{{Cite web|url=https://nizista.com/views/article2?id=84e27140a28d11e6a516534a54ca5a0c|title=藍原ことみインタビュー（前編） 「小さい頃から2次元の男の子になりたくて、それなら“声優”だと思いました」|date=2016-11-19|accessdate=2016-11-21|publisher=Nizista (ニジ★スタ)}}</ref>。幼稚園の頃からその傾向があり、『[[美少女戦士セーラームーン (テレビアニメ)|美少女戦士セーラームーン]]』に登場する男の子っぽいキャラクターのセーラーウラヌスのグッズを多く買ってもらっていた<ref name ="zenpen" />。

地元の大学に通っていて声優を目指す勉強ができなかったが、母親の勧めでオーディションを受けることを決め、[[こえ部]]のオーディションで特別賞を受賞する。[[声優アワード]]の新人発掘オーディションの決勝にこえ部の枠で出場し、アトミックモンキー声優・演技研究所に特別枠で通える資格を得る。養成所では講師の[[長沢美樹]]から指導を受ける<ref name ="zenpen" />。

== 人物 ==
憧れの声優は[[緒方恵美]]。小さい頃から声でキャラクターを好きになる癖があり、無意識にセーラーウラヌスなど緒方が演じるキャラクターを好きになっている<ref name ="zenpen" />。また、[[新山志保]]にも同じくらい影響を受けている<ref name ="kouhen">{{Cite web|url=https://nizista.com/views/article2?id=b96b2eb0a29211e6a1e7f19d3441c335|title=藍原ことみインタビュー（後編） 「演技も歌も、技術以上に気持ちが大切だと思っています」|date=2016-11-20|accessdate=2016-11-21|publisher=Nizista (ニジ★スタ)}}</ref>。

地声が低く、女の子の役を演じる時は自信がなく構えてしまうが、男の子の役は普段の声とリアクションで演じられる<ref name ="kouhen" />。

趣味は歌うことで、地元では歌が上手いと言われていた。アニソンも好きで、よく[[林原めぐみ]]の曲を歌っている<ref name ="kouhen" />。[[水樹奈々]]の曲も好きで、声優アワードの自己PRでは「[[SCARLET KNIGHT]]」を歌っている<ref name ="zenpen" />。

トークが好き。裏表がなく、ちょっとしたことでも嘘がつけず顔に出るタイプ<ref name ="kouhen" />。早口言葉が得意<ref name ="fresh" />。

ホラーゲームや推理ゲームが好きで、『[[ミステリート (第1作)|ミステリート]]』を繰り返しプレイしている<ref name ="kouhen" />。『[[LINE:ディズニー ツムツム|ツムツム]]』や『[[テトリス]]』などのパズルゲームが得意<ref name ="fresh" />。

90年代のアニメをこよなく愛しており、『カードキャプターさくら』や『美少女戦士セーラームーンセーラースターズ』が好き<ref name ="kouhen" />。
