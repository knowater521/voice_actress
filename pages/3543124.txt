'''橘 えみり'''（たちばな えみり、[[1995年]][[12月21日]]<ref name="prof">{{Cite web|url=http://swallow-p.com/va_tachibana.html|title=橘えみり プロフィール - スワロウ|accessdate=2017-01-09|publisher[[スワロウ]]}}</ref> - ）は[[日本]]の[[女性]][[声優]]。

[[ポニーキャニオンアーティスツ|スワロウ]]所属<ref name="prof" />。[[埼玉県]]出身<ref name="prof" />。身長160cm<ref name="prof" />。

== 来歴 ==
[[アミューズメントメディア総合学院]]声優タレント科専科を[[2014年]]に卒業し、卒業後は[[81プロデュース]]の研究生となった。

[[2015年]]から81プロデュースでジュニア所属<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/583104886075609088|title=Twitter / 81プロデュースのジュニア所属になりました。|date=2015-04-01 12:13|accessdate=2017-01-08}}</ref>となるが、[[2016年]][[3月31日]]<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/715721766857940994|title=Twitter / ◎お知らせです|date=2016-04-01 11:05|accessdate=2017-01-07}}</ref>に退所し、フリー期間を経て、同年[[6月1日]]にスワロウ<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/737827523220766721|title=Twitter / ◎所属のご報告|date=2016-06-01 11:06|accessdate=2017-01-07}}</ref>へ移籍した。

デビュー当時の芸名は「英 えみり（はなぶさ えみり）」<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/737827523220766721|title=Twitter / ◎所属のご報告|date=2016-06-01 11:06|accessdate=2017-01-07}}</ref>だったが、事務所移籍の際に現在の名前に変更した。

== 人物 ==
趣味は[[占い]]。特技は[[ダンス]]。

仲の良い声優は、同じ事務所の[[遠藤ゆりか]]、[[加地綾乃]]、[[花守ゆみり]]<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/749111053343088641|title=Twitter / 妹の話で盛り上がった|date=2016-07-02 14:22|accessdate=2017-01-07}}</ref>と前の事務所の時に仲良くなった、[[田中美海]]<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/646260490033696768|title=Twitter / 田中美海ちゃんとごはん食べてこれから映画みます！|date=2015-09-22 18:51|accessdate=2017-01-08}}</ref><ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/676032705935486976|title=Twitter / 友達の田中美海ちゃんが呼んでくれて観に行きました！|date=2015-12-13 22:35|accessdate=2017-01-08}}</ref>と[[芹澤優]]<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/615531259293339649|title=Twitter / 芹澤優ちゃんとご飯行ってきました～|date=2015-06-29 23:44|accessdate=2017-01-08}}</ref>、[[高柳知葉]]<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/657923371007512577|title=Twitter / プロフィットの高柳知葉さまと遊びました|date=2015-10-24 23:15|accessdate=2017-01-08}}</ref><ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/665156155715383296|title=Twitter / 高柳知葉さまと遊びました！|date=2015-11-13 22:16|accessdate=2017-01-08}}</ref>がいる。

本人の部屋には、大量の[[マイメロディ]]<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/809030389071126528|title=Twitter / まいめろ部屋|date=2016-12-14 22:40|accessdate=2017-01-07}}</ref>グッズがある。

小さい頃に[[ミュージカル]]の経験<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/765884611826114560|title=Twitter / みりんと立ち話|date=2016-08-17 21:14|accessdate=2017-01-07}}</ref>がある。

家族では、妹<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/749111053343088641|title=Twitter / 妹の話で盛り上がった|date=2016-07-02 14:22|accessdate=2017-01-07}}</ref>と弟<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/807956059411259394|title=Twitter / 弟がおーばーうぉっち？|date=2016-12-11 23:31|accessdate=2017-01-07}}</ref>と犬<ref>{{Cite web|url=https://twitter.com/twinkleflower_e/status/688660413743955968
|title=Twitter / 新しい服|date=2016-01-17 18:53|accessdate=2017-01-08}}</ref>がいる。

[[冷やし中華]]が好物。
