'''村上 奈津実'''（むらかみ なつみ、[[1995年]]<ref name="vip-times">{{Cite web |url=http://www.vip-times.co.jp/?talent_id=W17-0198 |title=村上奈津実 - 日本タレント名鑑 |publisher=VIPタイムズ社 |accessdate=2017-02-04}}</ref>[[9月7日]]<ref name="official">{{Cite web |url=http://app.global-websystem.net/list/main.php?s0=20007B255001&s1=1&a2=19950907 |title=村上奈津実 - YU-RIN PRO Talent Profile |publisher=[[ゆーりんプロ]] |accessdate=2017-02-04}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。[[ゆーりんプロ]]所属。[[東京都]]出身<ref name="vip-times" /><ref name="official" />。

== 経歴 ==
声優を目指すようになったきっかけは、中学校の英語の授業で行われた『[[不思議の国のアリス]]』の絵に声を当てる朗読劇。アリス役を演じた際にクラスメイトから「アリスの声がかわいい」という感想をもらい、演じるということに興味を持ち声の仕事を意識するようになった<ref name="2017.2">「超！アニ」連動新コーナー「お前は誰だ？」vol.1、『[[アニメディア]]』2017年2月号 (2017年1月10日)、pp.132-133、[[学研プラス]]</ref><ref name="seigura1">{{Cite web |url=http://seigura.com/senior/message/20170906_22739/1.html |title=村上奈津実｜先輩から学ぼう！｜声優グランプリweb（第1回） |publisher=[[声優グランプリ]] |date=2017-09-06 |accessdate=2017-09-08}}</ref>。もともと好きだったアニメの見方も変わり、『[[DEATH_NOTE_(アニメ)|DEATH NOTE]]』で[[宮野真守]]の声の存在感に感銘を受けた<ref name="2017.2" />。

中学生の時点ではアルバイトができずレッスンに通う資金が無かったため学校の勉強に集中し、受験を終えた後に声優の勉強ができる学校を探し始めた<ref name="seigura1" /><ref name="ddnavi">{{Cite web |url=https://ddnavi.com/interview/436876/a/ |title=「村上奈津実」声優インタビュー＆ミニグラビア【声優図鑑】 |publisher=ダヴィンチ・ニュース |date=2018-02-16 |accessdate=2018-02-17}}</ref>。複数の専門学校や養成所の体験授業に参加し、その中で見学した[[よこざわけい子]]の授業は良い緊張感があり基礎からしっかり学べると思い、[[よこざわけい子 声優・ナレータースクール]]に入学することを決めた<ref name="seigura1" />。

『[[アイカツスターズ!]]』の早乙女あこ役でアニメ作品に初出演<ref name="2017.2" /><ref>{{Cite web |url=https://twitter.com/yourinpro/status/730748067960393728 |title=（株）ゆーりんプロ (@yourinpro) |publisher=[[Twitter]] |date=2016-05-12 |accessdate=2017-02-04}}</ref>。

== 人物 ==
=== 趣味・趣向 ===
* 趣味・特技は[[釣り]]、[[けん玉]]<ref>{{Cite web |url=http://www.nicovideo.jp/watch/1484824213 |title=【新人声優動画】村上奈津実さんコメント＆特技披露！【アニメディア連動企画「お前は誰だ？」】 |publisher=超！アニメディア - [[ニコニコチャンネル]] |date=2017-01-20 |accessdate=2017-02-04}}</ref>、[[ドラム]]ほか<ref name="2017.2" /><ref>{{Cite web |url=https://app.famitsu.com/20171013_1157680/ |title=『ラブライブ！スクールアイドルフェスティバルALL STARS（スクスタ）』キャストインタビューを掲載！ |publisher=ファミ通App |date=2017-10-13 |accessdate=2017-10-14}}</ref>、[[節約]]など<ref name="ddnavi" />。特に釣りは始めた時に魚が多く釣れたことをきっかけに大きくはまっている<ref name="2017.2" />。

* 好きな食べ物は[[食肉|肉]]や[[白米]]<ref name="2017.2" />、[[カレーライス]]<ref name="ddnavi" />。料理もよくすると語っている<ref name="ddnavi" />。

* 好きなアニメは『[[カードキャプターさくら]]』で、幼稚園の頃から繰り返し見ている<ref name="2017.2" />。

* 好きなアーティストは[[aiko]]<ref name="2017.2" /><ref name="ddnavi" />。

* 感銘を受けた本は『[[博士の愛した数式]]』<ref name="2017.2" />。

* 所有資格は[[日本漢字能力検定|漢字検定]]準2級<ref name="official" />。

* [[イヌ|犬]]を飼っており、名前はモカ<ref name="2017.2" /><ref>{{Cite web |url=https://twitter.com/natyaaaaaaan07/status/856354176367906816 |title=村上奈津実 (@natyaaaaaaan07) |publisher=[[Twitter]] |date=2017-04-23 |accessdate=2017-04-28}}</ref>。

=== 声優として ===
* 仕事をする上でのモットーおよび座右の銘は「なるようになる」<ref name="2017.2" /><ref name="ddnavi" />。

* 声優としてのターニングポイントになった役に『アイカツスターズ!』の早乙女あこを挙げており、人生で大切な存在と語っている<ref name="2017.2" /><ref name="ddnavi" />。また、印象に残っている役に初めて声のみで泣く芝居を行った『[[クイズRPG 魔法使いと黒猫のウィズ]]』のウォーブリンガー＝ミリィを挙げている<ref name="ddnavi" /><ref name="seigura2">{{Cite web |url=http://seigura.com/senior/message/20170906_22739/2.html |title=村上奈津実｜先輩から学ぼう！｜声優グランプリweb（第2回） |publisher=[[声優グランプリ]] |date=2017-09-20 |accessdate=2017-09-22}}</ref>。

* 「声が特徴的でヒロイン向きじゃない」と言われることもある。目標はラブコメディで正統派のヒロインを演じること<ref name="2017.2" />。

* 具体的に尊敬している人物はいないが、周りに流されず思ったことをはっきり言える人物に憧れ尊敬している<ref name="2017.2" />。また、演技方面では憧れの声優に[[大谷育江]]と[[釘宮理恵]]を挙げている<ref name="ddnavi" />。
