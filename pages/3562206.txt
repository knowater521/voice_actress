'''小出 ひかる'''（こいで ひかる、[[1999年]][[7月4日]] - ）は、[[日本]]の[[声優]]。[[アース・スター エンターテイメント]]所属。

== 略歴 ==
2014年にアース・スター エンターテイメントが主催した国民的声優グランプリにて審査員特別賞を受賞<ref>{{Cite web |date= 2014-12-16|url= http://www.anime-recorder.com/ArticleDetail.aspx?seq_no=1231|title= アース・スターまつり2014にて「国民的声優グランプリ」受賞者9名が発表。『高宮なすのです！』のテレビアニメ化も発表|publisher= AnimeRecorder|accessdate=2017-02-13}}</ref>し、[[アース・スター ドリーム]]のメンバーとして活動。また、2017年1月より同じくアース・スター ドリームの[[高尾奏音]]とグループ内ユニット「[[ひかのん]]」を結成している。グループ内でのイメージカラーは[[黄色]]。

2018年3月に退所を発表し、3月末日で退所予定<ref>{{Cite web |date= 2018-03-02|url= http://www.earthstar.jp/talent/news.php#201803021800-1|title= 重要なお知らせ|work= 所属声優・タレント - ニュース|publisher= アース・スター エンターテイメント|accessdate=2018-03-08}}</ref>。

== 人物 ==
好きなアニメは『[[銀魂 (アニメ)|銀魂]]』で、好きな声優は[[小松未可子]]。小松との共演を目標として挙げている<ref>{{Cite web |url= http://www.charapedia.jp/article/show/9421/|title= 【キャラペディアンサー】『アース・スター ドリーム』黄色担当「小出ひかる」さんの場合。|publisher= キャラペディア|accessdate=2017-02-20}}</ref>。

猫好きである。
