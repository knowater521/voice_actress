'''小坂井 祐莉絵'''（こざかい ゆりえ）は、[[日本]]の[[声優]]。[[愛知県]][[東浦町]]出身。[[クロコダイル (声優事務所)|クロコダイル]]所属<ref name="prof" />。

東浦超観光大使であり、知多半島のご当地キャラクター「知多娘。」の2代目・東浦未来の声優としても活動<ref>{{Cite web |date=2018-02-08|url=https://deview.co.jp/News?am_article_id=2105450|title= 愛知県・知多半島のご当地萌えキャラ「知多娘。」声優オーディション　知多半島を守る完全武装のアンドロイド「大田メディ」役を募集 |publisher= Deview|accessdate=2018-02-20}}</ref>。

== 略歴 ==
2012年、『知多娘。ご当地アイドル声優オーディション』に東浦未来役に合格し、声優活動を開始<ref>{{Cite web |date=2014-02-18|url= http://kamiya-a.cocolog-nifty.com/turezure/2014/02/web-tv-2-b742.html|title= 知多娘。ご当地アイドル声優オーディション開催！|publisher= 知多娘。公式WEBサイト|accessdate=2018-02-20|archive url=https://web.archive.org/web/20120506085415/http://www.chita-musume.com/blog/2012/05/audition/|archive date=2012-06-02}}</ref><ref>{{Cite web |date=2012-12-30|url= http://dengekionline.com/elem/000/000/579/579648/index-2.html|title= 夢は努力してつかむもの――愛知県知多半島のご当地声優“知多娘。”による夢の東京ライブを徹底レポート！|publisher= 電撃オンライン|accessdate=2018-02-20}}</ref>。2014年、[[東浦町]]の『東浦”超”観光大使』に任命される<ref>{{Cite web |date=2014-02-18|url= http://kamiya-a.cocolog-nifty.com/turezure/2014/02/web-tv-2-b742.html|title= 東浦町 WEB-TV 東うらやまシティ 第2回|publisher= つれづれログ|accessdate=2018-02-20}}</ref>。2017年2月19日、[[クロコダイル (声優事務所)|クロコダイル]]所属となる<ref>{{Cite web |date=2017-04-04|url= http://www.town.aichi-higashiura.lg.jp/soshiki/kohojoho/kohotokei/gyomu/koho/topics/h28/1488793567057.html|title= 平成28年度3月のトピックス|publisher= 東浦町ホームページ|accessdate=2018-02-20}}</ref><ref>{{Cite web |date=2017-02-19|url= https://www.crocodile-ltd.com/single-post/2017/02/19/%25E6%2589%2580%25E5%25B1%259E%25E3%2581%25AE%25E3%2581%258A%25E7%259F%25A5%25E3%2582%2589%25E3%2581%259B|title= 所属のお知らせ|publisher= 声優事務所 クロコダイル|accessdate=2018-02-20}}</ref>。

大学在学中は仕事の度に上京していた<ref>{{Cite web |date=2017-03-27|url=http://kamiya-a.cocolog-nifty.com/turezure/2017/03/post-6729.html |title= 東浦未来役の小坂井祐莉絵さんが来てくれました。|publisher= つれづれログ|accessdate=2018-02-20}}</ref>。

台湾国際旅展（ITF）にて三年連続ベストパフォーマンス賞を受賞<ref>{{Cite web |url=http://anyuken.tv/ |title= 生徒名簿|publisher= 『アニュ研！ 秋葉原アニメ・アミューズメント研究所』公式サイト|accessdate=2018-02-20}}</ref>。
