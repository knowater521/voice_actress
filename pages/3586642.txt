'''江口 菜子'''（えぐち なこ、[[1993年]]<ref>{{Cite web |url=https://twitter.com/75nakoe/status/732895728809320449 |title=江口菜子 (@75nakoe) |publisher=[[Twitter]] |date=2016-05-18 |accessdate=2017-04-22}}</ref>[[5月18日]]<ref name="about">{{Cite web |url=http://7575blog.tumblr.com/about |title=About - 75blog |publisher=[[Tumblr]] |accessdate=2017-04-22}}</ref><ref name="official">{{Cite web |url=http://www.81produce.co.jp/list.cgi?lady+0413324115000 |title=江口菜子：所属俳優：81produce |publisher=[[81プロデュース]] |accessdate=2017-04-22}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]。[[81プロデュース]]所属。[[福岡県]]出身<ref name="about" /><ref name="official" />。

== 経歴 ==
小さい頃からアニメや漫画が好きで、小学生の時に複数の少女漫画雑誌の漫画コンテストに応募するなど作る側への憧れを持っていた<ref name="2017.5">「超！アニ」連載「お前は誰だ？」vol.4、『[[アニメディア]]』2017年5月号 (2017年4月10日)、pp.152-153、[[学研プラス]]</ref>。

高校生の時に[[VOCALOID]]に夢中になり、声の仕事に興味を持つようになった<ref name="2017.5" />。また、'''ひななた'''名義で音楽活動も行っていた<ref>{{Cite web |url=http://i-style.surpara.com/blog/archives/2105 |title=ラピス＆江口菜子が歌う「わーなびっ.jk」主題歌集CDのクロスフェードを公開 |publisher=VOCALOID3 i-style Project |date=2015-10-02 |accessdate=2017-04-22}}</ref>。

18歳の時<ref>{{Cite web |url=http://www.dtmm.co.jp/item/877 |title=4月6日発売！VOCALOID3「蒼姫ラピス」 |publisher=[[DTM magazine]] |date=2012-03-26 |accessdate=2017-04-22}}</ref>、
就職活動と並行して受けた「[[VOCALOID#ヤマハ|VOCALOID3 蒼姫ラピス]]」の歌声を決める一般公募オーディションに合格<ref name="2017.5" />。

社会人になった後も蒼姫ラピスの思い出が残り続け、次第に声の仕事への思いも強くなり、作品づくりに携わりたいという思いから声優を目指すことを決めた<ref name="2017.5" />。

『[[レゴ ニンジャゴー (テレビアニメ)|レゴ ニンジャゴー]]』のキャシー役、メイドA役でテレビアニメ作品に初出演<ref>{{Cite web |url=http://7575blog.tumblr.com/post/121437794469/ |title=最近のこと☆2 - 75blog |publisher=[[Tumblr]] |date=2015-06-14 |accessdate=2017-04-22}}</ref>。

2016年4月1日付で81プロデュースにジュニア所属として加入<ref>{{Cite web |url=http://www.81produce.co.jp/news/ |title=ニュースバックナンバー：81produce |publisher=[[81プロデュース]] |date=2016-04-01 |accessdate=2017-06-02}}</ref>。

== 人物 ==
趣味・特技は[[散歩]]、[[絵画|絵]]（[[絵描き歌]]づくり）<ref>{{Cite web |url=http://www.nicovideo.jp/watch/1492743245 |title=【新人声優動画】江口菜子さんコメント＆特技披露！【アニメディア連動企画「お前は誰だ？」】 |publisher=超！アニメディア - [[ニコニコチャンネル]] |date=2017-04-21 |accessdate=2017-04-22}}</ref>、[[卵焼き]]づくり、[[温泉]]巡り<ref>{{Cite web |url=http://7575blog.tumblr.com/post/168075538564/ |title=『温泉むすめ』北郷このみ役 ☆ - 75blog |publisher=[[Tumblr]] |date=2017-12-01 |accessdate=2017-12-06}}</ref>など（2017年4月時点<ref name="2017.5" />）。

好きな食べ物・飲み物は[[竹輪|ちくわ]]、[[たい焼き]]、[[ビール]]で<ref name="2017.5" /><ref name="about" />、特にビールはほぼ毎日飲んでいる<ref name="2017.5" />。

感銘を受けた本は[[中島敦]]の『狐憑』<ref name="2017.5" />。

養成所では[[落語]]も習い、普段は[[寄席]]にも通っている<ref>{{Cite web |url=http://enterstage.jp/news/2017/04/006946.html |title=声のプロによる最高のエンターテインメント！Zepp×81pro Live Vol.1「渋谷に“声”の神来たる」落語＆朗読劇レポート |publisher=エンタステージ |date=2017-04-18 |accessdate=2017-04-22}}</ref>。

声優としてのターニングポイントになった役に蒼姫ラピスと『[[アイカツスターズ!]]』の花園きららを挙げている<ref name="2017.5" />。
