'''菅 愛理'''（かん あいり、[[10月19日]]<ref name="profile" /> - ）は、[[日本]]の[[女性]][[声優]]。[[レオパード・スティール|LEOPARD STEEL]]所属<ref name="profile" />。[[福岡県]]出身<ref name="profile"/>{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。プロジェクト:声優#冒頭部分にてガイドラインが制定されていますので、そちらも参照して下さい。}}。

== 経歴 ==
子供の頃からアニメが好きで、大学時代にゼミの発表会で発表を行った際に周りから聞き取りやすかったと褒められたことをきっかけに声優の道を目指した<ref name="声グラ">{{Cite web |url=http://seigura.com/senior/training/20160414_17157/1.html|title=ただいま修行中 菅 愛理さん vol.34|accessdate=2017-10-26}}</ref>。

短大卒業後、[[代々木アニメーション学院]]福岡校で2年間学び、学内オーディションを経て、[[松濤アクターズギムナジウム]]に入所した<ref name="sagatv">{{Cite web |url=http://www.sagatv.co.jp/miranba/anime/staff/
|title=がんばれ！ミランバくん|accessdate=2017-10-26}}</ref><ref name="声グラ" />。

2017年4月1日付で[[レオパード・スティール|LEOPARD STEEL]]の所属となった<ref>{{Cite web |url=https://twitter.com/LEOPARD_STEEL/status/848118796220891137
|title=【お知らせ④】２０１７年４月１日付で・・・|accessdate=2017-10-26}}</ref>。

2017年6月より[[明神カフェ]]のキャストとして働いている<ref>{{Cite web |url=https://twitter.com/Kan_Airi/status/880791075744399360
|title=本日から #明神カフェ にてキャストとして働かせて頂くことになりました|accessdate=2017-10-26}}</ref>。

== 人物 ==
* ファッション系の短大を出ているため、服装にこだわりを持っており、[[コスプレ]]にも自信があると語っている<ref name="声グラ" />。また、洋服を作ることも特技としている<ref name="パーソナリティ">{{Cite web |url=https://twitter.com/leopard_school/status/873444921029533696|title=パーソナリティ紹介その1・・・|accessdate=2017-10-26}}</ref>。
* 趣味は水族館に行くこと<ref name="パーソナリティ" />。
* お姉さん系や少年系の役が得意だと自負している<ref name="声グラ" />。
* ラジオで共演している松元菜々海からは素朴で自分に素直、澤潟詠里からは愛すべきアホの子と評されている<ref name="ラジオ" />。
* [[沢城みゆき]]を目標とし、[[神田沙也加]]を追い越す存在になりたいと語っている<ref name="声グラ" />。
