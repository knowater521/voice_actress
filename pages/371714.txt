'''福原 香織'''（ふくはら かおり、[[1986年]][[8月11日]]<ref name="officialwebprof">{{Cite web |url=http://fukuharakaori.jp/profile.html |title=PROFILE |work=福原香織オフィシャルウェブサイト |accessdate=2017-04-01 |archiveurl=https://web.archive.org/web/20160310123944/http://fukuharakaori.jp/profile.html |archivedate=2016-03-10}}</ref><ref name="W09-1057">{{Cite web |url=http://www.vip-times.co.jp/?talent_id=W09-1057 |title=福原香織 |work=日本タレント名鑑 |publisher=VIPタイムズ社 |accessdate=2017-04-01}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[ナレーター]]<ref name="W09-1057" />。[[千葉県]]出身<ref name="officialwebprof" /><ref name="W09-1057" />。[[身長]]151cm、[[ABO式血液型|血液型]]は[[ABO式血液型|AB型]]<ref name="officialwebprof" /><ref name="W09-1057" />。[[フリーランス|フリー]]<ref name="ameblo20160928" />。以前は[[エイベックス・プランニング&デベロップメント]]に所属していた<ref name="officialwebprof" />。

代表作に『[[らき☆すた (アニメ)|らき☆すた]]』（柊つかさ）、『[[咲-Saki-]]』（天江衣）、『[[Aチャンネル]]』（百木るん）、『[[新妹魔王の契約者]]』（成瀬万理亜）などがある<ref name="W09-1057" />。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

「'''[[福原香織 #ディスコグラフィ|FUNCTION6ch]]'''」<ref>{{Cite web |url=http://function6ch.com/ |title=FUNCTION6ch 公式サイト |accessdate=2017-04-01}}</ref>、「'''[[かと*ふく]]'''」<ref>{{Cite web |url=http://katofuku.dive2ent.com/ |title=かと＊ふく 公式サイト |accessdate=2017-04-01}}</ref>として音楽活動も行っている。

== 来歴 ==
幼い頃から声優を志し、15歳で単身上京し<ref>[http://www.oricon.co.jp/special/50165/ 声優・福原香織はどのように夢を叶えたのか？ デビューから12年、仕事への向き合い方]</ref>、16歳の時に『[[ボイスニュータイプ]]』誌主催の「声優オーディションツアー」でVOICE NEW TYPE&avex mode賞を受賞し、それを機に17歳から芸能活動を始める。アニメデビューする以前は、テレビ、CMナレーション、舞台出演が主だった。エイベックスアーティストアカデミー・声優コースの第2期[[特待生]]となる。

2005年、[[テレビアニメ]]『[[うえきの法則]]』プティング役でテレビアニメデビュー<ref name="W09-1057" />し、2007年に第12回[[アニメーション神戸]]にて、主題歌賞（ラジオ関西賞）を受賞<ref>[https://web.archive.org/web/20071011200637/http://www.anime-kobe.jp/anime-kobe/ アニメーション神戸賞・主題歌賞（ラジオ関西賞）「もってけ！セーラーふく」 - アニメーション神戸]（2007年10月11日時点の[[インターネットアーカイブ|アーカイブ]]）</ref>。2009年に、第2回[[声優アワード]]にて歌唱賞を受賞<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_02 |title=第二回声優アワード受賞者 |publisher=声優アワード |accessdate=2017-04-01}}</ref>。

2012年に[[劇団ひとり]]プロデュースで、ダンスユニット[[リアルアキバボーイズ]]と共に「福原香織とRAB」としてCDデビューし、2014年12月1日より「FUNCTION6ch」に改名し、[[エイベックス]]より活動をスタートする<ref>{{Cite web|url=http://function6ch.com/|title=FUNCTION6ch|work=エイベックス公式サイト|accessdate=2015-03-19}}</ref>。[[加藤英美里]]と[[声優ユニット]]「[[かと*ふく]]」を結成。

2016年9月28日、[[エイベックス・プランニング&デベロップメント]]を退所し、フリーで活動していることを報告した<ref name="ameblo20160928">{{Cite web|date=2016-09-28|url=http://ameblo.jp/fukuharakaori/entry-12204384201.html|title=ご報告|work=福原香織オフィシャルブログ|author=|publisher=[[サイバーエージェント]]|accessdate=2016-09-30}}</ref>。

== 人物 ==
愛称は“かおりん”<ref name="yomecolle" />。水やお茶が飲めないほか、野菜や果物の大半も食べられない。だが、徐々に克服してきている<ref name="officialwebprof" />。{{要出典範囲|[[タバスコ]]を1週間で使い切るほどの|date=2017年4月}}辛い物好き<ref name="officialwebprof" />。{{要出典範囲|声優を志すきっかけとなった|date=2017年4月}}『[[美少女戦士セーラームーン]]』のファンである<ref name="officialwebprof" />。

テレビアニメ『らき☆すた』で共演した[[白石稔]]が彼女をモデルに作詞、作曲した『[[白石みのるの男のララバイ|かおりんのテーマ]]』という曲が存在する。[[中川翔子]]のファンで、「[[しょこたん☆ぶろぐ]]」に自分のことが書いてあったときはとても喜んでいた<ref>{{Cite web |author=福原香織 |date=2008-02-04 |url=http://blog.livedoor.jp/kaorin_f/archives/50909017.html |title=ギザ思い出 |work=福原香織のブログ |accessdate=2011-08-30 |archiveurl=https://web.archive.org/web/20080208234243/http://blog.livedoor.jp/kaorin_f/archives/50909017.html |archivedate=2008-02-08}}</ref>。

アロマテラピー検定2級取得<ref name="officialwebprof" /><ref>{{Cite web |author=福原香織 |date=2009-06-16 |url=http://blog.livedoor.jp/kaorin_f/archives/2009-06.html |title=合格 |work=福原香織のブログ |publisher=livedoor Blog |accessdate=2011-09-11 |archiveurl=https://web.archive.org/web/20100127100028/http://blog.livedoor.jp/kaorin_f/archives/2009-06.html |archivedate=2010-01-27}}</ref>。{{要出典範囲|高校時代は野球部の[[マネージャー]]をやっていたことがあり|date=2017年4月}}、さらには{{要出典範囲|バンドを組み[[ベース (弦楽器)|ベース]]と[[ボーカル]]を担当していた|date=2017年4月}}。また、{{要出典範囲|[[ピアノ]]を11年間習っていたり|date=2017年4月}}、{{要出典範囲|[[トランペット]]の経験もあり|date=2017年4月}}、{{要出典範囲|[[ギター]]も弾ける|date=2017年4月}}。
