'''池羽 悠'''（いけば ゆう、[[2月24日]]<ref name="profile">{{Cite web |url=http://gunbar.net/ikeba.html |title=所属事務所公式サイト内プロフィールページ|accessdate=2018-01-17}}</ref> - ）は、[[日本]]の[[女性]][[声優]]。
[[茨城県]]出身<ref name="profile" />。[[ABO式血液型|血液型]]は[[ABO式血液型|O型]]<ref name="profile" />。[[ガンバール]]に所属。

ファンの呼称は「'''悠牧民'''」<ref>{{Twitter status|ikeba_yuu|931168123549466624}}</ref><ref group="注">2017年11月16日放送の「【第3回】池羽悠のゆうゆうチャンネル」内にて決定。</ref>。

== 略歴 ==
* 2013年、プロ・フィット基礎科所属。<ref name="profile" />
* 2017年6月1日付けで[[ガンバール]]所属タレントとして活動を開始する。<ref>{{Cite web |url=http://gunbar.net/ |title=株式会社ガンバールofficialsite 2017年6月1日付けのNEWSより|accessdate=2018-01-17}}</ref>

== 人物 ==
* 実家は[[茨城県]]の[[梨]]農家である。<ref>{{Twitter status|ikeba_yuu|867961296037134336}}</ref>
* 演技をする事でそれぞれの役と一緒に生きられる事を夢に思い描き声優の仕事に就いた。<ref name="hisho3">{{Cite web |url=http://www.dreamcatch.atosuta.net/backnumber_dch03.html |title=Dream Catch 飛翔 Vol.3出演時のインタビュー|accessdate=2018-02-14}}</ref>
* [[シナモロール]]が子どもの頃から大好きで、池羽悠として仕事をする時は大抵ぬいぐるみを連れて仕事場に行っている。<ref>{{Twitter status|ikeba_yuu|938661821455597568}}</ref><ref>{{Twitter status|ikeba_yuu|953646134068563969}}</ref>
* 様々な仕事に関わっていく内、色々な事に興味が湧いてきて色々な事に挑戦したくなる等、強い好奇心と向上心を持っている。<ref name="hisho3" />
* 日々の自身にまつわる出来事に着目し、[[Twitter]]上で#1日1チャレンジとして毎日報告している。<ref>{{Twitter status|ikeba_yuu|909733225735450624}}</ref><ref>{{Cite web |url=https://gigazine.net/news/20171009-liverevolt-machiasobi19/ |title=オタ芸OKで観客とアーティストが一緒にライブを作り出す「ライブレボルト」とは何なのか？ - GIGAZINE|accessdate=2018-03-12}}</ref>
