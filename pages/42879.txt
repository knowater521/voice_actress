'''石田 ゆり子'''（いしだ ゆりこ、[[1969年]][[10月3日]]<ref name="profile">{{Cite web |url=http://www.yuriko-ishida.com/profile/ |title=公式プロフィール |work=Yuriko Ishida Official Homepage |accessdate=2017-01-03}}</ref> - ）は[[日本]]の[[俳優|女優]]・[[エッセイスト]]。本名・'''石田 百合子'''（いしだゆりこ）<ref name="profile" />。愛称は「りり」<ref name="ishida">[[#石田 2018|石田 2018]] 154頁</ref>「ゆりちゃん」<ref name="ishida" />「ゆり」<ref name="ishida" />「ゆりっぺ」<ref name="ishida" />。
== 来歴・人物 ==
[[東京都]]出身<ref name="profile" />。{{要出典範囲|[[東京都立桜町高等学校]]|date=2016年1月}}。[[女子栄養大学短期大学部]]卒業<ref>{{Cite web |url=https://www.minkou.jp/university/school/senior/20130/ |title=女子栄養大学 出身の有名人 |publisher=みんなの大学情報 |accessdate=2017-11-09}}</ref>。

{{要出典範囲|愛知県名古屋市にて出生後すぐに東京で生活。その後も父親の仕事の都合で（[[東京都]]→[[兵庫県]][[西宮市]]→[[神奈川県]][[川崎市]]→[[台湾]]）などと転居し15歳の時に東京に戻る|date=2017年1月}}。小学校の時に水泳（平泳ぎ）で[[ジュニアオリンピック]]に出場し第8位の記録を持つ<ref>{{cite web|url=http://www.excite.co.jp/News/entertainment_g/20150209/Techinsight_20150209_103480.html|title=石田ゆり子はバラエティ向き？　生番組に元カレから投書が来たと勘違い。「え！」と目を剥く。|publisher=エキサイトニュース|date=2015-02-13|accessdate=2017-03-31}}</ref>からか「（私の）どこかに『スポ根』があるんですよ」と発言<ref>ぴったんこカン・カン 「石田ゆり子＆安住アナが今夜だけのスペシャル恋ダンスに挑戦！」 2016年12月9日{{出典無効|title=検証できません。|date=2017年1月}}</ref>{{出典無効|title=検証できません。|date=2017年1月}}。

高1の時に自由が丘・亀屋万年堂の前でスカウトされて[[モデル (職業)|モデル]]になる。1987年に全日空の沖縄キャンペーンガールに選ばれてテレビCMやポスターなどに登場。1988年11月にドラマ「海の群星」に出演し女優としてデビュー。出演している映画「悲しい色やねん」に1988年12月に公開。1993年4月に連続ドラマ「彼女の嫌いな彼女」でドラマ初主演。「[[北の零年]]」にて第29回日本アカデミー賞優秀助演女優賞を受賞。

[[ボックスコーポレーション]]<ref>(株)東京ニュース通信社TVガイド臨時増刊平成7年4月10日号 フレッシュスター名鑑'95の65ページに記載</ref>に所属していたが実妹である女優・[[石田ひかり]]と共に個人事務所「風鈴舎」を1999年に設立して<ref>[http://webmagazine.gentosha.co.jp/backnumber/vol28_20010601/ishidayuriko/ishidayuriko.html Webマガジン幻冬舎: おぉ、素晴らしき日常]</ref><ref>[https://web.archive.org/web/20040901160808/http://www.prati.info/c/back/interview/62.html 東本願寺真宗会館 広報紙『サンガ』 インタビュー 石田ひかりさんに聞く]（2004年9月1日時点の[[インターネットアーカイブ|アーカイブ]]）</ref>社長兼女優として活動している。

最もジュエリーが似合う人に贈られる日本ジュエリーベストドレッサー賞を二度にわたり受賞している<ref>{{Cite web |url=https://mdpr.jp/news/detail/1658055 |title=石田ゆり子、21年ぶり2度目の快挙に感慨　変わらぬ美貌を披露 |publisher=modelpress |accessdate=2017-11-09}}</ref>。

ペットに1匹の犬<ref>{{Cite web |date= |url=http://www.1101.com/hananokoto/ |title=花のこと。 |work=ほぼ日刊イトイ新聞 |publisher= |accessdate=2016-01-09}}</ref>（名前は「雪」）と２匹の猫（名前は「たび」「はにお」）を飼っている。女優・[[板谷由夏]]の飼っている猫と兄弟である<ref>{{cite web|url=https://www.1101.com/yukichan/|title=ばあやでございます。|publisher=ほぼ日刊イトイ新聞「ゆきちゃん。」|date=2016-07-28|accessdate=2017-03-31}}</ref>。妹・石田ひかりは石田ゆり子を「りり」と呼んでいる<ref name="ishida" />。{{要出典範囲|本名の由来である花（[[ユリ]]＝英語:lily）に因むのではなく石田ひかりは「小さい頃“ゆり”って発音出来なくて“りり”って呼んでいてそのまま『りり』と呼んでいる」と述べている|date=2017年1月}}。
