'''大後 寿々花'''（おおご すずか、[[1993年]][[8月5日]] - ）は、[[日本]]の[[俳優|女優]]。本名、同じ<ref>{{Cite web |url=http://www.vip-times.co.jp/popularity/%e5%a4%a7%e5%be%8c%e3%80%80%e5%af%bf%e3%80%85%e8%8a%b1/ |title=大後 寿々花 |work=[[日本タレント名鑑]] |publisher=VIPタイムズ社 |accessdate=2016-09-29}}</ref>。

[[神奈川県]][[横浜市]]出身<ref name="Tw20130511" />。[[慶應義塾大学環境情報学部]]卒業。[[アルファエージェンシー]]所属。

== 略歴 ==
[[2000年]]（平成12年）11月、[[劇団ひまわり]]に所属し、[[明治座]]秋の演劇祭『[[国盗り物語]]』でデビュー。

[[2005年]]（平成17年）、[[行定勲]]監督の『[[北の零年]]』に出演する。また、同作品で共演した[[渡辺謙]]に推薦され映画『[[SAYURI]]』に出演。[[章子怡|章子怡（チャン・ツィイー）]]が演じるヒロインの子供時代を演じ、11歳でハリウッドデビューする<ref>{{Cite news |title=13歳大後寿々花、日テレ連ドラヒロイン |url=http://www.nikkansports.com/entertainment/p-et-tp0-20070221-159642.html |date=2007-02-21 |newspaper=nikkansports.com |publisher=日刊スポーツ新聞社 |accessdate=2014-01-31 |archiveurl=https://web.archive.org/web/20071215030258/http://www.nikkansports.com/entertainment/p-et-tp0-20070221-159642.html |archivedate=2007年12月15日 |deadlinkdate=2017年9月 }}</ref>。

[[2006年]]（平成18年）12月、劇団ひまわりを退団し、[[CATAMARAN]]に移籍<ref>{{Cite web |url=http://www.catamaran2007.co.jp/profile/ |title=[CATAMARAN] プロフィール 所属一覧 |archiveurl=https://web.archive.org/web/20130124131959/http://www.catamaran2007.co.jp/profile/ |archivedate=2013年1月24日 |accessdate=2013-06-02 |deadlinkdate=2017年9月 }}</ref>。

[[2007年]]（平成19年）4月、『[[セクシーボイスアンドロボ#テレビドラマ|セクシーボイスアンドロボ]]』で連ドラ初ヒロインを務め、[[北川昌弘]]から「主演に値する」と評せられるほどの好演を見せた<ref>[[ザテレビジョン]]第53回「ドラマアカデミー賞」助演女優賞受賞の際の講評より</ref>。

[[2010年]]（平成22年）1月、[[新春ドラマスペシャル]]『[[福助 (漫画)#テレビドラマ|福助]]』でテレビドラマ初主演。

[[2012年]]（平成24年）4月、{{要出典範囲|date=2016年9月|[[AO入試]]にて}}[[慶應義塾大学環境情報学部]]に入学<ref>オフィシャルサイト内DIARYより</ref><ref>{{Cite web |url=https://twitter.com/Ohgo_Suzuka |title=ツイッター自己紹介「慶應義塾大学在学中」  |author=[https://twitter.com/Ohgo_Suzuka 大後寿々花] |date=2013-5-11|accessdate=2013-5-11}}</ref>、[[2016年]]（平成28年）3月に卒業<ref>{{Cite web |author=大後寿々花 |date=2006-03-31 |url=http://ameblo.jp/suzukaohgo/entry-12145288390.html |title=卒業 |publisher=大後寿々花オフィシャルブログ「Suzuka's Diary Ⅱ」 |accessdate=2016-09-29}}</ref><ref>{{Cite web  |date=2016-08-12 |url=http://www.bs-asahi.co.jp/jinsei/prg_047.html |title=♯47 大後寿々花×世界遺産・高野山 |work=[[人生を変える7日旅]] |publisher=[[BS朝日]] |accessdate=2016-09-29}}</ref>。

[[2013年]]（平成25年）2月、CATAMARANから[[アルファエージェンシー]]に移籍。
