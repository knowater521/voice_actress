'''下田 麻美'''（しもだ あさみ、[[1986年]]<ref>{{Cite web|date=2012-01-30|url=http://yaplog.jp/asaponmax/archive/3358|title=ありがとうの日|work=あさぽんの小屋|author= 公式ブログ|publisher=[[yaplog!]]|accessdate=2012-10-11}}</ref>[[1月30日]]<ref name="artsvisionprof">{{Cite web |url=http://www.artsvision.co.jp/talent/813/ |title=下田麻美 |publisher=アーツビジョン |accessdate=2017-01-05}}</ref> - ）は、[[日本]]の[[女性]][[声優]]、[[舞台女優]]、[[歌手]]。[[アーツビジョン]]所属<ref name="artsvisionprof" />。[[鳥取県]][[鳥取市]]<ref>{{Twitter status|shimoda_asami|569494535799046144|下田麻美（@shimoda_asami）2015年2月22日 22:50のツイート}}</ref>出身。愛称は「あさぽん」<ref name="yaplog578" />など。
代表作に『[[IS 〈インフィニット・ストラトス〉]]』（凰鈴音）<ref name="oriconprofile">{{Cite web |url=http://www.oricon.co.jp/prof/379741/profile/ |title=下田麻美（出典：VIPタイムズ社） |work=ORICON STYLE |publisher=オリコン |accessdate=2017-01-05}}</ref>、『[[THE IDOLM@STER]]』（[[双海亜美・真美]]）<ref name="oriconprofile" />、『[[VOCALOID|VOCALOID2]]』（[[鏡音リン・レン]]）<ref name="oriconprofile" />、『[[お兄ちゃんだけど愛さえあれば関係ないよねっ]]』（猿渡銀兵衛春臣）<ref name="oriconprofile" />などがある。<!--代表作の追加・削減・変更はリンク先に変更があった場合か、信頼できる情報源に基づく出典の明記を伴った上でお願いします。-->{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
2006年まで年1回行われていた「アーツビジョン無料新人育成オーディション」に合格し、{{要出典範囲|[[日本ナレーション演技研究所#特待生オーディション合格者リスト|17期特待生]]|date=2017年1月|title=「17期」とされる信頼できる情報源からの出典をお願いいたします。}}として[[日本ナレーション演技研究所]]を経て、[[アーツビジョン]]所属となった。

2005年に稼動した業務用ゲーム『[[THE IDOLM@STER]]』の[[双海亜美]]・[[双海真美]]役で本格的にデビュー（{{要出典範囲|収録開始は2003年から|date=2017年1月}}）。

2009年6月に本人名義としては初のソロアルバム『[[Prism (鏡音リン・レンfeat. 下田麻美のアルバム)|Prism]]』をリリース。それに連動して[[ニコニコ動画]]に「下田麻美ちゃんねる」が2009年4月27日に開設された<ref>{{Cite web |url=http://ch.nicovideo.jp/ch207 |title=下田麻美ちゃんねる |work=ニコニコ動画 |publisher=DWANGO |accessdate=2017-01-05}}</ref>。

ユニット活動は、個人的な活動の「elements」（エレメンツ）が声優の[[狩野茉莉]]と2007年6月頃から2010年3月まで、2008年のテレビアニメ『[[今日の5の2]]』の出演声優によるユニット「[[Friends (声優ユニット)|Friends]]」は番組終了後の2009年10月まで。

2007年に発売された[[VOCALOID2]]『[[鏡音リン・レン]]』の声データの提供者で、キャラクターとしての鏡音リン・レンの声の他、追加ライブラリのための音声収録に参加している。

2013年7月13日、故郷である鳥取県から「とっとりふるさと大使」を委嘱された<ref>{{Cite web|work=とりネット／鳥取県公式ホームページ|url=http://www.pref.tottori.lg.jp/148973.htm|title=とっとりふるさと大使/広報課/とりネット/鳥取県公式サイト|accessdate=2015-03-09}}</ref>。

2017年12月26日、自身のオフィシャルブログで一般人男性と入籍したことを発表した。

== 人物・エピソード ==
ニックネーム「あさぽん」はラジオ番組にゲストで出演した時に、リスナーの投稿で「あさぽん」と呼ばれたことをきっかけに本人が決めた<ref name="yaplog578">{{Cite web|date=2007-05-18|url=http://yaplog.jp/asaponmax/archive/578|title=小屋!|work=あさぽんの小屋|author= 公式ブログ|publisher=[[yaplog!]]|accessdate=2012-10-11}}</ref>。他の愛称に「アーサー」{{Refnest|group="注"|声優養成所の同級生につけられた愛称<ref>『DJCD ラジオdeアイマCHU!! EXTRA』でのトークより。</ref>。主に[[長谷川明子]]がこの愛称を使っている<ref>[http://blog.livedoor.jp/onigiriwa315/archives/64924254.html 長谷川明子ブログ2008年2月12日]</ref>。}}、「げったん」<ref>[http://wakachoku.way-nifty.com/blog/2008/06/post_4dad.html 若林直美ブログ2008年6月9日]</ref>、「ぽんちゃん」<ref group="注">ニコニコ生放送の番組（番組名に変遷があるので詳細は[[#ラジオ・Web等配信番組|ラジオ・Web等配信番組]]節を参照）内で[[奥井雅美]]から、『ラジオdeアイマCHU!!』で[[仁後真耶子]]からこの愛称で呼ばれている。</ref>など。

高校生のときに声優養成所の特待生に合格し、高校在学中は住まいのある鳥取と東京を[[夜行バス]]などで週1回往復する生活をした<ref name="interview01">{{Cite web|author=稲元徹也|date=2007-07-17|url=http://trendy.nikkeibp.co.jp/article/special/20070709/1001482/|title=「とかちつくちて」と歌った理由を、声優・下田麻美さんがファイナルアンサー!?|publisher=日経トレンディネット|pages=1-2 |accessdate=2008-01-13 }}</ref>。この当時の様子を[[NHK鳥取放送局]]が取材しており、そのときの担当がアニメ好きとして知られる[[塩澤大輔]]アナウンサー（現・[[NHK仙台放送局|仙台局]]）だったことから、下田・塩澤の両者から当時のことについての言及が見られる<ref>{{Twitter status|shimoda_asami|283003658055077889|下田麻美（@shimoda_asami）2012年12月24日 9:18のツイート}}</ref><ref>{{Cite web|url=http://www.nhk.or.jp/kochi/program/blog/archives/1312.html#D24-2|title=輝け！キミノミライ|work=NHK高知放送局 アナウンサー・キャスターブログ|date=2013-12-24|accessdate=2014-05-27 |archiveurl=https://web.archive.org/web/20140527214736/http://www.nhk.or.jp/kochi/program/blog/archives/1312.html |archivedate=2014-05-27}}</ref>。

ゲーム『[[THE IDOLM@STER]]』に収録された「エージェント夜を往く」という曲で、彼女が歌うゲームバージョンは「溶かしつくして」という歌詞が「とかちつくちて」に聞こえる<ref name="interview01" /><ref name="itmedia20111123">{{Cite news|author=池谷勇人|date=2011-11-23|url=http://gadget.itmedia.co.jp/gg/articles/1111/23/news007.html|title=「公式が病気」「良い意味でおかしい」――ばんえい競馬の「アイマス」コラボがやりすぎだと話題に|work=ITmediaガジェット|publisher=[[ITmedia]]|accessdate=2012-01-07}}</ref>ことが2007年に動画サイト等で話題になる。

[[サザンオールスターズ]]や[[桑田佳祐]]のファンで<ref name="yaplog3714">{{Cite web|date=2013-03-18|url=http://yaplog.jp/asaponmax/archive/3714|title=アイアムファン|work=あさぽんの小屋|author= 公式ブログ|publisher=[[yaplog!]]|accessdate=2017-01-11}}</ref>、新宿タワーレコードのキャンペーンに行った姿が本人のブログ<ref name="yaplog3713">{{Cite web|date=2013-03-17|url=http://yaplog.jp/asaponmax/archive/3713|title=うぎゃあああああ|work=あさぽんの小屋|author= 公式ブログ|publisher=[[yaplog!]]|accessdate=2017-01-11}}</ref>だけでなく、サザンオールスターズの公式ツイッターでも一般人として紹介されている<ref name="twi">{{Twitter status|sasfannet|313213286848012289|サザンオールスターズ official（@sasfannet）2013年03月17日 18:00のツイート}}. 2017年1月11日閲覧。</ref>。また[[尾崎豊]]のファンでもある<ref>{{Cite web|date=2012-09-19|url=http://yaplog.jp/asaponmax/archive/3533|title=尾崎豊特別展|work=あさぽんの小屋|author= 公式ブログ|publisher=[[yaplog!]]|accessdate=2014-01-30}}</ref><ref>{{Cite web|date=|url=http://www.famitsu.com/news/201204/18012973.html|title=ファーストシングル「awake／もう一度」のリリースが決定した声優・下田麻美にインタビュー！|publisher=ファミ通.com|accessdate=2014-01-30}}</ref>。
