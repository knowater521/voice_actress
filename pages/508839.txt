'''壁谷 明音'''（かべや あかね、[[1990年]][[11月25日]] - ）は、[[日本]]の[[ファッションモデル]]、[[歌手]]、[[俳優|女優]]。[[愛知県]]出身。大学の[[社会学部]]卒業<ref>{{Cite web |author=壁谷明音 |url=http://ameblo.jp/a-a-a-x/entry-10440106742.html |title=Questhon2  |accessdate=2017-3-31 |date=2010-1-22 |work=オフィシャルブログ「Are you Happy ?」}}</ref><ref>{{Cite web |author=壁谷明音 |url=http://ameblo.jp/a-a-a-x/entry-10623202297.html |title=Answer2♢ |accessdate=2017-3-31 |date=2010-8-18 |work=オフィシャルブログ「Are you Happy ?」}}</ref><ref>{{Cite web |author=壁谷明音 |url=http://ameblo.jp/a-a-a-x/entry-11155443365.html |title=Answer★ |accessdate=2017-3-31 |date=2012-2-4 |work=オフィシャルブログ「Are you Happy ?」}}</ref><ref>{{Cite web |author=壁谷明音 |url=http://ameblo.jp/a-a-a-x/entry-11498070640.html |title=無事に4年間通った大学を卒業してきました!! |accessdate=2017-3-31 |date=2013-3-25 |work=オフィシャルブログ「Are you Happy ?」 }}</ref>。

== 略歴 ==
[[2002年]]、[[ファッション雑誌]]「[[ピチレモン]]」（[[学研ホールディングス|学研]]）の専属モデルオーディションでグランプリを受賞<ref name="tam">{{Cite web |url=http://www.tambourineartists.com/md_kabeya.html |title=壁谷明音 公式プロフィール |accessdate=2017-3-31 |date=2010-3-8 |work=タンバリンアーティスツ |publisher= |archiveurl=https://web.archive.org/web/20100308142914/http://www.tambourineartists.com/md_kabeya.html |archivedate=2010-3-8 }}</ref>。同年5月から[[2007年]]8月号まで専属モデルとして活動。人気モデルの1人で表紙起用は歴代モデルの中でトップで、その内2回が単独での起用であった。

[[2003年]]に'''[[ピポ☆エンジェルズ]]'''の一員として歌手デビューを果たした<ref name="tam"/>。詳しくは[[ピポ☆エンジェルズ]]の項を参照。

[[2007年]][[2月]]、[[ミランカ]]より配信されるWEBシネマ「[[リアルシスター]]」で女優デビューを果たす<ref name="tam"/>。

[[2010年]][[1月18日]]に[[タンバリンアーティスツ]]との契約を終了。

[[2014年]]12月、18歳から務めていたMERCURYDUOを退職<ref>{{Cite web |author=壁谷明音 |url=http://ameblo.jp/a-a-a-x/entry-11959760974.html |title=♡:最後のご挨拶 |accessdate=2017-3-31 |date=2014-12-2 |work=オフィシャルブログ「Are you Happy ?」}}</ref>。

2014年に入籍、2015年に男児を出産した<ref>{{Cite web |author=壁谷明音 |url=http://ameblo.jp/ahh-o/day-20150601.html |title=♡:ご挨拶とご報告 |accessdate=2017-4-1 |date=2015-6-1 |work=PrivateBlog Ahh♡ }}</ref>。
