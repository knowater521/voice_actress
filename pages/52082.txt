== 来歴 ==
=== 生誕からデビューまで ===
[[広島女学院中学校・高等学校]]、[[慶應義塾大学環境情報学部]]卒業<ref>{{Cite web|author=松来未祐|date=2008-09-11|url=http://blog.excite.co.jp/matsukimiyu/8593812/|title=学園祭出演☆|work=松来未祐日記|accessdate=2015-11-03}}<br />{{Cite web |date= |url=http://www.sossfc.net/miyumiyu/index.html |title=松来未祐トークイベント〜みゆみゆ先生の特別授業 in SFC〜 |work=SFC CLIP（学生によるサイト） |publisher=慶應義塾大学湘南藤沢キャンパス |accessdate=2010-12-13 |archiveurl=https://web.archive.org/web/20101004132821/http://www.sossfc.net/miyumiyu/index.html |archivedate=2010年10月4日 |deadlinkdate=2017年10月 }}<br />{{Cite web |url=http://sfcclip.net/message2008100301/ |title=秋祭での松来未祐トークイベントについて |work=SOS@SFC（学生によるニュースサイト） |publisher=慶應義塾大学湘南藤沢キャンパス |date=2008-10-03 |accessdate=2015-11-03 }}</ref>。

声優の勉強をするために上京することを両親から猛反対されたが、大学進学を条件として上京を許されたため、[[慶應義塾大学環境情報学部]]に合格。大学に通う傍ら、[[日本ナレーション演技研究所]]や[[アクセント (芸能プロダクション)|アクセント]]付属養成所シャインに通い、演技の勉強をする。

本名のまま、[[1998年]]に[[PlayStation (ゲーム機)|PlayStation]]用ゲーム『[[御神楽少女探偵団]]』の檜垣千鶴役で声優デビュー。同作以外にもしばらくは本名で活動するが、後に本名だと芸能人としては画数が良くないからとの理由で「松来未祐」へ改名する。

=== 声優としてのキャリア ===
[[2000年]]、[[ケイエスエス]]による新人オーディション企画『あっぷ² MY GIRL 2000』に合格し、ユニット「あっぷ²（あっぷあっぷ）」として様々な活動を行う。また、同社のグループ「西五反田フォークキャンプ」のメンバーとして「ちいさな翼」などの楽曲を発表する（同曲は後に「Memories〜Miyu Matsuki Best Songs〜」に収録された）。

[[2002年]]、『[[七人のナナ]]』で初のアニメレギュラーを獲得。同年6月、アクセントから[[81プロデュース]]へ移籍する。同年10月より始まったラジオ番組『[[(有)櫻井工房|（有）櫻井工房]]』は『[[(有)チェリーベル|（有）チェリーベル]]』へ続き長期番組となった。

[[2003年]]から[[2004年]]にかけては『[[D.C. 〜ダ・カーポ〜 (アニメ)|D.C. 〜ダ・カーポ〜]]』『[[ダイバージェンス・イヴ]]』シリーズでアニメレギュラーを獲得し、2004年1月の『[[変身3部作|超変身コス∞プレイヤー]]』で初の主演を務める。この年、ラジオ番組『RADIOデコピンないと』で共演した[[金田朋子]]と[[声優ユニット]]「[[松来未祐と金田朋子のRADIOデコピンないと2#SD☆Children|SD☆Children]]」を組み、CDリリースや各種イベントなどの活動を行った。その他にも多数のアニメに出演すると共に、最多で週4本のラジオ番組を持っていた。また、『[[さよなら絶望先生 (アニメ)|さよなら絶望先生]]』シリーズ（藤吉晴美役）や『[[ひだまりスケッチ (アニメ)|ひだまりスケッチ]]』シリーズ（吉野屋先生役）など、[[新房昭之]]が監督を務める[[シャフト (アニメ制作会社)|シャフト]]制作作品の常連声優の1人となっていた。

[[2010年]]より自主活動[[声優ユニット]]の「[[N±]]」を結成した。

2012年の『[[這いよれ! ニャル子さん]]』（クー子役）では、声優ユニット「[[後ろから這いより隊|後ろから這いより隊G]]」の一員としてオープニングテーマを歌唱。[[2013年]][[3月1日]]、「後ろから這いより隊G」の一員として第7回[[声優アワード]]歌唱賞を受賞した<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_07 |title=第七回声優アワード受賞者 |publisher=声優アワード |accessdate=2015-11-03}}</ref>。

=== 病気療養の背景 ===
2013年頃より夜中に39℃の熱を出しては翌朝には下がるなどの症状を訴え始め、その後も同様の症状が続いたことから両親に体調不良を訴える。1週間経つと症状はなくなっていた。2015年の正月に広島の実家に帰省した際には、首のリンパ節が腫れ疲労も訴えた。その後も体調が回復せずに深刻な体調不良が続いたため、複数の大きな病院で何度も検査を受け、その際にガン検診も複数回受けたが、異常は見当たらず原因不明であった（この頃、過去に[[自己免疫疾患]]で入退院を繰り返していた[[後藤邑子]]に病状の件で相談していたという<ref>{{Cite web |url=http://ameblo.jp/goto-yuko/entry-12091826782.html |title=君が好きだと叫びたい！てゆうか何度も叫んだ！ |work=後藤邑子オフィシャルブログ |publisher=アメーバ|date=2015-11-04 |accessdate=2016-02-09}}</ref>）。松来自身も、仕事を減らさざるを得ないことになる事を危惧していたため、事務所にも入院の直前まで伝えていなかったという。

後述のEBウイルス感染症の疑いが浮上したのは同年6月に入ってからで、7月2日に改めて精密検査を受ける予定であったが、6月30日に咳が止まらず呼吸困難に陥り、救急車に電話し病院に搬送された。その後、急性[[肺炎]]で命の危機状態に陥りそのまま入院、本格的な療養生活に突入した。それに伴い、7月4日に出演予定だった「[[蒼穹のファフナー]]」初回放送記念イベントを欠席することを、7月1日公式サイト、ブログで発表された。[[7月14日]]に自らのブログで「今年に入って熱が下がらず、病院で検査を繰り返していたのですが、急性の肺炎で検査を重ねたところで、少しでも早く治療に取り組まないといけない病気であることが分かりました」と語り、活動休止を発表した<ref>{{Cite news |url=http://www.sponichi.co.jp/entertainment/news/2015/07/14/kiji/K20150714010733950.html |title=声優の松来未祐 病気治療のため活動休止…肺炎の検査から判明 |newspaper=スポニチ Sponichi Annex |publisher=スポーツニッポン新聞社 |date=2015-07-14 |accessdate=2015-07-14}}</ref><ref>なお、活動休止の発表時点で放送されていた7月期のテレビアニメの役については、休養前にすべての収録を済ませていたため、そのまま放送された。</ref>。

=== 死去 ===
入院後は抗がん剤投与などの治療を行い、病状が改善されたこともあり、[[9月4日]]に一旦退院。[[9月14日]]には極々身内の者と共に誕生パーティを開いたことをブログ上にアップし回復ぶりをアピールしたが<ref>{{Cite web |url=http://blog.excite.co.jp/matsukimiyu/22169446/|title=ありがとう|work=松来未祐日記|publisher=exciteブログ|date=2015-09-14 |accessdate=2017-06-03}}</ref>、[[9月18日]]には容体が急激に悪化し再入院。この時には既に末期の[[悪性リンパ腫]]を罹患していたことも判明、同時に[[クモ膜下出血]]も併発して緊急手術を受けるなど、益々病状は悪化していった。だが、亡くなる直前まで意識があり、言葉を発することはできなかったが、問いかけには応じて頷いていたという<ref>{{Cite web |author=81プロデュース スタッフ一同 |date=2015-12-15 |url=http://blog.excite.co.jp/matsukimiyu/22662440/ |title=松来未祐を応援してくださった皆様へ |work=松来未祐日記 |accessdate=2015-12-15}}</ref><ref>{{Cite news |url=http://www.sponichi.co.jp/entertainment/news/2015/12/15/kiji/K20151215011690890.html |title=声優の松来未祐さん 難病だった…遺族が同じ病気の早期発見願い公表 |newspaper=[[スポニチアネックス]] Sponichi Annex |publisher=[[スポーツニッポン新聞社]] |date=2015-12-15 |accessdate=2015-12-15}}</ref><ref>{{Cite news |url=http://www.oricon.co.jp/news/2063888/full/ |title=松来未祐さん、家族の意向で病名公表 死因は「悪性リンパ腫」 |newspaper=オリコンスタイル ORICON STYLE |publisher=[[オリコン]] |date=2015-12-15 |accessdate=2015-12-15}}</ref>。

2015年10月27日午後10時18分、松来は入院先の東京都内の病院で死去。享年38。通夜、葬儀、告別式は、遺族の意向により家族と親族のみで執り行われた<ref name="ORICON2061682"/><ref>{{Cite web |author=81プロデュース スタッフ一同 |date=2015-11-02 |url=http://blog.excite.co.jp/matsukimiyu/22444146/ |title=皆様へ |work=松来未祐日記 |accessdate=2015-11-02}}</ref>。その後、四十九日が経過した同年[[12月15日]]に遺族から病名公表の要望があったことから（松来自身の生前の意向でもあった）、難病に近い非常に希少な病気である[[慢性活動性EBウイルス感染症]]（CAEBV）に感染していたこと、最終的な死因は[[悪性リンパ腫]]であったことが81プロデュースによって初めて公表された。<ref>{{Cite web |author=81プロデュース スタッフ一同 |date=2015-12-15 |url=http://blog.excite.co.jp/matsukimiyu/22662440/ |title=松来未祐を応援してくださった皆様へ |work=松来未祐日記 |accessdate=2015-12-15}}</ref><ref>{{Cite news |url=http://www.sponichi.co.jp/entertainment/news/2015/12/15/kiji/K20151215011690890.html |title=声優の松来未祐さん 難病だった…遺族が同じ病気の早期発見願い公表 |newspaper=[[スポニチアネックス]] Sponichi Annex |publisher=[[スポーツニッポン新聞社]] |date=2015-12-15 |accessdate=2015-12-15}}</ref><ref>{{Cite news |url=http://www.oricon.co.jp/news/2063888/full/ |title=松来未祐さん、家族の意向で病名公表 死因は「悪性リンパ腫」 |newspaper=オリコンスタイル ORICON STYLE |publisher=[[オリコン]] |date=2015-12-15 |accessdate=2015-12-15}}</ref>

2015年11月2日に公表された訃報を受け、声優仲間をはじめとして親交の深かった関係者たちがTwitterやブログなどを通じて松来への哀惜の言葉を綴り<ref>{{Cite news |url=http://www.tokyo-sports.co.jp/entame/anime/467169/ |title=松来未祐さん死去に植田佳奈、緒方恵美ら声優仲間が悲しみ |newspaper=東スポWeb |publisher=東京スポーツ新聞社 |date=2015-11-02 |accessdate=2015-11-03}}</ref><ref>{{Cite news |url=http://www.sponichi.co.jp/entertainment/news/2015/11/02/kiji/K20151102011435550.html |title=本名陽子 松来未祐さん訃報に沈痛「信じたくない」声優仲間が追悼 |newspaper=スポニチ Sponichi Annex |publisher=スポーツニッポン新聞社 |date=2015-11-02 |accessdate=2015-11-03}}</ref><ref>{{Cite web |url=http://www.crank-in.net/game_animation/news/39681 |title=声優・松来未祐さん死去 吉田尚記アナ、植田佳奈ほか早すぎる死に悲しみの声 |work=クランクイン！ |publisher=ハリウッドチャンネル |date=2015-11-02 |accessdate=2015-11-03}}</ref>、ファンからも急逝を悼む書き込みが見られたほか、Yahoo! JAPANのリアルタイム検索も一時的に上位20ワードが松来やその出演作に関するもので占められた<ref>{{Cite web |url=http://www.oricon.co.jp/news/2061689/full/ |title=声優・松来未祐さん死去「若すぎるよ」 追悼の声続々 |work=ORICON STYLE |publisher=オリコン |date=2015-11-02 |accessdate=2015-11-03}}</ref><ref name="tospo">{{Cite news |url=http://www.tokyo-sports.co.jp/entame/anime/467236/ |title=松来未祐さん死去 最後のブログで「この病気が治ったら本気で婚活するんだ…！」 |newspaper=東スポWeb |publisher=東京スポーツ新聞社 |date=2015-11-02 |accessdate=2015-11-03}}</ref>。同年11月19日には、[[超A&G+]]で特別番組『[[超!A&G+スペシャル|松来未祐〜メモリーズ〜]]』が配信された<ref>{{Cite web |date=2015-11-13 |url=http://www.agqr.jp/topics/archives/post_556.php |title=阿澄さん、神田さんも出演「松来未祐〜メモリーズ」 |publisher=[[超!A&G+]] |accessdate=2015-11-21}}</ref>。また、松来の休養後も復帰を待つ形で放送を続けていた『[[(有)チェリーベル|（有）チェリーベル]]』も、松来の急逝を受けて2015年11月6日をもって終了した<ref>{{Cite web |url=http://www.sponichi.co.jp/entertainment/news/2015/11/06/kiji/K20151106011462990.html|title=松来未祐さん出演ラジオ12年の歴史に幕「3人だからこそチェリーベル」|accessdate=2015-11-07}}</ref>。

=== 死後 ===
2016年9月7日、超!A&G+ 文化放送モバイルplus presents 生放送！で、『文化放送モバイルplus presents サンキュー！ 未祐ちゃんの生放送！』を配信し、親交が深かった神田朱未、儀武ゆう子、大坪由佳がイベント告知や病気の事や松来とのエピソードを語る。9月11日に東京・科学技術省サイエンスホールにて開催される松来未祐愛悼イベント 「サンキュー！未祐ちゃん」を開催<ref>{{Cite web |date=2016-9-5 |url=http://www.agqr.jp/topics/archives/_plus_presents_3.php |title=サンキュー！ 未祐ちゃんの生放送！ |publisher=[[超!A&G+]] |accessdate=2016-9-8}}</ref><ref>{{Cite web |date=2016-7-9 |url=http://39miyu-chan.official.jp/ |title=松来未祐さん愛悼イベント 39!未祐ちゃん |publisher=81プロデュース |accessdate=2016-9-8}}</ref>。15日、ゲーム『[[ペルソナ5]]』（御船千早役で出演<ref>{{Cite web |url=https://www.youtube.com/watch?v=xUhAf2DuHGA|title=『ペルソナ5』Cooperation キャラクター「御船 千早」|accessdate=2016-7-19|author=アトラス|date=2016-6-15|work=YouTube}}</ref>）が発売。

2017年5月9日放送の『[[ザ!世界仰天ニュース]]』の特集「世界が知らない恐ろしい病スペシャル」で「病名が分からない苦しみと闘った女性」として松来の闘病生活が取り上げられた<ref>{{Cite web |url=http://www.ntv.co.jp/gyoten/backnumber/article/20170509_02.html |title=病名が分からない苦しみと闘った女性 |work=ザ！世界仰天ニュース |publisher=[[日本テレビ放送網]] |accessdate=2017-05-09}}</ref><ref>{{Cite news |url=http://nlab.itmedia.co.jp/nl/articles/1705/03/news035.html |title=稀有な病「CAEBV」で逝去した声優・松来未祐さん　「世界仰天ニュース」で闘病生活を特集 |newspaper=ねとらぼ |publisher=[[アイティメディア]] |date=2017-05-03 |accessdate=2017-05-03}}</ref>。

== 人物 ==
広島県出身ということもあり、『[[たまゆら (アニメ)|たまゆら]]』シリーズの八色ちもや『[[咲-Saki-]]』の佐々野いちごなどのように[[広島弁]]を使うキャラクターに採用されることがあった。また[[白石涼子]]が『咲-Saki-』で広島弁を使う染谷まこ役を演じた際には個人的に方言指導を行った<ref>{{Twitter status|RyokoShiraishi_|661068826272862208}}</ref>。

食べることが好きで、ブログではその日食べたものの写真を掲載することが多く、病にかかって療養生活に入るまでも食欲はあったという。｢ポン太｣と｢チー｣という2匹の猫を飼っている愛猫家でこの2匹の様子や写真を掲載することも多く、亡くなる直前までつづられていた<ref name=nikki20150907>{{Cite web |url=http://blog.excite.co.jp/matsukimiyu/22122970/ |title=9月！？ |accessdate=2016-11-9 |author=松来未祐 |date=2015-9-7 |work=松来未祐日記}}</ref>。松来の療養生活以降は友人に引き取られ、友人の手で育てられている<ref name=nikki20150714>{{Cite web |url=http://blog.excite.co.jp/matsukimiyu/21948132/ |title=みなさまへ |accessdate=2016-11-9 |author=松来未祐 |date=2015-7-14 |work=松来未祐日記}}</ref><ref name=nikki20150907/>。

地元が広島出身であることから、[[サンフレッチェ広島]]や[[広島東洋カープ]]のファンであった。サンフレッチェは亡くなった年の12月に[[2015年のJリーグチャンピオンシップ|チャンピオンシップ]]を制し、J1優勝を果たしている。カープは前述の追悼イベントが行われる前日の2016年9月10日に、対[[読売ジャイアンツ]]戦（[[東京ドーム]]）に勝利し、[[1991年]]以来25年ぶりに[[セントラル・リーグ|セ・リーグ]]優勝を果たしている<ref name=nikki20150504>{{Cite web |url=http://blog.excite.co.jp/matsukimiyu/21760760/ |title=☆幸腹、参加！！☆ |accessdate=2016-11-9 |author=松来未祐 |date=2015-5-4 |work=松来未祐日記}}</ref>。

『ザ!世界仰天ニュース』では、声優として活動して以降、毎年父親の誕生日にネクタイをプレゼントしていた事が明らかになり、亡くなる直前、父は「早く元気になって、来年のネクタイ買ってくれよ」と頼んでいたという。
