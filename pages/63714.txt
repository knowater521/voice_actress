== 概要 ==
=== 活動内容 ===
; 声優
: 声質は[[メゾソプラノ]]<ref name="haikyoprof" />。基本的に大人の女性を演じることが多く、母親役を演じる機会も頻繁である。それ以外にも脱力系や少年、猫まで演技の幅は広い。{{要出典範囲|キャラ設定や性格をイメージしたうえで、自然に出てきた声で芝居しているという|date=2017年4月}}。
; ナレーター
: ナレーションでは、アニメで演じているような個性を極力抑え、地声よりやや低いトーンで話す。番組やCMのナレーションを多数担当しており、さらには携帯電話[[Au (携帯電話)|au]]などの音声案内、[[京王電鉄|京王]]・[[京浜急行電鉄|京急]]・[[相模鉄道|相鉄]]などの関東私鉄の駅構内アナウンスや店内アナウンスなど、様々な媒体に大原の音声が起用されている。特に関東圏における鉄道アナウンス音源の使用率は、2010年現在少なくとも5割を占めているとされる<ref>{{Cite web |author=大原さやか |date=2010-05-07 |url=http://www.sayaka-ohara.jp/nisshi/2010/05/100507.htm |title=2010.5.7 ラジオ収録とナレーションな日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813233702/http://www.sayaka-ohara.jp/nisshi/2010/05/100507.htm |archivedate=2014-08-13}}</ref>。
: 2000年代前半は、ナレーターから格上げで、番組内でのリポーターやアシスタントとして映像出演することがあった。
; DJ・ラジオパーソナリティ
: [[ベイエフエム|bayfm]]の帯番組『[[NOW HITS STREET]]』を1999年から現在まで[[ディスクジョッキー#ディスクジョッキー（司会者）|DJ]]を担当し、毎回多種多様な音楽アーティストの楽曲の解説をしている他、アーティスト本人へのインタビューを行っている。
: [[アニラジ]]では[[ラジオパーソナリティ]]を務めることがある。
: かつて、[[広橋涼]]と共に担当した『[[インターネットラジオ カレイドスター そらとレイラの すごい ○○|すごらじ]]』は、{{要出典範囲|『[[うたわれるものらじお]]』開始までは音泉の各番組の中で最もアクセス数（＝聴取率）の高い番組であった|date=2017年4月}}。

=== 来歴 ===
* 中高6年間は演劇部に所属し<ref>{{Cite web |url=http://king-cr.jp/special/tonagra/taidan02_.html |title=となグラ! キャストアンケートその1 |work=となグラ! |publisher=キングレコード |accessdate=2016-08-16}}</ref>、おこづかいをつぎ込んでミュージカルを観に行くなどしていた<ref>{{Cite web |author=大原さやか |date=2003-11-24 |url=http://www.sayaka-ohara.jp/nisshi/2003/11/031124.htm |title=2003.11.24 「オケピ！」をみた。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032535/http://www.sayaka-ohara.jp/nisshi/2003/11/031124.htm |archivedate=2016-08-15}}</ref>。{{要出典範囲|子供の頃から芝居が好きであったが、「声優」という職業を意識し始めたのは、中学生か高校生になった頃で、CDドラマなどを聴かせてくれた弟の影響があるとのこと|date=2017年4月}}。
* {{要出典範囲|[[青山学院大学]][[文学部]][[英米文学科]]|date=2017年4月}}に入学。当時は[[オーケストラ|管弦楽団]]に所属していた。
* 1997年、{{要出典範囲|大学3年生の時に|date=2017年4月}}、[[俳協ボイスアクターズスタジオ]]に第11期生として入所した。{{要出典範囲|同期には[[山岸功]]が在籍していた|date=2017年4月}}。{{要出典範囲|声優の勉強と並行して就職活動も行っていたが、就職活動をする自分に違和感を覚えるようになり、声優にかけようと思ったとのこと|date=2017年4月}}。{{要出典範囲|声優を目指すことについて、両親（特に父親）に反対されて喧嘩をしていたが、最後は土下座をして許して貰った|date=2015年10月25日 (日) 15:47 (UTC)}}。
* {{要出典範囲|1998年|date=2017年4月}}、{{要出典範囲|大学・俳協ボイスアクターズスタジオ卒業後、[[東京俳優生活協同組合]]に所属となり、声優デビュー|date=2017年4月}}。デビュー前後には、舞台出演を経験している。しかし、すぐに仕事は来なかったため、オーディションを受けつつ、アルバイトをこなしながら、友人と劇団を立ち上げて芝居をしていた<ref>{{Cite web |author=大原さやか |date=2011-06-12 |url=http://www.sayaka-ohara.jp/nisshi/2011/06/110612.html |title=2011.6.12 ゲーム収録な日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813185730/http://www.sayaka-ohara.jp/nisshi/2011/06/110612.html |archivedate=2014-08-13}}</ref>。テレビアニメのデビュー作は同年放送の『[[名探偵コナン (アニメ)|名探偵コナン]]』第126・127話の白井ゆり役。翌1999年4月より、『[[NOW HITS STREET]]』（[[ベイエフエム|bayfm]]）のDJを担当している。
* デビュー後数年間はナレーション関連の仕事が主であったが、2000年代に入ってから、アニメ出演が増えていき、2003年の『[[ぽぽたん]]』の「あい」で{{要出典範囲|初めて主役を演じた|date=2017年4月}}。
* 2009年12月6日、自身の誕生日に入籍したことを、公認サイト「大原省」にて発表した。相手の男性は1歳年上の役者<ref>{{Cite web|author=大原さやか|date=2009-12-06|url=http://www.sayaka-ohara.com/saimondo/ohara/091206_oshirase_x.htm|title=いつも応援してくださっているみなさまへ|publisher=大原省|accessdate=2016-08-15|archiveurl=https://web.archive.org/web/20091208105024/http://www.sayaka-ohara.com/saimondo/ohara/091206_oshirase_x.htm|archivedate=2009年12月8日|deadlinkdate=2017年9月}}</ref>。
* [[2013年]][[3月1日]]、第7回[[声優アワード]]にて助演女優賞を受賞した<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_07 |title=第七回声優アワード受賞者 |publisher=声優アワード |accessdate=2016-08-15}}</ref>。

== 人物 ==
自他共に使われる愛称は{{要出典範囲|「'''さぁや'''」|date=2017年4月}}。他に{{要出典範囲|「'''さやさや'''」|date=2017年4月}}・{{要出典範囲|「'''さや姉'''」|date=2017年4月}}などと呼ばれることがある。

視力はあまり悪くはないが、コンタクトレンズを使用している<ref>{{Cite web |author=大原さやか |date=2009-02-23 |url=http://www.sayaka-ohara.jp/nisshi/2009/02/090223.htm |title=2009.2.23 アフレコ2本と、ラジオな日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032516/http://www.sayaka-ohara.jp/nisshi/2009/02/090223.htm |archivedate=2016-08-15}}</ref>。

[[幼児洗礼]]を受けて以来、[[キリスト教徒|クリスチャン]]（[[カトリック教会|カトリック]]）である<ref>{{Cite web |author=大原さやか |date=2006-10-02 |url=http://www.sayaka-ohara.jp/nisshi/2006/10/061002.htm |title=2006.10.2 オーディションと、「永遠の法」打ち上げパーティーな日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032521/http://www.sayaka-ohara.jp/nisshi/2006/10/061002.htm |archivedate=2016-08-15}}</ref>。

=== 特技・趣味・嗜好 ===
特技は[[チェロ]]演奏<ref name="haikyoprof" />。{{要出典範囲|母親の薦めで始めた|date=2017年4月}}。{{要出典範囲|同時に[[ピアノ]]や[[ソルフェージュ#聴音|聴音]]も習っていたが、すぐに挫折|date=2017年4月}}。{{要出典範囲|チェロを選んだ理由は人間の声にもっとも近いという音域と深い音色に魅せられたとのこと|date=2017年4月}}。大学生時代は[[管弦楽団]]に所属し、大学の管弦楽団の先輩の誘いで現在は地元・神奈川を拠点に活動しているサファリオーケストラの活動にも参加した経験があり、パーティーでBGMとしてカルテット演奏のバイトを何回かしたことがある<ref>{{Cite web |author=大原さやか |date=2007-06-30 |url=http://www.sayaka-ohara.jp/nisshi/2007/06/070630.htm |title=2007.6.30 打ち合わせな日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032516/http://www.sayaka-ohara.jp/nisshi/2007/06/070630.htm |archivedate=2016-08-15}}</ref>。{{要出典範囲|その縁で同オーケストラの公演の司会を務めたことがある|date=2017年4月}}。また、自身が声を担当した『[[テイルズ オブ リバース]]』のヒルダ・ランブリングはチェロが得意という設定である。

{{要出典範囲|声優業界きっての|date=2017年4月}}[[京都市|京都]]好きであり、暇を見つけては度々訪れている。一人で行くことが多く、朝4時起きで6時過ぎの[[のぞみ (列車)|のぞみ]]に乗って、1泊してから（次の昼に仕事が入っていたため）朝一番に帰ることもあった<ref>{{Cite web |author=大原さやか |date=2005-11-08 |url=http://www.sayaka-ohara.jp/nisshi/2005/11/051108.htm |title=2005.11.8 秋の、善峰寺へ続く道。あきれるくらいの陽射しと、別世界のような静寂。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032507/http://www.sayaka-ohara.jp/nisshi/2005/11/051108.htm |archivedate=2016-08-15}}</ref><ref>{{Cite web |author=大原さやか |date=2005-11-09 |url=http://www.sayaka-ohara.jp/nisshi/2005/11/051109.htm |title=2005.11.9 午後からアフレコがあるので、大急ぎでホテルで朝食を取り、のぞみに飛び乗った。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032453/http://www.sayaka-ohara.jp/nisshi/2005/11/051109.htm |archivedate=2016-08-15}}</ref>。

[[京都・観光文化検定]]（通称・京都検定）の2級と{{要出典範囲|3級|date=2017年4月}}に合格している<ref name="haikyoprof" />。{{要出典範囲|検定自体は2004年から始まったが、2004年は仕事の都合で、2005年はスケジュールを空けたのに申し込むのを忘れていて受験できなかった|date=2017年4月}}。{{要出典範囲|2006年は念願かなって2級と3級を受験し、ダブル合格を果たした|date=2017年4月}}。京都以外にも温泉地など、様々な土地へ出かけており、旅行自体も好きである。

[[和服]]を好んでいる。着付けが得意で、着付け講師認定準1級を所持している<ref name="ooharashouprof" />。

読書を好んでおり、年に数回、書店で本を大量に買い込むことがある<ref>{{Cite web |author=大原さやか |date=2004-05-04 |url=http://www.sayaka-ohara.jp/nisshi/2004/05/040504.htm |title=2004.5.4 とあるお店でみつけた、おもしろお菓子たち！ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813214426/http://www.sayaka-ohara.jp/nisshi/2004/05/040504.htm |archivedate=2014-08-13}}</ref>。『[[詩とメルヘン]]』を中学時代から休刊になるまで購読していた<ref>{{Cite web |author=大原さやか |date=2003-07-04 |url=http://www.sayaka-ohara.jp/nisshi/2003/07/030704.htm |title=2003.7.4 「詩とメルヘン」の最終号が届く。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032532/http://www.sayaka-ohara.jp/nisshi/2003/07/030704.htm |archivedate=2016-08-15}}</ref>。

英語が堪能で、[[実用英語技能検定|英検]]準1級を取得している<ref name="haikyoprof" />。パーソナリティを務める『[[『うみねこのなく頃に』Episode R -Radio of the golden witch-]]』では、冒頭の番組タイトルコールで、サブタイトルを流暢な発音で読み上げている。

学生時代から演劇・ミュージカル好きで、同業者が出演している舞台に足を運んでいる。『[[レ・ミゼラブル (ミュージカル)|レ・ミゼラブル]]』は中学生時代からのファン<ref>{{Cite web |author=大原さやか |date=2007-06-09 |url=http://www.sayaka-ohara.jp/nisshi/2007/06/070609.htm |title=2007.6.9 「レ・ミゼラブル」な日♪ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813232347/http://www.sayaka-ohara.jp/nisshi/2007/06/070609.htm |archivedate=2014-08-13}}</ref>で、この世界を目指すきっかけになっており<ref>{{Cite web |author=大原さやか |date=2007-01-21 |url=http://www.sayaka-ohara.jp/nisshi/2007/01/070121.htm |title=2007.1.21 アフレコな日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032440/http://www.sayaka-ohara.jp/nisshi/2007/01/070121.htm |archivedate=2016-08-15}}</ref>、[[博多]]で行われた公演へ観に行くほどの熱の入れようである<ref>{{Cite web |author=大原さやか |date=2007-10-20 |url=http://www.sayaka-ohara.jp/nisshi/2007/10/071020.htm |title=2007.10.20 博多へ！ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032458/http://www.sayaka-ohara.jp/nisshi/2007/10/071020.htm |archivedate=2016-08-15}}</ref>。

劇団の中でも[[音楽座]]のファンで、17歳の頃にファンクラブに入り、劇団の役者と友達になり、千秋楽の舞台セットの解体まで手伝っていたことがある。同劇団の作品の中でも「[[マドモアゼル モーツァルト]]」に対しての思い入れは深い<ref>{{Cite web |author=大原さやか |date=2005-08-01 |url=http://www.sayaka-ohara.jp/nisshi/2005/08/050801.htm |title=2005.8.1 音楽座…Rカンパニーのミュージカル「マドモアゼル・モーツァルト」をみにいった。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813205929/http://www.sayaka-ohara.jp/nisshi/2005/08/050801.htm |archivedate=2014-08-13}}</ref>。

=== 共演・交流関係 ===
『[[カレイドスター]]』で共演した{{要出典範囲|[[広橋涼]]とは親交が深く、大原は広橋のことを「涼さん」と呼んでいる|date=2017年4月}}。{{いつ範囲|現在|date=2017年4月}}でも{{要出典範囲|一緒に芝居を観に行ったり、食事に行く様子などが日誌に掲載されている|date=2017年4月}}。また、[[伊藤静]]とも仲が良く、{{要出典範囲|家も近所とのことだが、おおむね、酒飲み友達という|date=2017年4月}}。2009年7月26日、伊藤が行ったイベント「伊藤静 Talk&Live『あそびにおいで(*´艸`*)』」にゲスト出演した。多くの候補曲から伊藤が選んだ[[中島みゆき]]の『[[EAST ASIA#収録曲|糸]]』を伊藤と共に歌った。大原は『糸』について、中島が歌ったオリジナルのものではなく、[[Bank Band]]がカバーしたもので初めて知り、好きになったという<ref>{{Cite web |author=大原さやか |date=2009-07-26 |url=http://www.sayaka-ohara.jp/nisshi/2009/07/090726.htm |title=2009.7.26 伊藤静嬢のソロライブ「あそびにおいで」な日♪ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032441/http://www.sayaka-ohara.jp/nisshi/2009/07/090726.htm |archivedate=2016-08-15}}</ref>。

『[[XXXHOLiC]]』で共演した[[菊地美香]]とも交流が多い。菊地が出演した『[[超忍者隊イナズマ!]]』の{{要出典範囲|メイキング番組のナレーションを大原が担当|date=2017年4月}}したり、大原の好きな『レ・ミゼラブル』に菊地が出演していたりと、お互いに縁が深い。菊地から自身の[[結婚披露宴]]の司会を頼まれ、2009年12月6日（大原自身の誕生日であり、入籍した日でもある）に式を担当した<ref>{{Cite web |author=大原さやか |date=2009-12-06 |url=http://www.sayaka-ohara.jp/nisshi/2009/12/091206.htm |title=2009.12.6 菊地美香ちゃん＆岸祐二さん Wedding な日♪ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032524/http://www.sayaka-ohara.jp/nisshi/2009/12/091206.htm |archivedate=2016-08-15}}</ref>。また、2007年5月13日に『カレイドスター』でアソシエイトプロデューサーを務めていた[[池田東陽]]の人前式、披露宴の司会をしたこともあり<ref>{{Cite web |author=大原さやか |date=2007-05-13 |url=http://www.sayaka-ohara.jp/nisshi/2007/05/070513.htm |title=2007.5.13 池田東陽プロデューサーの結婚式な日！ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032441/http://www.sayaka-ohara.jp/nisshi/2007/05/070513.htm |archivedate=2016-08-15}}</ref>、友人・知人からイベントの司会を頼まれる機会が多い。

[[豊口めぐみ]]とは同じ和服・ミュージカル・演劇好きということで、収録現場で逢うと関連話（特に『レ・ミゼラブル』）で盛り上がる<ref>{{Cite web |author=大原さやか |date=2003-09-15 |url=http://www.sayaka-ohara.jp/nisshi/2003/09/030915.htm |title=2003.9.15 そうなんだよね。休日なんだよね、今日は。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813235454/http://www.sayaka-ohara.jp/nisshi/2003/09/030915.htm |archivedate=2014-08-13}}</ref><ref>{{Cite web |author=豊口めぐみ |url=http://toyoguchi-megumi.cocolog-nifty.com/blog/2009/05/fatezero-f2fb.html |title=Fate/Zero ラジオマテリアル |work=めぐぶろぐ |date=2009-05-26 |accessdate=2013-10-07 |archiveurl=https://web.archive.org/web/20090603013149/http://toyoguchi-megumi.cocolog-nifty.com/blog/2009/05/fatezero-f2fb.html |archivedate=2009年6月3日 |deadlinkdate=2017年9月 }}</ref>。

=== 本人に関して ===
大学時代にいくつものアルバイトを経験している。[[パーツモデル|背中専門のモデル]]<ref>{{Cite web |author=大原さやか |date=2010-01-01 |url=http://www.sayaka-ohara.jp/nisshi/2010/01/100101.htm |title=2010.1.1 あけまして、おめでとうございます！ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032505/http://www.sayaka-ohara.jp/nisshi/2010/01/100101.htm |archivedate=2016-08-15}}</ref>、エキストラのアルバイトで『[[ラブジェネレーション]]』（フジテレビ系）、『[[ふぞろいの林檎たち]]IV』（TBS系）などに出演していたことがある。横浜のイタリアレストランでもアルバイトをしたことがあったが、店長に泣かされるなど「大人の理不尽さ」や「虐げられる弱い立場」を思い知らされ、トラウマになっているという。レストランに行った際、その店で働いていた元社員がいて驚いたことがあった<ref>{{Cite web |author=大原さやか |date=2004-03-20 |url=http://www.sayaka-ohara.jp/nisshi/2004/03/040320.htm |title=2004.3.20 思わず、凝視してしまった。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032453/http://www.sayaka-ohara.jp/nisshi/2004/03/040320.htm |archivedate=2016-08-15}}</ref>。

{{要出典範囲|[[ADSL]]などの[[ブロードバンド]]環境が既に広く普及していた頃でも[[ダイヤルアップ接続]]でインターネットをやっていて、電話料金などの請求書が数万に達した事がある|date=2017年4月}}。{{要出典範囲|その後、関係者やファンなどの助言でブロードバンド接続にようやく変更した|date=2017年4月}}。

=== 仕事 ===
{{要出典範囲|オーディションを受けた帰り際、そこで使った台本を破って捨てるようにしている。台本を家に何日も置いておくと落ちるというジンクスがあると考えているため|date=2017年4月}}。

『[[NOW HITS STREET]]』担当開始時から、自ら局に出向いて過去放送分をオンエアテープからダビングし、何回も聞いて復習していた<ref>{{Cite web |author=大原さやか |date=2004-04-29 |url=http://www.sayaka-ohara.jp/nisshi/2004/04/040429.htm |title=2004.4.29 …あれ、今日って休日！？ |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032453/http://www.sayaka-ohara.jp/nisshi/2004/04/040429.htm |archivedate=2016-08-15}}</ref>。その後、2006年12月1日に収録スタジオが幕張の新スタジオが幕張ワールドビジネスガーデンに移転にともなうデジタル化で、この作業ができなくなった<ref>{{Cite web |author=大原さやか |date=2006-11-17 |url=http://www.sayaka-ohara.jp/nisshi/2006/11/061117.htm |title=2006.11.17 bayfmな日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032530/http://www.sayaka-ohara.jp/nisshi/2006/11/061117.htm |archivedate=2016-08-15}}</ref>。京都出身のアーティストがゲストに来た場合、京都話で盛り上がり、京都のお薦めの場所を質問して教えてもらっている。ただ、京都出身だから京都に詳しいとは限らず、逆に大原が名所の説明をして、相手が感嘆することもあるという<ref>{{Cite web |author=大原さやか |date=2004-08-16 |url=http://www.sayaka-ohara.jp/nisshi/2004/08/040816.htm |title=2004.8.16 事務所でオーディションテープを録ったあと、bayfmへ。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032504/http://www.sayaka-ohara.jp/nisshi/2004/08/040816.htm |archivedate=2016-08-15}}</ref>。

『[[大人の趣味と生活向上◆アクトオンTV|ケイコとマナブChannel]]』の仕事では、30秒番宣のナレーションから、マスコットキャラ「テレビくん」の声、局宣番組で初めての顔出し出演までしていた。番組撮影では、メイクアーティストとスタイリストが付かなかったため、自身でヘアメイクをし、自前で用意した4〜5パターンの衣装をシーンごとに着替えていた。さらにチェロ演奏まですることになり、[[ヨハン・ゼバスティアン・バッハ|バッハ]]や[[ヴォルフガング・アマデウス・モーツァルト|モーツァルト]]はなぜか「著作権の問題がある」と言われ、即興で5〜6小節のオリジナル曲を演奏したという<ref>{{Cite web |author=大原さやか |date=2004-03-19 |url=http://www.sayaka-ohara.jp/nisshi/2004/03/040319.htm |title=2004.3.19 ナレーションとアフレコの日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://web.archive.org/web/20140813201202/http://www.sayaka-ohara.jp/nisshi/2004/03/040319.htm |archivedate=2014-08-13}}</ref>。

アニメ『[[スクラップド・プリンセス]]』にてラクウェル・カスールの声を演じると共に次回予告ナレーションを担当していた。{{要出典範囲|当初、監督は「次回予告に割り当てられたごくわずかな時間内でこの文字数を読み上げるのはさすがに無理だろう」と取り敢えず提示した原稿を大原は読み上げた|date=2017年4月}}。{{要出典範囲|このことから回を追うごとに次回予告ナレーションの文字数が増やされると共に、それの読み上げに成功した場合には監督からプレゼントが出されるようになった|date=2017年4月}}。

2005年9月頃にフジテレビから生番組への出演依頼が来たが、断ったことがある。依頼内容は「au携帯電話の音声案内の声は誰がやっているか」というコーナーへの出演であったが、当時大原の音声が使用されていたのは一部（全面切り替えは同年12月1日以降）であったことと、「ナレーターとしての顔出しもどうかと思う」ということを理由に辞退した<ref>{{Cite web |author=大原さやか |date=2005-09-20 |url=http://www.sayaka-ohara.jp/nisshi/2005/09/050920.htm |title=2005.9.20 ナレーション2本の日。 |work=さやかの業務日誌 |publisher=大原省 |accessdate=2016-08-15 |archiveurl=https://archive.is/20160815032530/http://www.sayaka-ohara.jp/nisshi/2005/09/050920.htm |archivedate=2016-08-15}}</ref>。

『[[となグラ!]]』最終話は、大原の提案で出演者全員が[[浴衣]]でアフレコに臨んだ<ref>{{Cite web |url=http://king-cr.jp/special/tonagra/taidan11_.html |title=となグラ! 最終回アフレコインタビュー |work=となグラ! |publisher=キングレコード |accessdate=2016-08-16}}</ref>。

{{要出典範囲|2006年8月18日18時41分から同年8月25日17時41分まで[[Yahoo! オークション]]の[[チャリティー]]で劇場版『[[XXXHOLiC#映画|xxxHOLiC]]』のアフレコサイン入り台本が出品され、落札金額（122,000円）は「[[メイク・ア・ウィッシュ]] オブ ジャパン」に全額寄付された|date=2017年4月}}。
