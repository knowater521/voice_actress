== 来歴・人物 ==
{{節stub}}
[[2004年]]に[[PlayStation 2]]用ゲームソフト『[[幻想水滸伝IV]]』のフレデリカ役で声優デビュー。

妹と14歳離れた弟<ref>{{Cite web |date= 2007年8月22日 12:20 |url= http://filn.jp/news.pl?news=216 |title= TVアニメ『風のスティグマ』のWEB動画バラエティ＜WBR＞『炎のツンデレ女子高生・綾乃の放課後ケーキバイキング＜WBR＞』公開録画レポート |work= Filnニュース|author=|publisher= コンテンツイノベーション |accessdate= 2009-10-09}} </ref>の3人姉妹弟である。黒猫の携帯ケースを愛用している<ref>{{Cite web |date= 2009年3月12日 |url= http://yaplog.jp/saki-ohana/archive/3111 |title= かわいい★ |work= 藤田咲の電子庭園☆〜さっきぃのおはなばたけ〜 |author= [[藤田咲]] |accessdate= 2009-10-09}}</ref>。

== エピソード ==
; アニメ「あさっての方向。」関連
:『[[あさっての方向。]]』の五百川からだ役で、初めてヒロインを務めた。また同時にキャラクターソングやラジオパーソナリティなども初めて経験することとなった。
; その他仕事関連など
: 尊敬する声優として[[家弓家正]]と[[一城みゆ希]]を挙げている<ref name="push" />。特に家弓は藤村に[[芝居]]への探究心を教えてくれた役者でもあり、日常の隅々までアンテナを張っている家弓の姿に憧れているとのこと<ref name="push" />。また、『[[真マジンガー 衝撃! Z編]]』では両者と共演を果たしている。藤村は主役や正義側の声を演じることは多いものの、[[悪役]]や敵側のキャラクターの声を演じることはあまりない。そのため「悪役をやりたい」と語ったこともある<ref name="push">{{Cite web|url=http://www.kenproduction.co.jp/push/22_fujimura.html|title=マンスリープッシュ ｜ 第22回 藤村 歩|publisher=賢プロダクション|accessdate=2013-08-21}}</ref>。『[[BLEACH (アニメ)|BLEACH]]』で[[端役]]ではあるものの敵役の声を演じた経験はあり、『[[翠星のガルガンティア]]』では、事実上のラスボス的存在であるロボット（マシンキャリバー）・ストライカーを演じている。また「この役は藤村がやったからこそ、もっと面白くなったと思われるようになりたい」とも語っている<ref name="push" />。
