'''坂本 美里'''（さかもと みさと）は、[[日本]]の[[女性]][[声優]]。[[フリーランス|フリー]]。[[高知県]]出身<ref name="Official profile">{{Cite web |url=http://misatostyle.web.fc2.com/profile.html |title=Profile * 坂本美里 Official Web * |accessdate=2012-10-13}}</ref>。[[ABO式血液型|血液型]]は[[ABO式血液型|A型]]<ref name="Official profile" />。

== 来歴 ==
*[[日本工学院専門学校]]（蒲田校）演劇俳優科声優コース在学中に[[新谷良子]]、[[清水愛]]と共にユニット「[[めぇめぇもぉ]]」を結成。（2000年9月）
*2007年3月31日付けでT・S・Pを退社し、4月1日より[[トーゴープロダクション]]に所属<ref>{{Cite web |url=http://31style.dreamlog.jp/archives/51391906.html |title=坂本美里　Official Web みぃすたぃる : information |accessdate=2012-10-13}}</ref>。2008年7月よりフリー<ref>{{Cite web |url=http://31style.dreamlog.jp/archives/51392040.html |title=坂本美里　Official Web みぃすたぃる : information |accessdate=2012-10-13}}</ref>。
