'''塩野 アンリ'''（しおの あんり、[[5月2日]] - ）は、[[日本]]の[[声優|女性声優]]。[[千葉県]]出身。[[ABO式血液型|血液型]]は[[ABO式血液型|A型]]<ref>{{Cite web |url=http://profile.ameba.jp/anri-shiono/ |title=しおののプロフィール |accessdate=2012-08-23}}</ref>。身長は153cm。旧芸名は、'''吉川 友佳子'''（よしかわ ゆかこ）。

== 来歴・人物 ==
2007年7月まで[[81プロデュース]]に所属<ref>{{Cite web |url=http://ameblo.jp/anri-shiono/entry-10277052153.html |title=お知らせです！｜しおのDiary |accessdate=2012-08-23}}</ref>。2007年10月に現在の芸名に改名<ref>{{Cite web |url=http://ameblo.jp/anri-shiono/entry-10277045413.html |title=お知らせです！だぶるだよ☆｜しおのDiary |accessdate=2012-08-23}}</ref>。2008年10月から2011年3月まで[[JTBエンタテインメント]]に所属していた<ref>{{Cite web |url=http://ameblo.jp/anri-shiono/entry-10277046431.html |title=事務所所属のお知らせです☆｜しおのDiary |accessdate=2012-08-23}}</ref><ref name="blog">{{Cite web |url=http://ameblo.jp/anri-shiono/entry-10848616830.html |title=お知らせ｜しおのDiary |accessdate=2012-08-23}}</ref>。

2011年3月、[[青山学院大学]][[経済学部]]卒業。声優から引退しデザイン会社勤務<ref name="blog" />。
