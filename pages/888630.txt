'''戸松 遥'''（とまつ はるか、[[1990年]][[2月4日]]<ref name="tomatsu">{{Cite web |url=http://www.tomatsuharuka.com/bio/index.html |title=戸松遥｜バイオグラフィ |work=BIOGRAPHY |publisher=戸松遥 Special website |accessdate=2010-11-15}}</ref> - ）は、[[日本]]の[[声優]]、[[俳優|女優]]、[[歌手]]、[[ナレーター]]。[[愛知県]]出身<ref name="tomatsu" />。身長165cm<ref name="tomatsu" />、[[ABO式血液型|血液型]]は[[ABO式血液型|B型]]<ref>{{Cite web |url=http://www.vip-times.co.jp/?talent_id=W13-0066 |title=戸松遥 |work=日本タレント名鑑 |publisher=VIPタイムズ社 |accessdate=2017-01-04}}</ref>。<!--この節に加筆される場合は、必ずノートでの議論に参加してください。-->[[ミュージックレイン]]所属<ref>{{Cite web |url=http://musicrayn.com/tomatsu_haruka |title=戸松遥 |publisher=ミュージックレイン |accessdate=2017-01-04}}</ref>。

代表作に『[[あの日見た花の名前を僕達はまだ知らない。]]』（安城鳴子）<ref name="oriconprofile">{{Cite web |url=http://www.oricon.co.jp/prof/422881/profile/ |title=戸松遥（出典：VIPタイムズ社） |work=ORICON STYLE |publisher=オリコン |accessdate=2017-01-04}}</ref>、『[[ソードアート・オンライン]]』（アスナ/結城明日奈）<ref name="oriconprofile" />、『[[ハピネスチャージプリキュア!]]』（氷川いおな / キュアフォーチュン）<ref name="oriconprofile" />、『[[妖怪ウォッチ]]』（ケータ / 天野景太）<ref name="oriconprofile" />がある。{{VOICE Notice Hidden|冒頭部分に記載する代表作は、編集合戦誘発の原因となりますので、多数の出典で確認できるものに限ってください。[[プロジェクト:芸能人#記事の書き方]]にてガイドラインが制定されていますので、そちらも参照して下さい。}}

== 経歴 ==
<!--経歴に関しては主なもののみ記載。CDの発売記念イベントなどは記載しないでください。CDなども第1作以外は記載しないでください。-->
[[2005年]]から[[2006年]]にかけて開催された「ミュージックレイン スーパー声優オーディション」に合格<ref>{{Cite web|date=|url=http://anime.eiga.com/person/11041/|title=戸松 遥|publisher=アニメハック Powered by 映画.com|accessdate=2017-11-24}}</ref><ref>{{Cite web|date=|url=http://www.sonymusic.co.jp/sd/ss03/|title=第3回ミュージックレインスーパー声優オーディション|publisher=株式会社ソニーミュージックエンタテイメント|accessdate=2017-11-24|archiveurl=https://web.archive.org/web/20171124004727/http://www.sonymusic.co.jp/sd/ss03/|archivedate=2017-11-24}}</ref>し、声優としての活動を始めた。{{要出典範囲|[[2006年|翌年]]1月に開催された[[東宝芸能]]の[[東宝「シンデレラ」オーディション|「シンデレラ」オーディション]]では、3万7443人の中から最終選考（15名）まで進出|date=2017年1月}}。

[[2007年]]『[[がくえんゆーとぴあ まなびストレート!]]』のスイーツ生徒役でアニメデビュー。同年4月に放送された『[[神曲奏界ポリフォニカ]]』のヒロイン・コーティカルテ役に抜擢された<ref name ="interview" />。

[[2008年]]に高校卒業と共に上京。1人暮らしを始め、大学へ通いながら活動を続ける<ref>{{Cite web|date=2009-01-01|url=https://ameblo.jp/tomatsuharuka-blog/entry-10802335066.html|title=さよならありがとう２００８年|publisher=ハルカレンダー Powered by Ameba|accessdate=2017-11-24}}</ref>。同年4月に放送された『[[絶対可憐チルドレン (アニメ)|絶対可憐チルドレン]]』の三宮紫穂役、『[[To LOVEる -とらぶる-]]』のララ・サタリン・デビルーク役で知名度が上昇。同年[[9月3日]]、自身も新田美恵子役で出演したテレビドラマ『[[ここはグリーン・ウッド|ここはグリーン・ウッド 〜青春男子寮日誌〜]]』のエンディングテーマ『[[naissance]]』で単独名義での歌手デビューを果たす（所属レーベルはミュージックレイン）。同年10月に放送された『[[かんなぎ (漫画)|かんなぎ]]』のナギ役で初主演。

[[2009年]][[2月15日]]に開催された「ミュージックレインgirls 春のチョコまつり」において、同じ事務所に所属している[[寿美菜子]]・[[高垣彩陽]]・[[豊崎愛生]]とともに[[声優ユニット]]「[[スフィア (声優ユニット)|スフィア]]」の結成を発表。以降、ユニットとしての音楽活動も行っている（所属レーベルは[[ランティス#GloryHeavenレーベル所属者|GloryHeaven]]）。同年[[3月7日]]、第3回[[声優アワード]]新人女優賞を受賞<ref>{{Cite web|publisher=声優アワード|url=http://www.seiyuawards.jp/200903/07210000.php|title=第三回声優アワード 受賞者発表|accessdate=2009-03-07|archiveurl=https://web.archive.org/web/20090311024859/http://www.seiyuawards.jp/200903/07210000.php|archivedate=2009年3月11日|deadlinkdate=2017年10月}}</ref>。同年10月9日に自身初の写真集『HARUKAs（ハルカ）』を発売。

[[2010年]][[2月24日]]に自身初のアルバム『[[Rainbow Road]]』を発売。この年から『[[トリハダ（秘）スクープ映像100科ジテン]]』をはじめ、散発的にナレーションの仕事をするようになってきている{{refnest|group="注"|本人はブログにて色々なナレーションをさせてもらって嬉しいと吐露している<ref>{{Cite web |publisher=戸松遥オフィシャルブログ「ハルカレンダー」 |url=http://ameblo.jp/tomatsuharuka-blog/day-20110111.html |title=トリハダでるよ☆ |accessdate=2011-01-11 |archiveurl=https://web.archive.org/web/20110728033645/http://ameblo.jp/tomatsuharuka-blog/day-20110111.html |archivedate=2011-07-28}}</ref>。}}。

[[2011年]][[7月17日]]から、初の単独ライブツアー『オレンジ☆ロード』が、[[神戸市|神戸]]・[[名古屋市|名古屋]]・[[横浜市|横浜]]の3か所で開催された。

[[2013年]][[3月1日]]、第7回声優アワード助演女優賞を受賞<ref>{{Cite web |url=http://www.seiyuawards.jp/winning/winning_07 |title=第七回声優アワード受賞者 |publisher=声優アワード |accessdate=2017-01-04}}</ref>。

2015年10月1日より、戸松遥のLINE公式アカウントがスタートした。

愛称は「とまっちゃん」<ref>{{cite web|url=https://akiba-souken.com/article/28145/|title=ニューシングル「モノクロ/Two of us」には、今までにない「とまっちゃん」が！|publisher=アキバ総研|date=2016-10-25|accessdate=2017-01-17}}</ref>、「はるちゃん」<ref>{{cite web|url=http://www.sponichi.co.jp/entertainment/news/2016/03/28/kiji/K20160328012300140.html|title=人気声優・戸松遥　ＮＨＫ動画投稿番組新ナレーターに「とても光栄」|publisher=[[Sponichi Annex]]|date=2016-03-28|accessdate=2017-01-17}}</ref>、「ハルカス」などがある。なお、2011年夏ごろより[[大阪府]][[大阪市]][[阿倍野区]]の[[超高層ビル]]である『[[あべのハルカス]]』と連動させたフレーズやギャグを自身のブログやスフィアのライブMCなどで多用するようになる<ref>[http://ameblo.jp/tomatsuharuka-blog/entry-11002376870.html ハルカレンダー「ハルカス！」] [http://ameblo.jp/tomatsuharuka-blog/entry-11022615876.html ハルカレンダー 「クライマックス！」]</ref>。なお、あべのハルカスには『[[戸松遥のココロ☆ハルカス]]』の取材の一環で来訪し、グランドオープン前の展望台に上がっており、この模様は同番組の『RADIO FANDISK 2』に収録されている<ref>[http://harukas.secondshot.jp/ 戸松遥のココロ☆ハルカス公式サイト] 2015年1月28日閲覧。</ref>。

いつも元気で明るくお調子者<ref name="hightouch">{{Cite book |和書 |date=2010-02 |title=戸松遥写真集High Touch! |page=71 |publisher=[[学研]] |isbn=978-4-05-404381-7}}</ref>。[[スフィア (声優ユニット)|スフィア]]のムードメーカーである。一旦こうと決めたら真っすぐ突き進むタイプで、自分には妥協を許さない<ref name="photobook">{{Cite book |和書 |date=2010-11-17 |title=スフィアフォトブック天体観測〜Sphere observation〜 |pages=68, 74, 84頁 |publisher=[[学研]] |isbn=978-4-05-404755-6}}</ref>が、その反面、適当なところがあり、おしゃれや流行にはあまり興味がなく<ref>リスアニ！LIVE2010公式パンフレット(p.27) 戸松遥　Q&A50</ref>、不器用である<ref name="daibouken">{{Cite book |和書 |date=2010-10-16 |title=戸松遥 はるかの大冒険 |page=61 |publisher=[[主婦の友社]] |isbn=978-4-07-275524-2}}</ref>。

趣味、特技は料理、洋服をたたむこと、[[ベース (弦楽器)|ベース]]、サメに関する知識、体を鍛えること、セリフを覚えること、くちびるがやわらかいこと<ref name="tomatsu"></ref>、数独<ref>戸松遥のココロ☆ハルカス第199回放送</ref>。好きなアーティストは[[阿部真央]]と[[サザンオールスターズ]]<ref name="daibouken" />。{{要出典範囲|好きな食べ物は肉で、焼き肉が大好物|date=2017年1月}}。{{要出典範囲|「マジか！」「さっぱりピーマン」が口癖|date=2017年1月}}。

=== 声優 ===
<!--この節に加筆される場合は、必ずノートでの議論に参加してください-->
声優を志したきっかけは、中学3年生の頃に芝居に興味を持ったこと、[[スタジオジブリ]]の作品に出演したいと思ったことによるとのこと<ref name="seipri">{{Cite book |和書 |date=2009-07  |chapter=スフィア |title=声優プリンセス |pages=32-59 |publisher=[[近代映画社]] |isbn=978-4-7648-7153-3}}</ref>。そうした気持ちを漠然と抱いていたところ、友人に奨められてオーディションを受けることになったとのこと<ref name="seipri" />。色々と考えず、感じたままに演技をしているという<ref name = "interview">{{Cite web|publisher=ニュース アニメイトTV|url=http://www.animate.tv/news/details.php?id=1241741099|title=連続企画 (3) 大型ユニット・スフィアから戸松遥さんインタビュー|accessdate=2009-04-14}}</ref>。

[[花澤香菜]]とは『[[To LOVEる -とらぶる-]]』『[[狂乱家族日記]]』から共演する機会が多く、プライベートでも仲がよい。尊敬する人物として、同郷の女優や声優として活動している[[戸田恵子]]を挙げている<ref>オトナアニメ (vol.10) P.23「ナギ役・戸松遥INTERVIEW」</ref>。

=== 歌手 ===
前述の声優ユニット「スフィア」としての歌手活動の他、個人名義としても歌手活動を行っている。自身もナギ役で出演した『[[かんなぎ (漫画)|かんなぎ]]』では、オープニングテーマ・エンディングテーマの両方を担当し、オープニングテーマとなった2ndシングル『[[motto☆派手にね!]]』は週間[[オリコンチャート]]10位にランクインした<ref name="minp" />。2013年1月16日発売の2ndアルバム『[[Sunny Side Story]]』で自身初のオリコンウィークリーのTOP5入りを果たす。また売上げ枚数も1.4万枚で、これまでの記録を更新した<ref> {{Cite web|publisher=biglobeニュース|url=http://news.biglobe.ne.jp/entertainment/0122/nrn_130122_3190548613.html|title=声優の戸松遥が初のTOP5入り、2ndアルバム「Sunny Side Story」で。|accessdate=2013-01-22}}</ref>。更に14thシングル『[[Courage]]』は初動2.4万を売り上げオリコンチャート4位を獲得。初動売上・順位共に自身最高記録を更新した<ref name="minp">{{Cite web|date=2014-12-09|url=http://www.minp-matome.jp/news/detail/62115|title=戸松遥「courage」がオリコン週間4位獲得、自己最高位を更新|work=minp!|author=|publisher=|accessdate=2015-01-28}}</ref>。

楽曲によって声質が変わる傾向にあると語っている<ref>{{Cite web|publisher=アニカンジェイビー|url=http://www.anican.jp/anicanfreeplus/vol81_harukatomatsu/vol81_harukatomatsu.aspx|title=戸松遥ロングインタビュー!!|accessdate=2010-08-31}}</ref>。
