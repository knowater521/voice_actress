'''結城 しのぶ'''（ゆうき しのぶ、[[1953年]][[11月25日]]<ref name="W00-0682">{{Cite web |url=http://www.vip-times.co.jp/?talent_id=W00-0682 |title=結城しのぶ |work=日本タレント名鑑 |publisher=VIPタイムズ社 |accessdate=2017-01-12}}</ref> - ）は、[[日本]]の[[俳優|女優]]。[[千葉県]]<ref name="W00-0682" /><!-- [[松戸市]] --><!-- 「松戸市」と確認のできる信頼できる情報源からの出典明記をお願いいたします。 -->出身。身長157cm、[[ABO式血液型|血液型]]はA型<ref name="W00-0682" />。

== 人物 ==
特技：[[手話]]、[[乗馬]]<ref name="antenneprof">{{Cite web |url=http://www1.bbweb-arena.com/antenne/antenne_008.htm |title=結城しのぶ |publisher=アンテーヌ |accessdate=2017-01-12}}</ref>。

1972年 {{要出典範囲|和洋女子大学を中退して|date=2017年1月}}東宝現代劇17期生に入る<ref name="antenneprof" />。

1973年 芸術座『湯葉』で初舞台<ref name="antenneprof" />。

1974年 TBS『[[大岡越前 (テレビドラマ)|大岡越前]]』第四部でTV出演。

1975年 NHK『[[新・坊っちゃん]]』に[[ギランバレー症候群]]で降板した[[大原麗子]]の代役としてマドンナ役で出演。

1979年 [[角川映画]]『[[蘇える金狼#1979年|蘇える金狼]]』では[[ヌード]]で[[濡れ場]]に出演。

1986年 {{要出典範囲|男児を出産|date=2017年1月}}。{{要出典範囲|これを機に芸能活動を一時休止|date=2017年1月}}。

2000年 {{要出典範囲|仕事を本格的に再開|date=2017年1月}}。女優業の傍ら、子育てや介護、家庭生活に関する講演活動も行っている<ref name="antenneprof" />。
