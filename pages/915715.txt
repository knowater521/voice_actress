'''竹達 彩奈'''（たけたつ あやな、[[1989年]]<ref>{{Cite web|date=2010-06-24|url=http://ameblo.jp/strawberry-ayana/day3-20100624.html#main|title=21。|work=Strawberry Candy|author=竹達彩奈|publisher=[[サイバーエージェント]]|accessdate=2013-01-16}}</ref>[[6月23日]] - ）は、[[日本]]の[[女性]][[声優]]、[[歌手]]、[[ナレーター]]。

[[プロ・フィット#リンク・プラン|リンク・プラン]]（事務所）、[[ポニーキャニオン]]（レコード会社）に所属。愛称は主に'''あやち'''、'''あやにゃん'''。[[埼玉県]]出身<ref name="linkplan" />。[[ABO式血液型|O型]]。本名同じ。公式ファンクラブは「'''あやな公国'''」<ref>{{Cite web|url=http://ayanakoukoku.com/|title=竹達彩奈Official Mobile Fanclub|accessdate=2017-07-22}}</ref>。

== 概要 ==
声優としては、2009年スタートのテレビアニメ『[[けいおん!]]』の中野梓役でブレイクして（2010年に第2期が製作され、2011年に劇場版が上映された）、その後も、『[[俺の妹がこんなに可愛いわけがない (アニメ)|俺の妹がこんなに可愛いわけがない]]』の高坂桐乃役（2010年に第1期、2013年に第2期）、『[[神のみぞ知るセカイ]]』の高原歩美役（2010年～2013年にかけて、第3期までが制作された）、『[[たまゆら (アニメ)|たまゆら]]』の沢渡楓役（2010年～2016年）など、コンスタントに主演、ヒロイン役やメインキャストを担当している<ref name="ayanabible2017">『日経エンタテインメント　アニメSpecial　声優バイブル2017』、86頁～91頁（竹達彩奈のインタビューページ）。</ref>。

演技力に加えて、「かわいらしい声質」と「キュートなルックス」をも併せ持つことも、大きな特徴であり、多面的に活動の幅を拡げている<ref name="ayanabible2017" />。近年は特に活発に音楽活動を展開させており、2012年からソロ歌手として活動しているほか、2013年からは「[[petit milady]]」（[[悠木碧]]との2人組の[[声優ユニット]]）としての活動も並行させている<ref name="ayanabible2017" />。また、2013年から2014年にかけて展開された[[吉野家]]とのコラボ企画も話題になり、2014年には[[コマーシャルメッセージ|Web CM]]が制作された<ref>{{Cite web|publisher=おたスケ|url=http://www.ota-suke.jp/news/113292|title=吉野家の「アタマの大盛」が500万食突破、竹達彩奈と再びコラボ|accessdate=2017-01-30}}</ref><ref>{{Cite web|publisher=ORICON STYLE|url=http://www.oricon.co.jp/news/2043042/full/|title=人気声優・竹達彩奈が吉野家とのコラボで“大食いキャラ”をいかんなく発揮!!|accessdate=2017-01-30}}</ref><ref>[https://www.youtube.com/watch?v=LR8bEkD7WDM]2017年1月30日閲覧。</ref>。

== 略歴 ==
14歳（中学2年生）のとき、[[アイムエンタープライズ]]系列の[[日本ナレーション演技研究所]]の特待生オーディションを受け同所に入所する。その後、劇団ヴォアレーヴ（アイム社系の養成を目的とした附属劇団）の在籍を経て、15歳（中学3年生）の春にアイムエンタープライズに所属した。当時は最年少記録だった<ref name="声優グランプリ2011年6月">{{Cite journal|和書 |author=  |year= 2011 |title= |journal=声優グランプリ|issue=2011年6月号 |pages=14-15 |publisher= [[主婦の友社]]|id=雑誌 05887‐6}}</ref>。

初仕事は、事務所に所属してから2か月後、2005年6月19日に行われたアイムエンタープライズ8周年記念の「アイム祭」である<ref name="声優グランプリ2011年6月" />。その後、[[OVA|OAD]]『[[kiss×sis]]』のヒロイン・住之江あこ役で声優としてデビュー。そして『[[けいおん!]]』の中野梓役で知名度が上昇<ref>2009年5月『けいおん!』に初登場した回が放送されてからWikipedia、ブログのアクセス数が急増した。[http://stats.grok.se/ja/200905/%E7%AB%B9%E9%81%94%E5%BD%A9%E5%A5%88 2009年5月Wikipedia「竹達彩奈」ページアクセスグラフ]、[http://ameblo.jp/strawberry-ayana/day5-20090523.html 2009年5月23日竹達彩奈ブログ]</ref>した。

[[2010年]][[3月6日]]に開催された第4回[[声優アワード]]授賞式で、テレビアニメ『けいおん!』の劇中で結成したユニット“放課後ティータイム”として、[[豊崎愛生]]、[[日笠陽子]]、[[佐藤聡美]]、[[寿美菜子]]とともに歌唱賞を受賞した。10月10日に[[OVA]]『[[たまゆら (アニメ)|たまゆら]]』の舞台となった[[広島県]][[竹原市]]の一日市長に就任した<ref>{{Cite web |url = http://gigazine.net/news/20101012_the_day_of_tamayura/

== 活動 ==
声優を目指すようになったきっかけとして、幼少期にテレビアニメ『[[カードキャプターさくら]]』や『[[美少女戦士セーラームーン (テレビアニメ)|美少女戦士セーラームーン]]』、ゲーム『[[Kanon (ゲーム)|Kanon]]』や『[[シスター・プリンセス]]』に触れたことや、[[堀江由衣]]への憧れを挙げており、それらを通して、アニメやゲームに携わりたいと思うようになったとのことである<ref name="ayanabible2017" /><ref name="声優プリンセス">{{Cite journal|和書 |author =  |year = 2009 |title = PICK-UP FRESH VOICE ACTRESS |journal = 声優プリンセス |pages = 76-78 |publisher = [[近代映画社]] |isbn=978-4-7648-7153-3}}</ref>。

本人曰く、声優を目指す後押しも、現在の精力的な活動も、「私の一番のファンである母の後押しのおかげ」と述べているが、当初は「（母は）私を子役にして、芸能界で顔出しの仕事をして欲しかった」とのことであり、また、養成所の学費への懸念をも含めて、声優を目指すことを反対されていたとのことである<ref name="ayanabible2017" /><ref name="声優プリンセス" /><ref name="ayanapickupvoice2016">『Pick-up Voice』2016年8月号、38頁～41頁（竹達彩奈のインタビューページ）。</ref>。しかし、「私はお芝居がしたいしアニメが好きだから、声優がいいの！」と粘り強く説得して、最終的には快く送り出してくれたとのことであり、14歳のとき、声優の養成所に入る<ref name="ayanabible2017" /><ref name="声優プリンセス" /><ref name="ayanapickupvoice2016" />。養成所から事務所に入るまでは「すごく早かった」とのことだが、そこからアニメデビューの『[[kiss×sis]]』（住之江あこ役）までは、およそ3年を要したとのことであり、しかも事務所に所属はしていたものの、養成所のレッスンも受ける身であったとのことで、高校時代は「学業、アルバイト（雑貨屋さんと飲食店）、養成所のレッスン、事務所のレッスン」の「4足のわらじ」であったという<ref name="ayanabible2017" />。

自身の基本的な活動姿勢として、「'''（声優としては、）お芝居の主役はキャラクターであって、私ではない。完全に裏方に徹して、自分は自分の仕事をする。でも音楽活動は、自分名義で出すならば、声優である自分でなければできない音楽を、自分の音楽として届けたい'''」という趣旨のことを述べている<ref name="ayanabible2017" />。また、「ものを作る、作品を作る瞬間に関われている、その時間がすごく好きだし楽しい」ことが、（声優、歌手等の）活動の原動力になっていると述べている<ref name="ayanalivedoor2016">{{Cite web|publisher=ライブドアニュース|url=http://news.livedoor.com/article/detail/12347683/|title=竹達彩奈「笑ってここまで来られたかと言うと、そうでもなかった」――3rdアルバム『Lyrical Concerto』に至る道のり|accessdate=2017-01-30}}</ref>。

自身が出演した作品はどれも思い入れがあるとのことだが、その中でも特に、『[[kiss×sis]]』（住之江あこ役。2008年の[[OAD]]版で声優としてデビュー。2010年にテレビアニメ化された）及び『けいおん!』への思い入れが強いことを、インタビューやライブにおいてしばしば表現している<ref name="ayanabible2017" /><ref name="ayananewyearlive2017">{{Cite web|publisher=ナターシャ|work=音楽ナタリー|url=http://natalie.mu/music/news/217002|title=過去最多曲数を披露した“ニューあやち”中野サンプラザワンマン完遂|accessdate=2017-01-30}}</ref>。特に『kiss×sis』では、アフレコ経験もない中で、過激なお色気やキスシーンなどを演じたため、本人曰くその当時は「衝撃的だった」とのことだが、その後は「これをやったらなんでもできるだろう」と思うようになったとのこと<ref name="声優プリンセス" />。

『けいおん!』では、元々は軽音部の初期メンバー4人の役でオーディションに参加し、新入部員の中野梓役に選ばれた。梓の登場は第8話からであるが、アフレコ現場にはそれまでの回の収録にも見学に来ていた<ref>{{Cite web|url=http://ameblo.jp/strawberry-ayana/entry-10265559702.html|title = けいおん! |work = 竹達彩奈オフィシャルブログ「Strawberry Candy」|author = 竹達彩奈 |publisher = [[アメーバブログ|Ameba]] |date = 2009-05-22|accessdate=2010-07-28}}</ref>。『けいおん!』の[[インターネットラジオ|Webラジオ]]『[[らじおん!]]』の企画で、自身らが声を演じるキャラクターと同様に実際にバンド活動を行っている。練習もほかの4人と同時に始めていたが、『らじおん!』の公開録音「ろくおん!」のサプライズのために、公式サイトで後編が配信されるまで非公開であった<ref>{{Cite web|url = http://www.tbs.co.jp/anime/k-on/k-on_tv/radio/radio22.html|title = 『らじおん!』第22回（09/07/31配信） |work = TBSアニメーション・けいおん!公式ホームページ/Webラジオ|publisher = [[TBSテレビ|TBS]] |author = |date = 2009-07-31 |accessdate = 2013-10-14}} </ref><ref>{{Cite web |url = http://ameblo.jp/strawberry-ayana/entry-10312065416.html|title = ありがとうございました!! |work = 竹達彩奈オフィシャルブログ「Strawberry Candy」|author = 竹達彩奈 |publisher = [[アメーバブログ|Ameba]] |date = 2009-08-01|accessdate = 2009-11-09 }}</ref>。ファンからは、思わずペロペロしたくなるほど梓がかわいいと感じる言葉として「あずにゃんペロペロ」という用語ができて、竹達自身も使用している<ref>{{Cite web|url=http://datazoo.jp/tv/%E5%8D%88%E5%89%8D%E9%9B%B6%E6%99%82%E3%81%AE%E5%B2%A1%E6%9D%91%E9%9A%86%E5%8F%B2/697615|title=TVでた蔵「2014年1月29日放送　午前零時の岡村隆史」|publisher=[[ワイヤーアクション (企業)|ワイヤーアクション]]|date=2014-01-29|accessdate=2015-03-12}}</ref>。

音楽活動について、2012年にソロ歌手としてのメジャーデビューを果たすが、当初は自ら希望した訳ではなく、『[[けいおん!]]』のスタッフから、ソロ歌手デビューの話を頂いたことが出発点とのことだが、「1人で決めるにはキャパオーバーと思った」ことから、自宅に持ち帰って母に相談したら、「やりなさいよ。あんたの好きな[[堀江由衣]]さんだってやっているじゃないの。ファンの人も喜んでくれるし、いっぱい頑張れるでしょ」との後押しを頂いたことで、歌手活動を始める決心ができたという<ref name="ayanabible2017" />。

ソロ歌手としての音楽活動において、先述のように、「声優だからこそできる音楽」にこだわりたいことと共に、1つのアルバムが完成するたびに1つのシーズン（サイクル）と位置付けており、そのたびに何らかのテーマを掲げて音楽をつくるとのことである<ref name="ayanabible2017" /><ref name="ayanalivedoor2016" />。また、アルバムのタイトルには基本的に何らかの音楽用語を挿入しているという<ref name="ayanalivedoor2016" />。なお本人曰く、1stシーズンは「コアでマニアックな音楽」、2ndシーズンは「プリンセス」、3rdシーズンは「二次元」をテーマにしているとのことである<ref name="voicegirls">『 B.L.T. VOICE GIRLS』VOL.28, p.75.</ref><ref name="ayanalivedoor2016" /><ref>{{Cite web|publisher=リスアニ！WEB|url=http://www.lisani.jp/0000007324/|title=竹達彩奈、2ndアルバムのタイトルが『Colore Serenata』に決定！参加する豪華クリエイター陣も発表に！|accessdate=2017-01-30}}</ref>。尤も本人曰く、自身の音楽活動に向き合うにあたって、「正直な話、音楽に対して相当後ろ向きなところからスタートしたので、前向きになるまでにやっぱりすごい時間がかかりました」<ref name="ayanalivedoor2016" />とのことだが、その過程で、「一つ一つの楽曲の世界観を大切にして、楽曲によって声質や歌い方に細部までこだわるようになった」<ref name="ayanalivedoor2016" />と述べている。

声質やルックスの「かわいらしさ」は、自身の代名詞といわれることが少なくないが、本人曰く、子ども時代や『けいおん!』にて演じていたときは、「自分の表面ばかりが強調されている感じがして嫌だった、素直に受け止められなかった」と、声質やルックスがむしろコンプレックスに感じることが少なくなかったとのことである<ref name="ayanabible2017" /><ref name="声優プリンセス" /><ref group="注">本人曰く、子ども時代は周囲（特に友達）から「アニメっぽい声」としばしばいわれており、それを嫌に感じることが少なからずあったとのことである。</ref>。しかし、2016年のインタビューでは、「（声優や歌手などの活動を通して）『かわいいといわれているうちが花だな』と思えるようになった」「（それが活動の原動力になっていると思うこと、及び活動の過程で、）私たちの活動のレールを敷いてくれた先輩方の背中を見て、求めてもらえるならば、全力で頑張ろうと思えるようになった」と述べている<ref name="ayanabible2017" />。

先述のように、「憧れであり目標」の声優として[[堀江由衣]]を挙げている<ref name="ayanabible2017" />。堀江のライブには、声優になる以前から通っていたとのことであり、子ども時代はCDを[[アニメイト]]で買うほどであったが、直接会ったときには本人にはファンであることを言えなかったという<ref>{{cite episode|title=俺の妹が（ラジオでも）こんなに可愛いわけがない|network=「俺の妹がこんなに可愛いわけがない」アニメ公式サイト|number=2|minutes=}}</ref>。自身が声優及び歌手として活動するようになってからは、「（日程が合えば）コンサート会場の物販コーナーに一ファンとして並んでいる」一方で、「純粋に好きな気持ちは勿論持ち続けていますが、勉強になるという思いもあってライブ鑑賞をしています」ということを、2016年のインタビューにて述べている<ref name="ayanabible2017" />。なお、2017年1月15日に開催した、[[中野サンプラザ]]での自身のワンマンライブにて、堀江の楽曲「翼」を歌唱（カバー）したが、歌唱の際は感極まる場面もあったとのことである<ref name="ayananewyearlive2017" />。

== 人物 ==
愛称は主に「'''あやち'''」<ref name="ayananewyearlive2017" /><ref>{{Cite web|publisher=SPICE|url=https://spice.eplus.jp/articles/99103|title=チーム「竹達」が中野の夜を「あやち」色に染める 竹達彩奈『Lyrical Concerto』ライブレポート|accessdate=2017-01-30}}</ref>と呼ばれることが多いとのことであり、自身のイメージ映像の作品のタイトルにも用いられている。

自身の性格として、「毒の塊」と述べている<ref name="ayanabible2017" />。本人曰く、このことの理由として、子ども時代から「思ったことをはっきりとすぐに口にする人間」であったことに由来するとのこと<ref name="ayanabible2017" /><ref name="ayanapickupvoice2016" />。この性格は活動における原動力にもなっているとのことで、活動をする過程で「何を言おうが自分自身に返ってくる訳だから、私が責任を取ればいいのかなと（思えるようになった）」と述べている<ref name="ayanapickupvoice2016" />。また、「石橋を叩いて渡る」タイプ<ref name="ayanalivedoor2016" />とも述べている。

3人兄妹の末っ子で兄が二人いる。本人曰く、親しい人には甘えん坊なところがある「典型的な妹キャラ」とのことである<ref name="声優グランプリ2011年6月">{{Cite journal|和書 |author=  |year= 2011 |title= |journal=声優グランプリ|issue=2011年6月号 |pages=14-15 |publisher= [[主婦の友社]]|id=雑誌 05887‐6}}</ref>。

インタビュー等にて、アニメ、ゲーム好きであることをしばしば述べており、先述のように、幼少時から、『Kanon』や『シスター・プリンセス』等の、いわゆる「ギャルゲー」「乙女ゲー」を遊んでいたことが声優を目指す大きなきっかけの一つになったが<ref name="ayanabible2017" />、積極的に公表するようになったのは『[[俺の妹がこんなに可愛いわけがない (アニメ)|俺の妹がこんなに可愛いわけがない]]』の[[俺の妹がこんなに可愛いわけがないの登場人物#高坂桐乃|高坂桐乃]]を演じたことがきっかけであるとのこと<ref name="Voice Angel" /><ref name="ayanabible2017" />。アニメ、ゲームのグッズを集めていた学生時代、竹達が家に帰ると大事にしていた作品の非売品のノートに勝手に判子が押されていた。これは竹達の父が新しく判子を作った試し押しをしたもので本人にも見せながら判子を押して見せた。このときの感想を「（自分の判子を作ってもらって）嬉しいようで悲しいような複雑でした（笑）」と語っている<ref>『B.L.T. VOICE GIRLS Vol.4』（[[東京ニュース通信社]]）竹達彩奈記事・インタビューより。</ref>。また、カメラも趣味の一つとのことである<ref name="Voice Angel">{{Cite journal|和書 |author=  |year= 2011 |title= |journal=Voice Angel |pages=17 |publisher= [[竹書房]] |isbn=978-4-8124-4805-2}}</ref>。特技として、公式プロフィールでは“カラオケ・読書・ショッピング”となっている。

自称引きこもりのインドア派で家で過ごすことが多い。極度の人見知りで、会話が苦手である<ref name="声優アニメディア2012年2月号">{{Cite journal|和書 |author=  |year= 2012 |title=  |journal=声優アニメディア |issue=2012年2月号 |pages=49-51 |publisher=[[学研]] |id=雑誌 05705-02-2605705022}}</ref>。

その一方で、2016年11月のインタビューにて、休みの日には「本屋さんに行って、気になった作品や話題になっている作品である漫画を、片っ端から大量に買います」と述べている<ref name="ayanalivedoor2016" />。理由として、「これ嫌い、つまんない、おもしろい、好きっていうのは、自分でお金を出してからこそ言える言葉だと思っている」からであると述べている<ref name="ayanalivedoor2016" />。

好きな食べ物は肉<ref>{{Cite journal|和書 |author=  |year= 2011 |title= |journal=[[声優グランプリ]] |issue=2011年6月号 |pages=12 |publisher=[[主婦の友社]]}}</ref>。好きなアーティストは[[奥華子]]<ref>『[[けいおん!#Webラジオ|らじおん!!]]』2010年[[10月15日]] 第26回配信（「文化祭に呼んでみたいミュージシャン」として本人の発言）より。</ref>。また[[宇徳敬子]]が好きで、カラオケではよく歌っていたという<ref>{{Cite journal|和書|month=4|journal=[[声優グランプリ]]|issue=2012年5月号|pages=16|author=|year=2012|publisher=[[主婦の友社]]}}</ref>。

地毛は若干焦茶色である<ref>{{Cite web|url=http://ameblo.jp/strawberry-ayana/day-20110105.html|title = まっくろくろすけ♪|date = 2010-01-05|work = 竹達彩奈オフィシャルブログ「Strawberry Candy」|author = 竹達彩奈|publisher = [[アメーバブログ|Ameba]]|accessdate = 2011-01-06}}</ref>。

『[[俺の妹がこんなに可愛いわけがない (アニメ)|俺の妹がこんなに可愛いわけがない]]』第1期に出演していた時期には、[[花澤香菜]]と特に親しくしていた<ref>{{cite episode|title=[[俺の妹がこんなに可愛いわけがない (アニメ)#Webラジオ|俺の妹が（ラジオでも）こんなに可愛いわけがない]]|network=「俺の妹がこんなに可愛いわけがない」アニメ公式サイト|airdate=2011-07-22|number=24|minutes=}}</ref>。その花澤との関係について「業界の先輩であり嫁でありマイスィートエンジェル」と当時述べていた<ref name="声優グランプリ2011年9月">{{Cite journal|和書 |author=  |year= 2011 |title= |journal=声優グランプリ |issue=2011年9月号 |pages=15 |publisher= [[主婦の友社]] |id=雑誌 05887‐9}} </ref>。
