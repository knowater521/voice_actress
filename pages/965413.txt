'''藤屋 裕子'''（ふじや ゆうこ、[[11月24日]]）は、[[日本]]の[[女性]][[声優]]。[[フリーランス|フリー]]。[[北海道]]出身。血液型は[[ABO式血液型|O型]]<ref>{{Cite web |url=http://profile.ameba.jp/yukoringo-v-fujiya/ |title=ペコりんのプロフィール |accessdate=2012-11-07}}</ref>。

== 人物 ==
[[81プロデュース]]に所属していたが、2009年1月をもって退所<ref>{{Cite web |url=http://ameblo.jp/yukoringo-v-fujiya/entry-10201049674.html |title=大切なお知らせとお詫びと感謝｜のびポジってことなの♪ |accessdate=2012-11-07}}</ref>。[[フリーランス|フリー]]期間、TVやラジオにて活動を続けつつ講師として働き、2011年10月にはC&Oプロダクションに所属<ref>{{Cite web |url=http://ameblo.jp/yukoringo-v-fujiya/entry-11069592576.html |title=最近シリーズvol.4 ～おNEWな出発～｜のびポジってことなの♪ |accessdate=2012-11-07}}</ref>。2014年8月より[[フリーランス|フリー]]となり、現在も講師として働いている。

2011年8月に、同年2月から一緒に住んでいた彼氏にプロポーズされ、10月に婚姻届を提出した<ref>{{Cite web |url=http://ameblo.jp/yukoringo-v-fujiya/entry-11057175046.html |title=最近シリーズvol.２ ～幸せな入籍～｜のびポジってことなの♪ |accessdate=2012-11-07}}</ref>。
