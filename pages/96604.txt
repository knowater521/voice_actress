== 経歴 ==
{{要出典範囲|[[松濤アクターズギムナジウム]]声優部1期生卒業後|date=2017年4月}}、[[ケッケコーポレーション]]所属。

{{要出典範囲|1997年4月、養成所内のオーディションに合格し、[[セガサターン]]用ソフト「雀帝」でデビュー|date=2017年4月}}。2000年の『[[六門天外モンコレナイト]]』の大矢門斗役、『[[ちっちゃな雪使いシュガー]]』の[[ソルト]]役などで知名度を上げていった。

[[2005年]][[2月]]、持病の心臓疾患（[[不整脈]]の一種である[[発作性上室性頻拍症]]）の治療に専念するとして芸能活動の休業を発表<ref>{{Cite web |url=http://www.broccoli.co.jp/topics/050210_saekitomo.html |title=サエキトモさん活動休業のお知らせ |publisher=[[ブロッコリー (企業)|ブロッコリー]] |date=2005-02-10 |accessdate=2010-07-10}}</ref>。{{要出典範囲|[[2007年]][[春]]に事務所との契約が解除された|date=2017年4月}}。

[[2010年]][[8月6日]]、[[1314 V-STATION]]の過去の人気番組を1カ月にわたって復活放送する[[大阪放送|ラジオ大阪]][[特別番組]]『1314 V-STATION ザ・ラジオスター』で、『[[大河ラジオだ ブシロード]]』『[[アベノ橋魔法☆商店街#ラジオ番組|ラジオアベノ橋魔法☆商店街]]』パーソナリティーとして出演<ref>{{Cite web |url=http://vstation.net/news/archives/300 |title=【緊急速報!】Vステの名番組・名パーソナリティが8月に続々復活!! |publisher=ラジオ大阪 |date=2010-07-06 |accessdate=2010-07-10 |archiveurl=https://archive.is/20130502152105/http://vstation.net/news/archives/300 |archivedate=2013-05-02}}</ref>。{{要出典範囲|同時に収録したラジオ大阪・[[文化放送]]『[[ガイナックス電波]]』（[[8月7日]]放送）にもゲストパーソナリティーとして出演した|date=2017年4月}}。

『ガイナックス電波』第15回（2010年[[10月16日]]放送）でラジオパーソナリティー活動を再開<ref>{{Cite web |author=サエキトモ |date=2010-10-15 |url=http://tomosaeki.exblog.jp/14213425/ |title=そしてメインパーソナリティの座へ・・・ |work=サエキトモ生存確認場。 |accessdate=2017-04-30}}</ref>。さらに[[10月17日]]付の自身のブログで、療養生活を支えた一般人の5歳年上の男性と[[結婚]]したことを発表した<ref>{{Cite web |author=サエキトモ |date=2010-10-17 |url=http://tomosaeki.exblog.jp/14222903/ |title=モルディブの事実 |work=サエキトモ生存確認場。 |accessdate=2017-04-30}}</ref>。

{{要出典範囲|2010年[[11月1日]]、前所属事務所であるケッケコーポレーションに再所属|date=2017年4月}}。
